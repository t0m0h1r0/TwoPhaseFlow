# 実験仕様書：仮想時間陰解法CCDによる多角的な数値解法精度の検証

## 1. 実験の目的

本実験の目的は、6次精度コンパクトスキームであるCCD法と仮想時間陰解法の組み合わせが、ポアソン方程式以外の楕円型・放物型方程式に対しても高い精度と収束性を有することを確認することにある。特に、勾配が急峻な解や、移流が卓越する系における数値振動の抑制能力を、標準的な差分法と比較検証する。

## 2. 対象とする方程式と検証ケース

### Case 1: ポアソン方程式（基本性能）

高周波成分を含む定常解の再現性を確認する。

- 方程式: $\nabla^2 p = S(x, y)$
- 厳密解: $p_{\text{exact}} = \sin(n\pi x) \sin(m\pi y)$
- 目的: 空間分解能に対する収束次数（理論値6次）の確認。

### Case 2: 非定常拡散方程式（放物型方程式）

熱伝導などの時間発展問題における、仮想時間反復の有効性を検証する。

- 方程式: $\frac{\partial T}{\partial t} = \alpha \nabla^2 T$
- 物理シナリオ: 領域中央に配置されたガウス分布の拡散。
- 厳密解: $T(x, y, t) = \frac{1}{4\pi \alpha t} \exp\left( -\frac{(x-x_0)^2 + (y-y_0)^2}{4\alpha t} \right)$
- 検証の肝: 各実時間ステップ内の仮想時間反復により、大きな $\Delta t$ での安定性と精度を両立できるか。

### Case 3: 定常移流拡散方程式（特異摂動問題）

CCD法が低拡散（High-resolution）であることを確認する。

- 方程式: $\mathbf{u} \cdot \nabla \phi = \Gamma \nabla^2 \phi$
- 設定: 速度場 $\mathbf{u}$ が斜め方向（例：45度）の一定流、境界条件として不連続な値（Step function）を与える。
- 検証の肝: ペクレ数 $Pe = |\mathbf{u}|L/\Gamma$ が大きい条件下での、数値拡散（解のなまり）と数値振動（ギブス現象）の抑制性能。

## 3. 数値計算アルゴリズムの仕様

### 3.1 空間離散化

- CCD法: 変数 $\phi$ とその勾配 $\phi_x, \phi_y$、および2階微分 $\phi_{xx}, \phi_{yy}$ を連立して解く。
- 境界処理: 境界付近での精度低下を防ぐため、資料に基づいた5次〜6次精度の境界スキームを適用する。

### 3.2 時間進行（仮想時間陰解法）

実時間を $t$、仮想時間を $\tau$ とすると：

$$
\frac{\partial \phi}{\partial \tau} + \left[ \frac{\partial \phi}{\partial t} + \mathbf{u} \cdot \nabla \phi - \Gamma \nabla^2 \phi \right] = 0
$$

実時間項 $\frac{\partial \phi}{\partial t}$ は、2次精度の後退差分（BDF2）等で近似する。仮想時間 $\tau$ に関して陰解法を適用し、定常状態（ブラケット内がゼロ）に達するまで反復する。

## 4. 実験手順と評価指標

- 格子収束性試験: 格子サイズ $h$ を半分にするごとに誤差が $1/2^6 = 1/64$ に減少するかをプロットする。
- 位相誤差・振幅誤差解析 (Case 3): 移流問題において、解のピーク値が維持されているか（数値拡散の評価）、および解の背後に不自然な振動が出ていないか（数値分散の評価）を目視および $L_\infty$ ノルムで確認する。
- 計算効率の評価: 反復回数 $k$ に対する残差 $R^k$ の減少率を、ヤコビ法やガウス・ザイデル法等の標準的な緩和法と比較する。

## 5. 期待される結果

- 空間精度: すべてのケースで、2次精度差分法 (CDS2) よりも圧倒的に少ない格子数で同等の精度を得られること。
- 安定性: 移流卓越場においても、標準的な中央差分で見られるような激しい振動が抑制され、安定して収束すること。
- 汎用性: 同一のCCD演算子ライブラリを用いて、異なる物理方程式（ポアソン、拡散、移流拡散）を統一的に扱えること。

### 補足

特に Case 3（移流拡散）は、気液界面の移流や速度場の計算において重要となる。CCD法を用いることで、界面付近の急峻な勾配を精度良く捉えられることが期待される（資料 §10.4 参照）。
