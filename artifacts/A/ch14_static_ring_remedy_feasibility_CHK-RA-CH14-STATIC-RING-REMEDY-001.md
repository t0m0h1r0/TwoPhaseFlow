# CHK-RA-CH14-STATIC-RING-REMEDY-001

Date: 2026-05-07

Scope: decide whether there is a physically and mathematically legitimate path
to eliminate the tiny static-droplet velocity ring.  The rule is strict:
do not hide the residual by damping, smoothing, curvature caps, CFL tuning,
benchmark branches, blanket range projection, or post-hoc force subtraction.

## Short Answer

Yes, there is a credible path, but it is not a local force tweak.

The ring can be eliminated only by making the initial "static droplet" a
static solution of the same finite-dimensional variational system used by the
production pressure corrector:

```text
h(q,mu) = Z_A(q) [ s(q) - B(q) mu ] = 0,
V(q) = V0.
```

Here `q` is the pre-reinit interface state, `s=-M_A^{-1}T_h^*dS_h`, `B` is
the component-volume reaction cochain, and `Z_A` is the pressure-Hodge
complement generated by the active `D_f,G_A,M_A`.  A sampled continuum circle
is not guaranteed to satisfy this finite equation.  The solver is therefore
allowed to create a tiny ring unless we construct a discrete critical state.

## Feasibility Check

For N=32 periodic FCCD in the current node-centred grid:

```text
face shapes                    (32,33), (33,32)
face dof                       2112
pressure/node equation dof      1089
psi plus mu unknowns            1090
expected pressure rank upper    1088
expected Hodge dimension lower  1024
```

Adding the volume constraint gives about `1025` scalar conditions for
`1090` unknowns before accounting for gauge/profile/stratum freedoms.  This
does not prove existence, but it rules out the most pessimistic objection that
the exact static-state problem is obviously overdetermined.

The more serious risks are regularity and theorem consistency:

1. `q` must stay on one regular marching-squares stratum during the nonlinear
   solve, or the derivative being solved is not the derivative used by the
   equations.
2. `s` and `B` must be built with the same pressure-adjoint `M_A` used by
   `Z_A`; arithmetic kinetic weights are not enough near affine jump faces.
3. The material state must be the same endpoint as the capillary geometry:
   `q_c=q_T`, not post-reinit `q_R`, unless an endpoint-equivalence gate
   passes.
4. The corrector must receive the same corrected cochain `s-Bmu` used in the
   residual solve.

## Mathematical Construction

The cleanest finite-dimensional construction is a constrained root problem:

```text
F(q,mu) =
[
  W_h^{1/2} Z_A(q) (s(q)-B(q)mu)
  alpha (V(q)-V0)
]
= 0.
```

`W_h` is the active face metric `M_A` restricted to the Hodge complement.  The
root can be solved by matrix-free damped Newton or Levenberg--Marquardt, but
the damping here belongs to the nonlinear initializer solve, not to the
physical momentum equation.  The accepted output is a new static initial
condition `q_static`, not a runtime force correction.

The derivative is available by finite-dimensional chain rules:

```text
delta s =
  -delta(M_A^{-1}) T^* dS
  -M_A^{-1} delta(T^*) dS
  -M_A^{-1} T^* d^2S delta q

delta B =
   delta(M_A^{-1}) T^* dV
  +M_A^{-1} delta(T^*) dV
  +M_A^{-1} T^* d^2V delta q
```

For a first implementation, matrix-free JVPs can be checked by finite
differences on a fixed stratum.  Promotion requires an analytic or
GPU-vectorized JVP for performance, but finite differences are acceptable as a
diagnostic proof tool.

## Why This Eliminates the Ring

The pressure corrector computes, up to sign convention,

```text
u_f^{n+1} = u_f^* + dt [ c_corrected - G_A p ].
```

If the static constructor returns `q_static,mu` satisfying

```text
c_corrected = s(q_static)-B(q_static)mu,
Z_A(c_corrected) = 0,
```

then there exists a pressure `p` with

```text
G_A p = c_corrected.
```

For zero predictor velocity and zero external force, the corrected face
acceleration is therefore zero by the same operator used in production.  This
is not a projection trick: the full variational surface cochain still exists,
but the static constrained shape makes it entirely representable by pressure
and component reactions.

## Problem Identification

The remaining static ring is best identified as:

```text
a nonzero finite-dimensional constrained surface-energy gradient of the
sampled initial droplet, measured in the pressure-adjoint Hodge quotient.
```

It is not a failure of incompressibility, not a PPE tolerance artifact, and
not a visualization issue.  It is also not a sign that capillarity should be
weakened.  The finite-N static reference itself must be constructed from the
same equations whose zero it is supposed to test.

## Implementation Path

1. Add an explicit conservative static-criticality diagnostic using the active
   `G_A,M_A` and component saddle residual.
2. Replace arithmetic cochain division in the conservative Riesz builder by
   active pressure-adjoint face weights, or pass those weights in explicitly.
3. Implement a fail-close static initializer:
   - input: target volume, initial guess `psi`, active grid policy;
   - residual: `Z_A(s-Bmu)` plus volume;
   - constraints: fixed regular stratum, no reinit arrow, no topology change;
   - output: `psi_static` plus diagnostic certificate.
4. Validate:
   - manufactured pure range plus component still gives zero velocity;
   - sampled circle reports nonzero residual before construction;
   - constructed static state gives `capillary_hodge_weighted_l2` at solver
     tolerance and zero-speed release from rest;
   - arbitrary noncritical perturbations, including non-elliptic modes, still
     produce nonzero capillary drive.

## Verdict

The prospect is good enough to justify implementation work.  The mathematically
correct way to remove the ring is not to change the physical force during the
simulation, but to construct the static initial condition as a discrete
constrained critical point of the exact production capillary/Hodge complex.

The irreducible risk is not existence; the dimension count is favorable.  The
main risks are operator consistency and fixed-stratum regularity.  Both are
testable and can fail closed.
