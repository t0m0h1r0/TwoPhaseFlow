## Question

Complete the theory after recognizing that Ridge-Eikonal reinitialization can
change the measured droplet deformation even when the velocity is zero.

The target is not to disable reinitialization forever.  The target is a generic
theory in which:

- interface transport,
- Ridge-Eikonal/profile repair,
- capillary surface-energy work,
- pressure projection,
- static equilibrium,
- dynamic capillary release

all live in one consistent variational/Hodge contract without benchmark-name
logic.

## Verdict

The complete theory is a split-map variational theory:

```text
q^n -- T_h(u_f) --> q_T
q_T -- Pi_h    --> q^{n+1}
```

where:

- `T_h(u_f)` is the physical projection-native interface transport driven by
  the canonical face velocity;
- `Pi_h` is the Ridge-Eikonal/profile/metric projection into the admissible
  representation manifold;
- capillary work is paired only with the part of interface motion generated by
  `u_f`, unless `Pi_h` is explicitly promoted to a modeled physical
  dissipative process with its own energy budget.

In fixed-topology capillary validation, the required production contract is:

```text
Pi_h preserves the interface trace and trace surface energy:
Gamma(q^{n+1}) = Gamma(q_T),
S_h(Gamma(q^{n+1})) = S_h(Gamma(q_T)).
```

Then reinitialization is correctly "renormalized" into the state
representation: it repairs the gauge/profile, but it cannot create or remove
capillary work.  Under this contract the same Hodge gate remains decisive:

```text
static equilibrium        <=> P_h c_sigma = 0
dynamic capillary release <=> P_h c_sigma != 0
```

If `Pi_h` changes `Gamma` or `S_h` during fixed-topology motion, it is not a
harmless reinitialization.  It is an unaccounted metric/topology operation.  It
must either fail the step or be charged to an explicit non-physical projection
defect/dissipation ledger.  It must not be silently included as capillary
force.

## State Variables

Use three levels of interface state:

```text
a       trace/interface degrees of freedom, representing Gamma
q       conservative diffuse carrier, e.g. psi
phi     metric/gauge/profile field, e.g. signed-distance-like representation
```

The physical surface energy depends on trace geometry:

```text
S_h = S_h(a) = |Gamma_h(a)|.
```

The diffuse/profile fields are representation variables.  They are allowed to
support transport, reconstruction, curvature estimation, and conditioning, but
they are not allowed to change surface energy unless their induced trace change
is explicitly accounted for.

This separation is the key completion.  Reinitialization is dangerous only when
`q` and `phi` are treated as if every change in them were physical interface
motion.  It becomes admissible when it is a constrained projection that leaves
`a` fixed in fixed-topology periods.

## Physical Transport Map

Let `D_f` be the projection-native face divergence and `P_f` the face-value
operator used by the interface transport.  The physical transport step is:

```text
q_T - q^n + dt D_f( (P_f q^theta) u_f ) = 0.
```

For any virtual face velocity `w_f`,

```text
delta q_T[w_f] = -dt D_f( (P_f q^theta) w_f ).
```

This is the only fixed-topology interface displacement generated by momentum.
Therefore this is the only displacement that may be paired with reversible
capillary work.

## Ridge-Eikonal/Profile Projection

After transport, reinitialization is a constrained closest-point projection:

```text
(q^{n+1}, phi^{n+1}) = Pi_h(q_T)
```

with a schematic KKT form:

```text
minimize    1/2 ||q - q_T||_W^2
subject to  q = H_eps(-phi),
            E_h(phi) = 0                 in the interface band,
            V_h(q) = V_h(q^n),
            Gamma(q) = Gamma(q_T)        fixed topology,
            S_h(Gamma(q)) = S_h(Gamma(q_T)).
```

The multipliers of this projection live inside the representation map.  They
are not hidden body forces.  They do not enter the Navier-Stokes momentum
equation unless the model explicitly declares a physical metric/topology
dissipation law.

### Identity-Preserving Reinit

In ordinary fixed-topology capillary benchmarks, the admissible projection is
identity-preserving on the trace:

```text
Delta_Gamma_Pi = Gamma(q^{n+1}) - Gamma(q_T) = 0,
Delta_S_Pi     = S_h(Gamma(q^{n+1})) - S_h(Gamma(q_T)) = 0.
```

It may change off-interface gauge/profile data.  It may improve `|grad phi|`.
It may restore `q=H_eps(-phi)`.  But it may not move the interface or change
surface energy.

### Defect-Accounting Reinit

If the projection cannot preserve the trace, define:

```text
Delta_S_Pi = S_h(Gamma(q^{n+1})) - S_h(Gamma(q_T)).
```

Then:

- fixed topology: `|Delta_S_Pi|` above tolerance is a failure;
- topology/regularization event: `Delta_S_Pi` is charged to a named metric
  defect or dissipation budget `D_metric`;
- static/dynamic capillary validation: a nonzero `Delta_S_Pi` cannot be used
  as evidence of physical capillary motion.

This is why the observed deformation change under zero velocity is a theory
violation in the current oscillating-droplet validation: it is a projection
trace defect, not a capillary oscillation.

## Capillary Work Cochain

The capillary cochain is the transport adjoint of the trace surface-energy
variation.  Construct a discrete gradient `g_T` satisfying:

```text
<g_T, q_T - q^n> = S_h(Gamma(q_T)) - S_h(Gamma(q^n)).
```

Then define the capillary force covector by virtual work:

```text
<F_sigma, w_f>_f = -sigma <g_T, delta q_T[w_f]>.
```

Using the transport equation above:

```text
<F_sigma, w_f>_f
  = sigma dt <g_T, D_f((P_f q^theta) w_f)>
  = -sigma dt <(P_f q^theta)(-D_f)^T g_T, w_f>_f,
```

up to the exact face/cell pairings and sign convention.  The corresponding
face acceleration/cochain is the mass-inverted version used by the
PPE/corrector.  The important point is invariant:

```text
c_sigma is built from T_h^* dS_h, not from an unlabelled
(transport + reinitialization) endpoint.
```

If `Pi_h` is trace preserving, then:

```text
S_h(Gamma(q^{n+1})) - S_h(Gamma(q^n))
  = S_h(Gamma(q_T)) - S_h(Gamma(q^n)),
```

so the reinitialized state may be stored as `q^{n+1}` without changing the
capillary work identity.

If `Pi_h` is not trace preserving, the energy split is:

```text
Delta S_total = Delta S_T + Delta S_Pi,
Delta S_T     = S_h(Gamma(q_T))       - S_h(Gamma(q^n)),
Delta S_Pi    = S_h(Gamma(q^{n+1}))   - S_h(Gamma(q_T)).
```

Only `Delta S_T` belongs to reversible capillary work.  `Delta S_Pi` belongs
to projection accounting.

## Pressure-Hodge Projection

Let `M_f` be the physical face mass/metric pairing.  Let `A_f G_f p` be the
same pressure-gradient acceleration used by the PPE and velocity corrector.

The capillary pressure projection is:

```text
D_f A_f G_f p = r_h + D_f c_sigma,
a_f = A_f G_f p - c_sigma,
u_f^{n+1} = u_f^* - dt a_f.
```

The range/Hodge decomposition is:

```text
Pi_R c_sigma = A_f G_f p_sigma,
D_f A_f G_f p_sigma = D_f c_sigma,
h_sigma = c_sigma - Pi_R c_sigma.
```

The production force must be the physical `c_sigma`.  The projection
`Pi_R c_sigma` is a diagnostic and an equilibrium certificate.  Replacing
`c_sigma` by `Pi_R c_sigma` in the corrector is a force deletion and freezes
all zero-predictor capillary releases.

The generic gate is:

```text
static equilibrium        <=> ||h_sigma||_M <= tol_static ||c_sigma||_M
dynamic capillary release <=> ||h_sigma||_M >= tol_release ||c_sigma||_M
```

with `M_f`, `D_f`, `A_f`, and `G_f` identical to the production projection.

## Completed Discrete Energy Law

For a fixed-topology step with no external work, the target energy law is:

```text
K^{n+1} - K^n
+ sigma [S_h(Gamma(q_T)) - S_h(Gamma(q^n))]
+ D_visc
+ D_metric
= R_num.
```

For identity-preserving reinit:

```text
D_metric = 0,
S_h(Gamma(q^{n+1})) = S_h(Gamma(q_T)).
```

For accepted topology/metric events:

```text
D_metric accounts for Delta S_Pi and any declared metric regularization.
```

For capillary validation:

```text
D_metric must be zero or separately bounded;
R_num must converge with grid/time refinement;
Delta S_Pi must not explain the observed droplet oscillation.
```

This energy law prevents two opposite errors:

1. treating reinitialization-induced shape change as physical capillary motion;
2. deleting the physical Hodge component of capillary force to keep static
   droplets quiet.

## Theorems

### Theorem 1 - Gauge Invariance of Reinit

If:

```text
S_h = S_h(Gamma(q)),
Gamma(Pi_h(q_T)) = Gamma(q_T)
```

in a fixed-topology step, then Ridge-Eikonal/profile projection does not change
capillary work:

```text
dS_h[Pi_h T_h(u)]/du = dS_h[T_h(u)]/du.
```

Therefore reinit can be fully included in the algorithm while remaining absent
from the reversible capillary force.

### Theorem 2 - Projection Defect Is Not Capillary Force

If `Pi_h` changes `Gamma` at zero face velocity, then there is no reversible
fluid work pairing that can explain the change as capillary motion, because:

```text
delta q_T[w_f] = 0 for w_f = 0,
but Delta S_Pi may be nonzero.
```

The change must be recorded as metric/projection defect or rejected.  It cannot
be folded into `c_sigma` without introducing a fictitious force.

### Theorem 3 - Static/Dynamic Separation Survives Reinit

Under identity-preserving reinit and a variational transport cochain:

```text
P_h c_sigma = 0
```

if and only if the constrained surface-energy first variation vanishes on the
admissible incompressible face velocity space.  Thus:

- constant generalized curvature states remain static;
- non-equilibrium deformations release capillary acceleration;
- no benchmark-name branch is needed.

### Theorem 4 - Range Replacement Is Still Invalid

Even in the reinit-aware theory, replacing the production force by
`Pi_R c_sigma` yields:

```text
a_f = A_f G_f p_sigma - Pi_R c_sigma = 0
```

for zero predictor.  This freezes every dynamic release.  Reinit accounting
does not rescue range replacement as a production law.

