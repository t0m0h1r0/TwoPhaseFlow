Date: 2026-05-07

Scope: identify how to eliminate the tiny velocity-ring residual in the

## Theorem Contract

Continuum Young--Laplace equilibrium is a constrained variational statement:

```text
delta (sigma S - lambda V) = 0.
```

For a smooth closed component this gives constant curvature.  In the code,
however, the decisive object is the finite-dimensional pressure-adjoint
capillary cochain.  With the selected discrete endpoint, pressure operator,
and kinetic metric,

```text
s_h      = -M_A^{-1} T_h^* dS_h
B_h      =  M_A^{-1} T_h^* dV_h
c_h(mu)  = s_h - B_h mu
h_h(mu)  = Z_A c_h(mu)
```

where `Z_A` is the weighted pressure-Hodge complement generated by the same
`D_f`, `A_f`, `G_A`, and `M_A` used by the corrector.  A droplet is discrete
static if and only if there exists a component pressure reaction `mu` such
that

```text
h_h(mu) = 0.
```

This is shape-free.  It does not ask whether the interface is a circle,
ellipse, or named mode.  It asks whether the chosen discrete surface-energy
covector is a constrained critical covector for the chosen transport endpoint
and pressure-Hodge quotient.

## Problem Identification

The tiny velocity ring exists because the initial "static droplet" is a
sampled continuum circle, not a solution of the selected finite-dimensional
Euler--Lagrange equation.

At finite `N`, the discrete surface functional, volume functional, transport
endpoint, pressure range, and mass metric define their own static manifold.
The continuum circle is only a convergence target.  Unless the initialized
interface is constructed to satisfy

```text
Z_A(s_h - B_h mu) = 0
```

on the actual fitted grid and with the actual corrector metric, the solver is
mathematically correct to generate a small velocity: it is moving the interface
along the residual constrained surface-energy gradient.

There is also a theorem-consistency risk that must be removed before claiming
roundoff staticity: the cochain builder and the pressure-Hodge projection must
use the same `M_A`.  The current checks show this is not the dominant
magnitude source, but the exact theorem requires one metric object throughout.

## Valid Remedies

### R1. Discrete Static-State Constructor

Construct the static droplet as a discrete constrained critical point, not as a
sampled analytic circle.

Unknowns:

```text
psi, q_c, mu
```

or, on a fixed regular stratum,

```text
trace vertices z, grid q_c, mu
```

Residual:

```text
R_static(psi, q_c, mu) = Z_A(q_c) [s_h(psi,q_c) - B_h(psi,q_c) mu]
V_h(psi,q_c) - V0 = 0
```

Acceptance:

```text
||R_static||_{M_A} <= solver-roundoff-scaled tolerance
volume error <= volume tolerance
same regular stratum, or fail-close
```

This is the most physical and mathematical remedy.  It does not classify the
shape.  It lets the discrete equations decide which interface is static.

### R2. Metric Unification

Build `s_h` and `B_h` using the same face metric `M_A` used by the affine-jump
pressure-Hodge projection:

```text
s_h = -M_A^{-1} T_h^* dS_h
B_h =  M_A^{-1} T_h^* dV_h
```

This is necessary for exact proof even if it is not sufficient to eliminate
the ring for a noncritical initial shape.

### R3. Conservative Static-Criticality Diagnostic

Export a conservative-endpoint static gate, analogous to the trace diagnostic:

```text
capillary_conservative_static_residual_weighted_l2
capillary_conservative_static_residual_ratio
```

The current `capillary_static_critical_residual_ratio=0` in the cached run is
not evidence of staticity; it means the conservative criticality diagnostic is
not yet wired.  The fail-close rule should require this diagnostic for static
validation.

## Invalid Remedies

The following would hide the symptom rather than solve the theorem:

- damping the velocity ring;
- tightening CFL to make it visually smaller;
- curvature caps or smoothing;
- replacing the production cochain by its pressure-range projection;
- special-casing circles;
- imposing a constant Young--Laplace jump as a benchmark branch;
- subtracting the initial residual as a force correction.

All of these can make the static picture quieter while breaking the
surface-energy virtual-work contract or the oscillating-droplet drive.

## Verdict

The nearest correct path is:

```text
1. unify the capillary cochain metric with M_A;
2. add a conservative static-criticality gate;
3. implement a fail-close discrete static-state constructor;
4. require N32 static validation to satisfy Z_A(s_h-B_h mu) ~= 0 before
   interpreting zero velocity as a physical result.
```

Only step 3 can eliminate the finite-N ring by theorem.  Steps 1 and 2 are
required so that the theorem being solved is exactly the theorem used by the
production pressure corrector.
