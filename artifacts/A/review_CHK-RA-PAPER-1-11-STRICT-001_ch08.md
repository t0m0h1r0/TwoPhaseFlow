## Round 1 Findings

### MAJOR: 章の中心概念が `face cochain / pressure-jump / face complex` のまま残る

第8章は「どの量を同じ面空間で釣り合わせるか」が主題だが、`face cochain`, `pressure-jump`, `face complex`, `face slot`, `surface-energy cochain`, `component-corrected cochain` がそのまま残り、読者が概念を掴みにくい。

対応:

- `面共鎖`, `圧力ジャンプ`, `面複体`, `面補正枠`, `表面エネルギー面共鎖`, `成分補正済み面共鎖` に統一。
- 章冒頭、構成リスト、Balanced--Force 条件、FCCD 面フラックス部分系の説明を同じ語彙に揃えた。

### MAJOR: コロケート格子の2つの破綻原因が英語のまま曖昧

`decoupling`, `face complex`, `PPE/corrector`, `RHS` の混在により、演算子不整合と格子スケールの圧力・速度分離が別問題であることが見えにくかった。

対応:

- `格子スケールの圧力・速度分離`, `同じ面複体による PPE/補正段共有`, `右辺発散` として整理。
- Rhie--Chow の説明も `面速度補間` に修正。

### MAJOR: P-1--P-7 が API 契約のように読める

`残差契約`, `標準契約`, `精度契約`, `gate`, `range projection`, `raw` が混在し、査読者には実験・実装メモの名残として見える。

対応:

- `残差受理条件`, `標準受理条件`, `精度受理条件`, `判定量`, `値域射影`, `未補正` に修正。
- 固定回数 `k=3` は実用上限であり、標準 BF 閉包の受理条件ではないことを明確化。

## Round 3 Findings

### MAJOR: `面共鎖` への統一だけでは、Balanced--Force の物理的主張が読者に届かない

前回までの修正は英語語彙の日本語化としては進んだが、章の主語が `圧力面共鎖`、`毛管ジャンプ面共鎖`、`面複体` のままで、読者は「同じ面で圧力補正と毛管補正を釣り合わせる」という物理的な核を掴みにくい。これは単なる語彙問題ではなく、章冒頭から七原則、FCCD 部分系までのナラティブが抽象語に寄りすぎている MAJOR 指摘である。

対応:

- 章冒頭の問いを `圧力による面加速度` と `毛管ジャンプ補正量` の釣合に書き換えた。
- `面共鎖` は本文の主語から外し、`面上補正量`、`毛管補正量`、`面上の釣合`、`面空間` として読める表現に変更した。
- P-1/P-3/P-4/P-6/P-7 と FCCD 部分系の表・アンチパターンを同じ語彙でそろえ、式の記号と本文説明を分離した。

### MAJOR: Ch9 の Hodge/Riesz/アフィンジャンプ細部が Ch8 の失敗モード説明を圧迫している

第8章は Balanced--Force の必要条件を説明する章であり、Hodge 平衡判定やアフィンジャンプ面複体を主語にすると、失敗モード F-1--F-5 と原則 P-1--P-7 の対応が見えにくくなる。詳細な毛管理論は §9 へ送るべきである。

対応:

- `アフィンジャンプ` は本文では `圧力ジャンプ` とし、GFM/HFE の接続だけを残した。
- `Hodge 平衡判定` は `毛管平衡判定` に置き換え、第8章では「非平衡毛管成分を消してはいけない」という設計上の意味を前面に出した。
- 固定回数 `k=3` の観測記述を削り、DC の受理条件を高次残差の低下に戻した。
