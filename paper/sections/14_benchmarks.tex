\section{二相流ベンチマーク}
\label{sec:validation}
\label{sec:benchmarks}

第~\ref{sec:verification}章では，単相 NS，静止液滴，密度比 sweep，
非一様格子，CLS 強変形移流を用いて，個々の離散閉包が
設計通りに働く範囲を確認した．
本章では，その部品検証を越えて，移動界面・表面張力・変密度投影・浮力が
同時に作用する物理ベンチマークを扱う．
対象は，毛管波，Rayleigh--Lamb 型振動液滴，単一気泡上昇，
および Rayleigh--Taylor 不安定性の 4 本である．

本章のナラティブは，界面力学の難しさを段階的に増やすことで統一する．
毛管波では，PhaseRegion graph chart が
同じ相領域所有・表面エネルギー第二変分から小振幅復元力を再現するかを見る．
振動液滴では，PhaseRegion closed chart が同じ変分原理で
閉じた界面の mode-2 復元を 1 周期で再現できるかを見る．
単一気泡上昇では，表面張力に加えて浮力残差と圧力射影が同じ面位置を共有するかを読むが，
本稿では形状破綻が現れる前の初期窓 $0\le t\le0.03\,\mathrm{s}$ に掲載範囲を限定する．
Rayleigh--Taylor では，浮力が安定化ではなく不安定成長を作る配置で，
線形増幅から非線形形状形成までを追う．
静止円形液滴は標準構成の静止平衡検証として第~\ref{sec:verification}章で扱ったため，
本章では重複させない．

\paragraph{本章の式・グラフの読み方．}
本章のベンチマークは，数値が理論曲線に近いかだけを見る章ではない．
毛管波では，界面変位が PhaseRegion の graph chart 上で
$q_h=Q_h(\Omega_h)$ と $E_h[\Omega_h]$ の同じ状態から出て，
線形毛管波の位相とエネルギーを再現するかを見る．
振動液滴では，同じ変分原理を closed chart へ移したとき，
面積保存制約，finite-volume 測度，mode-2 表面エネルギーが
1 周期で整合するかを見る．
気泡上昇では，浮力残差，圧力射影，界面輸送が同じ面速度を共有しているかを見る．
Rayleigh--Taylor では，浮力が意図通り不安定成長を作り，
数値拡散や圧力補正がその成長を不自然に消していないかを見る．
つまり各グラフは見栄えの確認ではなく，保存性，初期応答，有界性，
形状発展の 4 軸が同時に破綻していないかを読む診断である．

\subsection{共通数値構成と判定方針}
\label{sec:val_grid_policy}
% ═══════════════════════════════════════════════════════════════════

4 本のベンチマークは，実験ごとに幾何・境界条件・重力を変える一方で，
次の共通数値構成を共有する：
\begin{itemize}[noitemsep,leftmargin=2em]
  \item 界面適合非一様格子 $\alpha=2$，
        移動界面追従ケースでは各時間段で格子を再構築，
  \item 標準 CLS/NS 経路では FCCD 保存形面フラックス輸送，
        PhaseRegion 毛管経路では $\Omega_h$ を所有し
        $q_h=Q_h(\Omega_h)$ を測度として読む chart stepper，
  \item UCCD6 対流 + IMEX--BDF2，CCD 粘性 + 陰的 BDF2 + 欠陥補正，
  \item PhaseRegion 毛管経路：
        相領域 $\Omega_h$ と境界 atlas $\Gamma_h$ を主状態とし，
        $q_h=Q_h(\Omega_h)$，$E_h=\sigma\,\mathrm{Per}(\Omega_h)$，
        chart 第一変分，測度残差，および
        \texttt{force\_admissible} を明示的に出力，
  \item Ridge--Eikonal 再初期化を NS ベンチマークの設定に従って適用し，
        PhaseRegion 毛管波と振動液滴では
        chart から測度と境界を再構成する，
  \item 向き付き圧力ジャンプ
        $j_{gl}=p_g-p_l=-\sigma\kappa_{lg}$ を現在の切断面法則で評価する
        分相 FCCD PPE + アフィンジャンプ補正，
  \item 射影と同じ面フラックスと正準面速度による界面輸送，
        および格子再構築時の射影後面量の移送．
\end{itemize}
したがって本章で比較するのは求解器選択の違いではなく，
同じ離散閉包を異なる物理状況へ置いたときの応答である．
ただし，PhaseRegion graph/closed chart 実験は現時点では
圧力・速度場へ力を消費しない reduced route であり，
\texttt{force\_admissible=0} を合格条件に含める．
したがって旧 production runtime で得た速度場・圧力場は，
PhaseRegion 正規ルートの成功根拠としては用いない．

非一様格子を採用できる条件は，格子密度比 $\alpha$ そのものではなく，
投影・浮力残差・速度補正・界面輸送が同じ面位置を共有することである．
投影形 PPE が $D_fA_fG_f$ を用い，求解形では $-D_fA_fG_f$ を解くなら，
補正段は同じ $A_fG_f$ で速度を補正し，
浮力残差も同じ面勾配系列上で評価する
（式~\eqref{eq:same_face_projection_closure}）．
さらに，射影で得た正準面速度を
式~\eqref{eq:projection_native_psi_transport} に直接渡す．
節点へ再構成した速度から界面フラックスを作り直す経路は，
同じ離散化の別表現ではなく，面速度空間の受理条件から外れる．

各ベンチマークは，時間履歴診断とスナップショット診断を組み合わせて判定する．
時間履歴では体積保存，運動エネルギー，界面振幅，変形量，重心，
平均上昇速度などを読む．
スナップショットでは $\psi$，速度場，圧力代表を同じ時刻で読み，
体積保存だけでは見えない圧力ジャンプ/浮力の不整合を検出する．
判定は，単なる完走ではなく，
\textbf{保存性，理論で決まる初期応答，長時間の有界性，形状発展の物理的一貫性}
の 4 軸で行う．
