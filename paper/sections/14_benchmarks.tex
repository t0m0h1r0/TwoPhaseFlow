\section{二相流ベンチマーク}
\label{sec:validation}
\label{sec:benchmarks}

第~\ref{sec:verification}章では構成要素ごとの精度と NS 連成の物理的整合性を検証した．
本章では，研究用 YAML として維持する本章の本番検証セットを，
毛管波（§~\ref{sec:val_capillary}），浮力駆動気泡上昇
（§~\ref{sec:val_rising_bubble}），および Rayleigh--Taylor smoke 設定の
3 本に限定する．
いずれも第~\ref{sec:verification}章の部品検証には含まれない
移動界面・表面張力・変密度投影の結合問題である．
したがって本章の目的は単なる「動いた/動かない」の報告ではなく，
第~\ref{sec:two_to_one}章の浮力分解，
第~\ref{sec:time_int}章の IMEX--BDF2 時間発展，
第~\ref{sec:split_ppe_framework}章の分相 PPE，
および第~\ref{sec:ridge_eikonal_nu_grid}章の Ridge--Eikonal 非一様 FMM が
同一の離散不変量を共有しているかを確認することである．

\subsection{本章の YAML セットと格子方針}
\label{sec:val_grid_policy}
% ═══════════════════════════════════════════════════════════════════

本章で維持する YAML は次の3本である：
\begin{itemize}[noitemsep,leftmargin=2em]
  \item \texttt{experiment/ch14/config/ch14\_capillary.yaml}
  \item \texttt{experiment/ch14/config/ch14\_rising\_bubble.yaml}
  \item \texttt{experiment/ch14/config/ch14\_rayleigh\_taylor.yaml}
\end{itemize}
これらは代表物性（毛管波・上昇気泡は水--空気；
Rayleigh--Taylor は水（重相，上）／シリコーン油 5cSt（軽相，下）），
非一様 interface-fitted 格子 $\alpha=2$，Ridge--Eikonal 再初期化，
FCCD 圧力勾配，HFE 曲率，pressure-jump 型分相 PPE，
UCCD6 対流，および IMEX--BDF2/implicit-BDF2 時間発展を共有する．
RT については，YAML スキーマ上 \texttt{physics.phases.liquid} スロットに下層シリコーン油，
\texttt{physics.phases.gas} スロットに上層水を割り当てる
（スキーマのスロット名と物理流体名を分離した運用である）．
時間積分の安定境界は YAML \texttt{run.time.cfl} を理論 CFL 定数への乗数として与え，
本章の3本はいずれも乗数 $1.0$ で，移流 $C_{\mathrm{adv}}=0.10$，毛管 $C_{\mathrm{cap}}=0.05$，
粘性 $C_{\mathrm{visc}}=1.0$ に解決される
（定数定義は \texttt{src/twophase/simulation/config\_run\_builder\_sections.py:20--22}；
毛管 CFL 定数の物理的根拠は~\cite{DennerVanWachem2015}）．

非一様格子を採用できる条件は，格子密度比 $\alpha$ そのものではなく，
投影・浮力残差・速度補正が同じ face locus を共有することである．
具体的には投影形 PPE が $D_fA_fG_f$ を用い，solve 形では $-D_fA_fG_f$ を解くなら，
Corrector は同じ $A_fG_f$ で速度を補正し，
浮力残差も同じ face-gradient chain 上で評価する
（式~\eqref{eq:same_face_projection_closure}）．
さらに interface transport は，射影で得た正準面速度を
式~\eqref{eq:projection_native_psi_transport} に直接渡す．
すなわち，節点へ再構成した速度から界面 flux を作り直す経路は
同じ YAML stack の別選択肢ではなく，projection-native な離散契約の破れである．
この閉包がない場合，非一様格子では「同じ物理メトリクスを使った」だけでは不十分で，
面係数・壁面行・界面 cut face の扱いの差が投影後発散として残る．

出力は YAML のみで規定する．各実験は時系列診断に加え，
一定間隔で \texttt{psi}，速度場，圧力場の snapshot PDF/NPZ を保存する．
これは，体積保存だけでは pressure-jump/buoyancy の不整合を検知できず，
速度場・圧力場・界面場を同時に読む必要があるためである．

静止円形液滴 $N=64$ の原因切り分けでは，節点再構成 velocity による
旧 interface transport が final cut-face $\kappa$ std $1.07\times10^{1}$ を生じたのに対し，
式~\eqref{eq:projection_native_psi_transport} の正準面速度更新では
$1.86$ まで低下した．同じ初期界面を固定した control では
$1.66\times10^{-2}$ であったため，主要な failure mode は
曲率式そのものではなく，projection face state と interface flux の
表現不一致であった．この結果は CFL や smoothing の調整ではなく，
face-space 契約の履行として扱う．

\subsection{毛管波：向き付き affine jump の符号検証}
\label{sec:val_capillary}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と理論基準：符号が復元力を決める}

初期界面を平坦界面に正弦波擾乱として与える：
\begin{equation}
  y_\Gamma(x,0)=y_0 + A_0\cos(kx+\theta_0),
  \qquad
  k=\frac{2\pi m}{L_x}.
  \label{eq:capillary_wave_initial}
\end{equation}
YAML では $y_0=0.5$，$A_0=0.01$，$m=2$，$L_x=1$，$\theta_0=0$ とする．
本設定の二流体毛管波 benchmark は DNS 規約に倣う~\cite{Tryggvason2011}．
小振幅の二流体毛管波では，粘性と壁の影響を無視した最初の基準として
\begin{equation}
  \omega^2
  = \frac{(\rho_l-\rho_g)g\,k+\sigma k^3}{\rho_l+\rho_g},
  \qquad
  g=0 \ \text{for this YAML},
  \label{eq:flat_capillary_dispersion}
\end{equation}
が成り立つ~\cite{Prosperetti1981}．
本設定では重力を切るため，観測すべき復元力は界面曲率と圧力ジャンプだけから来る．
したがってこのケースは「毛管波を正しく振動させられるか」を見るだけでなく，
\textbf{Ridge--Eikonal による距離関数品質，HFE 曲率，向き付き affine jump，
IMEX--BDF2 投影}の動的閉包が同時に働くかを切り出す実験である．

本稿の向き付き規約は
\[
  \psi=1\ \text{液相},\quad \psi=0\ \text{気相},\quad
  \kappa_{lg}=\nabla_\Gamma\!\cdot\hat{\bm n}_{lg},\quad
  j_{gl}=p_g-p_l=-\sigma\kappa_{lg}
\]
である．毛管波の山では gasward 曲率が正になり，$j_{gl}<0$ となるため，
初期加速度は
\begin{equation}
  A''(0)=-\omega^2A_0
  = -1.427056761315\times10^{-3}
  \label{eq:capillary_initial_accel_reference}
\end{equation}
でなければならない．符号が逆なら，同じ大きさの毛管剛性が
復元力ではなく anti-restoring 力として働く．

\begin{itemize}[noitemsep,leftmargin=2em]
  \item グリッド：$[0,1]^2$，$32^2$，interface-fitted $\alpha=2$，壁面境界
  \item 物性：$\rho_l=1000$，$\rho_g=1.2$，$\mu_l=10^{-3}$，$\mu_g=1.8\times10^{-5}$，
        $\sigma=0.072$，$g=0$
  \item 時間：$T=10.0$，CFL 乗数 $1.0$
        （移流 $C_{\mathrm{adv}}=0.10$／毛管 $C_{\mathrm{cap}}=0.05$／粘性 $C_{\mathrm{visc}}=1.0$），
        界面移流は TVD-RK3，運動量は IMEX--BDF2
  \item 幾何：\texttt{ridge\_eikonal} を 20 ステップ毎に適用し，
        $\varepsilon_\text{scale}=1.4$，$\sigma^{\mathrm{ridge}}_0=3.0$
  \item 界面厚さ：\texttt{thickness.mode=local}，\texttt{base\_factor=1.5}；
        曲率は HFE（\texttt{psi\_direct\_hfe}），キャップ $5.0$
  \item 圧力結合：\texttt{phase\_separated} + \texttt{affine\_jump}．
        \texttt{InterfaceStressContext} は gas-minus-liquid pressure jump
        $j_{gl}$ を保存し，raw な $\sigma\kappa$ は保存しない
  \item 出力：\texttt{interface\_amplitude}，体積保存，運動エネルギー，
        さらに \texttt{psi}・速度・圧力 snapshot を $0.05$ 間隔で保存
\end{itemize}

\subsubsection{原因連鎖：Balanced Force ではなくジャンプ契約が主要因}

旧 \texttt{jump\_decomposition} 経路は，鋭い界面ジャンプを
regular 圧力場 $J$ として持ち込んだため，PPE が
$p_\mathrm{base}\simeq -J$ を選び，合成圧力
$p_\mathrm{total}=p_\mathrm{base}+J$ をほぼゼロにした．
一ステップ診断の
\[
  \frac{\operatorname{ptp}(p_\mathrm{total})}{\operatorname{ptp}(J)}
  =9.022527\times10^{-4}
\]
は，長時間不安定・再初期化履歴・曲率キャップ以前に起きる
代数的相殺であり，毛管復元力が PPE に届かないことを示す．

次に導入した \texttt{affine\_jump} は
$G_\Gamma(p;j)=G(p)-B_\Gamma(j)$ により相殺を除いたが，
当初は $p_g-p_l$ として $+\sigma\kappa$ を渡していた．
そのため $N=32,T=10$ run は $t=4.882542161581$，$884$ step で
運動エネルギー guard に停止し，早期 signed-mode 加速度は
$A''_\mathrm{obs}=+1.502517625523\times10^{-3}$ であった．
式~\eqref{eq:capillary_initial_accel_reference} と大きさは一致するが符号が逆であり，
毛管剛性は存在するが anti-restoring に作用していた．
したがって主要因は Balanced Force 原理そのものではなく，
Balanced Force が受け取る物理ジャンプ $j_{gl}$ の向きである．

\subsubsection{判定と残課題}

本節の判定は「毛管波分散関係の完全成功」ではなく，
\textbf{旧 failure の主要因であった regular jump cancellation と符号反転を解消し，
$N=32,T=10$ の短時間 run で正しい初期復元加速度を確認した}である．
一方で，\texttt{interface\_amplitude} は成長し，
\texttt{kappa\_max} は cap $5.0$ に到達しているため，
Prosperetti 分散関係に対する位相・周期の定量一致はまだ主張しない．
次の検証は，この向き付き affine jump を固定した上で，
曲率 cap，HFE 曲率平滑化，壁面境界，毛管 CFL，解像度を分離し，
幾何エネルギー安定性を確認する sweep として設計すべきである．

\subsection{単一気泡上昇}
\label{sec:val_rising_bubble}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と理論的焦点}

気泡（気相円，半径 $R=0.25$）を
$[0,1]\times[0,2]$ の液相中 $(0.5,0.5)$ に置き，
小さい重力 $g=10^{-3}$ の下で上昇を計算する．
この設定は Hysing 型の単一気泡配置~\cite{Hysing2009}を幾何的原型とし，
diffuse-interface 系の対応 benchmark~\cite{AlandVoigt2019} と整合するが，
物性は水--空気級の高密度比に戻し，
時間区間は長時間 closure と上昇ダイナミクスの両方を観察する $T=70.0$ に取る．

本ケースの理論的焦点は，浮力をそのまま nodal predictor に入れないことである．
浮力分解 $\rho\bm{g}=-\bnabla(\rho\Phi_g)+\Phi_g\bnabla\rho$
（第~\ref{sec:two_to_one}章で導出）の閉包により，
hydrostatic な勾配成分は圧力空間に残し，
密度変化に支えられた残差のみを face residual acceleration として組み立てる．
この残差は PPE と同じ $A_f,G_f,D_f$ 上で評価する必要がある；
そうでなければ投影で消えるべき gradient force が非ソレノイダルな速度として残る
（balanced-force 表面張力との同類概念は~\cite{Francois2006}）．

\begin{itemize}[noitemsep,leftmargin=2em]
  \item グリッド：$[0,1]\times[0,2]$，$128\times256$，interface-fitted $\alpha=2$，壁面境界
  \item 物性：$\rho_l=1000$，$\rho_g=1.2$，$\mu_l=10^{-3}$，$\mu_g=1.8\times10^{-5}$，$\sigma=0.072$，$g=10^{-3}$
  \item 時間：$T=70.0$，CFL 乗数 $1.0$
        （移流 $C_{\mathrm{adv}}=0.10$／毛管 $C_{\mathrm{cap}}=0.05$／粘性 $C_{\mathrm{visc}}=1.0$），
        界面移流は TVD-RK3，運動量は IMEX--BDF2
  \item 幾何：\texttt{ridge\_eikonal} を 4 ステップ毎に適用し，
        $\varepsilon_\text{scale}=1.4$，$\sigma^{\mathrm{ridge}}_0=3.0$
  \item 界面厚さ：\texttt{thickness.mode=local}，\texttt{base\_factor=1.5}；
        曲率は HFE（\texttt{psi\_direct\_hfe}），キャップは YAML 既定値
        （\texttt{interface.geometry.curvature.cap} の明示無し；
        毛管波・RT は \texttt{cap=5.0} を明示する点との差分に注意）
  \item 浮力：\texttt{balanced\_buoyancy} predictor assembly
  \item 投影係数：\texttt{phase\_separated}，\texttt{pressure\_jump}
  \item 面状態：\texttt{face\_flux\_projection}，
        \texttt{canonical\_face\_state}，
        \texttt{face\_native\_predictor\_state}
  \item 出力：重心 $y_c(t)$，平均上昇速度，体積保存，変形量，
        さらに \texttt{psi}・速度・圧力 snapshot を $0.05$ 間隔で保存
\end{itemize}

\subsubsection{受入基準と想定診断}

\textit{（実験結果は計算完了次第掲載）}

本ケースは「終端速度 benchmark」そのものではなく，
水--空気級密度比・非一様格子・pressure-jump・浮力残差分解を同時に有効化したときに
投影不整合による早期ブローアップが起きないことを確認する閉包 benchmark を
長時間まで延長した位置づけである．想定する受入基準は次の通り：
\begin{enumerate}[noitemsep,leftmargin=2em]
  \item \textbf{保存性} \texttt{volume\_conservation} の最終値
        $|\Delta V|/V_0 < O(10^{-4})$．
  \item \textbf{重心} \texttt{bubble\_centroid}（\texttt{yc}）が単調上昇に推移．
  \item \textbf{平均上昇速度} \texttt{mean\_rise\_velocity} が時間とともに
        終端値へ漸近．
  \item \textbf{変形量} \texttt{deformation} は弱重力 $g=10^{-3}$ の下では
        狭い範囲で推移し，急激な振動が出ない．
\end{enumerate}
物理 benchmark としての長時間終端速度・形状収束を Hysing 数値解と直接比較する場合は，
$g$，$T$，および無次元数を Hysing benchmark と一致させた別 YAML で評価する．

\subsection{Rayleigh--Taylor 不安定性}
\label{sec:val_rayleigh_taylor}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と理論基準}

水（重相，上）／シリコーン油 5cSt（軽相，下）の二層配置を平坦界面に正弦波擾乱として与え，
重力下で Rayleigh--Taylor 不安定の線形成長から非線形 mushroom 領域までを観察する．
線形理論では，小振幅擾乱 $A_0$ は
\begin{equation}
  A(t) = A_0 \, e^{\gamma t},
  \qquad
  \gamma = \sqrt{\mathrm{At}\, g\, k},
  \qquad
  \mathrm{At} = \frac{\rho_{\text{w}}-\rho_{\text{oil}}}{\rho_{\text{w}}+\rho_{\text{oil}}},
  \qquad
  k = \frac{2\pi m}{L_x},
  \label{eq:rt_linear_growth}
\end{equation}
の指数増大を示す（粘性・表面張力・有限振幅の影響は線形近似では落とす；
RT 線形安定論の正準的取り扱いは~\cite{Chandrasekhar1961}）．
本 YAML の数値代入では
$\mathrm{At}=(1000-950)/(1000+950)\approx 0.0256$（低 Atwood），
$k=2\pi/0.5=4\pi\approx 12.57\ \mathrm{m^{-1}}$，
$\gamma\approx \sqrt{0.0256\times 9.81\times 4\pi}\approx 1.78\ \mathrm{s^{-1}}$
となり，最終時刻 $T=7.0\ \mathrm{s}\approx 12.5/\gamma$ は
線形指数領域を十分超え，非線形 mushroom 領域に到達する設定である
（低 Atwood RT の DNS benchmark 規約は~\cite{Tryggvason1988}）．

スロット規約は §\ref{sec:val_grid_policy} に従う．
すなわち YAML スキーマ上は \texttt{physics.phases.liquid} スロットに
下層シリコーン油，\texttt{physics.phases.gas} スロットに
上層水を割り当てる．
擾乱形状は毛管波と同型であるため
\texttt{initial\_condition.type=capillary\_wave} ハンドラを流用するが，
重力下で重相が上にある不安定な層化のため，
本ケースの物理は RT である（毛管波ではない）．

\begin{itemize}[noitemsep,leftmargin=2em]
  \item グリッド：$[0,0.5]\times[0,2.0]$，$128\times512$，
        interface-fitted $\alpha=2$，壁面境界
  \item 物性（下層シリコーン油 5cSt）：
        \texttt{phases.liquid}: $\rho=950$，$\mu=5\times10^{-3}$
  \item 物性（上層水）：
        \texttt{phases.gas}: $\rho=1000$，$\mu=10^{-3}$
        （スキーマのスロット名は \texttt{gas} だが物理は水）

\end{enumerate}

§~\ref{sec:error_budget} で特定した残存限界との対応：
\begin{itemize}[noitemsep]
  \item \textbf{$\sigma > 0$ 移動界面結合}——各 YAML は pressure-jump 分相 PPE を用い，
        CSF 体積力ではなく向き付き圧力ジャンプ
        $j_{gl}=p_g-p_l=-\sigma\kappa_{lg}$ として表面張力を扱う．
        毛管波 run では旧 regular jump cancellation と符号反転を解消し，
        初期復元加速度の符号・大きさを確認した（\textbf{部分補完}）．
  \item \textbf{再初期化--移流蓄積}——Ridge--Eikonal + local thickness により，
        毛管波では非一様 $\alpha=2$ のまま $T=10$ run と
        $|\Delta V|/V_0^{\max}=6.75\times10^{-4}$ を確認した．
        気泡上昇・RT については各受入基準で同じ
        $|\Delta V|/V_0 < O(10^{-4})$ を確認する（\textbf{補完}）．
  \item \textbf{毛管波 CFL 安定境界}——YAML \texttt{run.time.cfl} は理論 CFL 定数への
        乗数であり，本章 3 本の乗数 $1.0$ は移流 $0.10$／毛管 $0.05$／粘性 $1.0$ に
        解決される（実装は
        \texttt{src/twophase/simulation/}\allowbreak\texttt{config\_run\_builder\_sections.py:43}）．
        今回の $32^2$ run はこの CFL で $T=10$ まで完走し，
        式~\eqref{eq:flat_capillary_dispersion} の初期加速度符号を再現したが，
        振幅成長と曲率キャップ $5.0$ 到達が残る．
        毛管 CFL 制約の最近の体系的議論~\cite{DennerVanWachem2022} と合わせて，
        解像度・CFL 乗数・曲率キャップを分離した sweep で切り分ける（\textbf{継続}）．
  \item \textbf{浮力--圧力閉包}——気泡上昇では hydrostatic 勾配を圧力空間に残し，
        face residual のみを同一面投影へ渡すことで早期発散を抑制する（\textbf{補完}）．
  \item \textbf{RT 不安定 閉包}——低 Atwood（$\mathrm{At}\approx 0.026$）かつ
        強重力 $g=9.81$ 下で，浮力残差分解と pressure-jump 閉包が
        式~\eqref{eq:rt_linear_growth} の線形指数領域から非線形 mushroom
        領域までの長時間発展を支える想定である（\textbf{継続}）．
  \item \textbf{物理 benchmark の未完了点}——毛管波は「向き付き affine jump の符号 pass／分散応答は継続」
        として記録し，気泡の終端速度および RT の線形成長率検証は
        実行完了後に同じ形式で判定する．毛管波については，
        oriented affine jump を固定した resolution/CFL/curvature-cap sweep で
        物理応答の閉包を確定する（\textbf{継続}）．
\end{itemize}
