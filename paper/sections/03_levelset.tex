\section{Conservative Level Set (CLS) 法による界面追跡}
\label{sec:levelset}

本章では，界面追跡変数 $\psi = H_\varepsilon(\phi)$ の定式化から
幾何量（曲率・法線）の高精度計算まで，CLS 法の全体像を整備する．

\begin{enumerate}
  \item \textbf{§~\ref{sec:why_cls}（なぜ CLS か）：}
        標準 LS 法の2つの独立した問題——Eikonal 条件の崩れと質量損失——の
        根本原因を示し，CLS 法がどちらを解決するかを明確化する．
  \item \textbf{§~\ref{sec:cls_advection}（保存形移流）：}
        $\psi$ の保存形（発散形）移流方程式と，
        発散定理による連続レベルの質量保存，および離散実装での条件付き保存性を示す．
  \item \textbf{§~\ref{sec:reinit}（再初期化）：}
        移流後のプロファイル崩れを修復する圧縮--拡散方程式と，
        界面幅パラメータ $\varepsilon$ の設定指針を述べる．
  \item \textbf{§~\ref{sec:levelset_mapping}（$\psi$--$\phi$ 対応）：}
        ロジット解析逆変換（$\psi \to \phi$）の実装と，
        Extension PDE で必要な法線計算 $\hat{\bm{n}} = \bnabla\phi/|\bnabla\phi|$ への応用を示す．
  \item \textbf{§~\ref{sec:curvature_direct_psi_cls}（曲率の直接計算）：}
        ロジット逆変換を省略して $\psi$ 微分から曲率を直接求める高効率手法と
        その数値安定性を述べる．
\end{enumerate}

\subsection{標準 Level Set 法の問題点：なぜ CLS が必要か}
\label{sec:why_cls}

第~\ref{sec:intro}章では，界面のにじみ・質量損失（§~\ref{sec:failure_modes}）という
\textbf{現象面での問題}を示した．
本節では，これらが標準 Level Set 法の移流方程式の\textbf{数学的構造}に起因することを明確にし，
CLS 法がどのようにそれを解決するかを段階的に示す．

\subsubsection{標準 Level Set 法の移流と \texorpdfstring{$|\bnabla\phi|\neq 1$}{|grad phi| neq 1} 問題}

標準 Level Set 法では，符号付き距離関数 $\phi$ の移流方程式
\begin{equation}
  \pd{\phi}{t} + \bu\cdot\bnabla\phi = 0
  \label{eq:ls_standard}
\end{equation}
を解く．この方程式は\textbf{非保存形}（発散形でない）であるため，
数値積分の過程で $\int\phi\,dV$ の保存が保証されない．

さらに，移流計算の数値誤差によって，$\phi$ が持つべき「符号付き距離関数」の性質
\[
  |\bnabla\phi| = 1 \quad \text{（Eikonal 条件）}
\]
が時間とともに崩れていく．

\paragraph{$|\bnabla\phi|\neq 1$ が引き起こす問題}
Eikonal 条件の崩れ $|\bnabla\phi|\neq 1$ は，\textbf{幾何量の精度}に直結する：
\begin{enumerate}
  \item \textbf{法線ベクトルの精度低下：}
        $\hat{\bm n} = \bnabla\phi/|\bnabla\phi|$ の計算で，
        $|\bnabla\phi|$ が 1 から大きく外れると，
        法線方向が不正確になる．
  \item \textbf{曲率の誤計算（致命的）：}
        曲率 $\kappa = -\bnabla\cdot(\bnabla\phi/|\bnabla\phi|)$ の計算精度が
        $|\bnabla\phi|$ の誤りに強く依存するため，
        表面張力の計算が壊滅的に悪化し，寄生流れが生じる．
\end{enumerate}
\textbf{重要：} Eikonal 条件の崩れが直接「体積（質量）の損失」を引き起こすわけではない．
質量損失は，次の §~\ref{sec:why_cls_massloss} で述べる\textbf{独立した原因}に由来する．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{標準 Level Set 法における質量損失の真の原因：非保存形の離散化誤差}
\label{sec:why_cls_massloss}
% ─────────────────────────────────────────────────────────────────────────────

質量損失は，$|\bnabla\phi|\neq 1$ とは\textbf{独立した}問題である．
その根本原因は，標準 LS 移流方程式
\begin{equation}
  \pd{\phi}{t} + \bu\cdot\bnabla\phi = 0
  \label{eq:ls_noncons}
\end{equation}
が\textbf{非保存形（non-conservative form）}であることにある．

\paragraph{連続解析での保存性と離散での非保証}

非圧縮条件 $\bnabla\cdot\bu = 0$ が厳密に成立するとき，
非保存形と保存形は恒等式
\[
  \bu\cdot\bnabla\phi
  = \bnabla\cdot(\phi\bu) - \phi\underbrace{\bnabla\cdot\bu}_{=\,0}
  = \bnabla\cdot(\phi\bu)
\]
によって解析的に等価である．したがって連続レベルでは，
液相体積 $V_l = \int_\Omega H(-\phi)\,dV$ の時間微分は

\[
  \frac{dV_l}{dt}
  = -\int_\Omega H'(-\phi)\pd{\phi}{t}\,dV
  = \int_\Omega \delta_\varepsilon(\phi)\,\bu\cdot\bnabla\phi\,dV
  \overset{|\bnabla\phi|=1}{=}
  \int_\Gamma u_n\,dS
  = \int_{\Omega_l} \bnabla\cdot\bu\,dV = 0
\]

となり，非圧縮流では体積が保存される．
第3の等号は Eikonal 条件 $|\bnabla\phi|=1$ のもとで $\delta_\varepsilon(\phi)|\bnabla\phi|^{-1} = \delta_s$（界面面積測度）を用いる coarea 公式による．
第4の等号は Gauss の定理を液相領域 $\Omega_l$ に適用し，$\bnabla\cdot\bu=0$ を使う．

\paragraph{離散化による等価性の崩れ}

数値計算では，$\bu\cdot\bnabla\phi$ と $\bnabla\cdot(\phi\bu)$ の離散近似は一般に一致しない：
\begin{equation}
  \bigl[\bu\cdot\bnabla\phi\bigr]_h
  \;\neq\;
  \bigl[\bnabla\cdot(\phi\bu)\bigr]_h
  \label{eq:discrete_noncons}
\end{equation}
その差は $\phi_h\,\bigl[\bnabla\cdot\bu\bigr]_h$ に相当し，
Projection 法が数値的に残す微小な離散発散 $[\bnabla\cdot\bu]_h \sim \mathcal{O}(\Delta t)$ と，
$\bu\cdot\bnabla$ 演算子の打ち切り誤差が組み合わさって，
$\int\psi\,dV$ を時間ステップごとにわずかにドリフトさせる．
このドリフトが時間積分とともに蓄積し，\textbf{体積の単調損失（または増加）}として現れる．

\begin{tcolorbox}[mybox, title={2つの問題の独立性：実装者への指針}]
\begin{center}
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lll}
\hline
問題 & 原因 & 影響 \\
\hline
Eikonal 条件の崩れ & 移流誤差による $|\bnabla\phi|\neq 1$ & 曲率誤差 → 寄生流れ \\
質量損失 & 非保存形の離散化誤差の蓄積 & 液相体積の時間ドリフト \\
\hline
\end{tabular}
\end{center}
\medskip
\textbf{実装上の含意：}
\begin{itemize}
  \item 再初期化で Eikonal 条件を完全に維持しても，非保存形を使う限り体積ドリフトは残る．
  \item CLS 法の保存形 $\partial_t\psi + \bnabla\cdot(\psi\bu) = 0$ は，
        壁面・周期境界条件のもとで Gauss の定理により $\int\psi\,dV$ の保存を，
        連続レベルでは速度場の発散値に依らず保証する（Eikonal 条件とは独立）．
        離散レベルの保存性は，フラックス発散・境界処理・clamp/再初期化補正の整合に依存する．
\end{itemize}
\end{tcolorbox}

\subsubsection{標準 Level Set 法の「Sussman 再初期化」とその問題}

この $|\bnabla\phi|\neq 1$ 問題を解決するため，標準 LS 法では
「Sussman 再初期化」~\cite{Sussman1994} と呼ばれる操作を各タイムステップ後に行う：
\begin{equation}
  \pd{\phi}{\tau} + \operatorname{sgn}(\phi_0)\bigl(|\bnabla\phi| - 1\bigr) = 0
  \label{eq:sussman_reinit}
\end{equation}
この方程式を仮想時間 $\tau$ に関して解き，$|\bnabla\phi|=1$ を回復させる．
しかし，この操作には2つの残存問題がある：
\begin{itemize}
  \item \textbf{界面位置の微小移動：} 再初期化の過程で $\phi=0$ の等値面がわずかに移動し，
        体積変化の\textbf{追加的な誤差源}となる（§~\ref{sec:why_cls_massloss} の離散化誤差とは別）．
  \item \textbf{Eikonal 条件の不完全な回復：} 強い変形を受けた界面近傍では，
        $|\bnabla\phi|=1$ が完全には回復しないことがある．
\end{itemize}

\subsubsection{CLS 法の解決策：保存形への変換}

Conservative Level Set (CLS) 法~\cite{OlssonKreiss2005} は，
$\phi$ の代わりに，滑らかな\textbf{ヘヴィサイド関数} $\psi = H_\varepsilon(\phi) \in [0,1]$
を輸送変数として採用することでこの問題を解決する．

$\psi$ の移流方程式は\textbf{保存形（発散形）}で記述される（式~\eqref{eq:cls_advection}，§~\ref{sec:cls_advection} で詳述）．
これにより連続レベルでは $\int\psi\,dV =$ const が成り立ち，
固定領域では液相体積 $\int(1-\psi)\,dV$ の保存と同値になる．
離散実装では保存形フラックスを直接評価することで，標準 LS の非保存形離散化に由来する体積ドリフトを大きく抑制する．
