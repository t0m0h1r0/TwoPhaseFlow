\section{\texorpdfstring{$\psi \to \phi$}{ψ→φ} 逆変換におけるニュートン法の局所二次収束}
\label{app:newton_quad_conv}
% ═══════════════════════════════════════════════════════════════════════════════

本稿の §\ref{sec:newton_inversion} で述べたように，
$\psi \to \phi$ の逆変換はロジット関数 $\phi = \varepsilon\ln(\psi/(1-\psi))$ により
ほとんどの格子点で解析的に求まる（残差ゼロ，ニュートン法不要）．
ニュートン法（式~\eqref{eq:newton_inversion}）は実装の主経路ではなく，
ロジット逆変換の診断・補助としてのみ用いる．
以下の証明は，ニュートン法を使う場合に成立する標準的な局所収束条件を明示するために掲載する．

\textbf{本証明のスコープ：} 以下の二次収束証明は，
真の解 $\phi^*$ の近傍に初期値があり，かつ
$H_\varepsilon'(\phi)$ が小さすぎない局所区間内で反復が進む場合に成立する．
実装上の飽和処理（clamp）は $|\phi| \le \phi_{\max}$ に制限して
オーバーフローを防ぐための防御策であり，
それだけで大域収束や二次収束を保証するものではない．
大域的な収束解析は Kelley \cite{Kelley2003} 等のニュートン法標準テキストを参照のこと．

以下では，上記の局所区間においてニュートン法が二次収束することを示す．

\subsection*{前提}

$F(\phi) \equiv H_\varepsilon(\phi) - \psi_i = 0$ を根とする問題を考える．
$\phi^*$ を真の解（$F(\phi^*)=0$），$e_k = \phi^{(k)} - \phi^*$ を第 $k$ 反復での誤差とする．

\subsection*{二次収束の証明}

テイラー展開より：
\[
  F(\phi^{(k)}) = F'(\phi^*) e_k + \frac{F''(\xi_k)}{2} e_k^2
  \quad (\xi_k \in [\phi^*, \phi^{(k)}])
\]
ニュートン更新 $\phi^{(k+1)} = \phi^{(k)} - F(\phi^{(k)})/F'(\phi^{(k)})$ に対して
誤差方程式を導くと（$F'(\phi^{(k)}) = F'(\phi^*) + \Ord{e_k}$ を用いて
$\Ord{e_k^2}$ の寄与を吸収した主要項として）：
\[
  e_{k+1} = -\frac{F''(\xi_k)}{2F'(\phi^{(k)})} e_k^2 + \Ord{e_k^3}
\]

各因子を評価する．
\begin{itemize}
  \item $F'(\phi) = H_\varepsilon'(\phi) = \dfrac{1}{\varepsilon}H_\varepsilon(\phi)\bigl(1-H_\varepsilon(\phi)\bigr) > 0$
        （$H_\varepsilon \in (0,1)$ であるから常に正）．

  \item 局所区間 $I$ 上で $H_\varepsilon'(\phi)$ が小さすぎないとき，
        $F'$ は有限の正の下界 $F'_{\min} > 0$ を持つ：
        \[
          F'_{\min} = \inf_{\phi\in I} \frac{1}{\varepsilon}H_\varepsilon(\phi)\bigl(1-H_\varepsilon(\phi)\bigr) > 0
        \]

  \item $F''(\phi) = \dfrac{1}{\varepsilon^2}H_\varepsilon(1-H_\varepsilon)(1-2H_\varepsilon)$ は
        局所区間 $I$ 上で有界である（$H_\varepsilon \in [0,1]$ の連続関数）．
        $H_\varepsilon(1-H_\varepsilon) \le 1/4$ かつ $|1-2H_\varepsilon| \le 1$ であるから，
        $\|F''\|_\infty \le 1/(4\varepsilon^2)$ が成立する．
\end{itemize}

以上より，収束定数
\[
  C = \frac{\|F''\|_\infty}{2 F'_{\min}}
    < \infty
\]
は有限であり，
\[
  |e_{k+1}| \le C\,|e_k|^2
\]
が成立する．これはニュートン法の\textbf{二次収束}を示す．\qed

\subsection*{実装上の含意}

飽和領域（$\psi \to 0,1$）では $H_\varepsilon'$ が小さくなり，
Newton 更新は丸め誤差に敏感になる．
したがって本稿の実装ではロジット逆変換を主手法とし，
$\psi$ を $[\delta_{\text{clamp}},1-\delta_{\text{clamp}}]$ に制限して
$|\phi|\le\phi_{\max}$ に飽和させる．
界面から遠い飽和領域は曲率計算にも寄与しないため，Newton 反復に依存しない．
