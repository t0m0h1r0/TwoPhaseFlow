\subsection{\texorpdfstring{{\FCCD}：面ジェット共通プリミティブ}{FCCD：面ジェット共通プリミティブ}}
\label{sec:fccd_def}
% ═══════════════════════════════════════════════════════════════════════════════

本節では\textbf{面中心 Combined Compact Difference}（{\FCCD}）演算子族を，
\textbf{4 設計原理} → \textbf{4 次 cancellation 導出} → \textbf{行列定式化}
→ \textbf{周期 DFT 解析}の順に提示して定義する．
{\FCCD} は節点値から
\textbf{面値・面導関数} $\mathcal{J}_f(u)=(u_f, u'_f, u''_f)$ を
$\Ord{H^4}$ で構成する基底 CCD の派生スキームであり，
本節の数値特性は 4 設計原理と DFT 正準解析で完結する．

\medskip\noindent\textbf{なぜ面中心化が必要か．}
基底 CCD（§~\ref{sec:ccd_def}）が節点 $i$ で $(f_i, f'_i, f''_i)$ を
同時連立するのに対し，
{\FCCD} は面 $f=i-1/2$ で同等の 3 成分量を構成する．
これにより，後続章で要求される
「面位置に値を持つ離散演算子族」（移流フラックス・面平均勾配など）が
基底 CCD と同じ Hermite 補間に基づく $\Ord{H^4}$ 整合性を保ったまま
構築できる．バルクの節点 CCD は $\Ord{h^6}$ を維持するため，
{\FCCD} は bulk 精度の縮退ではなく\textbf{面ジェット供給の設計選択}である．
出力形式の API 定義は §~\ref{sec:face_jet_def} に集約する．

\noindent\textbf{歴史的経緯の保全．}
{\FCCD} 初版の動機（FVM 面平均勾配とのメトリック整合，H-01 診断）と
Exp-1/Exp-2 実測値は付録~\ref{sec:fccd_motivation_h01}
（§B.4 H-01 診断の歴史的ケーススタディ）に保全されている．

% -------------------------------------------------------------------------------
\subsubsection{4 つの設計原理}
\label{sec:fccd_design_principles}
% -------------------------------------------------------------------------------

{\FCCD} は以下の 4 原理に従う：

\begin{enumerate}[label=\textbf{P-F\arabic*.}]
\item \textbf{風上限定．}
      $a>0$ において $\{u_{i-1}, u_i\}$ のみ使用，
      中心・下流成分に依存しない．
\item \textbf{ステンシル非拡大．}
      $u_{i-2}$ 以遠は参照せず，3 点コンパクト性を CCD から継承．
\item \textbf{Combined Compact 構造．}
      高階補助未知数（$u'''_f$）を導入し，
      Chu--Fan 同様の\textbf{微分関係式の組合せ}で精度を取得．
\item \textbf{面中心化．}
      一次量は全て面 $f = i-1/2$ で定義し，後続章で消費される
      面位置の他の離散演算子と\textbf{同一面ロカス}を共有する．
\end{enumerate}

% -------------------------------------------------------------------------------
\subsubsection{4 次 Cancellation 導出}
\label{sec:fccd_derivation}
% -------------------------------------------------------------------------------

1 次差分の面上展開：
\begin{equation}
  \frac{u_i - u_{i-1}}{\Delta x}
  \;=\; u'_f + \frac{\Delta x^2}{24}\,u'''_f + \Ord{\Delta x^4}.
  \label{eq:fccd_upwind_expansion}
\end{equation}
Chu--Fan 流の組合せ演算子を定義：
\begin{equation}
  D^{\FCCD} u_f
  \;=\; \frac{u_i - u_{i-1}}{\Delta x}
  \;-\; \lambda\,\Delta x^2\,u'''_f.
  \label{eq:fccd_operator}
\end{equation}
式~\eqref{eq:fccd_upwind_expansion} を代入して \textbf{cancellation 条件}
$\lambda = 1/24$ を課せば：
\begin{equation}
  \boxed{\;
  D^{\FCCD} u_f \;=\; u'_f + \Ord{\Delta x^4}
  \;}
  \label{eq:fccd_fourth_order}
\end{equation}
となり，ステンシル拡大・風上構造の放棄なしに\textbf{4 次精度}を獲得する．
$u'''_f$ は近傍節点の $q_i = u''_i$（CCD 出力）から
$u'''_f \approx (q_i - q_{i-1})/\Delta x$ の 2 次中心差で閉じる．

% -------------------------------------------------------------------------------
\subsubsection{行列定式化}
\label{sec:fccd_matrix}
% -------------------------------------------------------------------------------

行列表記を用いると，面勾配ベクトル
$\mathbf{d}^{\FCCD} \in \mathbb{R}^N$ は節点値 $\mathbf{u}\in\mathbb{R}^{N+1}$ から：
\begin{equation}
  \mathbf{d}^{\FCCD}
  \;=\; \mathbf{D}_1\,\mathbf{u} - \mathbf{D}_2\,\mathbf{q},
  \qquad \mathbf{q} = \mathbf{S}_{\CCD}\,\mathbf{u},
  \label{eq:fccd_matrix_system}
\end{equation}
で得られる．ここで
$\mathbf{D}_1, \mathbf{D}_2 \in \mathbb{R}^{N\times(N+1)}$ は面↔節点
\textbf{bidiagonal}（行あたり 2 非零）：
\begin{align}
  (\mathbf{D}_1)_{f_{i-1/2},\,i-1} &= -1/H, &\quad (\mathbf{D}_1)_{f_{i-1/2},\,i} &= +1/H, \label{eq:fccd_D1}\\
  (\mathbf{D}_2)_{f_{i-1/2},\,i-1} &= -H/24, &\quad (\mathbf{D}_2)_{f_{i-1/2},\,i} &= +H/24. \label{eq:fccd_D2}
\end{align}
$\mathbf{D}_2$ の符号は，
$u_f'''\approx(q_i-q_{i-1})/H$ の中心差分から
cancellation 項 $-(H^2/24)u_f'''=-(H/24)(q_i-q_{i-1})$
として導かれる（§\ref{sec:fccd_def}, eq:~\eqref{eq:fccd_operator}）．
$\mathbf{S}_{\CCD} = \Pi_q\,\mathbf{A}_{\CCD}^{-1}\mathbf{B}_{\CCD}$ は
Chu--Fan 結合システムを節点 2 次導関数へ射影した作用素で，
\textbf{形式上は密行列}だが実行時は block-Thomas（壁）または
block-circulant LU（周期）で $\Ord{N}$ 計算．
従って合成演算子：
\begin{equation}
  \boxed{\;
  \mathbf{M}^{\FCCD}
  \;=\; \mathbf{D}_1 - \mathbf{D}_2\,\mathbf{S}_{\CCD}
  \;\in\; \mathbb{R}^{N\times(N+1)}
  \;}
  \label{eq:fccd_composite_matrix}
\end{equation}
の適用は 2 段階：
(i) $\mathbf{q} \leftarrow \mathbf{S}_{\CCD}\mathbf{u}$（既存 CCD ソルバ再利用），
(ii) $\mathbf{d}^{\FCCD} \leftarrow \mathbf{D}_1\mathbf{u} - \mathbf{D}_2\mathbf{q}$（面上局所ステンシル）．
全体コスト $\Ord{N}$，追加メモリは $\mathbf{q}$ のみ．

% -------------------------------------------------------------------------------
\subsubsection{周期境界での DFT 解析}
\label{sec:fccd_periodic_dft}
% -------------------------------------------------------------------------------

周期格子で $\mathbf{M}^{\FCCD}$ は\textbf{ブロック循環行列}（block-circulant）となり，
離散 Fourier 変換で対角化される．
波数 $\xi = k H$ に対する Fourier シンボル：
\begin{equation}
  \hat M^{\FCCD}(\xi)
  \;=\; \frac{2i\sin(\xi/2)}{H}
        \;\cdot\;
        \left[1 + \frac{\omega_2(\xi)^2}{24}\right]
  \label{eq:fccd_fourier_symbol}
\end{equation}
ここで前段因子 $2i\sin(\xi/2)/H$ は bidiagonal node-to-face 差分
$(D_1 u)_{f_{i-1/2}} = (u_i-u_{i-1})/H$（式~\eqref{eq:fccd_D1}）の
面位置における Fourier シンボル（純虚数で散逸ゼロ；
節点間中央差分 $i\sin(\xi)/H$ とは別演算子）であり，
$\omega_2(\xi)$ は CCD 2 階微分の無次元有効波数
$\hat q_i = -\omega_2(\xi)^2\hat u_i/H^2$（§\ref{sec:ccd_def}）である．
括弧内因子は $\hat D_2 = (iH/12)\sin(\xi/2)$（$(\pm H/24)$ bidiagonal ステンシルの面シンボル）と
$\hat S_{\CCD} = -\omega_2^2/H^2$（CCD ソース）から
$\hat M^{\FCCD} = \hat D_1 - \hat D_2\hat S_{\CCD}$ を整理して
$(2i\sin(\xi/2)/H)[1 + \omega_2^2/24]$ に到達する．
Taylor 展開 $2\sin(\xi/2) = \xi - \xi^3/24 + \Ord{\xi^5}$ および
$\omega_2(\xi)^2 = \xi^2 + \Ord{\xi^8}$（式~\eqref{eq:ccd_omega_12}；
均一格子上で $H \equiv h$，$\xi \equiv \theta = kh$ と読み替える）を代入すると，
括弧内 $1+\xi^2/24$ が前段因子の $-\xi^3/24$ 切断項を相殺し
$\hat M^{\FCCD}(\xi) = i\xi/H + \Ord{\xi^5/H}$ を得る．純虚数性（散逸ゼロ）が
保たれたまま式~\eqref{eq:fccd_fourth_order} の 4 次精度が Fourier レベルで確認される．

\subsubsection{演算子整合性：同一面ロカスでの $\Ord{\Delta x^4}$ 性質}
\label{sec:fccd_h01_remedy}
% -------------------------------------------------------------------------------

{\FCCD} を異なる入力場（例として 2 つの場 $p, \psi$）に適用すると：
\begin{equation}
  D^{\FCCD}\,p,
  \qquad
  D^{\FCCD}\,\psi,
  \label{eq:fccd_bf_unification}
\end{equation}
両者は\textbf{同一面ロカス・同一メトリック・同一 Fourier シンボル}となり，
これらの離散勾配の整合誤差は：
\begin{equation}
  \bigl\|\,D^{\FCCD}\,p - g(\psi)\cdot D^{\FCCD}\,\psi\,\bigr\|_\infty
  \;=\; \Ord{\Delta x^4}
  \qquad\bigl(g(\psi):\text{平滑関数}\bigr)
  \label{eq:fccd_bf_residual_order}
\end{equation}
まで圧縮される．
これは {\FCCD} 出力が 4 つの異なる消費点で共有されたとしても
$\Ord{\Delta x^4}$ メトリック整合が崩れない
\textbf{$\lambda=1/24$ cancellation の構造的帰結}である．
具体的な物理量適用（圧力勾配・表面張力・界面ジャンプ条件など）における
活用方法は当該用途章で展開される．
