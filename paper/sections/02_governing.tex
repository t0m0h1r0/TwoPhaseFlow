\section{支配方程式系と表面張力項}
\label{sec:governing}

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{主要変数の定義：\texorpdfstring{$\phi$}{phi}（符号付き距離関数）と \texorpdfstring{$\psi$}{psi}（保存形 Level Set 変数）}
\label{sec:notation}
% ═══════════════════════════════════════════════════════════════════════════════

本稿では，界面を表現するために二つの関連する変数，
符号付き距離関数 $\phi$ と保存形レベルセット変数 $\psi$ を用いる．
両者の役割を明確に区別することが，本章以降のアルゴリズムを理解する上で極めて重要である．
以降の全ての方程式で登場するこれらの変数を，本節で事前に定義しておく．

\begin{tcolorbox}[defbox, title={変数 $\phi$ と $\psi$ の役割分担（本稿全章共通）}]
\begin{itemize}
  \item \textbf{符号付き距離関数 $\phi(\bm{x}, t)$}：
        界面からの最短距離に符号を付けた関数（$\phi=0$ が界面，$\phi<0$ が液相内部）．
        その性質 $|\bnabla\phi|=1$（Eikonal 条件）は，
        法線ベクトル $\hat{\bm{n}} = \bnabla\phi$ や
        曲率 $\kappa = -\bnabla\cdot\hat{\bm{n}}$ といった
        \textbf{幾何学的情報}の計算精度を保証するために不可欠である．
        本稿では，曲率計算（§\ref{sec:curvature}）や
        再初期化方程式における法線ベクトルの算出に $\phi$ を用いる．

  \item \textbf{保存形レベルセット変数 $\psi(\bm{x}, t)$}：
        $\phi$ を滑らかなヘヴィサイド関数 $H_\varepsilon$ で変換したもので，
        液相で $\psi \approx 0$，気相で $\psi \approx 1$ を取る
        （第\ref{sec:levelset}章 §\ref{sec:cls_advection} 参照）．
        その移流方程式は保存形で書けるため，\textbf{質量（体積）保存性}を向上させる．
        密度 $\rho(\psi)$ や粘性 $\mu(\psi)$ などの
        \textbf{物理的物性値}の補間にも直接用いられる．
\end{itemize}

\medskip
\textbf{両変数の関係：}
両者は
\[
  \psi = H_\varepsilon(\phi)
  \qquad \bigl(H_\varepsilon \text{：滑らかなヘヴィサイド関数，第\ref{sec:levelset}章で詳述}\bigr)
\]
の関係で結ばれる．逆に $\psi$ から $\phi$ を得るには，
ロジット関数 $\phi = \varepsilon\ln\bigl(\psi/(1-\psi)\bigr)$ により
解析的に求まる（第\ref{sec:levelset}章参照）．
\end{tcolorbox}

\begin{tcolorbox}[warnbox, title={符号規約の統一：$\kappa$ の符号と Young-Laplace 則（本稿全章共通）}]
\textbf{本稿の符号規約（全章共通）：}
\begin{itemize}
  \item $\phi < 0$：液相内部，$\phi > 0$：気相，
        $\hat{\bm{n}} = \nabla\phi/|\nabla\phi|$ は\textbf{液相から気相への外向き}法線．
  \item $\kappa = -\nabla\cdot\hat{\bm{n}}$：
        球状液滴（内部が液相）では $\kappa = -1/R < 0$．
\end{itemize}

\textbf{Young-Laplace 則の確認：}
$[p] = p_\text{gas} - p_\text{liq} = \sigma\kappa = \sigma\cdot(-1/R) = -\sigma/R < 0$
より，$p_\text{liq} > p_\text{gas}$（液滴内部の圧力が外部より高い）は物理的に正しい．
$|[p]| = \sigma/R$ を「ラプラス圧力の大きさ」と呼ぶが，符号付き定義
$[p] = \sigma\kappa$ を正式表記として用いる．
\end{tcolorbox}

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{Two-Fluid から One-Fluid へ：統一支配方程式の導出}
\label{sec:two_to_one}
% ═══════════════════════════════════════════════════════════════════════════════

「なぜ2種類の流体を1つの方程式で記述できるのか？」
これは One-Fluid 定式化を初めて学ぶ際に誰もが抱く正当な疑問である．
本節では，まず各相の方程式を別々に書き，界面条件を経て統一形式に至る
導出のロードマップを示す．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{各相の支配方程式（Two-Fluid 形式）}
% ─────────────────────────────────────────────────────────────────────────────

液相（添字 $l$）と気相（添字 $g$）はそれぞれ，各相の占有する領域 $\Omega_l$，$\Omega_g$ で
別々の Navier-Stokes 方程式に従う：
\begin{align}
  \bnabla\cdot\bu &= 0 \quad \text{（各相内部，非圧縮条件）}
  \label{eq:cont_each} \\
  \rho_k\!\left(\pd{\bu}{t} + \bu\cdot\bnabla\bu\right)
  &= -\bnabla p + \bnabla\cdot[\mu_k(\bnabla\bu+\bnabla\bu^T)] + \rho_k\bm{g}
  \quad (k = l,\, g)
  \label{eq:NS_each}
\end{align}
これら2組の方程式は，界面 $\Gamma(t)$ 上で以下の\textbf{接合条件（jump conditions）}で結ばれる：
\begin{align}
  [\bu]_\Gamma &= 0
  \quad \text{（速度の連続性：スリップなし）}
  \label{eq:jump_vel} \\
  [-p\mathbf{I} + \mu(\bnabla\bu+\bnabla\bu^T)]\cdot\hat{\bm n}\big|_\Gamma &= \sigma\kappa\hat{\bm n}
  \quad \text{（応力ジャンプ = 表面張力）}
  \label{eq:jump_stress}
\end{align}
式 \eqref{eq:jump_stress} は\textbf{Young-Laplace 則}の力の形への表現であり，
「界面を挟む法線応力の不連続が表面張力 $\sigma\kappa$ に等しい」ことを意味する．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{Two-Fluid → One-Fluid への変換：$\delta$ 関数による統合}
% ─────────────────────────────────────────────────────────────────────────────

Two-Fluid 形式は各相で別々の方程式を解く必要があり，
界面の移動追跡（Front Tracking）を常に必要とするため，実装が非常に煩雑になる．

\textbf{One-Fluid 定式化}は，全領域 $\Omega = \Omega_l \cup \Omega_g \cup \Gamma$ で
\textbf{一つの方程式系}を成立させるアイデアである．
鍵となるのは，界面上の表面張力（面力）を「界面上にのみ集中する体積力」として
\textbf{界面デルタ関数 $\delta_s(\bm{x})$} を用いて表現することである：
\begin{equation}
  \bm{f}_\sigma = \sigma\kappa\hat{\bm n}\,\delta_s(\bm{x})
  \quad \text{（全空間で定義された体積力，界面以外でゼロ）}
  \label{eq:csf_intro}
\end{equation}
また，物性値 $\rho, \mu$ を界面指標関数 $\psi \in [0,1]$ で補間する
（$\psi \approx 0$ が液相，$\psi \approx 1$ が気相）：
\begin{equation}
  \rho(\psi) = \rho_l + (\rho_g - \rho_l)\psi, \qquad
  \mu(\psi) = \mu_l + (\mu_g - \mu_l)\psi
  \label{eq:interp_intro}
\end{equation}
これにより，各相の方程式 \eqref{eq:NS_each} と界面条件 \eqref{eq:jump_stress} が
\textbf{一本の方程式}に統合される：
\begin{equation}
  \rho(\psi)\!\left(\pd{\bu}{t} + \bu\cdot\bnabla\bu\right)
  = -\bnabla p + \bnabla\cdot[\mu(\psi)(\bnabla\bu+\bnabla\bu^T)]
  + \rho(\psi)\bm{g} + \bm{f}_\sigma
  \label{eq:NS_onefluid}
\end{equation}

Two-Fluid 形式から One-Fluid 形式への変換で，以下の操作が行われている：
\begin{enumerate}
  \item 各相の\textbf{別々の方程式}（$k=l,g$）が\textbf{1本の方程式} \eqref{eq:NS_onefluid} に統合された．
  \item 界面での\textbf{応力ジャンプ条件} \eqref{eq:jump_stress} が
        右辺の\textbf{CSF 体積力項} $\bm{f}_\sigma$（CSF モデル，§\ref{sec:csf}）として吸収された．
  \item 界面の\textbf{速度連続条件} \eqref{eq:jump_vel} は，
        全領域で同一の速度場 $\bu$ を解くことで自動的に満たされる．
\end{enumerate}
代償として，$\psi$ の時間発展方程式（第\ref{sec:levelset}章，CLS 法）を別途解く必要がある．

\medskip
\noindent\textbf{One-Fluid 化の数学的正当性（付録~\ref{app:onefluid_1d}）：}
上記の統合の数学的正当性は，1次元拡散方程式による簡約モデルで厳密に確認できる．
Two-Fluid 系（各相の方程式 ＋ 応力ジャンプ条件）と，デルタ関数的な界面力項を加えた
One-Fluid 方程式は，弱形式において\textbf{等価}であり，界面条件が CSF 体積力項に
「吸収」される仕組みを定量的に示す（詳細は付録~\ref{app:onefluid_1d} 参照）．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{One-Fluid 定式化と非圧縮条件}
\label{sec:onefluid}
% ═══════════════════════════════════════════════════════════════════════════════

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{One-Fluid モデルの仮定}
% ─────────────────────────────────────────────────────────────────────────────

本稿では，気相と液相を「1つの流体」として統一的に記述する
\textbf{One-Fluid（単一流体）定式化}を採用する．
このアプローチは以下の物理的仮定に基づいている．

本稿の One-Fluid 定式化は以下の仮定に基づく：
\begin{enumerate}
  \item \textbf{非混合（immiscible）：}
        気相と液相は混合せず，明確な界面が存在する．
  \item \textbf{界面厚さは数値的：}
        物理的な界面は厚さゼロだが，数値計算上はグリッド数セル分の厚みを持つ
        遷移層として表現する（CLS 変数 $\psi$ の役割）．
  \item \textbf{速度の連続性：}
        界面を挟んで速度は連続 $[\bm{u}]_\Gamma = 0$（スリップなし）．
  \item \textbf{圧力のジャンプ条件（Young-Laplace 則）：}
        界面での圧力は表面張力により不連続になる：
        \begin{equation}
          [p]_\Gamma \equiv p_\text{gas} - p_\text{liq} = \sigma\kappa
          \label{eq:young-laplace}
        \end{equation}
        ここで $\sigma$ は表面張力係数，$\kappa$ は界面の平均曲率
        （§\ref{sec:notation} の符号規約では液滴界面で $\kappa < 0$）．
\end{enumerate}

仮定3・4は連続体レベルの条件であり，One-Fluid 定式化ではこれらを
\textbf{暗黙的に}（界面での物性補間と CSF モデルによって）
自動的に満足する形で離散化される．

\paragraph{なぜ物性値を補間できるのか？}
仮定1（非混合）と仮定2（数値的界面）から，
界面近傍の各格子点には実際には気相または液相のどちらかが占有されているが，
有限の界面幅の中では「どの割合で液相か」を $\psi$ で表現する．
すなわち，$\psi\in[0,1]$ は「液相の体積分率」と解釈でき，
加重平均による物性補間は有限体積的に整合している．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{支配方程式（連続の式と運動量保存）}
% ─────────────────────────────────────────────────────────────────────────────

非圧縮条件は連続の式：
\begin{equation}
  \bnabla \cdot \bu = 0
  \label{eq:continuity}
\end{equation}
で与えられる．
物性値は界面指標 $\psi \in [0,1]$（$\psi \approx 0$ が液相，$\psi \approx 1$ が気相）により連続的に補間する：
\begin{align}
 \rho(\psi)&=\rho_l+(\rho_g-\rho_l)\,\psi,
 &
 \mu(\psi)&=\mu_l+(\mu_g-\mu_l)\,\psi.
\end{align}
ここで $\psi$ は Conservative Level Set 法における
ヘヴィサイド関数 $H_\varepsilon(\phi)$ に相当し（§\ref{sec:notation} 参照），
その時間発展は第\ref{sec:levelset}章の CLS 法で計算される．

\begin{tcolorbox}[warnbox, title={粘性係数 $\mu$ の算術平均と調和平均：選択根拠と定量比較}]
\textbf{$\rho^{-1}$ には調和平均，$\mu$ には算術平均——その理由を厳密に導く}

\medskip
\noindent\textbf{まず調和平均が正しい場合を理解する（密度・PPE）}

圧力 Poisson 方程式 $\bnabla\cdot(\rho^{-1}\bnabla p) = \cdots$ における係数 $\rho^{-1}$ の面補間は，
\textbf{鋭い不連続界面でのフラックス連続条件}から調和平均が厳密に導かれる．
1次元定常問題 $d/dx(\rho^{-1}\,dp/dx)=0$ でセル $L$（$\rho=\rho_l$）と
セル $R$（$\rho=\rho_g$）が隣接するとき，
フラックス $\rho^{-1}dp/dx$ の面連続条件を解くと：
\[
  (\rho^{-1})_f^{\mathrm{harm}}
  = \frac{2\rho_l^{-1}\rho_g^{-1}}{\rho_l^{-1}+\rho_g^{-1}}
  = \frac{2}{\rho_l+\rho_g}
\]
これは直列抵抗則に対応し，PPE の離散化において\textbf{厳密解}を与える．

\medskip
\noindent\textbf{「では粘性の法線方向フラックスも調和平均では？」という正当な疑問}

\textbf{答え：鋭い界面（VOF・フロント追跡法）では正しい．しかし CLS（拡散界面法）では算術平均が正しい．}

理由を以下に示す．

\medskip
\noindent\textbf{CLS における $\mu$ の FVM 面平均：算術平均が厳密}

CLS 法では $\mu(\psi) = \mu_l + (\mu_g - \mu_l)\psi$ を採用する（式 \eqref{eq:interp_intro}）．
これは $\psi$ の\textbf{線形関数}である．

有限体積法では，セル界面 $f$ での面粘性 $\mu_f$ として
「面を挟む半セル上での $\mu$ の体積平均」を採用するのが自然である：
\[
  \mu_f
  = \frac{1}{\Delta x}\int_{x_f - \Delta x/2}^{x_f + \Delta x/2} \mu(\psi(x))\,dx
\]
界面近傍で $\psi(x)$ が線形に変化する（$\psi$ が滑らかな CLS プロファイルを持つ）と仮定すると，
$\psi$ の区間平均は端点の算術平均 $(\psi_L + \psi_R)/2$ に等しく：
\begin{align}
  \mu_f
  &= \mu_l + (\mu_g - \mu_l)\cdot\frac{\psi_L + \psi_R}{2} \notag \\
  &= \frac{\bigl[\mu_l + (\mu_g-\mu_l)\psi_L\bigr]
           + \bigl[\mu_l + (\mu_g-\mu_l)\psi_R\bigr]}{2}
   = \frac{\mu(\psi_L) + \mu(\psi_R)}{2}
   = \mu_f^{\mathrm{arith}}
   \label{eq:mu_arith_derivation}
\end{align}
\textbf{結論：} $\mu(\psi)$ が $\psi$ の線形関数（= CLS の定義）であり，
$\psi$ が界面を連続的に変化する（= 拡散界面の仮定）ならば，
算術平均は FVM の体積平均として\textbf{厳密}である．

\medskip
\noindent\textbf{調和平均が「正しい」のは鋭い界面に限られる}

調和平均が厳密なのは，$\mu$ が界面でステップ状に不連続（$\mu = \mu_l$ または $\mu_g$，中間値なし）な
\textbf{鋭い界面仮定}のもとで，法線方向フラックスの連続条件を解いた場合に限られる．
CLS は界面を厚さ $2\varepsilon$ の滑らかな遷移層として表現するため，
この仮定は成立せず，調和平均は過小評価を与える．

\medskip
\noindent\textbf{定量確認：$\mu_l/\mu_g = 100$，界面中心 $\psi_L=0,\,\psi_R=1$}

\[
  \mu^{\mathrm{arith}}_f = \frac{\mu_l + \mu_g}{2} = \frac{101\mu_g}{2} = 50.5\,\mu_g
\]
\[
  \mu^{\mathrm{harm}}_f
  = \frac{2\mu_l\mu_g}{\mu_l+\mu_g}
  \approx 1.98\,\mu_g
\]
調和平均は算術平均の約 $1/25$ であり，液相の高粘性効果をほぼ消失させる（非物理的）．
式 \eqref{eq:mu_arith_derivation} に基づき，本稿では $\mu$ の補間に算術平均を採用する．

\medskip
\noindent\textbf{密度との選択の一貫性（確認）}

密度の面補間は PPE に現れる $\rho^{-1}$ の調和平均であり，
$\mu$ の算術平均と一見矛盾するように見える．
しかし両者の根拠は一貫している：

\begin{center}
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lll}
\hline
物性 & 採用する補間 & 根拠 \\
\hline
$\rho^{-1}$（PPE 演算子の係数）& 調和平均 & フラックス連続条件（拡散演算子の直列抵抗則） \\
$\mu$（粘性応力の係数）& 算術平均 & CLS 拡散界面での FVM 体積平均（式 \eqref{eq:mu_arith_derivation}） \\
\hline
\end{tabular}
\end{center}
\end{tcolorbox}
