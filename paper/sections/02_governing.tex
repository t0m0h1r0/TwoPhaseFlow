\section{支配方程式系：One-Fluid 定式化と表面張力モデル}
\label{sec:governing}

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{主要変数の定義：\texorpdfstring{$\phi$}{phi}（符号付き距離関数）と \texorpdfstring{$\psi$}{psi}（保存形 Level Set 変数）}
\label{sec:notation}
% ═══════════════════════════════════════════════════════════════════════════════

本稿では，界面を表現するために二つの関連する変数，
符号付き距離関数 $\phi$ と保存形 Level Set 変数 $\psi$ を用いる．
両者の役割を明確に区別することが，本章以降のアルゴリズムを理解する上で極めて重要である．
以降の方程式で用いるこれらの変数を本節で定義する．

\textbf{符号付き距離関数 $\phi(\bm{x}, t)$} は界面からの最短距離に符号を付けた関数である：
\begin{equation}
  \phi < 0\text{：液相内部},\quad \phi = 0\text{：界面},\quad \phi > 0\text{：気相}
  \label{eq:phi_def}
\end{equation}
Eikonal 条件 $|\bnabla\phi|=1$ が成り立ち，法線ベクトルと曲率を以下で定義する（§~\ref{sec:curvature}）：
\begin{equation}
  \hat{\bm{n}} = \frac{\bnabla\phi}{|\bnabla\phi|} = \bnabla\phi, \qquad
  \kappa = -\bnabla\cdot\hat{\bm{n}}
  \label{eq:kappa_def}
\end{equation}

\textbf{保存形 Level Set 変数 $\psi(\bm{x},t)$} は $\phi$ の平滑化ヘヴィサイド変換
$\psi = H_\varepsilon(\phi) \in [0,1]$ として定義される界面指標である
（$\psi \approx 0$：液相，$\psi \approx 1$：気相）．
$\psi$ の定義・性質・移流方程式は第~\ref{sec:levelset}章で詳述する．
密度・粘性の補間（式~\eqref{eq:interp_intro}）および
曲率の直接計算（§~\ref{sec:curvature_direct_psi}）に用いる．

\begin{tcolorbox}[defbox, title={Eikonal 制約のトポロジー硬直性}]
$\phi=0$ を界面とする境界条件と組み合わせた符号付き距離関数 $\phi$ の定義は，
Eikonal 条件 $|\bnabla\phi|=1$ と対応しており，
これは零等位集合の\textbf{平行オフセット性}を強制する．
幾何的頑健性と一意性の代償として，Eikonal 制約は
極値とリッジを抑制し，零等位集合のトポロジーは
滑らかな時間発展の下で凍結される．
FMM/FSM/Godunov Hamilton--Jacobi の各再初期化は，
幾何学的に見ればこのトポロジー硬直性を反復強制する手続きである．
§~\ref{sec:levelset} で導入する Ridge--Eikonal ハイブリッドは，
この硬直性を緩め界面の融合・分裂に必要な自由度を
Gaussian 補助場（§~\ref{sec:ridge_eikonal} で定義）により回復しつつ，
Eikonal 由来の距離精度を保つ設計である．
\end{tcolorbox}

\begin{tcolorbox}[defbox, title={符号規約の統一：$\kappa$ の符号と Young-Laplace 則（本稿全章共通）}]
\textbf{符号規約の定義（全章共通）：}
\begin{itemize}
  \item $\phi < 0$：液相内部，$\phi > 0$：気相（式~\eqref{eq:phi_def}，§~\ref{sec:notation}）．
  \item $\hat{\bm{n}} = \bnabla\phi/|\bnabla\phi|$：
        液相内部では $\phi < 0$ → $\phi$ が増加する方向（液相→気相）を向くため，
        $\hat{\bm{n}}$ は\textbf{液相から気相への外向き}法線である．
  \item $\kappa = -\bnabla\cdot\hat{\bm{n}}$（曲率の定義，式~\eqref{eq:kappa_def}）．
\end{itemize}

\medskip
\textbf{円形液滴（2次元，半径 $R$，中心を原点）での具体的導出：}

$\phi(\bm{x}) = |\bm{x}| - R$（液相は $|\bm{x}| < R$ で $\phi < 0$，界面は $|\bm{x}|=R$ で $\phi=0$）とすると，

\[
  \bnabla\phi = \frac{\bm{x}}{|\bm{x}|} \equiv \hat{\bm{r}}, \qquad
  |\bnabla\phi| = 1, \qquad
  \hat{\bm{n}} = \hat{\bm{r}} \quad \text{（球面外向き = 液相→気相 $\checkmark$）}
\]

2次元円（$\bnabla\cdot\hat{\bm{r}} = 1/r$）では $r = R$ で：
\[
  \kappa = -\bnabla\cdot\hat{\bm{n}} = -\frac{1}{R} < 0
\]

\medskip
\textbf{Young-Laplace 則との整合確認：}
\[
  [p] \equiv p_\text{gas} - p_\text{liq}
     = \sigma\kappa = \sigma\!\cdot\!\left(-\frac{1}{R}\right) = -\frac{\sigma}{R} < 0
\]
$[p] < 0$ すなわち $p_\text{liq} > p_\text{gas}$
（液滴内部の圧力が外部より高い）は Laplace 圧力の物理と整合 $\checkmark$

$|[p]| = \sigma/R$ を「ラプラス圧力の大きさ」と呼ぶが，符号付き定義
$[p] = \sigma\kappa$ を正式表記として用いる
（本 tcolorbox の数式は有次元形；無次元化と *-付き表記は §~\ref{sec:nondim}，
無次元形は式~\eqref{eq:young_laplace_jump}）．
\end{tcolorbox}

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{Two-Fluid から One-Fluid へ：統一支配方程式の導出}
\label{sec:two_to_one}
% ═══════════════════════════════════════════════════════════════════════════════

One-Fluid 定式化の核心は，界面上の応力ジャンプ条件をデルタ関数的な体積力として吸収し，
全領域で一本の方程式系を成立させる点にある．
本節では，まず各相の方程式を別々に書き，界面条件を経て統一形式に至る
導出のロードマップを示す．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{各相の支配方程式（Two-Fluid 形式）}
% ─────────────────────────────────────────────────────────────────────────────

液相（添字 $l$）と気相（添字 $g$）はそれぞれ，各相の占有する領域 $\Omega_l$，$\Omega_g$ で
別々の Navier--Stokes 方程式に従う：
\begin{align}
  \bnabla\cdot\bu &= 0 \quad \text{（各相内部，非圧縮条件）} \\
  \rho_k\!\left(\pd{\bu}{t} + \bu\cdot\bnabla\bu\right)
  &= -\bnabla p + \bnabla\cdot[\mu_k(\bnabla\bu+\bnabla\bu^T)] + \rho_k\bm{g}
  \quad (k = l,\, g)
  \label{eq:NS_each}
\end{align}
これら2組の方程式は，界面 $\Gamma(t)$ 上で以下の\textbf{接合条件（jump conditions）}で結ばれる：
\begin{align}
  [\bu]_\Gamma &= 0
  \quad \text{（速度の連続性：スリップなし）}
  \label{eq:jump_vel} \\
  [-p\mathbf{I} + \mu_k(\bnabla\bu+\bnabla\bu^T)]_\Gamma\cdot\hat{\bm n} &= -\sigma\kappa\hat{\bm n}
  \quad \text{（応力ジャンプ = 表面張力）}
  \label{eq:jump_stress}
\end{align}
式~\eqref{eq:jump_stress} は\textbf{Young-Laplace 則}の力の形への表現であり，
「界面を挟む法線応力の不連続が，本稿の外向き法線・曲率規約では
$-\sigma\kappa\hat{\bm n}$ に等しい」ことを意味する．
ここでジャンプ $[\cdot]_\Gamma$ は，速度連続条件~\eqref{eq:jump_vel} と同様に
界面の両側での値の差 $(\cdot)_g - (\cdot)_l$ を表す．
すなわち，応力テンソル $\bm{\tau} \equiv -p\mathbf{I} + \mu(\bnabla\bu+\bnabla\bu^T)$ は，
各相でそれぞれの粘性 $\mu_l$, $\mu_g$ を用いて評価される．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{Two-Fluid → One-Fluid への変換：\texorpdfstring{$\delta$}{δ} 関数による統合}
% ─────────────────────────────────────────────────────────────────────────────

Two-Fluid 形式は各相に個別の方程式を解き，界面追跡（Front Tracking~\cite{Unverdi1992}）を要するため実装が煩雑である．

\textbf{One-Fluid 定式化}は，全領域 $\Omega = \Omega_l \cup \Omega_g \cup \Gamma$ で
一つの方程式系を成立させるアイデアである．
鍵となるのは，界面上の表面張力（面力）を「界面上にのみ集中する体積力」として
\textbf{界面デルタ関数 $\delta_s(\bm{x})$} を用いて表現することである：
\begin{equation}
  \bm{f}_\sigma = \sigma\kappa\hat{\bm n}\,\delta_s(\bm{x})
  \quad \text{（全空間で定義された体積力，界面以外でゼロ）}
  \label{eq:csf_intro}
\end{equation}
また，物性値 $\rho, \mu$ を界面指標関数 $\psi \in [0,1]$ で補間する
（$\psi \approx 0$ が液相，$\psi \approx 1$ が気相）：
\begin{equation}
  \rho(\psi) = \rho_l + (\rho_g - \rho_l)\psi, \qquad
  \mu(\psi) = \mu_l + (\mu_g - \mu_l)\psi
  \label{eq:interp_intro}
\end{equation}
これにより，各相の方程式~\eqref{eq:NS_each} と界面条件~\eqref{eq:jump_stress} が
一本の方程式に統合される：
\begin{equation}
  \rho(\psi)\!\left(\pd{\bu}{t} + \bu\cdot\bnabla\bu\right)
  = -\bnabla p + \bnabla\cdot[\mu(\psi)(\bnabla\bu+\bnabla\bu^T)]
  + \rho(\psi)\bm{g} + \bm{f}_\sigma
  \label{eq:NS_onefluid}
\end{equation}

\noindent\textbf{浮力項の圧力ロバスト分解．}
重力ポテンシャル $\Phi_g$ を $\bm{g}=-\bnabla\Phi_g$ で定義し，
任意の基準密度 $\rho_0$ に対して $\rho'=\rho-\rho_0$ と書くと，
重力項は
\begin{equation}
  \rho\bm{g}
  = \rho_0\bm{g} + \rho'\bm{g}
  = -\bnabla(\rho_0\Phi_g+\rho'\Phi_g)
    + \Phi_g\bnabla\rho'
  \label{eq:buoyancy_gradient_residual_split}
\end{equation}
に分解できる．第1項は完全な勾配であり，非圧縮投影法では圧力に吸収されるべき
hydrostatic 成分である~\cite{Chorin1968,Guermond2006}．
一方，$\Phi_g\bnabla\rho'$ は密度が変化する界面帯に支持を持つ
非勾配残差であり，実際に速度を変える浮力である．
したがって水--空気級の浮力駆動計算では，$\rho\bm{g}$ 全体を Predictor の節点体積力として
入れるのではなく，勾配成分は圧力空間へ，残差成分は投影と同じ面演算子上へ分配する必要がある．
この分解は後の Balanced--Force 条件~\cite{Francois2006}
（§~\ref{sec:bf_seven_principles}）と
分相 PPE 閉包（§~\ref{sec:split_ppe_framework}）の連続式側の出発点である．

この変換には以下の操作が含まれる：
\begin{enumerate}
  \item 各相の方程式（$k=l,g$）が 1 本の方程式~\eqref{eq:NS_onefluid} に統合された．
  \item 界面での応力ジャンプ条件~\eqref{eq:jump_stress}
        $[\bm{\tau}\cdot\hat{\bm n}]_\Gamma = -\sigma\kappa\hat{\bm n}$ が，
        Heaviside 分布の微分則による体積積分化で
        $\bm{f}_\sigma = -[\bm{\tau}\cdot\hat{\bm n}]_\Gamma\,\delta_s = \sigma\kappa\hat{\bm n}\,\delta_s$
        として運動量方程式右辺に吸収された．
        弱形式等価性 $\int_\Omega\bm{f}_\sigma\!\cdot\!\bm{v}\,dV = \int_\Gamma\sigma\kappa\hat{\bm n}\!\cdot\!\bm{v}\,dA$
        が任意のテスト関数 $\bm{v}$ で成立することにより，面積分から体積積分への変換が厳密に正当化される
        （§~\ref{sec:csf}；詳細は付録~\ref{app:onefluid_1d}）．
  \item 界面の速度連続条件~\eqref{eq:jump_vel} は，
        全領域で同一の速度場 $\bu$ を解くことで自動的に満たされる．
\end{enumerate}
代償として，$\psi$ の時間発展方程式（第~\ref{sec:levelset}章，CLS 法）を別途解く必要がある．

\medskip
\noindent\textbf{One-Fluid 化の数学的正当性（付録~\ref{app:onefluid_1d}）：}
上記の統合の数学的正当性は，1次元拡散方程式による簡約モデルで厳密に確認できる．
Two-Fluid 系（各相の方程式 ＋ 応力ジャンプ条件）と，デルタ関数的な界面力項を加えた
One-Fluid 方程式は，弱形式において等価である．
この等価性により，界面条件が CSF 体積力項（§~\ref{sec:csf}）に「吸収」される仕組みが定量的に示される（詳細は付録~\ref{app:onefluid_1d} 参照）．

\subsection{One-Fluid 定式化と非圧縮条件}
\label{sec:onefluid}
% ═══════════════════════════════════════════════════════════════════════════════

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{支配方程式（連続の式と運動量保存）}
% ─────────────────────────────────────────────────────────────────────────────

§~\ref{sec:two_to_one} では Two-Fluid 系から One-Fluid 形式（式~\eqref{eq:NS_onefluid}）への導出ロードマップを示した．
本節では，その方程式系を数値求解の観点から改めて整理するとともに，
§~\ref{sec:two_to_one} では概要に留めた\textbf{点値の物性補間とフェイス係数構成}——
$\rho(\psi),\mu(\psi)$ の点値補間と，$\rho^{-1}$ 係数・$\mu$ 係数の離散化で使う平均の違い，
およびその物理的根拠——を詳述する．

非圧縮条件は以下の連続の式で与えられる：
\begin{equation}
  \bnabla \cdot \bu = 0
\end{equation}
物性値は界面指標 $\psi \in [0,1]$（$\psi \approx 0$ が液相，$\psi \approx 1$ が気相）により
式~\eqref{eq:interp_intro} で連続的に補間する．
ここで $\psi$ は Conservative Level Set 法における
ヘヴィサイド関数 $H_\varepsilon(\phi)$ に相当し（§~\ref{sec:notation} 参照），
その時間発展は第~\ref{sec:levelset}章の CLS 法で計算される．

\begin{tcolorbox}[mybox, title={点値補間とフェイス係数：$\rho^{-1}$ には調和平均，$\mu$ には算術平均}]
\begin{center}
\renewcommand{\arraystretch}{\PaperTableArrayStretch}
\begin{tabular}{lll}
\hline
対象 & 採用する平均 & 根拠 \\
\hline
$\rho^{-1}$（圧力方程式のフェイス係数）& 調和平均 $\dfrac{2}{\rho_L+\rho_R}$ & フラックス連続条件（直列抵抗則） \\[6pt]
$\mu$（粘性応力のフェイス係数）& 算術平均 $\dfrac{\mu_L+\mu_R}{2}$ & CLS 拡散界面での FVM 体積平均 \\
\hline
\end{tabular}
\end{center}
\smallskip
\noindent\textbf{記号の注意：}
上表の $\rho_L, \rho_R$（および $\mu_L, \mu_R$）はセル界面の\textbf{左 (Left)・右 (Right) の値}を表し，
相を示す添字 $l$（液相）・$g$（気相）とは異なる．
点値の $\rho(\psi),\mu(\psi)$ は式~\eqref{eq:interp_intro} に従い，
上表はその点値からセル界面の離散係数を構成する段階を示す．
\medskip
$\mu_l/\mu_g = 100$ の水/空気界面では，算術平均 $\approx 50.5\mu_g$，
調和平均 $\approx 1.98\mu_g$（$1/25$ 倍）．
調和平均は液相の高粘性効果をほぼ消失させるため非物理的である．
$\mu(\psi)$ が $\psi$ の線形関数である CLS では，算術平均が FVM 体積平均として厳密に導かれる
（証明：付録~\ref{app:mu_interp_derivation}）．
$\rho^{-1}$ の調和平均と $\mu$ の算術平均という選択の物理的根拠については
\cite{Prosperetti1981, Sussman2000, Kang2000} も参照．
\end{tcolorbox}
