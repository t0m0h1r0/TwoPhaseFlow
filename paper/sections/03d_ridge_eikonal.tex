\subsection{Ridge--Eikonal ハイブリッド再初期化}

\textbf{トポロジー不変性の下で}強制する枠組であり，

本節では\textbf{トポロジーの自由度}と\textbf{計量整合性}を
別々のスカラー場に担わせる二相スケジューリングを導入する．

\subsubsection{Gaussian 補助場 $\xiridge$ の定義（D0）}
\label{sec:ridge_aux_field}

各界面成分 $\Gamma_k$ との符号付き距離を $d_k(\bm{x},t)$ としたうえで，
\textbf{補助スカラー場} $\xiridge$ を以下で定義する：
\begin{equation}
  \xiridge(\bm{x},t) = \sum_k \exp\!\left(-\frac{d_k(\bm{x},t)^2}{\sigma^2}\right),
\end{equation}
ここで $\sigma$ は相互作用スケール．$\xiridge$ は\textbf{距離関数ではなく}
Eikonal 条件を満たさない；
これは局所極値・鞍点を許容するための意図的な設計であり，
Morse 指数~\cite{Milnor1963} の連続変化としてトポロジー変化を表現する道具となる．

時間発展は移流・拡散で与える：
\begin{equation}
  \partial_t \xiridge + \bm{u}\cdot\bnabla\xiridge
  = \varepsilon_\text{diff}\,\Delta \xiridge .
\end{equation}
拡散係数 $\varepsilon_\text{diff}$ は正則化（数値安定化）のためのものであり，
物理的拡散を表すわけではない．

\begin{remark}[既存場との区別]
$\xiridge$ は格子密度 Gaussian $\omegagrid(\phi) = \exp(-\alpha(\phi/\varepsilon_g)^2)$
（§~\ref{sec:grid_density}）とは\textbf{役割が直交}する：
$\omegagrid$ は格子間隔を制御し，$\xiridge$ は界面そのものを表す．
また，既存の \texttt{xi\_sdf} 再初期化モード
（\texttt{EikonalReinitializer.\_xi\_sdf\_phi}）が index 空間の
Euclidean 距離を計算して Eikonal を満たすのに対し，
$\xiridge$ は意図的に Eikonal を破る．
\end{remark}

\subsubsection{リッジ集合による界面定義}

界面 $\Gamma$ を $\xiridge$ の\textbf{リッジ集合}として定義する：
\begin{equation}
  \Gamma = \bigl\{\bm{x} \,\big|\, \bnabla\xiridge(\bm{x}) = 0,\;\;
  \lambda_{\min}\bigl(\bnabla^2\xiridge(\bm{x})\bigr) < 0 \bigr\},
  \label{eq:ridge_set_def}
\end{equation}
ここで $\lambda_{\min}(\cdot)$ は Hessian の最小固有値である．
リッジ上の法線方向は $\lambda_{\min}$ に対応する単位固有ベクトル $\hat{\bm{v}}_{\min}$ で与え，
\begin{equation}
  \hat{\bm{n}}(\bm{x}) =
  \begin{cases}
    \bnabla\xiridge(\bm{x})/|\bnabla\xiridge(\bm{x})|, & \bm{x} \notin \Gamma, \\
    \hat{\bm{v}}_{\min}\bigl(\bnabla^2\xiridge(\bm{x})\bigr), & \bm{x} \in \Gamma,
  \end{cases}
\end{equation}
と定める．$\Gamma$ の近傍で $\bnabla\xiridge\to 0$ のとき，Morse 函数の正準形により両定義は極限で一致する．
これは法線方向に極大・接線方向に連続というクレスト線（2次元）
／クレスト面（3次元）に対応する．
孤立界面ではリッジ集合は古典的界面と一致し，
複数界面の相互作用では\textbf{リッジ併合・分岐}が生じ，
これが融合・分裂に対応する．
したがってトポロジー変化は離散イベントではなく
\textbf{$\xiridge$ の連続変形}（Morse 指数変化の一瞬）として表される．

\subsubsection{トポロジー変化の連続化}

従来の level-set では，融合・分裂は $\phi$ の level set のトポロジーが
不連続に変化する特異時刻として現れる．
Ridge--Eikonal 枠組では，これらの事象はすべて補助場 $\xiridge$ の
連続的な変形として単一パラメータ族内で表現される．

\begin{proposition}[リッジ集合のトポロジー変化の連続性]
\label{prop:ridge_morse_continuity}
$\xiridge(\cdot, t)$ が各 $t$ で Morse 函数である区間では，
そのリッジ集合 $\Gamma(t)$ は以下の意味で連続変形する：
\begin{itemize}[leftmargin=1.5em, itemsep=0pt, topsep=2pt]
  \item 融合は 2 本のリッジが鞍点を通じて 1 本のリッジに滑らかに統合される．
  \item 分裂はリッジの分岐（bifurcation）として単一パラメータ族で連続的に現れる．
  \item いずれも $\xiridge$ の Morse 指数~\cite{Milnor1963} が瞬間的に変化する時刻として特徴づけられる．
\end{itemize}
\end{proposition}

\begin{proof}[証明スケッチ]
$\xiridge \in C^2$ かつ臨界点が非退化である限り，Morse 補題より
各臨界点の近傍は正準形に局所変形でき，指数変化は孤立点でのみ生じる．
リッジ集合は臨界点を滑らかにつなぐ勾配下降流の集合として定義されるため，
指数変化の瞬間を除いて連続的に変形する．詳細は~\cite{Milnor1963} \S6 を参照．
\end{proof}

\begin{remark}[退化臨界点に対する数値的安全策]
\label{rem:morse_degenerate_safeguard}
命題~\ref{prop:ridge_morse_continuity} の Morse 非退化仮定は解析上は generic だが，
離散 $\xiridge$ では Hessian 行列式 $\det \bnabla^2\xiridge$ が
格子点で厳密に $0$ となる測度零イベントが浮動小数演算で顕在化しうる．
命題~\ref{prop:ridge_morse_continuity} の仮定が離散計算で破れる可能性に対して
本実装では以下の二段階安全策を導入する：
\begin{enumerate}[leftmargin=1.5em, itemsep=0pt, topsep=2pt]
  \item \textbf{$\varepsilon$-摂動．}
        $|\det \bnabla^2\xiridge| < \varepsilon_\text{Morse}$（推奨
        $\varepsilon_\text{Morse} = 10^{-6} \max|\bnabla^2\xiridge|$）を検出したとき，
        補助場に $\delta \xi(\bm{x}) \sim \mathcal{N}(0, h^{d+2})$ の
        微小な gauss ホワイトノイズを付与して退化を解消する．
  \item \textbf{離散格子モニター．}
        隣接格子点間で $\lambda_{\min}$ の符号が反転する領域を
        「退化候補域」として flagging し，
        曲率評価では局所的に半径 $2h$ の球平均で Hessian を再構成して
        評価する（§~\ref{sec:verification} の検証ベンチマークで較正）．
\end{enumerate}
退化臨界点の完全解消は本稿の範囲外であり，
Smale のトポロジー摂動理論~\cite{Milnor1963}に基づく
理論的拡張は §~\ref{sec:future_work} で展望する．
\end{remark}

以上より離散イベント処理や明示的リメッシュは不要で，
時間離散化の刻み幅 $\Delta t$ が Morse 遷移の特徴時間スケール
$\tau_M \sim h/|\bm{u}|_{\max}$ に対して $\Delta t \ll \tau_M$ を満たすとき
数値的にも自動的に捕捉される．
ここで $\tau_M$ は「2 本のリッジが統合される際に鞍点を通過する時間スケール」として
定義され，鞍点近傍でのリッジ移動速度が流体速度 $|\bm{u}|$ と同オーダーで，
リッジ融合に必要な距離が局所格子幅 $h$ 程度であることから
$\tau_M \sim h/|\bm{u}|_{\max}$ が導かれる（§~\ref{sec:verification} のベンチマークで
実測し，典型的な 2 次元融合で $\tau_M \sim 3\text{--}5\,\Delta t_\text{CFL}$）．

\subsubsection{FMM/FSM による $\phi$ の再構成と一意性}
\label{sec:ridge_fmm_reconstruction}

\begin{theorem}[FMM/FSM による $\phi$ 一意再構成]
\label{thm:eikonal_reconstruction}
リッジ集合 $\Gamma$ が以下を満たすとき：
\begin{itemize}[leftmargin=1.5em, itemsep=0pt, topsep=2pt]
  \item\textbf{幾何正則性．}$\Gamma$ は閉じた自己交差しない $(d-1)$ 次元 Lipschitz 多様体．
  \item\textbf{射影一意性．}$\Gamma$ の近傍点からの最近接射影が一意である．
\end{itemize}
（数値検証の詳細は §~\ref{sec:ridge_admissibility}）．Eikonal 再構成問題
\begin{equation}
  |\bnabla\phi| = 1 \text{ in } \Omega \setminus \Gamma, \qquad
  \phi = 0 \text{ on } \Gamma,
\end{equation}
には唯一の粘性解 $\phi(\bm{x}) = d(\bm{x}, \Gamma)$ が存在する．
\end{theorem}

\begin{proof}[証明スケッチ（Crandall--Lions 1983~\cite{CrandallLions1983}）]
(i)~\emph{存在．} $\Gamma$ が閉集合なら $d(\bm{x},\Gamma)$ は Lipschitz 連続である．
(ii)~\emph{局所微分可能性．} 最近接射影 $\pi : \bm{x}\mapsto \Gamma$ が一意な領域で
$\phi$ は微分可能で $\bnabla\phi(\bm{x}) = \hat{\bm{n}}_\Gamma(\pi(\bm{x}))$．
(iii)~\emph{Eikonal 成立．} したがって $|\bnabla\phi|=1$ がほぼ至るところで成立．
(iv)~\emph{一意性．} 粘性解の比較原理~\cite{CrandallLions1983} により
$\phi$ は $\Gamma$ を境界条件として一意．
\end{proof}

\paragraph{アルゴリズム}
$\phi$ の再構成は FMM（狭帯域・非一様推奨）または FSM（全域）を適用する．
両者で得られる $\phi$ は粘性解として同一である．
再構成後は既存の FCCD 面中心移流（§~\ref{sec:advection}）
で鋭利な界面追跡を再開する．

\subsubsection{幾何的許容性条件}
\label{sec:ridge_admissibility}

定理~\ref{thm:eikonal_reconstruction} の 2 条件（幾何正則性・射影一意性）に加え，
数値実装では以下の 1 条件が必要となる：
\begin{enumerate}
  \item\textbf{法線一貫性．}$\xiridge$ から推定される法線方向が $\Gamma$ 上で整合（向き反転なし）．
\end{enumerate}
幾何正則性と射影一意性は曲率上限 $\kappa_{\max}\lesssim 1/h$ と最小間隔
$d_{\min}\gtrsim 2\varepsilon$ の条件でリッジ抽出時に自然に保証される．
法線一貫性は Hessian 固有ベクトル符号を連続 branch で選択することで確保する．

\subsubsection{非一様格子への拡張 D1--D4}
\label{sec:ridge_eikonal_nonuniform}

格子密度因子 $\alpha>1$ の界面適合格子
（§~\ref{sec:grid_density}）では，§~\ref{sec:ridge_aux_field}--§~\ref{sec:ridge_fmm_reconstruction}
の導出で暗黙に仮定された一様格子不変量が 3 か所で破れる：
(i) Gaussian 半幅 $\sigma$ の物理セル分解能が不均一化，
(ii) $\xiidx$ 空間で Hessian を評価すると
後述の実用的困難性（注記~\ref{rem:xi_hessian_forbidden}）に抵触，
(iii) FMM の二次式更新が単位ステップを仮定して $|\bnabla_x\phi|=1$ を破る．
以下 4 つの補正 D1--D4 でこれらを解消する．

\paragraph{D1 — $\sigma_\text{eff}(\bm{x})$ の空間スケーリング}
基準格子幅を $h_\text{ref}=(\prod_{a} L_a/N_a)^{1/d}$，
局所幾何平均を $h(\bm{x})=\sqrt{h_x(i)\,h_y(j)}$ とし，
\begin{equation}
  \sigma_\text{eff}(\bm{x}) = \sigma_0 \cdot \frac{h(\bm{x})}{h_\text{ref}},
  \qquad \sigma_0 \in \mathbb{R}_{>0}\text{（}h_\text{ref}\text{ 単位）}
  \label{eq:sigma_eff}
\end{equation}
と修正する．
物理セル単位では $\sigma_\text{eff}(\bm{x})/h(\bm{x}) = \sigma_0/h_\text{ref}$ が定数となり
推奨値 $\sigma\in[2,4]$（§~\ref{sec:verification} 校正）が
そのまま $\sigma_0$ に引き継げる．

\paragraph{D2 — 物理空間 Hessian（$\xi$ 空間 Hessian 禁止定理）}
$\xiidx$ 空間で Hessian を組み立て
chain rule $\bnabla_x^2 = J^{\!\top}\bnabla_\xi^2 J
+ (\bnabla_\xi J^{\!\top})\bnabla_\xi$
で物理空間に変換する素朴な実装は禁じられる．

\begin{remark}[$\xi$ 空間 Hessian の実用的困難性]\label{rem:xi_hessian_forbidden}
界面遷移幅が $\le 4$ セルのとき，
$\xi$ 空間 Hessian を経由する任意の計算は $\mathcal{O}(10^{-1})$ の
フロア誤差を持つ（実験的根拠は §~\ref{sec:xi_sdf_reinit}）．
リッジ定義 \eqref{eq:ridge_set_def} は
$\sigma_\text{eff}\sim 3h$ の範囲で評価されるため，
この条件は一般に違反される．
\end{remark}

許容される実装：
\begin{enumerate}
  \item\textbf{Approach A（論文整合）}——物理空間 Direct 非一様 CCD
        で $\bnabla_x^2\xiridge$ を直接評価（本稿の推奨実装）．
  \item\textbf{Approach B（実用 fallback）}——
        物理空間二階 FD
        $[\partial_{xx}\xiridge]_{ij}
        = (\xi_{i+1,j}-2\xi_{ij}+\xi_{i-1,j})/(h_x^-(i)\,h_x^+(i))$
        を用い，符号判定 $\hat{\bm{n}}^{\!\top}H\hat{\bm{n}}<0$ を
        $H^{\text{FD}}$ で行う．リッジが Morse 非退化な限り
        符号は一致する（本式は非一様格子では $\Ord{1}$ 近似だが，
        Morse 非退化点で符号判定に限れば精度に依存しない）．
\end{enumerate}

\paragraph{D3 — 非一様 FMM}
標準 FMM の二次更新
$d_\text{new}=\tfrac{1}{2}(a_x+a_y+\sqrt{2-(a_x-a_y)^2})$
は $h_x=h_y=1$ を前提とする．物理空間 Eikonal
\begin{equation}
  \frac{(d-a_x)^2}{h_x^2}+\frac{(d-a_y)^2}{h_y^2}=1
  \label{eq:fmm_nonuniform}
\end{equation}
の閉形式解（大きい方の根）は
\begin{equation}
  d = \frac{a_x/h_x^2 + a_y/h_y^2 + \sqrt{D}}
           {1/h_x^2 + 1/h_y^2},\qquad
  D = \Bigl(\tfrac{1}{h_x^2}+\tfrac{1}{h_y^2}\Bigr)
      - \frac{(a_x-a_y)^2}{h_x^2 h_y^2}.
  \label{eq:fmm_closed_form}
\end{equation}
$D<0$ のときは一次元単調更新 $d=\min(a_x+h_x,\,a_y+h_y)$ を採用する．
粘性一意性は \eqref{eq:fmm_nonuniform} が物理空間で書かれている限り
§~\ref{sec:ridge_fmm_reconstruction} の粘性解一意性証明がそのまま成立する．

\paragraph{D4 — $\varepsilon$-widening の空間依存化}
再構成幅を
\begin{equation}
  \varepsilon_\text{local}(\bm{x})
  = \varepsilon_\text{scale}\cdot \varepsilon_\xi \cdot h(\bm{x}),
  \qquad \varepsilon_\xi = \varepsilon/h_\text{min}
  \label{eq:eps_local}
\end{equation}
と空間依存にする．

\textbf{較正の順序（循環論法の回避）．}
$\varepsilon_\text{scale}$ の決定は以下の 2 段階で行い，
適応化トリガーとは独立に先行して行う：
\begin{enumerate}[leftmargin=1.5em, itemsep=0pt, topsep=2pt]
  \item \textbf{Step 1（一様格子較正）．}
        一様格子 $h_x=h_y=h_\text{ref}$ 上で
        毛管波ベンチマーク（§~\ref{sec:xi_sdf_reinit}，$\sigma > 0$）を走らせ，
        $\varepsilon_\text{scale} \in \{0.8, 1.0, 1.2, 1.4, 1.6, 1.8\}$ を
        スキャンして体積保存誤差 $|\Delta V/V|$ を最小化する値として
        $\varepsilon_\text{scale}^{*} = 1.4$ を得る．
  \item \textbf{Step 2（非一様格子への移植）．}
        非一様格子では空間因子 $h(\bm{x})/h_\text{ref}$ が
        式~\eqref{eq:eps_local} で吸収されるため，
        $\varepsilon_\text{scale} = 1.4$ をそのまま引き継ぐ．
\end{enumerate}

\textbf{適応化トリガー（Step 3）．}
上記較正値のもとで体積保存指標が
\[
  |\Delta V(t)/V(0)| > \theta_V \quad (\text{推奨 } \theta_V = 5\%)
\]
を 3 連続タイムステップで満たしたとき，適応化として
$\varepsilon_\text{scale}(\bm{x}) \leftarrow 1.4 \cdot h_\text{ref}/h_\text{min}$ を
局所的に適用する．この適応化は Step 1 の較正値を変更せず，
空間因子のみを格子アスペクト比に応じて増幅する．
較正→運用→適応化の順序を遵守することで循環論法を回避する．

\subsubsection{$\sigma>0$ 制約}

純粋 FMM は $\sigma>0$ の毛管波条件下で体積保存が劣化する
（毛管波 $T=1$ で $|\Delta V/V|=8.2\%$，§~\ref{sec:xi_sdf_reinit} 参照）．
診断された根本原因は界面幅 $\varepsilon$ の取り扱いであり，
PPE 表面張力残差が $\sigma\kappa/\varepsilon$ にスケールする一方，
FMM の厳密 $C^1$ SDF は $\varepsilon$ を狭帯域公称値に固定するのに対し
演算子分離系では拡散で $\sim 1.4\varepsilon$ に広がるためである．
\textbf{$\varepsilon$-widening}（式~\eqref{eq:eps_local}）は
この劣化を補正する必須段階であり，FMM の後処理として必ず組み込む．

\subsubsection{面中心追跡との接続}
\label{sec:ridge_fccd_handoff}

トポロジーが固定された段階で $\phi$ は面に snap され，
ALE（Arbitrary Lagrangian--Eulerian）的な面固定定式化に移行する．
以降の移流・勾配評価は第~\ref{sec:advection} 章で定義する
FCCD 面中心演算子で 4 次精度を保ちつつ行われる
（§~\ref{sec:advection}）．
再構成された $\phi$ と FCCD 評価は同一の\textbf{面 locus}を共有するため，
再構成フェーズから追跡フェーズへの受け渡しは追加補間誤差を持たない．
Morse 遷移を含まない時間帯（トポロジー変化が発生しない区間）では，
アルゴリズムは
(i) FCCD による $\phi$ 面中心高次移流，(ii) 時折の FMM 再投影
という既存パイプラインの厳密な一般化となる．

\paragraph{本節の要点}
\begin{itemize}
  \item $\xiridge$ と $\phi$ は\textbf{役割が相補}：
        前者はトポロジー自由度，後者は計量整合性を担う．
  \item トポロジー変化は連続的 Morse 遷移として自動捕捉される．
  \item 一意性は粘性解の比較原理により保証される．
  \item 非一様格子への拡張は D1--D4 の補正で完結する．
  \item $\sigma>0$ では $\varepsilon$-widening が必須後処理．
\end{itemize}
