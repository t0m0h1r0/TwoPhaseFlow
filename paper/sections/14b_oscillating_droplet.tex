\subsection{振動液滴：PhaseRegion closed chart の正規ルート}
\label{sec:val_oscillating_droplet}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と理論基準}

振動液滴ベンチマークでは，毛管波と同じ PhaseRegion 正規経路を
閉じた界面 chart へ移す．
主状態は液滴を含む相領域 $\Omega_h$ であり，
界面 $\Gamma_h=\partial\Omega_h$ は closed radial chart
$X(\theta;A)$ で表示する．
有限体積量 $q_h$ は相領域から測られる cell measure であり，
$\phi$ は可視化とスナップショット用の gauge である．
したがって，検証する経路は
\begin{equation}
  X(\theta;A)
  \longmapsto q_l=Q_h(X(A)),
  \qquad
  q_g=|C|-q_l,
  \qquad
  E_h[X]=\sigma\,\mathrm{Per}(X),
  \label{eq:phase_region_droplet_route}
\end{equation}
である．
毛管波と振動液滴は別理論ではなく，
同じ perimeter variation を graph chart と closed chart で読んだものである．

初期界面は中心 $(0.01,0.01)\,\mathrm{m}$，半軸
$a=5.5\,\mathrm{mm},\ b=4.5\,\mathrm{mm}$ の液相楕円である．
面積等価半径と初期変形量を
\begin{equation}
  R_{\mathrm{eq}}=\sqrt{ab}=4.974937\,\mathrm{mm},
  \qquad
  D_0=\frac{a-b}{a+b}=0.10
  \label{eq:oscillating_droplet_initial}
\end{equation}
と定義する．
2D の $n=2$ Rayleigh--Lamb 基準として
\begin{equation}
  \omega_0^2
  = \frac{n(n^2-1)\sigma}{(\rho_l+\rho_g)R_{\mathrm{eq}}^3},
  \qquad n=2,
  \label{eq:oscillating_droplet_lamb}
\end{equation}
を用いる~\cite{Prosperetti1981}．
物性値は水--空気の実スケール値
$\rho_l=998.2\,\mathrm{kg/m^3}$，
$\rho_g=1.204\,\mathrm{kg/m^3}$，
$\mu_l=1.002\times10^{-3}\,\mathrm{Pa\,s}$，
$\mu_g=1.825\times10^{-5}\,\mathrm{Pa\,s}$，
$\sigma=0.0728\,\mathrm{N/m}$ である．
このとき $\omega_0=59.578473371\,\mathrm{s^{-1}}$，
\[
  T_\mathrm{RL}=\frac{2\pi}{\omega_0}=0.105460663082\,\mathrm{s}
\]
であり，正規 YAML はちょうど 1 周期を出力する．

\begin{itemize}[noitemsep,leftmargin=2em]
  \item 正規入口：\texttt{experiment/ch14/config/ch14\_oscillating\_droplet.yaml}
  \item 実行：\texttt{experiment/ch14/diagnose\_phase\_region\_oscillating\_droplet\_steps.py}
  \item グリッド：$[0,0.02]^2\,\mathrm{m}^2$，$32^2$，
        周期境界，界面適合 $\alpha=2$
  \item 時間：$T=T_\mathrm{RL}$，理論 CFL 係数 $1.0$，
        固定 \texttt{dt} なし
  \item PhaseRegion owner：\texttt{gas\_outside}
  \item PhaseRegion chart：\texttt{closed\_radial}
  \item PhaseRegion measurement：
        \texttt{closed\_radial\_cell\_integral\_total\_volume\_closed}
  \item エネルギー：\texttt{closed\_perimeter\_second\_variation\_fixed\_area}
  \item dynamics：\texttt{velocity\_verlet\_linear\_rayleigh\_lamb\_mode}
  \item 出力時刻：
        $0,\frac14T_\mathrm{RL},\frac12T_\mathrm{RL},
        \frac34T_\mathrm{RL},T_\mathrm{RL}$
\end{itemize}

この結果は，閉じた液滴が graph chart と同じ PhaseRegion 変分原理で
1 周期を進むことを示す．
振幅と速度は Rayleigh--Lamb 線形解へ一致し，エネルギー drift は
$6.25\times10^{-7}$ 未満に留まる．
PhaseRegion 測度残差は $0$ であり，
総体積 drift は丸め誤差まで落ちる．

同時に，この計算は重要な失敗知も明示する．
raw の P1 closed-radial gauge 測度をそのまま積分すると，
固定面積 closed chart の物理体積から最大
$1.822318492916\times10^{-7}$ だけ外れる．
これは $q$ を絶対視すれば閉曲線をガタガタにする種類の誤差であり，
今回の正規 route では total-volume-closed 測度として，
chart が所有する総体積制約で有限体積測度を閉じる．
この処理は tolerance 弱化，平滑化，減衰，CFL retuning ではない．
状態所有者を PhaseRegion に戻し，
$q$ をその測度として扱うための体積制約である．

なお，\autoref{fig:ch14_droplet_yaml_velocity_pressure_snapshots} の
速度・圧力は YAML 出力時刻に生成した reduced diagnostic field であり，
production PPE/corrector が PhaseRegion face cochain を消費したことを意味しない．
したがって本節の合格条件は
\texttt{force\_admissible=0} を含む closed-chart reduced route の合格である．
旧 \texttt{active\_geometry\_capillary} production runtime の
1 周期閉界面保存量 failure は V12
（\autoref{sec:closed_interface_volume_gate}）で負対照として残し，
本節の PhaseRegion 正規 route と混同しない．
PhaseRegion face cochain を圧力・速度へ消費するゲートが通るまでは，
closed chart の 1 周期成功を production force-admission 成功へ拡張して読まない．
