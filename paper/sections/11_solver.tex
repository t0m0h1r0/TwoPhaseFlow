\subsection{圧力ソルバの検証}
\label{sec:verify_solver}

前節の界面処理パイプライン（§~\ref{sec:verify_interface}）により，
曲率 $\kappa$ および延長物性場がアルゴリズム Step~5--7 の PPE ソルバに引き渡される．
本節では，PPE ソルバの中核をなす欠陥補正法（DC）の反復特性
（§\ref{sec:verify_ppe}），
Neumann 境界条件での精度（§\ref{sec:verify_ppe_neumann}），
分相 PPE + DC $k{=}3$ による密度比非依存な高精度 PPE
（§\ref{sec:verify_split_ppe_dc}），
変密度一括解法の参考データ（§\ref{sec:verify_ppe_varrho}）を順に検証する．

\subsubsection{欠陥補正法の反復回数と空間精度}
\label{sec:verify_ppe}
\label{sec:dc_iteration_accuracy}
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
§~\ref{sec:defect_correction_main} で導入した欠陥補正法（DC）の
反復回数 $k$ が解の精度をどの程度支配するかを確認する．

\noindent\textbf{評価対象：}
欠陥補正法（DC）の反復特性：$k$ vs 精度，DC vs FD 比較．

\noindent\textbf{手順とパラメタ：}
\begin{itemize}[noitemsep]
  \item 2 次元 Poisson 問題，参照解 $p^* = \sin(\pi x)\sin(\pi y)$（Dirichlet BC）
  \item $N = 8, 16, 32, 64, 128$ の一様格子
  \item (a) 反復回数 $k = 1, 2, 3, 5, 10$ の $L^\infty$ 誤差を計測
  \item (b) FD 5 点直接解法と DC $k{=}3$ の精度・コスト比較
\end{itemize}

$k = 1$ では初期値 $p^{(0)}=0$ に対する 1 回の反復が
$p^{(1)} = \omega\,L_L^{-1}b$ を返すのみであり，
$L_L$ の打ち切り誤差が支配するため収束次数は $\Ord{h^2}$ にとどまる．
$k = 2$ では中間的な $\Ord{h^4}$ となり，
$k \ge 3$ で CCD の設計精度が完全に発現して $\Ord{h^7}$ を達成する．
設計次数 $\Ord{h^6}$ を超える超収束は，Dirichlet 条件下で参照解が境界上で零となるため
境界スキームの $\Ord{h^5}$ 打ち切り誤差が消失することに起因する．
$k = 3$ と $k = 10$ の絶対誤差の差は高々 2 倍程度であり，
これ以上の反復は精度向上に対するコストが見合わない．
したがって以降のすべての PPE 求解には $k = 3$ を採用する．

$k = 3$ と $k = 10$ の絶対誤差の差は高々 2 倍であり，$k = 3$ を採用する．
DC $k{=}3$ は $N = 128$ で FD 比 $4.9 \times 10^7$ 倍高精度，コスト増 $3.4$ 倍．

\noindent\textbf{補足：DC $\omega$ 緩和とスペクトル収束}%
\label{sec:verify_dc_spectral}%
（付録~\ref{app:dc_convergence_theory}）．
変密度条件での DC 反復の $\omega$ 依存性を調べた結果，
等密度（$\rho_l/\rho_g = 1$）では $\omega \in [0.2, 0.7]$ で収束するが，
$\rho_l/\rho_g \geq 5$ ではすべての $\omega$ で停滞することが判明した．
この制約は分相 PPE 解法（§~\ref{sec:split_ppe_framework}）により完全に回避される
（§~\ref{sec:verify_split_ppe_dc}）．

CCD 境界スキーム（$\Ord{h^5}$）が全体精度のボトルネックであり，
二相流 PPE（Neumann 条件）の達成可能精度は $\Ord{h^5}$ が実質的上界である．

\subsubsection{分相 PPE + DC $k{=}3$：密度比非依存な高精度 PPE}
\label{sec:verify_split_ppe_dc}
% backward-compat
\label{sec:split_ppe_recovery}%
\label{sec:gfm_ppe_recovery}%
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
§~\ref{sec:split_ppe_framework} の分相 PPE 解法が，
等密度 DC $k{=}3$ の $\Ord{h^7}$ を密度比によらず回復することを実証する．

\noindent\textbf{評価対象：}
分相 PPE + DC $k{=}3$ vs 変密度一括 DC vs 分相 PPE + FD．

\noindent\textbf{手順とパラメタ：}
\begin{itemize}[noitemsep]
  \item 製造解 $p^* = \sin\pi x\sin\pi y$（Dirichlet BC），
        円形界面（$R = 0.25$，中心 $(0.5, 0.5)$），
        smoothed Heaviside 密度場（$\varepsilon = 1.5h$）
  \item 3 手法を比較：
        (a) 変密度一括 DC $k{=}3$，
        (b) 分相 PPE + FD 直接法（$\Ord{h^2}$），
        (c) 分相 PPE + DC $k{=}3$
  \item 液相内部（$\phi > 3h$）で $L^\infty$ 誤差を計測
  \item 密度比スイープ $\rho_l/\rho_g = 1$--$1000$（$N = 64$），
        格子収束 $N = 8$--$128$（$\rho_l/\rho_g = 1000$）
\end{itemize}

\end{enumerate}

分相 PPE + DC $k{=}3$ は全密度比（$\rho_l/\rho_g = 1$--$1000$）で
$9.4 \times 10^{-11}$（$N = 64$）に不変であり，
格子収束は厳密に $\Ord{h^{7.0}}$ で密度比に完全非依存である．
変密度一括 DC は $N \geq 64$ で収束が反転（負の次数）し，
分相 PPE + DC $k{=}3$ 比で $5.7 \times 10^7$ 倍低精度である．

\subsubsection{変密度 PPE の参考データ：滑らかな密度場での収束特性}
\label{sec:verify_ppe_varrho}
\label{sec:varrho_ppe_limitation}%
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
CCD 変密度演算子自体の精度特性を参考として記録する．

\noindent\textbf{評価対象：}
変密度 DC $k{=}3$（滑らかな正弦波密度場，Dirichlet BC）．

\noindent\textbf{手順とパラメタ：}
\begin{itemize}[noitemsep]
  \item 密度場 $\rho(x,y) = 1 + A\sin(\pi x)\cos(\pi y)$
        （$A = 0, 0.8, 0.98, 0.998$；$\rho_{\max}/\rho_{\min} \approx 1, 9, 99, 999$）
  \item DC $k{=}3$，Dirichlet BC，格子収束試験
\end{itemize}

\noindent\textbf{結果：}
$\rho_{\max}/\rho_{\min} \approx 999$ でも $\Ord{h^{5.8}}$ を維持した
（$N=128$ で $L^\infty = 8.5 \times 10^{-11}$）．

\noindent\textbf{結論：}
密度場が十分に滑らかである限り，CCD の高次精度は密度比によらず発現する．
界面型密度分布（smoothed Heaviside）では $\rho_l/\rho_g \geq 10$ で
DC が発散する本質的限界があり（§~\ref{sec:interface_crossing}），
これが分相 PPE 解法を必要とする根拠となっている．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{圧力場フィルタ適用禁止の定量的根拠}
\label{sec:verify_pressure_filter}
% ─────────────────────────────────────────────────────────────────────────────

§~\ref{sec:dccd_pressure_nofilt} で DCCD フィルタの圧力場への適用を禁止した
根拠を定量的に実証する．
製造解 $p = \sin(2\pi x)\sin(2\pi y)$（周期境界）に対し，
フィルタなしの CCD ラプラシアン精度と，DCCD フィルタ適用後のラプラシアン誤差を比較した．

\noindent\textbf{結果：}
フィルタなしの CCD ラプラシアンは厳密解に対して $\Ord{h^{6.0}}$ で収束する．
一方，DCCD フィルタ（$\varepsilon_d = 0.25$）を適用すると，
$\bnabla^2 p - \bnabla^2 \tilde{p}$（$\tilde{p}$: フィルタ後）の $L_\infty$ ノルムが
$\Ord{h^{2.0}}$ の発散誤差を生じる（$N = 32$: $1.51$，$N = 256$: $2.38 \times 10^{-2}$）．
$\varepsilon_d = 0.05$ でも同じ $\Ord{h^{2.0}}$ のスケーリングが確認された．

この誤差は速度補正ステップにおいて
$\bnabla \cdot \bu^{n+1}_{\mathrm{filtered}} =
(\bnabla^2 p - \bnabla^2 \tilde{p})/\Delta t \neq 0$
となり，非圧縮条件を $\Ord{h^2}$ で破壊する．
したがって，§~\ref{sec:dccd_pressure_nofilt} の禁止規則は必須である．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{PPE 条件数のスケーリング}
\label{sec:verify_ppe_condition}
% ─────────────────────────────────────────────────────────────────────────────

§~\ref{sec:ppe_bc_stability} で議論した変密度 PPE の条件数スケーリング
$\kappa \approx \Ord{(\rho_l/\rho_g) / h^2}$ を数値的に確認する．
FVM 変密度ラプラシアン（Neumann 境界，中心ゲージ固定）の条件数を
$N = 8$--$64$，$\rho_l/\rho_g = 1$--$1000$ で測定した．

\noindent\textbf{結果：}
等密度（$\rho_l/\rho_g = 1$）では $\kappa \propto N^{3.0}$（$\Ord{h^{-2}}$ に対応する
離散ラプラシアンの固有値分布に整合）．
$\rho_l/\rho_g \geq 10$ では，界面近傍の密度コントラストが
高周波固有モードを選択的に増幅し，$\kappa$ は $N$ に対する指数が
3--6 に増大する．
特に $\rho_l/\rho_g = 1000$，$N = 64$ では $\kappa \approx 2.2 \times 10^7$ に達し，
これが §~\ref{sec:varrho_dc_convergence} で報告した DC 発散の直接的原因である．

\noindent\textbf{結論：}
高密度比での条件数悪化は $\Ord{(\rho_l/\rho_g)/h^2}$ を超えるスケーリングを示し，
分相 PPE 解法（§~\ref{sec:split_ppe_framework}）による条件数低減の必要性を裏付ける．
