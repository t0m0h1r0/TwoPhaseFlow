\subsubsection{運動エネルギー減衰}
\label{sec:energy_conservation}
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
Predictor--PPE--Corrector パイプラインが，
既知の運動エネルギー減衰率を正しく再現し，
離散非圧縮条件を計算機精度水準で維持することを確認する．

\noindent\textbf{評価対象：}
NS ソルバ全体の保存性（PPE 投影 + 粘性項 + 対流項の結合）．

\noindent\textbf{手順とパラメタ：}
\begin{itemize}[noitemsep]
  \item Taylor--Green 渦：$[0, 2\pi]^2$，周期境界，単相流（$\rho = 1$，$\nu = 0.01$，$\mathit{Re} = 100$）
  \item 初期条件 $u = \sin x\cos y$，$v = -\cos x\sin y$，$p = -\tfrac{1}{4}(\cos 2x + \cos 2y)$
  \item 厳密解 $E_k(t) = E_k(0)\,e^{-4\nu t}$
  \item $N = 64$，$\Delta t = 0.01$，AB2+IPC，$T = 2.0$（200 ステップ）
  \item 合格基準：$E_k$ 相対誤差 $< 10^{-4}$，$\|\bnabla\cdot\bu\|_\infty < 10^{-10}$
\end{itemize}

\noindent\textbf{結果：}

\noindent\textbf{結論：}
運動エネルギーの相対誤差は $3.9 \times 10^{-8}$（$t = 2.0$）であり，
目標の $10^{-4}$ を 4 桁下回る（図~\ref{fig:tgv_energy}(a)）．
AB2+IPC スキームの時間離散化誤差（$\Ord{\Delta t^2}$，$\Delta t = 0.01$）が支配的であり，
FFT による PPE の厳密求解と CCD の $\Ord{h^6}$ 空間離散化の寄与は十分に小さい．
$\|\bnabla\cdot\bu\|_\infty < 10^{-13}$ が全ステップで維持され，
離散非圧縮条件は計算機精度水準で成立する（図~\ref{fig:tgv_energy}(b)）．
