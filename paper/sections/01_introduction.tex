\section{はじめに}
\label{sec:intro}

\subsection{気液二相流の数値計算における課題}

気液二相流とは，気相（ガス）と液相（液体）が共存し，
その間に明確な界面を形成しながら運動する流体現象である．
気泡塔，核沸騰，波浪，噴霧燃焼など，工学・自然界に広く現れる流体現象である．

この現象を数値的に扱う際には，以下の4つの本質的困難が存在する：

\begin{enumerate}
  \item \textbf{界面の捕捉}：密度・粘性が界面を挟んで不連続に変化する
        （$\rho_l/\rho_g \approx 1000$，$\mu_l/\mu_g \approx 100$）ため，
        界面位置を高精度かつ安定に追跡しなければならない．
  \item \textbf{表面張力の正確な評価}：表面張力は界面の曲率（$\kappa$）に比例する体積力である．
        曲率の計算には二階偏微分が必要となり，数値誤差が増幅されやすい．
  \item \textbf{非圧縮条件の高精度維持}：速度場の発散をゼロに保つ
        圧力場を毎ステップ正確に求解する必要がある．
  \item \textbf{密度成層の安定処理}：重力と密度差による静水圧分布を
        正確に扱わないと寄生流れ（parasitic currents）が発生する．
\end{enumerate}

\subsection{二相流数値計算の典型的失敗例}
\label{sec:failure_modes}

上記の困難を軽視すると，計算は「動く」が物理的に意味のない結果を出力する．
学部生が実装する際に最もよく遭遇する4つの失敗例を示す．

\begin{tcolorbox}[warnbox, title={失敗例1：寄生流れ（Spurious Currents）}]
\textbf{症状：} 静止しているはずの液滴が，界面に沿って渦状の速度場を持つ．速度の大きさは表面張力に比例して大きくなる．

\textbf{原因：} 曲率 $\kappa$ の計算誤差が表面張力項 $\sigma\kappa\bnabla\psi$ に誤差を生じさせる．
この誤差は圧力場と速度場を通じてフィードバックループを形成し，数値的な「流れ」として固定化する．
具体的には，``圧力ジャンプ条件 $[p]=\sigma\kappa$'' を離散的に正確に満足しないために発生する．

\textbf{本稿の対処：} CCD 法による高精度な$\kappa$計算（6次精度），
および Rhie-Chow 補正による圧力-速度連成の改善によって抑制する．
\end{tcolorbox}

\begin{tcolorbox}[warnbox, title={失敗例2：質量損失（Mass Loss）}]
\textbf{症状：} 時間が経つにつれて液滴が小さくなり，最終的に消える．

\textbf{原因：} 標準的な Level Set 法では，移流方程式 $\partial\phi/\partial t + \bu\cdot\bnabla\phi=0$ が
保存形ではないため，数値積分誤差が体積損失として蓄積する．
再初期化の不適切な実装も体積を変化させる．

\textbf{本稿の対処：} Conservative Level Set 法（CLS）により，
質量（体積）の離散的保存性を向上させる．
\end{tcolorbox}

\begin{tcolorbox}[warnbox, title={失敗例3：圧力-速度のデカップリング（Checkerboard 不安定）}]
\textbf{症状：} 圧力場が空間的に激しく振動し（チェッカーボードパターン），
速度場が非物理的な振動を示す．

\textbf{原因：} コロケート格子において，中心差分による圧力勾配の評価では
隣り合う格子点の圧力が連立されないため（1つ飛びの点しか参照しない），
高周波の圧力振動が速度場に伝播しない「幽霊モード」が存在する．

\textbf{本稿の対処：} Rhie-Chow 型フラックス補正により，
圧力と速度を適切に連成させる（6章参照）．
\end{tcolorbox}

\begin{tcolorbox}[warnbox, title={失敗例4：界面のにじみ（Interface Smearing）}]
\textbf{症状：} 時間とともに界面が広がり，密度・粘性の遷移領域が厚くなっていく．

\textbf{原因：} 移流スキームの数値拡散（numerical diffusion）により，
界面指標 $\psi$ のプロファイルが徐々にぼやける．
特に低次精度スキーム（1次風上差分など）では顕著．

\textbf{本稿の対処：} WENO5 スキーム（5次精度の上流差分）と
CLS 再初期化の定期的適用により，界面の鮮鋭さを維持する．
\end{tcolorbox}

\subsection{本稿のアプローチ}

上記の困難を克服するため，本稿では以下の手法を組み合わせる
（図\ref{fig:ns_solvers}参照）：

\begin{itemize}
  \item \textbf{界面追跡：} \textbf{Conservative Level Set (CLS) 法} \cite{OlssonKreiss2005}
        により，質量保存性に優れた界面追跡を行う．
        標準的な Level Set 法の弱点である体積損失を抑制する．
  \item \textbf{移流スキーム：} 移流項には \textbf{WENO5} \cite{JiangShu1996} を採用し，
        衝撃波的な界面の急峻な変化を振動なく捉える．
  \item \textbf{高精度空間微分：} 結合コンパクト差分法（CCD 法，Chu \& Fan \cite{ChuFan1998}）を用い，
        曲率計算などに必要な微分を $\Ord{h^6}$ 精度で評価する．
  \item \textbf{界面適合格子：} Level Set 値に連動したグリッド密度関数 $\omega(\phi)$ により，
        界面近傍のみを自動的に高解像度化する非一様格子を生成する．
  \item \textbf{格子構造：} チェッカーボード不安定を防ぐため
        コロケート格子を採用し，Rhie--Chow 型流束補正を適用する．
  \item \textbf{時間積分：} 対流項と CLS 移流には
        3次 TVD Runge--Kutta を，粘性項には Crank--Nicolson 半陰的処理を用いる．
  \item \textbf{圧力求解：} 変密度 Chorin Projection 法に基づいて圧力 Poisson 方程式を導き，
        FVM で差分化した後 BiCGSTAB で反復求解する．
\end{itemize}
