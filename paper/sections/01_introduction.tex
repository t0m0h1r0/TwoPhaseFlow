\section{はじめに}
\label{sec:intro}

% ─────────────────────────────────────────────────────────────────────────────
% 【論文の芯】本稿を貫く単一の設計哲学
% ─────────────────────────────────────────────────────────────────────────────

気液二相流の数値シミュレーションには多くの数値的困難が伴うが，
それらのうち最も根本的かつ克服が難しいのが
\textbf{寄生流れ（spurious currents）}——静止すべき液滴界面に現れる人工的な渦流れ——である．
従来の CSF（連続表面力）モデルでは，表面張力を体積力として近似するため
界面遷移層の厚さ $\varepsilon$ に起因する $O(\varepsilon^2) \approx O(h^2)$ の物理的誤差が残り，
格子を細分化しても寄生流れの下限が消えない（§\ref{sec:failure_modes}，§\ref{sec:balanced_force}）．

\textbf{本稿の技術的中核はこの問題の根本的解決にある．}
Ghost Fluid Method（GFM，§\ref{sec:gfm}）により界面を数学的不連続として扱い，
Young--Laplace ジャンプ条件 $[p]_\Gamma = \sigma\kappa$ を PPE に直接組み込むことで，
CSF 体積力を予測子ステップから完全に除去する．
これにより CSF 近似誤差 $O(\varepsilon^2)$ という寄生流れの物理的下限が本質的に排除される．
さらに結合コンパクト差分法（CCD，$O(h^6)$，第\ref{sec:CCD}章）による高精度曲率計算と
$O(h^6)$ 精度の圧力求解により，数値離散化誤差成分を界面処理精度が律速する水準まで低減する．

本稿に登場する他の全手法はこの中心課題を支えるために選択されている：
\begin{description}
  \item[Conservative Level Set + Dissipative CCD]（第\ref{sec:levelset}・\ref{sec:advection}章）
        界面の質量保存と数値拡散抑制——曲率計算の入力精度を維持するための基盤
  \item[DCCD 散逸によるチェッカーボード抑制]（第\ref{sec:collocate}章§\ref{sec:dccd_decoupling}）
        コロケート格子上の $2\Delta x$ モードを Dissipative CCD のスペクトル散逸で内在的に除去
  \item[CCD-PPE + 仮想時間陰解法]（第\ref{sec:pressure}章）
        変密度 Poisson 方程式を $O(h^6)$ 精度で求解し，Corrector の速度補正精度を保証
  \item[非一様格子]（第\ref{sec:grid}章）
        界面近傍を自動的に高解像度化し，界面処理の空間分解能を効率的に確保
\end{description}

以下 §\ref{sec:why_hard}〜§\ref{sec:roadmap} では，
なぜ単相流の方法がそのまま使えないか，どの困難がどの手法で解決されるかを順番に説明する．
全体アルゴリズムの俯瞰図は §\ref{sec:algo_overview} に先行提示する．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{単相流と気液二相流の本質的違い}
\label{sec:why_hard}
% ═══════════════════════════════════════════════════════════════════════════════

数値流体力学（CFD）を学んだ読者は，まず単相流（空気のみ，水のみ）の
Navier-Stokes 方程式を扱ってきたはずである．
\textbf{気液二相流}とは，気相（ガス）と液相（液体）が共存し，
その間に明確な界面を形成しながら運動する流体現象であり，
気泡塔・核沸騰・波浪・噴霧燃焼など，工学・自然界に広く現れる重要な現象である．
なぜ「2種類の流体が共存する」だけで，これほど困難が増すのか？
その答えは一言でいえば \textbf{「界面の存在」} に尽きる．

\label{tab:single_vs_two_phase}

\hline

\subsubsection{問題の核心：界面上の不連続}
% ─────────────────────────────────────────────────────────────────────────────

図\ref{fig:density_jump}に示すように，
気液界面を挟んで密度と粘性が急激に変化する．
数値計算では有限の格子上で計算するため，この不連続を直接扱うことはできない．
界面を格子内の有限厚さの遷移層として\textbf{滑らかに近似}することで計算可能にするが，
その近似の仕方によって計算の安定性・精度・質量保存性が大きく変わる．

\subsubsection{数値拡散と界面の「にじみ」（Smearing）：視覚的理解}
% ─────────────────────────────────────────────────────────────────────────────

単純な1次精度の上流差分で界面を移流した場合，
時間が経つにつれて界面が広がり（にじみ），
密度・粘性の遷移幅が拡大していく．

界面の鮮鋭さが失われると：
\begin{itemize}
  \item 曲率 $\kappa$ の計算精度が低下する
  \item 密度の勾配が弱まり，表面張力が正確に評価できなくなる
  \item 液相体積（質量）が徐々に減少する（質量損失）
\end{itemize}

本稿の Dissipative CCD（第\ref{sec:advection}章）+ CLS 再初期化（第\ref{sec:levelset}章）は，
このにじみを抑制するための中心的な手法である．
Dissipative CCD は散逸的スペクトルフィルタにより数値拡散を $\mathcal{O}(h^5)$ まで低減し，
CLS 再初期化は崩れた界面プロファイルを $\psi = H_\varepsilon(\phi)$ の形に定期的に回復させる（§\ref{sec:reinit}）．

\subsection{界面に起因する4つの数値的困難}
\label{sec:challenges}
% ═══════════════════════════════════════════════════════════════════════════════

気液二相流を数値的に扱う際には，以下の4つの本質的困難が存在する：

\begin{enumerate}
  \item \textbf{界面の捕捉と質量保存}：
        密度・粘性が界面を挟んで不連続に変化する
        （$\rho_l/\rho_g \approx 1000$，$\mu_l/\mu_g \approx 100$）ため，
        界面位置を高精度かつ安定に追跡しなければならない．
        標準的な Level Set 移流方程式 $\partial\phi/\partial t + \bu\cdot\nabla\phi = 0$ は
        \textbf{非保存形}であるため，数値積分誤差が体積損失として蓄積する
        （第\ref{sec:failure_modes}節，失敗例2参照）．
        さらに移流スキームの数値拡散により界面プロファイルが「にじみ」，
        曲率計算精度が低下する（同節，失敗例4参照）．

        \textbf{→ 本稿の対処：} Conservative Level Set 法（CLS 法）による
        質量（体積）の離散的保存性の向上，および Dissipative CCD スキームによる数値拡散の低減
        （第\ref{sec:levelset}章，第\ref{sec:advection}章）．

  \item \textbf{表面張力の正確な評価}：
        表面張力 $\sigma\kappa\hat{\bm n}$ は本来「界面上の面力（2次元面力）」であり，
        体積方程式（Navier-Stokes）に直接組み込めない．
        曲率 $\kappa$ の計算には二階偏微分が必要となり，数値誤差が増幅されやすい．
        デルタ関数的な集中力をどう離散化するかが精度を決定する．

        \textbf{→ 本稿の対処：} GFM（Ghost Fluid Method，§\ref{sec:gfm}）により
        Young--Laplace ジャンプ条件を PPE に直接組み込み，CSF 近似誤差を除去する．
        曲率 $\kappa$ は CCD 法による $O(h^6)$ 精度評価（第\ref{sec:CCD}章）で求める．

  \item \textbf{非圧縮条件の高精度維持}：
        速度場の発散をゼロに保つ圧力場を毎ステップ正確に求解する必要がある．
        単相流では均一係数のラプラシアン $\nabla^2 p = \cdots$ であるが，
        二相流では可変係数 $\nabla\cdot(1/\rho\,\nabla p) = \cdots$ となり，
        界面をまたぐ密度比（$\rho_l/\rho_g \approx 1000$）が係数の急峻な変化を生み，
        反復ソルバーの収束が著しく悪化する（第\ref{sec:pressure}章）．
        さらにコロケート格子では圧力と速度の数値的デカップリング
        （チェッカーボード不安定，第\ref{sec:failure_modes}節失敗例3参照）が
        追加の困難をもたらす．

        \textbf{→ 本稿の対処：} CCD product-rule 展開による $\Ord{h^6}$ 変密度 PPE 離散化
        および仮想時間陰解法（第\ref{sec:pressure}章），
        DCCD 散逸（$\varepsilon_d = 1/4$）によるチェッカーボードモードの内在的除去
        （第\ref{sec:collocate}章§\ref{sec:dccd_decoupling}）．

  \item \textbf{密度成層と寄生流れの安定処理}：
        重力と密度差による静水圧分布を正確に扱わないと，
        静止すべき液滴の界面に沿って渦状の人工的速度場（寄生流れ，spurious currents）が発生する
        （第\ref{sec:failure_modes}節，失敗例1参照）．
        寄生流れの根本原因は，運動量方程式において圧力勾配項 $-\nabla p$ と
        表面張力項 $\sigma\kappa\nabla\psi$ に\textbf{異なる差分演算子}を用いることによる
        離散化の不整合である：連続体レベルで両者が釣り合っていても，
        離散レベルで $\mathcal{O}(h^2)$ の不整合誤差が体積力として残り，
        人工的な速度場を駆動する（§\ref{sec:balanced_force} 参照）．

        \textbf{→ 本稿の対処：} GFM（§\ref{sec:gfm}）により CSF 体積力を除去し，
        CSF 近似誤差 $\mathcal{O}(\varepsilon^2)$ という寄生流れの物理的下限を排除する．
        CCD 法（第\ref{sec:CCD}章）による $\mathcal{O}(h^6)$ 精度の曲率計算と圧力求解により，
        数値離散化誤差成分を界面処理精度が律速する水準まで低減する．
\end{enumerate}

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{二相流数値計算の典型的失敗例}
\label{sec:failure_modes}
% ═══════════════════════════════════════════════════════════════════════════════

上記の困難を軽視すると，計算は「動く」が物理的に意味のない結果を出力する．
学部生が実装する際に最もよく遭遇する4つの失敗例を示す．

\paragraph{失敗例1：寄生流れ（Spurious Currents）}
\textbf{症状：} 静止しているはずの液滴が，界面に沿って渦状の速度場を持つ．
速度の大きさは表面張力に比例して大きくなる．

\textbf{原因：} 曲率 $\kappa$ の計算誤差が表面張力項 $\sigma\kappa\bnabla\psi$ に誤差を生じさせる．
この誤差は圧力場と速度場を通じてフィードバックループを形成し，数値的な「流れ」として固定化する．
具体的には，``圧力ジャンプ条件 $[p]=\sigma\kappa$'' を離散的に正確に満足しないために発生する．

\textbf{本稿の対処：} GFM（§\ref{sec:gfm}）により CSF 体積力を除去し寄生流れの物理的下限を排除する．
CCD 法による高精度な $\kappa$ 計算（$O(h^6)$）がこれを支える．

\paragraph{失敗例2：質量損失（Mass Loss）}
\textbf{症状：} 時間が経つにつれて液滴が小さくなり，最終的に消える．

\textbf{原因：} 標準的な Level Set 法では，移流方程式
$\partial\phi/\partial t + \bu\cdot\bnabla\phi=0$ が
保存形ではないため，数値積分誤差が体積損失として蓄積する．
再初期化の不適切な実装も体積を変化させる．

\textbf{本稿の対処：} Conservative Level Set 法（CLS）により，
質量（体積）の離散的保存性を向上させる．

\paragraph{失敗例3：圧力--速度のデカップリング（Checkerboard 不安定）}
\textbf{症状：} 圧力場が空間的に激しく振動し（チェッカーボードパターン），
速度場が非物理的な振動を示す．

\textbf{原因：} コロケート格子において，中心差分による圧力勾配の評価では
隣り合う格子点の圧力が連立されないため（1つ飛びの点しか参照しない），
高周波の圧力振動が速度場に伝播しない「幽霊モード」が存在する．
周期 $2\Delta x$ のチェッカーボードモード $p_j = P_0(-1)^j$ に対して
中心差分がゼロ勾配を返すことは代数的に示される（付録~\ref{app:checkerboard_mode}）．

\noindent\textbf{本稿の対処：} Dissipative CCD のスペクトル散逸（$\varepsilon_d = 1/4$）により，
$2\Delta x$ チェッカーボードモードをスキーム内部で除去する
（第\ref{sec:collocate}章§\ref{sec:dccd_decoupling} 参照）．

\paragraph{失敗例4：界面のにじみ（Interface Smearing）}
\textbf{症状：} 時間とともに界面が広がり，密度・粘性の遷移領域が厚くなっていく．

\textbf{原因：} 移流スキームの数値拡散（numerical diffusion）により，
界面指標 $\psi$ のプロファイルが徐々にぼやける．
特に低次精度スキーム（1次風上差分など）では顕著．

\textbf{本稿の対処：} Dissipative CCD スキーム（散逸的スペクトルフィルタ付き CCD；§\ref{sec:advection_motivation}）と
CLS 再初期化の定期的適用により，界面の鮮鋭さを維持する．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{界面捕捉法の分類と本稿の手法の位置付け}
\label{sec:interface_methods}
% ═══════════════════════════════════════════════════════════════════════════════

気液界面を数値的に追跡する手法は，大きく以下の3種類に分類される．
それぞれに一長一短があり，本稿の Conservative Level Set (CLS) 法は
それらの欠点を補う位置づけにある．

\begin{description}
  \item[\textbf{VOF（Volume of Fluid）法} \cite{HirtNichols1981}]
        各セル内の液相体積分率 $F \in [0,1]$ を輸送変数とする．
        保存形の輸送方程式を解くため\textbf{質量保存性に優れる}が，
        界面の幾何学的情報（法線 $\hat{\bm n}$，曲率 $\kappa$）を $F$ から
        再構成する精度が限られる．寄生流れが生じやすい．

  \item[\textbf{標準 Level Set（LS）法} \cite{OsherSethian1988}]
        界面からの符号付き距離関数 $\phi$（$|\nabla\phi|=1$）を輸送変数とする．
        $\hat{\bm n} = \nabla\phi$，$\kappa = -\nabla\cdot\hat{\bm n}$ と簡潔に書けるため
        \textbf{幾何計算精度が高い}．しかし移流方程式が非保存形のため，
        数値積分誤差が\textbf{質量損失}として蓄積するという根本的問題がある．

  \item[\textbf{Conservative Level Set（CLS）法} \cite{OlssonKreiss2005}（\textbf{本稿採用}）]
        $\phi$ を滑らかなヘヴィサイド関数 $\psi = H_\varepsilon(\phi) \in [0,1]$ に変換し，
        \textbf{保存形}の移流方程式を解く．
        VOF の質量保存性と標準 LS の幾何計算精度を両立する．
        ただし，幾何量（法線・曲率）の計算に $\phi$ を要するため，
        $\psi \to \phi$ の逆変換はロジット関数 $\phi = \varepsilon\ln(\psi/(1-\psi))$ により
        解析的に求まる（第\ref{sec:levelset}章参照）．
\end{description}

3手法の特性を表\ref{tab:method_comparison}に比較する．
CLS 法は質量保存性と幾何精度の両立という点で，最もバランスが良い手法の一つである．

\hline

曲率計算精度       & 劣  & 優      & 優 \\

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{寄生流れ抑制手法の先行研究と本稿の位置付け}
\label{sec:prior_work_spurious}
% ─────────────────────────────────────────────────────────────────────────────

寄生流れ抑制に対する従来アプローチは大きく二つに分類される．
第一は CSF モデルの枠組み内で演算子整合性を確保する Balanced-Force 条件であり，
Fran{\c{c}}ois ら \cite{Francois2006} が VOF 枠組みで体系化した．
しかし CSF モデルでは界面遷移層 $\varepsilon$ に起因する近似誤差 $O(\varepsilon^2) \approx O(h^2)$ が
寄生流れの物理的下限として残留し，格子精緻化だけでは本質的に除去できない
（§\ref{sec:balanced_force} に詳述）．

第二は Ghost Fluid Method（GFM）により界面を数学的不連続として扱い，
CSF 体積力を完全に除去するアプローチである．
Desjardins ら \cite{Desjardins2008} は GFM と高次 Level Set の組み合わせで
密度不連続を界面で鋭く扱う手法を示した．

本稿は後者の路線に立ち，GFM（§\ref{sec:gfm}）により CSF 近似誤差の下限自体を排除する．
本稿の新規性は，GFM を CCD 法（$O(h^6)$）と組み合わせる点にある：
CCD による陰的コンパクト差分は同一3点ステンシルで $O(h^6)$ 精度の1次・2次微分を提供し，
曲率計算と圧力求解の両方を高精度化する．
さらに Dissipative CCD（$\varepsilon_d = 1/4$）がコロケート格子のチェッカーボード不安定を
スキーム内部で抑制することで，Rhie--Chow 補正を不要とし，
PPE 右辺を CCD 発散 $+$ GFM ジャンプ補正のみから構成する
簡潔かつ数学的に整合した定式化を実現する（§\ref{sec:dccd_decoupling}）．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{本稿のアプローチ・構成・学習ロードマップ}
\label{sec:roadmap}
% ═══════════════════════════════════════════════════════════════════════════════

本稿は，気液二相流の CFD を初めて学ぶ読者のための橋渡し教材として設計されている．
一般的な CFD テキストが単相流で終わる部分から先—界面追跡・表面張力の離散化・
変密度圧力求解—を，理論と実装の両面から体系的に解説することを目的とする．

§\ref{sec:challenges} で述べた4つの困難を克服するため，
本稿では以下の手法を組み合わせる：

\begin{itemize}
  \item \textbf{界面追跡：} \textbf{Conservative Level Set (CLS) 法} \cite{OlssonKreiss2005}
        により，質量保存性に優れた界面追跡を行う．
        標準的な Level Set 法の弱点である体積損失を抑制する．

  \item \textbf{移流スキーム：} CLS 移流には \textbf{Dissipative CCD}（散逸的 CCD フィルタ，§\ref{sec:advection_motivation}）を採用し，
        CCD 中心差分のフォン・ノイマン不安定を抑制しながら界面の急峻な変化を安定に移流する．

  \item \textbf{高精度空間微分：} 結合コンパクト差分法（CCD 法，Chu \& Fan \cite{ChuFan1998}，第\ref{sec:CCD}章）を用い，
        曲率計算・圧力 Poisson 方程式の離散化・界面適合格子のメトリック係数評価に必要な
        1・2 階微分を $\Ord{h^6}$ 精度で一括評価する（同一 3 点ステンシルで $O(N)$ コスト）．

  \item \textbf{界面適合格子：} Level Set 値に連動したグリッド密度関数 $\omega(\phi)$ により，
        界面近傍のみを自動的に高解像度化する非一様格子を生成する．

  \item \textbf{格子構造：} コロケート格子を採用し，Dissipative CCD の内在的散逸
        （$\varepsilon_d = 1/4$，§\ref{sec:dccd_decoupling}）でチェッカーボード不安定を抑制する．

  \item \textbf{時間積分：} CLS 移流・再初期化には 3次 TVD Runge--Kutta を，
        NS Predictor の対流項には AB2 外挿を（$\mathcal{O}(\Delta t^2)$），
        粘性項には Crank--Nicolson 半陰的処理を（$\mathcal{O}(\Delta t^2)$）用いる．
        NS 全体の時間精度は AB2+IPC により $\mathcal{O}(\Delta t^2)$ を達成する
        （第\ref{sec:advection}章参照）．

  \item \textbf{圧力求解：} 変密度 IPC（van Kan 1986）に基づいて圧力増分 Poisson 方程式を導き，
        CCD 法で高精度差分化した後，仮想時間陰解法（§\ref{sec:ppe_pseudotime}）で反復求解する．
\end{itemize}

NS 方程式の各項とそれを処理するソルバーの対応関係は，
アルゴリズム全体を把握した後の第\ref{sec:algorithm}章（図\ref{fig:ns_solvers}）で整理する．

\subsubsection{対象読者と前提知識}
% ─────────────────────────────────────────────────────────────────────────────

本稿を効果的に読み進めるために，以下の前提知識を持っていることが望ましい．
\begin{itemize}
  \item \textbf{流体力学の基礎：}
        非圧縮性 Navier--Stokes 方程式，連続の式，Reynolds 数・Froude 数・Weber 数の概念
  \item \textbf{数値解析の基礎：}
        有限差分法（1次・2次精度の中心差分，前進差分）の概念，

  \item \textbf{単相流の数値計算（推奨）：}
        Chorin の Projection 法（圧力--速度分離），スタガード格子または
        コロケート格子の概念，陽的・陰的時間積分の基礎
\end{itemize}
これらに不安がある読者は，まず単相流の標準的な CFD テキスト
（例：Tryggvason ら \cite{Tryggvason2011}）を参照することを推奨する．

\subsubsection{学習ロードマップとアルゴリズム全体像}
\label{sec:algo_overview}
% ─────────────────────────────────────────────────────────────────────────────

本稿の内容は，以下の5ステップで段階的に習得することを推奨する（表\ref{tab:chapter_overview}の章構成と対応）．
\begin{enumerate}
  \item \textbf{前提知識の確認：}
        単相 Navier--Stokes 方程式の基礎，Projection 法による圧力--速度分離，有限差分法の基本概念
  \item \textbf{数学的基盤と界面追跡}（第\ref{sec:governing}--\ref{sec:advection}章）：
        One-Fluid 定式化，CSF モデル（§\ref{sec:csf}，歴史的導入）から GFM（§\ref{sec:gfm}）への移行，CLS 法，Dissipative CCD 移流スキーム
  \item \textbf{空間離散化スキーム}（第\ref{sec:CCD}--\ref{sec:pressure}章）：
        CCD 法，界面適合格子，Rhie--Chow 補正，変密度 PPE 求解
  \item \textbf{完全アルゴリズムの実装}（第\ref{sec:algorithm}章）：
        7ステップ離散計算フローの統合実装
  \item \textbf{コンポーネント検証}（第\ref{sec:verification}章）：
        CCD 精度・GCL・Zalesak 円板・PPE の単体検証
  \item \textbf{多相流ベンチマーク}（第\ref{sec:validation}章）：
        静止液滴・毛細管波・気泡上昇・Rayleigh--Taylor 不安定の物理検証
\end{enumerate}

以降の章を読み進める前に，各時間ステップで何が行われるかの実装の地図を把握しておくことが
不可欠である．アルゴリズム全体は（1）界面移流・再初期化・物性更新・曲率計算という
界面サブシステム（Step 1--4）と，（2）Predictor・PPE 求解・Corrector という
流体力学サブシステム（Step 5--7）の7ステップで構成される．
両サブシステムは物性値 $\rho,\mu$ と曲率 $\kappa$ を介して結合する．
各ステップの離散方程式・実装上の注意・収束判定の詳細は第\ref{sec:algorithm}章で述べる．
