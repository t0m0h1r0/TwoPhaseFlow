\subsection{変密度 PPE の破綻限界と GFM 必要性の定量化}
\label{sec:interface_crossing}
% ═══════════════════════════════════════════════════════════════════════════════

§~\ref{sec:advection_pressure_coupling} では $\rho_l/\rho_g \leq 10$ での
二相流時間積分を検証し，高密度比側で精度劣化の兆候を記録する．
本節では密度比をさらに系統的に増大させ，
smoothed Heaviside 一括解法の\textbf{破綻閾値}を定量的に特定する．
これは「失敗を肯定的に記録する」実験であり，GFM 導入の数値的根拠を提供する．

\noindent\textbf{目的：}
\begin{enumerate}[noitemsep]
  \item smoothed Heaviside 密度場での変密度 PPE が $\rho_l/\rho_g \geq \Ord{10}$ で
        発散することを定量データで示す
  \item 破綻閾値を特定し，GFM が必要となる密度比条件を明確にする
  \item §~\ref{sec:varrho_ppe_limitation}（§~\ref{sec:numerical_integrity}章）の
        「界面型密度場での検証は今後の課題」を本節で実施する
\end{enumerate}

\noindent\textbf{§~\ref{sec:varrho_ppe_limitation} との関係：}
§~\ref{sec:varrho_ppe_limitation} では\textbf{滑らかな密度場}
（$\rho = 1 + A\sin\pi x\cos\pi y$）での収束性を確認し，
「界面型密度場は状況が根本的に異なる」と述べた．
本節はその補完として，界面型（smoothed Heaviside）密度場に対する
製造解テストをパラメトリックスイープ形式で実施する．

\subsubsection{セットアップ：密度比パラメトリックスイープ}

\begin{itemize}[noitemsep]
  \item \textbf{計算領域}：$[0,1]^2$，壁面境界条件
  \item \textbf{初期条件}：静止円形液滴（$R = 0.25$，中心 $(0.5, 0.5)$）
  \item \textbf{製造解}：$p^* = \sin\pi x\sin\pi y$（Laplace 圧テストと同一）
  \item \textbf{密度場}：$\rho = \rho_g + (\rho_l - \rho_g)H_\varepsilon(\phi)$，$\varepsilon = 1.5h$
  \item \textbf{密度比}：$\rho_l/\rho_g = 2, 5, 10, 20, 50, 100$
  \item \textbf{格子}：$N = 64$（固定）
  \item \textbf{欠陥補正}：$k = 3$ 固定
  \item \textbf{評価指標}：DC 残差履歴 $\|r_k\|_\infty$（各反復），$\Delta p$ 相対誤差
\end{itemize}

\noindent\textbf{合否基準（「失敗」が合格）：}
\begin{itemize}[noitemsep]
  \item $\rho_l/\rho_g \leq 5$：DC 収束（$\|r_3\| / \|r_0\| < 10^{-4}$） → PASS
  \item $\rho_l/\rho_g \geq 20$：DC 発散（$\|r_k\|$ が増大） → \textbf{EXPECTED FAILURE}
  \item 破綻閾値が $\rho_l/\rho_g = 10 \pm 5$ の範囲に特定できること → 本節の合格基準
\end{itemize}

\subsubsection{技術的根拠と GFM への接続}

\noindent\textbf{なぜ破綻するか（理論的説明）：}
$1/\rho$ が遷移層幅 $\varepsilon = 1.5h$ で $\rho_l/\rho_g$ 倍のジャンプを持つとき，
界面を横切る FD/CCD ステンシルは $\Ord{1}$ の打ち切り誤差を生じる（§~\ref{sec:ppe_condition_number}）．
欠陥補正の低次前処理 $L_L^{-1}$ も同じステンシルで構成されるため，
反復行列 $I - L_L^{-1}L_H$ のスペクトル半径が 1 を超えて発散する
（反復回数 $k$ の増大や緩和係数 $\omega < 1$ による収束回復は不可能）．

\noindent\textbf{破綻の解決策：GFM 分相解法：}
各相で定数密度の PPE（$\rho_k^{-1}\nabla^2 p = q$）を独立に解き，
界面上のジャンプ条件 $[p] = \sigma\kappa$ を
最近接点 Hermite 補間（§~\ref{sec:extension_pde}）で課す
\textbf{GFM}（Ghost Fluid Method，§~\ref{sec:gfm}）が必要である．
GFM は不連続係数を横切るステンシルを排除するため，
$\rho_l/\rho_g$ によらず CCD の $\Ord{h^6}$ 精度が回復される．
GFM の実装と検証は今後の最優先課題である（§~\ref{sec:future_work}）．

\noindent\textbf{結論：}
本節のパラメトリックスイープにより，
smoothed Heaviside 一括解法の適用可能範囲（$\rho_l/\rho_g \lesssim 5$–$10$）と
GFM が必要となる密度比条件（$\rho_l/\rho_g \geq 10$）が定量的に特定される．
これは精度バジェット（§~\ref{sec:error_budget}）における
「二相流拡張では GFM 導入が不可欠」という結論を数値的根拠で裏付ける．
