\subsection{非一様格子 NS（V8/V9）と一様格子 CLS 移流（V10）}
\label{sec:nonuniform_grid_ns}

\noindent\textbf{検証カバレッジの境界．}
V8 は固定非一様格子上の静止液滴，V9 は §14 stack 条件下の短時間 nominal/local-$\varepsilon$ switch probe，
V10 は $\alpha=1$ の固定一様格子上 CLS 移流である．
したがって本節は「非一様格子を有効化すれば，長時間移動界面・格子再生成・表面張力・
分相 PPE/HFE・壁面 BC の組合せが production 済みになる」ことを主張しない．
非一様 Zalesak，非一様 single-vortex，移動液滴，上昇気泡，および
pressure-jump/HFE 長時間非一様 run は追加 gate として残す．

\subsubsection{V8：非一様 NS 静止液滴}
\label{sec:nonuniform_ns_static}

\paragraph{検証対象．}
非一様格子（界面付近細密; 引き伸ばしパラメータ $\alpha$）上で
静止液滴 BF 平衡が長時間維持されること（\autoref{sec:U3_nonuniform_suite}）．
$\alpha = 1$ で一様格子と一致し，$\alpha = 2$ で界面付近 $h_{\min} = 0.5\, h_{\mathrm{avg}}$．
本 V8 は §\ref{sec:verification} 縮約路線 (ii)（二相縮約 BF；CCD 微分 + Neumann FVM PPE
+ CSF $\kappa_{lg}$，pressure-jump PPE は含まない）を非一様格子に拡張した縮約検証である．
出力 $p$ は physical pressure であり，判定は Laplace 圧 $\sigma/(r\,\mathrm{We})$ に対する
相対誤差 $\Delta p_{\mathrm{rel}}$ を用いる．

\paragraph{設定．}
$L=1$, 半径 $r=0.25$，$\sigma=1$, $\mathrm{We}=10$,
$\rho_l=10$, $\rho_g=1$．
$\Delta p_{\mathrm{exact}} = \sigma/(r\,\mathrm{We}) = 0.4$．
$N \in \{48, 64, 96\}$, $\alpha \in \{1, 2\}$．
$\Delta t = 0.20\,h_{\min}$, $n_{\mathrm{step}} = 200$．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item $\alpha = 1$（一様）で $\Delta p_{\mathrm{rel}} \le 1.4\%$ → V3 結果と一致\checkmark．
  \item $\alpha = 2$（非一様）で $\Delta p_{\mathrm{rel}} \le 4\%$（$N=96$ で $2.86\%$）\checkmark — 設計通りの安定動作．
        ただし非一様格子上の Heaviside--$\delta$ プロファイル（U5）と CSF $\kappa_{lg}$ 推定の
        Jacobian 補正が要求精度を増やす．
  \item \textbf{$\alpha=1$ で $N=48\!\to\!64\!\to\!96$ の $u_\infty^{\mathrm{end}}$ N 軸 monotonicity 非単調}
        ($1.42\!\to\!1.35\!\to\!5.64\times10^{-4}$，$N=96$ で約 $4\times$ 上昇)：
        $\Delta p_{\mathrm{final}}$ 自体は $0.395$ 一定で BF 平衡は維持される．
        本効果は §\ref{sec:verification}「N 軸 vs.\ 時間軸 monotonicity の区別」 で
        記述した CCD 高次微分の短波長 spurious mode 励起特性に該当する Type-D 現象であり，
        V5 と同じ機序．圧力スケールは保たれているため BF 検証目的では合格，
        ただし長時間 spurious 抑制には HFE/pressure-jump 経路（U6, V6/V7）が必要．
\end{itemize}
V8 の合格は固定格子・CSF 縮約静止問題に対するものであり，
移動界面の非一様格子 production 合格ではない．
検証対応：\autoref{sec:U3_nonuniform_suite} の非一様格子定義に対し，
静止液滴の圧力誤差と終端速度を測定する．

\subsubsection{V9：§14 stack による nominal/local $\varepsilon$ switch 診断}
\label{sec:local_eps_validation}

\paragraph{検証対象．}
非一様格子上で Heaviside--$\delta$ 厚み $\varepsilon$ を
\textbf{(A) nominal} ($\alpha=1,\ \varepsilon=1.5h$),
\textbf{(B) nominal} ($\alpha=2,\ \varepsilon=1.5h$),
\textbf{(C) local} ($\alpha=2,\ \varepsilon_{ij}=1.5\max(h_{x,i},h_{y,j})$)
で比較する．本試験は「local $\varepsilon$ が nominal より必ず小さい寄生流れを与える」
ことの優劣比較ではなく，現行 §14 stack で nominal/local mode を切り替えても
短時間静止液滴が破綻しないこと，およびこの縮約経路で mode 差が実際に現れるかを測る
switch 診断である．本試験は旧 CCD/CSF 経路ではなく，
\textbf{§14 と同じ operator stack}（FCCD interface/pressure gradient,
UCCD6 convection, HFE curvature, pressure-jump phase-separated FCCD PPE,
face-flux projection）で再実行する．

\paragraph{設定．}
縮約静止液滴条件 ($r=0.25,\ \sigma=0.10,\ \rho_l/\rho_g=10$)．
$N\in\{24,32\}$, $n_{\mathrm{step}}=10$,
theory CFL multiplier $0.50$．圧力ジャンプ形式では返却される $p$ は
物理 Laplace 圧そのものではなく投影補正圧の性格を持つため，
主指標は $u_\infty^{\max}$ と体積分率 drift（$|\Delta V|/V_0$）とし，
$|\Delta p_{\mathrm{corr}}|/(\sigma/R)$ は補助診断としてのみ示す．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{§14 stack switch 診断として合格} (\checkmark)：nominal/local mode は
        §14 stack（FCCD/HFE/pressure-jump PPE）と同時に用いる条件下で，短時間静止液滴を破綻させない
        （表~\ref{tab:V_verdict_types} の §14 stack 診断；design criterion 合格・
        Type-A/B/D 再分類のいずれとも異なる no-regression 診断）．
        $N=32$ では B/C が同一で，local mode はこの縮約経路で nominal より静かな界面を
        与えたとは主張しない．C/A の $u_\infty^{\max}$ 比は $2.36$，
        C/B 比は $1.00$ である．
        体積保存指標 $|\Delta V|/V_0$ は A/B/C 全条件で
        $\le 9.16\!\times\!10^{-8}$（表~\ref{tab:V9_local_eps}）であり，
        local-mode switch による体積誤差悪化は見られない．
  \item ただし本 V9 は縮約静止液滴 probe であり，§14 の水--空気級 material benchmark
        そのものではない．local $\varepsilon$ の効果を採用理由として使うには，
        HFE curvature と pressure-jump PPE が同時に有効な構成で，かつ移動界面または
        reinitialization が実際に local thickness を経由する追加 gate を別途要求する．
        CSF 経路では使用禁止であり，移動界面の長時間採用には
        非一様移流 benchmark と pressure-jump/HFE 統計 gate を別途要求する．
\end{itemize}
検証対応：\autoref{sec:U5_heaviside_delta} の Heaviside--$\delta$ 定義に対し，
nominal/local mode switch 時の寄生流れ，体積 drift，および 2D hot spot を比較する．

\subsubsection{V10-a：FCCD/TVD--RK3 による Zalesak CLS 移流}
\label{sec:zalesak_cls_advection}

\paragraph{検証対象．}
古典的 Zalesak slotted disk advection 問題~\cite{Zalesak1979} を
FCCD flux-form + TVD--RK3 の CLS 移流として解き，
\textbf{conservative CLS mass 保存}と\textbf{重心追跡精度}を評価する．
\autoref{sec:cls_advection} 式~\eqref{eq:cls_advection} の advection 項に
一様格子上の高次保存形差分 (\autoref{sec:fccd_def}) を組み合わせた評価であり，
非一様格子の検証は V8/V9 に分離する．本 V10-a は §\ref{sec:verification}
「各 V test の判定再分類」に従い質量保存軸 \checkmark / 形状軸 $\triangle$ の
二軸構造で評価する．

\paragraph{設定．}
標準 Zalesak slotted disk: 半径 $0.15$, slot 幅 $0.05$, slot 深さ $0.25$,
中心 $(0.5, 0.75)$．背景流 $\bu(\bx) = (-2\pi(y-0.5), 2\pi(x-0.5))$
（剛体回転; 1 周期 = $1$ 時間単位）．
$N \in \{64, 128\}$, $\alpha=1$．
$\Delta t_{\mathrm{est}} = 0.25\,h_{\min}/\max|\bu|$ からステップ数を決め，
最終時刻が 1 周期に一致するよう $\Delta t$ を調整して積分する．
\textbf{Olsson--Kreiss 型 conservative CLS mass correction}~\cite{OlssonKreiss2005}
を 10 step ごとに適用する（Type-B alignment：production stack（§14）と同じ
質量修正 cadence を §13 検証 path にも反映；mass-only correction であり
界面位置や曲率を rounding しないため，sharp-corner advection の独立診断としての性格は維持される）．

\paragraph{判定と制約条件（V10-a 二軸構造）．}
script の plot 合格参照ライン (mass drift $< 0.5\%$ axhline, および centroid 誤差パネルの
$O(h^{1.5})$ 参照\emph{傾き}線；後者は収束 order の閾値ではなく log--log slope の比較線；
診断プロット上の guideline であり docstring 由来の正式 criterion ではない；
plot コードでは \texttt{Volume drift} ラベルが用いられるが，incompressible CLS では
mass drift と等価) は二つの \emph{独立} な観測軸を混在させている：
\textbf{(i) 質量保存}（$\int_\Omega \psi\,dV$ の単一周期にわたる drift）と
\textbf{(ii) 形状/位置精度}（$C^0$ コーナーをもつ slotted disk の centroid 収束次数）．
前者は Olsson--Kreiss 形 mass correction~\cite{OlssonKreiss2005} で
構造的に担保され，後者は slot 角部 ($C^0$ singularity) と $\varepsilon$-smoothing
の interaction による\textbf{原理的下限}~\cite{Zalesak1979,OlssonKreiss2005}
で律速される．本稿はこれらを別軸で評価する（§\ref{sec:verification}
「各 V test の判定再分類」と整合する Type-B / Type-D の二軸構造）：
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{質量保存軸（\checkmark, Type-B 修正後）}：$N=128$ で
        mass drift $4.70\times 10^{-4}\%$，$N=64$ で $0.0281\%$，両者とも
        合格基準 $0.5\%$ を 3 桁以上下回る（V10-a の主判定）．
        Olsson--Kreiss mass correction を 10 step ごとに適用した結果であり
        （表~\ref{tab:V_verdict_types} の Type-B 修正に該当），
        この修正なしでは無修正 CLS 移流の cumulative drift がここまで
        抑えられない．
  \item \textbf{形状精度軸（$\triangle$, Type-D 解像度律速）}：$N=128$ で
        centroid err $4.911\times 10^{-3}$, centroid order $\approx 1.22$，
        shape $L^1 = 8.307\times 10^{-3}$．Zalesak slot 幅 $0.05$ に対し
        $N=128$ で slot/h $= 6.4$，$\varepsilon=1.5h$ が slot 幅の $24\%$ を
        占めるため，corner smearing は $\varepsilon$ と $h$ の両方で律速され，
        定数係数の $h$ 収束は asymptotic regime に到達しない（$N$ sweep 拡張
        $\{32, 64, 128, 256, 512\}$ で漸近域に入るが，本稿の検証コストでは
        $N=128$ までを採用）．これは Type-D（連続方程式・離散構造に由来する
        理論的 hard limit）として表~\ref{tab:V_verdict_types} に分類される．
\end{itemize}
以上により V10-a は \textbf{質量保存軸 \checkmark / 形状精度軸 $\triangle$} の
二軸構造で記録する．非一様格子サポート ($\alpha>1$) は
\textbf{NS 静止問題} (\autoref{sec:U3_nonuniform_suite}, V8 $\alpha=2$ で
$\Delta p$ 誤差 $\le 5\%$) と local $\varepsilon$ 検証（V9）で扱うに留まり，
非一様格子上の Zalesak 形状追跡は未検証 gate として残す．

検証対応：式~\eqref{eq:cls_advection}，\autoref{sec:cls_advection} の CLS 移流，
\autoref{sec:fccd_def} の face value/grad に対し，mass drift と重心誤差を測定する．

\subsubsection{V10-b：単一渦 deformation reversal}
\label{sec:single_vortex_reversal}

\paragraph{検証対象．}
LeVeque 型単一渦 deformation 問題~\cite{LeVeque1996,RiderKothe1998} を
固定一様格子上で FCCD/TVD--RK3 + Ridge--Eikonal 再初期化 + Olsson--Kreiss
mass correction で解き，速度場反転後 ($t=T$) における初期円盤への形状復元性を
評価する．本 V10-b は §\ref{sec:verification}「各 V test の判定再分類」に従い
質量保存軸 \checkmark / 形状軸 $\triangle$ の二軸構造で評価する．

\paragraph{設定．}
LeVeque 単一渦速度場 $\bu(\bx, t) = (-\sin^2(\pi x)\sin(2\pi y),\,
\sin(2\pi x)\sin^2(\pi y)) \cos(\pi t/T)$．
$N=128$, $\alpha=1$，$T=8$（Rider--Kothe 1998 標準 $T=2$~\cite{RiderKothe1998,LeVeque1996}
の \emph{4 倍} のストレス領域で意図的に実施し，filament 厚 $<h$ の sub-grid 領域
をどこまで保つかを試す）．Ridge--Eikonal 再初期化と
Olsson--Kreiss 質量修正は 10 step ごとに適用．図示は初期円盤，最大変形時 ($t=T/2$)，

\paragraph{判定と制約条件（V10-b 二軸構造）．}
V10-a と同じく二軸分離評価を行う（§\ref{sec:verification}
「各 V test の判定再分類」と整合する Type-B / Type-D の二軸構造）：
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{質量保存軸（\checkmark, Type-B 修正後）}：mass drift $0.0428\%$ は
        合格基準 $0.5\%$ を 10 倍以上下回る．Olsson--Kreiss mass correction~\cite{OlssonKreiss2005}
        による構造的保証が効いている（表~\ref{tab:V_verdict_types} Type-B）．
  \item \textbf{形状復元軸（$\triangle$, Type-D 理論的下限）}：$L^1$ reversal error
        $2.248\times10^{-2}$ には正式 criterion が存在しない．
        $T=8$ は Rider--Kothe 1998 標準 $T=2$~\cite{RiderKothe1998,LeVeque1996}
        の \emph{4 倍のストレス} 領域で実施しており，最大変形時 ($t=T/2$) には
        filament 幅が $h$ を下回り，固定一様格子では原理的に
        追跡不能~\cite{EnrightFedkiwFerzigerMitchell2002}．
        さらに reinit + clip + mass correction はいずれも非可逆な
        空間平滑化を伴うため，可逆 deformation cycle に対し
        $L^1 \to 0$ は実装縮約上達成不能である．
        これは sub-grid filament tracking の理論的下限であり，
        Eulerian-only 固定格子では PLS / AMR / moving mesh
        ~\cite{EnrightFedkiwFerzigerMitchell2002} 級の物理モデル拡張なしには
        改善できない．本稿はこれを Type-D（理論的 hard limit；
        表~\ref{tab:V_verdict_types}）として記録し，
        $\triangle$ を理論証明付きで据え置く．
\end{itemize}
以上により V10-b は \textbf{質量保存軸 \checkmark / 形状復元軸 $\triangle$} の
二軸構造で記録する．

\autoref{sec:fccd_def} の face value/grad に対し，単一渦 reversal error と
mass drift を測定する．
