\subsubsection{U2：CCD--Poisson + 現行 PPE 境界条件 [Tier I]}
\label{sec:U2_ccd_poisson_ppe_bc}

\paragraph{検証対象．}
第~\ref{sec:ppe_solve}章 で導入した defect--correction 反復
（§\ref{sec:defect_correction_ppe}：$k$ 回反復で精度を段階的に押し上げる），
PPE Neumann 境界条件 + 1 点ゲージ固定（§\ref{sec:ppe_solve} 末尾）の
\textbf{単体精度}を確認する．
U2-a の Dirichlet 変種は CCD 楕円作用素 + 一般 DC 反復理論の
smooth reference として local FD ヘルパで構成し，
§\ref{sec:defect_correction_main} の段階収束理論を独立に検証する．
U2-b は \texttt{PPEBuilder} が組み立てる現行 FVM/spsolve 系の
Neumann PPE + 1 点ゲージを直接測る．
旧 \texttt{PPESolverCCDLU}（CCD Kronecker + sparse LU）は
滑らかな component/reference 用に限定されるため，本 U2-b の production 経路からは外す．

\paragraph{設定．}
2D 矩形領域 $[0,1]^2$ 上で MMS による格子収束（壁境界条件下）．
\begin{description}\setlength{\itemsep}{0pt}
  \item[U2-a] Dirichlet Poisson 参照．$p^* = \sin(\pi x) \sin(\pi y)$，
        $f = -2\pi^2 p^*$．DC 反復回数 $k \in \{1,2,3,5,10\}$．
        理論期待：$k=1 \to \Ord{h^2}$，$k=2 \to \Ord{h^4}$，
        $k \ge 3 \to \Ord{h^7}$（スーパー収束）．
        実装は CCD 楕円作用素を残差評価に，FD Dirichlet Laplacian を inner solve
        として用いる手組 DC 反復（production の Neumann PPE--DC 連の数学的代理）．
  \item[U2-b] 現行 FVM--PPE 作用素 + Neumann ゲージ固定．
        $p^* = \cos(\pi x)\cos(\pi y)$，$f = -2\pi^2 p^*$．
        \texttt{PPEBuilder} が定数密度 ($\rho\!\equiv\!1$) の 5 点 FVM Laplacian を
        Neumann 境界条件 + 1 点ゲージ固定で組み立てる．理論期待は
        現行 production baseline と同じ $\Ord{h^2}$．
\end{description}

\paragraph{判定と次章への接続．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item U2-a $k=1,2,3$：設計値 $\{2,\,4,\,7\}$ にそれぞれ
        $\{2.00,\,4.18,\,6.90\}$ で到達．$k=3$ の実測 $6.90$ は 5 点 log--log fit
        ($N\!=\!8\!\to\!128$) の典型的な $\pm 0.1$ 雑音帯内で設計 7 次に整合；
        $N\!=\!128$ で $k\!\ge\!3$ は $\Ord{10^{-12}}$ の機械精度床に達し，
        slope は床支配で頭打ちになる．$k=5,10$ は $k=3$ と同精度
        （DC が $k=3$ 時点で既に飽和；§\ref{sec:ppe_dc_accuracy} の段階理論と整合）．\checkmark
  \item U2-b 現行 FVM--PPE + Neumann ゲージ：slope $2.00$．
        ch12 以降の production baseline である \texttt{PPEBuilder}/FVM direct 経路が，
        定数密度 Neumann MMS で設計 2 次を再現することを確認．\checkmark
\end{itemize}
検証対応：§\ref{sec:defect_correction_main} の DC 反復式に対し，
Dirichlet MMS と現行 Neumann ゲージ固定 PPE の格子収束を測定する．
