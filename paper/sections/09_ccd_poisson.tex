\section{圧力 Poisson 方程式の解法}
\label{sec:ppe_solve}

本章では，§~\ref{sec:projection_derivation} で導出した変密度 PPE を
高精度・効率的に解く枠組みを整える．
本稿の主要 PPE 戦略は\textbf{分相 PPE 解法}であり，
界面ジャンプ条件を課した各相の定数密度ポアソン方程式として解くことで，
界面を横切る不連続ステンシルを排除し，
密度比に直接依存しない各相内精度を目指す．

\noindent\textbf{本章の構成：}
CCD による $\Ord{h^6}$ PPE 離散化（§~\ref{sec:ccd_poisson_matrix}），
分相 PPE 解法の定式化（§~\ref{sec:split_ppe_framework}），
その基盤技術である Hermite 場延長法（§~\ref{sec:field_extension}），
各相内の高精度ソルバとしての欠陥補正法（§~\ref{sec:defect_correction_main}），
境界条件と安定性（§~\ref{sec:ppe_bc_stability}）を順に示す．
精度の全体像と設計根拠は §~\ref{sec:pressure_accuracy_summary} にまとめる．

\begin{enumerate}
  \item \textbf{§~\ref{sec:ccd_elliptic}（CCD の楕円型ソルバーとしての役割）}：
        微分スキームとしての CCD が楕円型方程式の陰的ソルバーにもなる双対的役割を示す．
  \item \textbf{§~\ref{sec:ccd_poisson_matrix}（変密度対応 CCD-PPE の構築）}：
        PPE を CCD 法で $\Ord{h^6}$ 精度に離散化し，
        DCCD によるチェッカーボード抑制と界面 Balanced-Force 整合を整える．
  \item \textbf{§~\ref{sec:split_ppe_framework}（分相 PPE 解法）}：
        各相で定数密度の PPE を解き，圧力ジャンプと法線フラックス条件を
        HFE で界面近傍の片側データへ反映する
        ——本稿の主要 PPE 戦略．
  \item \textbf{§~\ref{sec:field_extension}（Hermite 場延長法）}：
        分相 PPE 解法の基盤技術として，CCD Hermite データによる $\Ord{h^6}$ 界面越し場延長を定式化する．
  \item \textbf{§~\ref{sec:defect_correction_main}（欠陥補正法）}：
        各相内の等密度ポアソンを「高精度で評価し，低精度で解く」原理により $\Ord{h^6}$ 精度を保ったまま
        低コストで求解する．
  \item \textbf{§~\ref{sec:ppe_bc_stability}（境界条件と数値安定性）}：
        全周 Neumann 境界の零モード対策と高密度比における条件数を整理する．
\end{enumerate}

\subsection{CCD 法の楕円型ソルバーとしての双対的役割}
\label{sec:ccd_elliptic}
%% ═══════════════════════════════════════════════════════════════════════════════

CCD 法は「既知の関数 $f$ の微分値 $f', f''$ を高精度に求めるスキーム」（役割1）に加えて，
\textbf{楕円型方程式の陰的ソルバー}（役割2）としても機能する．

\begin{itemize}
  \item \textbf{役割1（微分演算子）：} 既知の $f$（$\phi$，$\bu$ など）から $\Ord{h^6}$ 精度の
        $f', f'', f_{xy}$ を求める．曲率計算・粘性項評価に使用．
  \item \textbf{役割2（楕円型ソルバー）：} 未知の圧力 $p$ に対して CCD を適用し，
        $\Ord{h^6}$ 精度の演算子 $\mathbf{L}_{\mathrm{CCD}}^{\rho}$ を構成する（§~\ref{sec:ccd_poisson_matrix}）．
\end{itemize}

数学的構造の詳細（射影行列 $\mathbf{E}_1, \mathbf{E}_2$，演算子行列 $\mathbf{L}_{\mathrm{CCD}}^{\rho}$ の定義，
変密度 Poisson 演算子の導出）は付録~\ref{app:ccd_elliptic} を参照されたい．

\subsection{CCD-Poisson 演算子の行列構造と変密度拡張}
\label{sec:ccd_poisson_matrix}
%% ═══════════════════════════════════════════════════════════════════════════════

§~\ref{sec:projection_derivation} で得た PPE（式~\eqref{eq:PPE_varrho}）を，
CCD 法（第~\ref{sec:CCD}章）を用いて $\Ord{h^6}$ 精度で離散化する．
本節では2つの課題を同時に解決する：

\begin{enumerate}
  \item \textbf{$\Ord{h^2} \to \Ord{h^6}$ への精度向上：}
        圧力 $p$，1次導関数 $p'$，2次導関数 $p''$ を連立未知量として同時求解する
        CCD ブロック系（付録~\ref{app:ccd_elliptic}）により，
        速度補正の圧力勾配を $\Ord{h^6}$ 精度で評価する．
  \item \textbf{チェッカーボード（奇偶デカップリング）の抑制：}
        コロケート配置では格子スケール振動が生じる．
        本稿では Rhie--Chow 系のフェイス速度補正（§~\ref{sec:rc_balanced_force}）に加え，
        DCCD 散逸機構（§~\ref{sec:dccd_decoupling}；パラメタ設計は §~\ref{sec:dccd_params}）を用い，
        DCCD フィルタ済み発散（式~\eqref{eq:dccd_ppe_rhs}）を PPE 右辺として構成する．
\end{enumerate}

行列の代数的導出（$\mathbf{L}_{\mathrm{CCD}}^{(x)} = \mathbf{E}_2\,\mathcal{M}_{\mathrm{CCD}}^{-1}\,\mathcal{R}$，
スペクトル半径 $\rho h^2 = 9.6$；§~\ref{sec:verify_ccd_spectral_radius}）は付録~\ref{app:ccd_elliptic}，
Neumann 境界条件の組み込み手順は付録~\ref{sec:ccd_neumann_bc} を参照のこと．

\subsubsection{2次元変密度 CCD-Poisson 演算子の完全定式化}
\label{sec:ccd_2d_full}

2次元変密度 PPE $\bnabla\cdot(1/\rho\,\bnabla p) = q$ の CCD 離散化を示す．
$x$ 方向の演算子（式~\eqref{eq:L_split}）を格子点 $(i,j)$ で評価すると：
\begin{align}
  \bigl(\mathcal{L}_{\mathrm{CCD}}^{\rho,x}\bm{p}\bigr)_{i,j}
  &= \frac{1}{\rho_{i,j}}\bigl(D_x^{(2)} p\bigr)_{i,j}
  - \frac{(D_x^{(1)}\rho)_{i,j}}{\rho_{i,j}^2}\bigl(D_x^{(1)} p\bigr)_{i,j}
  \label{eq:Lx_varrho_discrete}
\end{align}
$y$ 方向も同様に：
\begin{align}
  \bigl(\mathcal{L}_{\mathrm{CCD}}^{\rho,y}\bm{p}\bigr)_{i,j}
  &= \frac{1}{\rho_{i,j}}\bigl(D_y^{(2)} p\bigr)_{i,j}
  - \frac{(D_y^{(1)}\rho)_{i,j}}{\rho_{i,j}^2}\bigl(D_y^{(1)} p\bigr)_{i,j}
  \label{eq:Ly_varrho_discrete}
\end{align}

合計演算子：
\begin{equation}
  \bigl(\mathcal{L}_{\mathrm{CCD}}^{\rho}\bm{p}\bigr)_{i,j}
  = \bigl(\mathcal{L}_{\mathrm{CCD}}^{\rho,x}\bm{p}\bigr)_{i,j}
  + \bigl(\mathcal{L}_{\mathrm{CCD}}^{\rho,y}\bm{p}\bigr)_{i,j}
  \label{eq:L_CCD_2d_full}
\end{equation}

Matrix-Free 評価手順（各反復で Thomas 法を $x$・$y$ 方向に交互適用，$\Ord{N_x N_y}$）は
付録~\ref{sec:ccd_matrix_free_eval} を参照のこと．

\subsubsection{CSF + Balanced-Force + CCD 勾配による寄生流れの抑制}
\label{sec:ccd_balanced_force}

§~\ref{sec:balanced_force} で示した Balanced-Force 条件の要請に従い，
CCD-PPE 実装で同一演算子を適用する方法を示す
（CSF 近似誤差 $\Ord{\varepsilon^2} \approx \Ord{h^2}$ の物理的下限については付録~\ref{app:csf_balanced_force} 参照）．

本稿では CSF 体積力（§~\ref{sec:csf}）を予測子ステップに維持し，Balanced-Force 条件（§~\ref{sec:balanced_force}）
により圧力勾配と表面張力の離散化を整合させる：
CCD 演算子 $D^{(1)}$ による
圧力勾配と表面張力体積力 $(\kappa/We)D^{(1)}\psi$ に\textbf{同一演算子}を使用する
（式~\eqref{eq:balanced_pressure}，\eqref{eq:balanced_surface}）．
各相内の PPE は欠陥補正法（DC 法，§~\ref{sec:defect_correction_main}）で解く：
低次演算子 $L_L$（2 次精度 FD）で補正方程式を厳密に求解しつつ，
収束先の精度は高精度演算子 $L_H$（CCD $\Ord{h^6}$）が保証する（§~\ref{sec:dc_accuracy}）．
速度補正の勾配 $\bnabla p$ には CCD $D^{(1)}$ を適用する（§~\ref{sec:ppe_discretization_choice}）．
これにより：
\begin{itemize}
  \item $\bnabla p$ と $\kappa\bnabla\psi$ の CCD 演算子が同一であるため，
        Balanced-Force 条件が $\Ord{h^6}$ 精度で成立する．
  \item PPE 右辺は DCCD フィルタ済み発散（式~\eqref{eq:dccd_ppe_rhs}）のみから構成される
        （Rhie--Chow Balanced-Force 補正はPredictor 段で適用；§~\ref{sec:rc_balanced_force}）．
  \item 射影で制御される発散は，PPE 右辺に用いた DCCD フィルタ済み発散である．
        非フィルタ発散 $D_\mathrm{CCD}\cdot\bu^{n+1}$ には格子スケール残差が残りうるため，
        これを完全な非フィルタ発散自由性とは区別する（§~\ref{sec:dccd_decoupling}）．
  \item 投影不整合（FD PPE と CCD 勾配の不整合；非フィルタ発散残差）は $\Ord{h^2}$ であり，
        全体精度の律速因子（CSF誤差 $\Ord{h^2}$）と同次のため実用上問題なく，
        DC 反復を収束させない単回補正運用ではこの不整合が残るため，
        本稿では §~\ref{sec:pressure_accuracy_summary} の精度目標に応じて $k=3$ を標準とする．
\end{itemize}
