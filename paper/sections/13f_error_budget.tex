\paragraph{誤差源スタック．}
\begin{enumerate}\setlength{\itemsep}{0pt}
  \item \textbf{空間離散誤差}．CCD 6 次（V2 で確証）．Kovasznay slope $6.00$，TGV では時間誤差床下．
  \item \textbf{時間離散誤差}．単相 NS は AB2/IMEX-BDF2 で $O(\Delta t^2)$（V1 slope $2.00$）．
        二相結合では界面 reinit + 曲率 lag で実効 $O(\Delta t^{0.56})$（V7; \autoref{sec:imex_bdf2_twophase_time}）．
  \item \textbf{界面表現誤差}．Heaviside--$\delta$ は固定 $\varepsilon$ 推奨（V9）；
        local $\varepsilon$ は寄生流れ 3 桁増加．
  \item \textbf{曲率推定誤差}．CSF $\kappa$ + CCD 勾配で寄生流れ V5: CCD/FD 比 $1/3$--$1/6$ 抑制．
  \item \textbf{CLS reinit 誤差}．非一様格子 $\alpha=2$ で 1 周期積分後 volume drift $+12\%$（V10）．
\end{enumerate}

\paragraph{合格判定要約．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \checkmark \textbf{合格}: V1-b (時間 2 次)，V2-a (空間 6 次)，
        V3 (BF 平衡)，V5 (CCD 寄生流れ抑制)，V6 (密度比ロバスト)，
        V8 (非一様静的), V10 ($\alpha=1$).
  \item $\triangle$ \textbf{条件付き合格}: V4-a (Galilean: 固定 Eulerian + 参照点固定 PPE で
        $O(\|\bm U\|)$ 残差), V4-b (RT: 線形抽出 spec 外設計).
  \item $\times$ \textbf{要改善 / 推奨せず / 制約条件}:
        V7 (二相時間 slope $0.56$ → 設計 $O(\Delta t^2)$ 未達; reinit 頻度差 + $\kappa$ explicit lag が律速),
        V9 (local $\varepsilon$ → 寄生流れ激増; fixed $\varepsilon$ が安全 default),
        V10 ($\alpha=2, N=128$ → volume drift $+12.17\%$; CLS-advection 非一様格子は $\alpha\!=\!1$ 推奨).
\end{itemize}

\paragraph{設計指針 (PR-5 Algorithm Fidelity)．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 単相 NS：CCD 6 次空間 + AB2/IMEX-BDF2 2 次時間が paper-exact 設計通り．
  \item 二相 NS：分相 PPE + DC + HFE 構成が密度比 $\le 833$ でロバスト（V6）．
        ただし時間方向の実効次数は界面処理に律速されるため，
        \textbf{$\Delta t$ を $h_{\min}$ より小さく取る安全ファクタ}（CFL $\le 0.4$）が推奨．
  \item 非一様格子：$\alpha = 1$ 推奨．$\alpha > 1$ 設定は CLS reinit と局所 $\varepsilon$ の
        相互作用に注意．
  \item 寄生流れ抑制：CCD 演算子は FD 比 $3$--$6$ 倍抑制（V5）；
        CSF + 高次圧力勾配の組み合わせが効果的．
\end{itemize}
\paragraph{A3 トレース統合．}
本節は V1--V10 全結果の横断統合点．eq → discretization → code の連鎖を以下の
親 → sub 接続で明示する（個別の式番号と実装パスは各 sub-section 末尾を参照）：
\begin{itemize}\setlength{\itemsep}{0pt}
  \item V1 (\autoref{sec:single_phase_ns}) — Navier--Stokes \eqref{eq:NS_onefluid}
        → AB2 / IMEX-BDF2（\autoref{ch:time_integration}） →
        \texttt{src/twophase/integrator/\{ab2,imex\_bdf2\}.py}．
  \item V2 (\autoref{sec:single_phase_ns}, \autoref{sec:kovasznay}) —
        CCD 6 次空間（\autoref{sec:ccd_def} 式~\eqref{eq:CCD_I}） →
        \texttt{src/twophase/ccd/ccd\_solver.py}．
  \item V3, V5 (\autoref{sec:bf_static_droplet}, \autoref{sec:spurious_multistep})
        — BF 力学的バランス \eqref{eq:bf_fccd_unification} + CSF
        \eqref{eq:csf_final} → \texttt{src/twophase/csf/curvature.py},
        \texttt{src/twophase/ccd/ccd\_solver.py}．
  \item V4 (\autoref{sec:galilean_invariance}, \autoref{sec:rt_linear_growth}) —
        CCD 移流項対称性 + PPE 参照点ゲージ → \texttt{src/twophase/poisson/ppe.py}．
  \item V6, V7 (\autoref{sec:density_ratio_sweep},
        \autoref{sec:imex_bdf2_twophase_time}) — 分相 PPE \eqref{eq:split_ppe} +
        DC + HFE → \texttt{src/twophase/poisson/\{split\_ppe,density\_correction\}.py},
        \texttt{src/twophase/integrator/imex\_bdf2.py},
        \texttt{src/twophase/cls/reinit.py}（律速）．
  \item V8, V9, V10 (\autoref{sec:nonuniform_grid_ns}) — 非一様 NS + local
        $\varepsilon$ + CLS-advection → \texttt{src/twophase/grid/nonuniform.py},
        \texttt{src/twophase/cls/\{advection,reinit\}.py}．
\end{itemize}

唯一の synthesis であり，他章 (\autoref{sec:conclusion}) からの inbound 参照は
本表をエンドポイントとする．
