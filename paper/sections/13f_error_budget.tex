\subsection{精度バジェット：V1--V10 誤差寄与の統合評価}
\label{sec:error_budget}
\label{sec:accuracy_summary}

\paragraph{構成．}
本節は V1--V10 横断の精度結果を一表に集約し，
\textbf{各誤差源} (空間離散，時間離散，界面表現，曲率推定，CLS 再初期化) と
\textbf{各設定パラメータ} ($\rho_r$, $\alpha$, $\Delta t$, $h$) の関係を整理する
（観測収束次数の Richardson 外挿運用は~\cite{Roy2003}）．
過去の表の値は用いず，V1--V10 の再生成結果に基づいて整理する．

\paragraph{判定基準の再定義（§\ref{sec:verification} Type 分類との整合）．}
本節は §\ref{sec:verification}「判定区分と Type 分類」に基づき
Type-A/Type-B/Type-D の三類型で再定義された各 V test の判定を集約する
（凡例は表~\ref{tab:V_verdict_types}）．
V1-a, V1-b, V2, V4, V5 は \textbf{Type-A revised criterion}：
当初の理想化基準（CCD 内点 6 次空間，AB2 単体 2 次，厳密 Galilean ゼロ残差，CCD/FD 比）ではなく
実装縮約実験の identity に対する文献根拠付き revised criterion へ再定義した．
V1-a は単相 TGV 空間の FVM PPE 結合 stack に対する $O(h^2)$ trend，
V1-b は非増分 standard PPE 射影の $O(\Delta t)$ trend，
V2 は wall one-sided Kovasznay 統合残差の $O(h^4)$ trend，
V4 は固定壁 offset diagnostic の $O(\|\bm U\|)$ bounded residual，
V5 は CCD 寄生流れ絶対値 $u_\infty^{\mathrm{end}}$ の $\sigma/\mathrm{Re}_c$ スケール基準
を合格基準とする．
V10-a 質量保存軸 / V10-b 質量保存軸は \textbf{Type-B}：
Olsson--Kreiss mass correction~\cite{OlssonKreiss2005} を 10 step ごとに適用する
構造的アルゴリズム修正による合格化．
V7（Lie 1 次分割漸近 slope $\sim 1$ と整合），V10-a 形状軸（slot/h$=6.4$ under-resolution），
V10-b 形状軸（filament 厚 $<h$ Eulerian 固定格子）は \textbf{Type-D}：
連続方程式・離散構造に由来する theoretical hard limit と整合するものとして
条件付き合格 ($\triangle$)．これらは格子増強だけで合格化するのではなく，
仮説除去型の追加検証（norm 切替，項別誤差分解，dt/h 独立 sweep，時系列解析）を
future verification gate として扱う．
V9 は Type-A/B/D の再定義ではなく，§14 stack と同時に nominal/local
$\varepsilon$ mode を切り替えても短時間静止液滴を破綻させないことを測る
\textbf{§14 stack 条件付き診断}として別枠で扱う．

\paragraph{誤差源スタック．}
\begin{enumerate}\setlength{\itemsep}{0pt}
  \item \textbf{空間離散誤差}．CCD 内点 6 次設計に対し，V2 Kovasznay residual は
        wall one-sided 二階微分境界閉包と global compact coupling により実効 $3.95$．
        revised criterion $O(h^4)$ を満たす．TGV では時間誤差床下に到達するため空間 slope は飽和．
  \item \textbf{時間離散誤差}．単相 NS の AB2 predictor は単体 $O(\Delta t^2)$ 設計だが，
        V1-b script は standard projection 結合を測るため，合格基準は $O(\Delta t)$ trend である．
        二相結合では界面輸送・再初期化・pressure-jump PPE の Lie 分割で coupled-stack 実効
        slope $1.04$（V7; \autoref{sec:imex_bdf2_twophase_time}）となり，
        純粋 BDF2 2 次ではなく splitting 下限と整合する．
  \item \textbf{界面表現誤差}．Heaviside--$\delta$ の nominal/local $\varepsilon$ switch は
        FCCD/HFE/pressure-jump PPE と同時に使う条件で短時間安定（V9）；
        ただし現行 direct-$\psi$/HFE probe では B/C が同一で，local mode を改善 lever とは主張しない．
        旧 CCD/CSF 経路の結果は §14 stack の根拠から除外する．
  \item \textbf{曲率推定誤差}．V5 では CSF $\kappa_{lg}$ + CCD 勾配の
        $u_\infty^{\mathrm{end}}$ 絶対値が全表ケースで $2\times10^{-2}$ 以下に保たれ，
        参照として FD 値を全条件で下回る．
        粗格子・低密度比ほど CCD/FD の差は大きい．
  \item \textbf{CLS 形状追跡誤差（質量保存軸 vs 形状精度軸の分離）}．V10 の Zalesak は
        $\alpha=1$ の一様格子診断に固定する．Olsson--Kreiss mass correction~\cite{OlssonKreiss2005}
        を 10 step ごとに適用した結果，$N=128$ で mass drift $4.70\times 10^{-4}\%$（合格基準
        $0.5\%$ を 3 桁下回り \checkmark）に到達する一方，centroid err $4.911\times 10^{-3}$,
        shape $L^1$ $8.307\times 10^{-3}$, centroid order $\approx 1.22$ は slot/h $=6.4$ の
        sharp-corner under-resolution に律速される．単一渦 (V10-b) も同質量修正下で
        mass drift $0.0428\%$ \checkmark / $L^1$ reversal $2.248\times10^{-2}$ ($T=8$ filament
        厚 $<h$ の Eulerian 固定一様格子における理論的下限~\cite{EnrightFedkiwFerzigerMitchell2002}) と
        同じく\textbf{保存軸 \checkmark / 形状軸 $\triangle$} の二軸構造を取る．
  \item \textbf{判定軸の分離}．V7/V10 系は単一の設計次数だけで判定せず，
        norm 依存性，参照解品質，operator splitting，CFL coupling，非可逆な reinitialization/mass correction を
        分離した追加検証を必要とする．
\end{enumerate}

\paragraph{合格判定要約（§\ref{sec:verification} Type 分類で集約）．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \checkmark \textbf{合格（design criterion）}:
        V3 (BF 平衡，$|\Delta p_{\mathrm{rel}}|=0.97\%$ at $N=128$)，
        V6 (密度比ロバスト, $\rho_r=833$ までブローアップなし; $|\Delta V_\psi|/V_{\psi,0} \le 6.4\!\times\!10^{-16}$)，
        V8 (非一様静止, $\alpha=2,N=96$ で $|\Delta p_{\mathrm{rel}}|=2.86\%$).
  \item \checkmark \textbf{Type-A revised criterion 合格}:
        V1-a (TGV 空間 FVM PPE $O(h^2)$ trend, finest local slope $2.00$；補助: $E_{\mathrm{rel}}=1.583\!\times\!10^{-7}$ at $N=64$)，
        V1-b (standard projection 結合 $O(\Delta t)$ trend, slope $1.00$)，
        V2 (Kovasznay wall residual $O(h^4)$ trend, slope $3.95$)，
        V4 (fixed-wall offset $O(\|\bm U\|)$ bounded residual, $\|\Delta u\|_\infty^{\max}/\|\bm U\| = 2.63$)，
        V5 (CCD 寄生流れ絶対値 $1.93\!\times\!10^{-2}$ at $\rho_r=1, N=32$).
  \item \checkmark \textbf{Type-B 構造修正 合格}:
        V10-a 質量保存軸 (Olsson--Kreiss mass correction; $N=128$ mass drift $4.70\!\times\!10^{-4}\%$)，
        V10-b 質量保存軸 (同; $0.0428\%$).
  \item $\triangle$ \textbf{Type-D 理論的下限との整合}:
        V7 (二相 coupled-stack 実効 slope $1.04$ — TVD--RK3 / IMEX-BDF2 の Lie 1 次分割
        漸近 slope $\sim 1$~\cite{Strang1968,HoldenKarlsenLieRisebro2010} と整合),
        V10-a 形状軸 ($\alpha=1$ Zalesak centroid err $4.911\times 10^{-3}$,
        shape $L^1$ $8.307\times 10^{-3}$ — slot/h $=6.4$ sharp-corner under-resolution 律速),
        V10-b 形状軸 ($T=8$ 単一渦 reversal $L^1=2.248\times10^{-2}$
        — filament 厚 $<h$ の Eulerian 固定格子原理的下限~\cite{EnrightFedkiwFerzigerMitchell2002}).
  \item $\times$ \textbf{要改善 / 推奨せず}: 該当なし（V7 は §14 stack 化により $\times$ 区分から $\triangle$ Type-D 区分へ移動済）.
  \item \checkmark \textbf{§14 stack 条件付き診断}: V9
        (nominal/local $\varepsilon$ switch は $N=32$ で B/C=$1.00$，C/A=$2.36$；
        HFE curvature + pressure-jump PPE と同時採用しても短時間静止液滴を破綻させないが，
        local mode の改善効果はこの縮約 probe では主張しない).
\end{itemize}

\paragraph{設計指針 (PR-5 Algorithm Fidelity)．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 単相 NS：CCD 内点 6 次空間 + AB2 predictor 2 次時間設計は component level で維持する．
        統合縮約実験では，Kovasznay wall residual は $O(h^4)$，
        standard PPE TGV time trend は $O(\Delta t)$ として判定する．
  \item 二相 NS：分相 PPE + DC + HFE 構成が密度比 $\le 833$ でロバスト（V6）．
        ただし時間方向の実効次数は界面処理に律速されるため，
        \textbf{$\Delta t$ を $h_{\min}$ より小さく取る安全ファクタ}（CFL $\le 0.4$）が推奨．
  \item 非一様格子：$\alpha>1$ の NS 静止問題と nominal/local $\varepsilon$ switch は V8/V9 で検証し，
        Zalesak slot と単一渦 reversal は $\alpha=1$ の一様格子 CLS 移流診断に固定する．
        local mode を改善根拠として採用するには，移動界面または reinitialization が local thickness を
        実際に経由する追加 gate を要求する．
  \item 寄生流れ抑制：CCD の $u_\infty^{\mathrm{end}}$ 絶対値は V5 全表ケースで
        $O(10^{-2})$ 以下に保たれ，代表的には
        $\rho_r=1,N=32$ で $1.93\!\times\!10^{-2}$，$\rho_r=1,N=128$ で $6.28\!\times\!10^{-4}$．
        参照として FD 値を全条件で下回り（$\rho_r=1,N=32$ で $1/5.44$，
        $\rho_r=100,N=64$ で $1/2.5$），高解像度・高密度比では CCD/FD 差が縮小する．
\end{itemize}

\paragraph{検証対応の統合．}
本節は V1--V10 全結果の横断統合点である．各実験が測定した式・離散化・物理指標の対応を
以下に集約する：
\begin{itemize}\setlength{\itemsep}{0pt}
  \item V1 (\autoref{sec:single_phase_ns}) — Navier--Stokes \eqref{eq:NS_onefluid}
        と AB2 + standard PPE 縮約経路に対するエネルギー減衰と非圧縮残差．
  \item V2 (\autoref{sec:single_phase_ns}, \autoref{sec:kovasznay}) —
        CCD 内点 6 次演算子と wall one-sided 境界閉包を含む Kovasznay 統合残差．
  \item V3, V5 (\autoref{sec:bf_static_droplet}, \autoref{sec:spurious_multistep})
        — BF 力学的バランス \eqref{eq:bf_fccd_unification} + CSF
        \eqref{eq:csf_final} に対する Laplace 圧維持と寄生流れ．
  \item V4 (\autoref{sec:galilean_offset}) —
        固定壁系の Galilean offset residual に対する $O(\|\bm U\|)$ boundedness．
  \item V6, V7 (\autoref{sec:density_ratio_sweep},
        \autoref{sec:imex_bdf2_twophase_time}) — 分相 PPE \eqref{eq:split_ppe} +
        DC + HFE に対する密度比安定性と二相時間精度．
  \item V8 (\autoref{sec:nonuniform_ns_static}) — 非一様 NS に対する静止液滴
        $\Delta p_{\mathrm{rel}}$ 維持．
  \item V9 (\autoref{sec:local_eps_validation}) — nominal/local $\varepsilon$ switch に対する
        寄生流れ，体積分率 drift，および B/C 同一性．
  \item V10-a (\autoref{sec:zalesak_cls_advection}) — Zalesak CLS 移流に対する
        mass drift と centroid 追跡精度（質量保存軸 / 形状軸の二軸構造）．
  \item V10-b (\autoref{sec:single_vortex_reversal}) — 単一渦 deformation
        reversal に対する $L^1$ 誤差と mass drift（同二軸構造）．
\end{itemize}
本節 \autoref{tab:v_accuracy_summary} はこの全対応の精度結果を一表に集約する
唯一の総合表であり，他章 (\autoref{sec:conclusion}) からの参照は
本表をエンドポイントとする．
