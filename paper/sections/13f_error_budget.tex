\label{sec:error_budget}

\paragraph{構成．}
本節は V1--V10 横断の精度結果を一表に集約し，
\textbf{各誤差源} (空間離散，時間離散，界面表現，曲率推定，CLS 再初期化) と
\textbf{各設定パラメータ} ($\rho_r$, $\alpha$, $\Delta t$, $h$) の関係を整理する
（観測収束次数の Richardson 外挿運用は~\cite{Roy2003}）．
過去の表の値は用いず，V1--V10 の再生成結果に基づいて整理する．

\paragraph{誤差源スタック．}
\begin{enumerate}\setlength{\itemsep}{0pt}
  \item \textbf{空間離散誤差}．CCD 6 次設計に対し，V2 Kovasznay slope は実効 $3.95$（圧力境界・界面外周の低次寄与）．TGV では時間誤差床下に到達するため空間 slope は飽和．
  \item \textbf{時間離散誤差}．単相 NS の AB2/IMEX-BDF2 は単体 $O(\Delta t^2)$ 設計だが，
        PPE 射影との fractional-step 結合で V1-b 時間 slope は $1.00$ に縮退する．
        二相結合では界面再初期化 + 曲率の陽的遅れ評価で coupled-stack 実効 $O(\Delta t^{1.58})$
        （V7; \autoref{sec:imex_bdf2_twophase_time}）まで回復するが，純粋 BDF2 2 次には未達．
  \item \textbf{界面表現誤差}．Heaviside--$\delta$ の local $\varepsilon$ は
        FCCD/HFE/pressure-jump PPE と同時に使う条件で有効（V9）；
        旧 CCD/CSF 経路の結果は §14 stack の根拠から除外する．
  \item \textbf{曲率推定誤差}．V5 では CSF $\kappa$ + CCD 勾配が全表ケースで FD 以下となり，
        粗格子・低密度比ほど寄生流れの抑制が大きい．
  \item \textbf{CLS 形状追跡誤差}．FCCD/TVD--RK3 では非一様格子 $\alpha=2$ の Zalesak volume drift は
        $0.030\%$ まで収まる一方，centroid order は $0.34$ に留まる．
        $\alpha=1$ では同一 $N=128$ で volume drift $1.535\%$ で，slot 解像度に律速される（V10）．
        単一渦では Ridge--Eikonal なしの質量補正のみで mass drift は $0.01431\%$ まで回復するが，
        $L^1$ reversal error は $5.150\times10^{-2}$ であり，形状復元誤差は質量保存とは別に残る．
\end{enumerate}

\paragraph{合格判定要約．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \checkmark \textbf{合格}: V3 (BF 平衡)，V5 (CCD 寄生流れ抑制)，
        V6 (密度比ロバスト, $\rho_r=833$ までブローアップなし)，
        V8 (非一様静止), V10-b ($\alpha=2$ Zalesak: volume drift $0.030\%$).
  \item $\triangle$ \textbf{条件付き合格}:
        V1-b (単相時間 slope $1.00$ — fractional-step 結合の設計制約; AB2/IMEX-BDF2 単体は 2 次設計),
        V2-a (Kovasznay 空間 slope $3.95$ — 設計 6 次に対し実効 4 次),
        V4 (固定 Eulerian + 参照点固定 PPE で $O(\|\bm U\|)$ 残差),
        V7 (二相 coupled-stack 実効 slope $1.58$ — 純粋 BDF2 2 次には未達だが旧 reduced proxy slope $0.56$ より大幅改善),
        V10-a ($\alpha=1$ Zalesak: volume drift $1.535\%$, centroid err $4.69\times 10^{-3}$),
        V10-c ($T=8$ 単一渦 reversal: Ridge--Eikonal なしの質量補正のみで
        mass drift $0.01431\%$, $L^1$ reversal $5.150\times10^{-2}$).
  \item $\times$ \textbf{要改善 / 推奨せず}: 該当なし（V7 は §14 stack 化により $\times$ 区分から $\triangle$ 区分へ移動済）.
  \item \checkmark \textbf{§14 stack 条件付き合格}: V9
        (local $\varepsilon$ は $N=32$ で C/A=$0.11$，C/B=$0.15$；
        HFE curvature + pressure-jump PPE と同時採用する).
\end{itemize}

\paragraph{設計指針 (PR-5 Algorithm Fidelity)．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 単相 NS：CCD 6 次空間 (Kovasznay 実効 slope $3.95$，TGV は時間誤差床下) + AB2/IMEX-BDF2 2 次時間設計．
        ただし V1-b の通り PPE 射影との fractional-step 結合では時間 slope は $1.00$ に縮退するため，
        単相設計 2 次性能は予測子単体に限定される．
  \item 二相 NS：分相 PPE + DC + HFE 構成が密度比 $\le 833$ でロバスト（V6）．
        ただし時間方向の実効次数は界面処理に律速されるため，
        \textbf{$\Delta t$ を $h_{\min}$ より小さく取る安全ファクタ}（CFL $\le 0.4$）が推奨．
  \item 非一様格子：$\alpha > 1$ でも FCCD/TVD--RK3 の体積保存は成立するが，
        Zalesak slot のような薄い幾何特徴では重心追跡が低次化する．
        local $\varepsilon$ は §14 stack と同時に採用し，2D snapshot 診断で hot spot を確認する．
  \item 寄生流れ抑制：CCD は V5 の全表ケースで FD 以下であり，
        代表的には $\rho_r=1,N=32$ で FD の $1/5.9$，
        $\rho_r=100,N=64$ で $1/2.5$．高解像度・高密度比では差が縮小する．
\end{itemize}
\paragraph{検証対応の統合．}
本節は V1--V10 全結果の横断統合点である．各実験が測定した式・離散化・物理指標の対応を
以下に集約する：
\begin{itemize}\setlength{\itemsep}{0pt}
  \item V1 (\autoref{sec:single_phase_ns}) — Navier--Stokes \eqref{eq:NS_onefluid}
        と AB2 / IMEX-BDF2（\autoref{ch:time_integration}）に対するエネルギー減衰と非圧縮残差．
  \item V2 (\autoref{sec:single_phase_ns}, \autoref{sec:kovasznay}) —
        CCD 6 次空間（\autoref{sec:ccd_def} 式~\eqref{eq:CCD_I}）に対する Kovasznay 残差．
  \item V3, V5 (\autoref{sec:bf_static_droplet}, \autoref{sec:spurious_multistep})
        — BF 力学的バランス \eqref{eq:bf_fccd_unification} + CSF
        \eqref{eq:csf_final} に対する Laplace 圧維持と寄生流れ．
  \item V4 (\autoref{sec:galilean_offset}) —
        固定壁系の Galilean offset 残差に対する参照系シフト後の速度場差．
  \item V6, V7 (\autoref{sec:density_ratio_sweep},
        \autoref{sec:imex_bdf2_twophase_time}) — 分相 PPE \eqref{eq:split_ppe} +
        DC + HFE に対する密度比安定性と二相時間精度．
  \item V8, V9, V10 (\autoref{sec:nonuniform_grid_ns}) — 非一様 NS + local
        $\varepsilon$ + CLS 移流に対する静止液滴，局所 $\varepsilon$ 比較，体積保存性．
\end{itemize}
本節 \autoref{tab:v_accuracy_summary} はこの全対応の精度結果を一表に集約する
唯一の総合表であり，他章 (\autoref{sec:conclusion}) からの参照は
本表をエンドポイントとする．
