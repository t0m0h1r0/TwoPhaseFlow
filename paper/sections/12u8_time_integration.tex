\subsubsection{U8：時間積分スイート（TVD--RK3 / AB2 / CN spectral / Layer A,B,C）[Tier V]}
\label{sec:U8_time_integration}

\paragraph{検証対象．}
第~\ref{sec:time_int}章 で導入した時間積分の全要素を単体検証する：
TVD--RK3~\cite{ShuOsher1988,JiangShu1996}（§\ref{sec:time_int}：3 次 SSP），
AB2（§\ref{sec:time_int}：2 次 multistep），
そして粘性項 Crank--Nicolson の \emph{周期 BC 用} 単体ベンチマークとして
\textbf{FFT 対角化スペクトル CN}~\cite{PeacemanRachford1955}．
本ベンチマークは固有値ベースの amplification factor
$F(K^2) = (1 - \tfrac{1}{2}\nu \mathrm{d}t K^2)/(1 + \tfrac{1}{2}\nu \mathrm{d}t K^2)$ を
直接適用するため ADI 分割誤差を含まず，CN の理論次数 $O(\mathrm{d}t^2)$ と
A--安定性をクリーンに測る．第~\ref{sec:time_int}章で採用する粘性時間発展は
implicit-BDF2 + DC であり，CN spectral は拡散安定性を測る補助基準である．
A/B/C 3 層粘性切替（§\ref{sec:viscous_3layer}：均一 / 中濃度差 / 強濃度差）も併せて検証する．

\paragraph{設定．}
\begin{description}\setlength{\itemsep}{0pt}
  \item[U8-a] TVD--RK3 単体．スカラー ODE $\dot{u} = -\lambda u, \lambda=1, T=1$．
        $n_t \in \{4,8,\dots,512\}$．設計次数 3．
        production の \texttt{twophase.time\_integration.tvd\_rk3} を直接駆動．
  \item[U8-b] AB2 単体．同 ODE．$n_t \in \{16,\dots,512\}$．設計次数 2．
        本テストは \emph{ODE ベンチマーク}（インライン textbook AB2 を用いる）であり，
        production の \texttt{Predictor} 経路（NS 連成・FlowState/ConvectionTerm/...
        を要求）はスカラー ODE には適用不可．production と同一の AB2 公式
        $u_{n+1} = u_n + \mathrm{d}t (\tfrac{3}{2} f_n - \tfrac{1}{2} f_{n-1})$
        を用いるため，設計次数 2 を露出する目的に対しては等価．
  \item[U8-c] CN spectral 拡散方程式．$\nu = 0.01$，1D 拡散方程式，
        $n_t \in \{10,\dots,320\}$．設計次数 2．
        \textbf{安定性}：CFL$_\nu \in \{0.5,1,5,10\}$ で CN は安定．
        FE は CFL$_\nu \le 1.0$ で発散（高波数モードが励起される領域）；
        CFL$_\nu \ge 5$ では smooth IC の低周波成分のみが進化するため
        見かけ上 growth $\sim 0.62$ に収まる（高周波成分の blowup は潜在）．
  \item[U8-d] A/B/C 3 層 CN diagonal + EE-explicit cross（界面 viscosity ratio test, 1D MMS）．
        $\mu_{\mathrm{ratio}} \in \{10, 100\}$，layer $\in \{$A, B, C$\}$，
        $n_t \in \{16,\dots,256\}$．
        $L_{\mathrm{full}} = \partial_x(\mu \partial_x \cdot)$ を
        $L_{\mathrm{diag}}$（定数 $\langle\mu\rangle \partial_x^2$，CN--implicit）と
        $L_{\mathrm{cross}} = L_{\mathrm{full}} - L_{\mathrm{diag}}$（EE--explicit）に分割．
        Layer A=均一 ($L_{\mathrm{cross}}\!=\!0$) / B=粘性勾配 (band $1.5h$) / C=HFE 平滑化 (band $5h$)．
        explicit cross 項は EE 安定窓 (本実装 $N=64,\,\mu_l/\mu_g\!\le\!100$) を保つため
        スケーリング係数 $\eta_{\mathrm{cross}} = 10^{-3}$ で減衰させる
        (式の有効 PDE: $u_t = L_{\mathrm{diag}}(u) + \eta_{\mathrm{cross}} L_{\mathrm{cross}}(u) + s$)；
        LTE の \emph{slope} はスケーリング不変で，絶対値のみ縮小する．
\end{description}

\paragraph{判定と次章への接続．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item U8-a TVD--RK3：slope $3.01$，設計 3 次完全達成．\checkmark
  \item U8-b AB2：slope $2.04$，設計 2 次完全達成．\checkmark
        $0.04$ の上振れは FE start--up step
        （AB2 の 1 step 目の Forward--Euler 起動）に由来する $\Ord{\mathrm{d}t^2}$
        ではなく $\Ord{\mathrm{d}t^3}$ の transient であり，
        $n_t$ を増やすほど該当寄与が相対的に小さくなって観測 slope が
        設計 2 次に上から漸近する典型的挙動．許容帯
        $|p_{\mathrm{measured}} - p_{\mathrm{design}}| \le 0.5$ の十分内側．
  \item U8-c CN spectral：slope $2.00$，FE 比 安定性 $\approx 21$ 桁優位
        ($\log_{10}(1.73\!\times\!10^{21}/0.6408) \approx 21.4$；CFL$_\nu=0.5$ にて)．
        §\ref{sec:time_operator_stiffness} の A--stable 分類と整合．\checkmark
  \item U8-d Layer A/B/C：A 層は完全 implicit CN ($L_{\mathrm{cross}}\!=\!0$) で完全 2 次．
        B/C 層は EE--explicit cross 項
        ($\eta_{\mathrm{cross}}\!=\!10^{-3}$) の LTE $\Ord{\mathrm{d}t}$ で実効 1 次に劣化．
        これは CN--implicit/EE--explicit 演算子分割の典型挙動であり
        §\ref{sec:viscous_3layer} の予測通り．$\triangle$（既知の劣化を定量化；
        絶対値は $\eta_{\mathrm{cross}}$ で抑制し EE 安定窓を確保）．
\end{itemize}
検証対応：
式~\eqref{eq:tvd_rk3}（SSP 形），AB2 multistep 公式，
CN spectral amplification factor に対し，ODE/MMS 収束と粘性安定性を測定する．
