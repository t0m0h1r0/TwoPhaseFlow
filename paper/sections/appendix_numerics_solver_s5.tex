\subsection{収束理論：反復行列のスペクトル解析}
\label{app:dc_convergence_theory}

本付録は §\ref{sec:defect_correction_main} で定義した欠陥補正法（式~\eqref{eq:dc_three_step}）の
収束性を反復行列のスペクトル解析に基づいて理論的に保証する．

\subsubsection*{反復行列のスペクトル半径}

欠陥補正法の更新式（式~\eqref{eq:dc_three_step}）を誤差 $e^{(k)} \equiv p^{(k)} - p^*$
（$p^* = L_H^{-1} b$ は真の解）について整理すると：
\begin{equation}
  e^{(k+1)} = \underbrace{\bigl(I - \omega\,L_L^{-1}\,L_H\bigr)}_{\displaystyle \equiv M}\,e^{(k)}
  \label{eq:dc_iteration_matrix}
\end{equation}
反復の収束条件はスペクトル半径 $\rho(M) < 1$ であり，
$k$ 回反復後の誤差は $\|e^{(k)}\| \leq \rho(M)^k\,\|e^{(0)}\|$ で上界される．

\subsubsection*{\texorpdfstring{$M$ の固有値構造}{M の固有値構造}}

$L_L$ と $L_H$ が同一の固有ベクトル系 $\{v_j\}$ を共有すると仮定する
（等方一様格子で近似的に成立）．
$L_L\,v_j = \mu_j\,v_j$，$L_H\,v_j = \lambda_j\,v_j$ とすると
$M$ の固有値は：
\begin{equation}
  \sigma_j = 1 - \omega\,\frac{\lambda_j}{\mu_j}
  \label{eq:dc_eigenvalue}
\end{equation}
ここで $\lambda_j / \mu_j$ は CCD 演算子と FD 演算子の固有値比であり，
低周波モード（$\xi \to 0$）では $\lambda_j / \mu_j \to 1$（両者とも $\Ord{\xi^2}$ で一致），
高周波モード（$\xi \to \pi$）では $\lambda_j / \mu_j \neq 1$ となる（高次精度項の差異）．

\subsubsection*{固有値比の解析}

$L_H$ の修正波数は式~\eqref{eq:lambda_fourier} より
$\lambda_j = \Lambda_H(\xi_j)/h^2$（$\Lambda_H(\xi) = 24\sin^2(\xi/2)/(3/2 + \sin^2(\xi/2))$），
$L_L$ の修正波数は標準2次精度中心差分 $\Lambda_L(\xi) = 4\sin^2(\xi/2)$ であるから：
\begin{equation}
  \frac{\lambda_j}{\mu_j}
  = \frac{\Lambda_H(\xi_j)}{\Lambda_L(\xi_j)}
  = \frac{24\sin^2(\xi_j/2)/(3/2 + \sin^2(\xi_j/2))}{4\sin^2(\xi_j/2)}
  = \frac{6}{3/2 + \sin^2(\xi_j/2)}
  \label{eq:dc_eigenvalue_ratio}
\end{equation}

この比は $\xi_j$ の単調減少関数であり：
\begin{itemize}
  \item $\xi_j \to 0$（低周波）：$\lambda_j/\mu_j \to 6/(3/2) = 4$
  \item $\xi_j = \pi$（Nyquist）：$\lambda_j/\mu_j = 6/(3/2+1) = 12/5 = 2.4$
\end{itemize}

\subsubsection*{\texorpdfstring{最適緩和係数 $\omega^*$ の導出}{最適緩和係数 ω* の導出}}

$|\sigma_j| = |1 - \omega\,\lambda_j/\mu_j| < 1$ が全モードで成立するために，
$\omega$ は $\lambda_j/\mu_j$ の最大値と最小値から決定される．
等方一様格子での固有値比の範囲は $[\alpha_{\min}, \alpha_{\max}] = [2.4, 4.0]$
であるから，$\rho(M)$ を最小化する $\omega^*$ は：
\begin{equation}
  \omega^* = \frac{2}{\alpha_{\min} + \alpha_{\max}}
  = \frac{2}{2.4 + 4.0}
  = \frac{2}{6.4}
  \approx 0.3125
  \label{eq:dc_omega_opt}
\end{equation}
このとき
\begin{equation}
  \rho(M^*) = \frac{\alpha_{\max} - \alpha_{\min}}{\alpha_{\max} + \alpha_{\min}}
  = \frac{4.0 - 2.4}{4.0 + 2.4}
  = \frac{1.6}{6.4}
  = 0.25
  \label{eq:dc_rho_opt}
\end{equation}

すなわち\textbf{最適緩和係数 $\omega^* \approx 0.31$ では1反復あたり $\rho(M) = 0.25$}であり，
$k=3$ 回の反復で $\rho^3 = 0.016$，$k=5$ で $\rho^5 \approx 10^{-3}$ となる．
これは §\ref{sec:dc_iteration_accuracy} で観測される「$k=3$ で $\Ord{h^6}$ に漸近」の理論的根拠となる．

\subsubsection*{\texorpdfstring{仮想時間刻み $\Delta\tau$ との関係}{仮想時間刻み Δτ との関係}}

§\ref{sec:dc_terminology} で述べたように，$\omega$ と $\Delta\tau$ は等価である．
式~\eqref{eq:dc_sweep_x}--\eqref{eq:dc_sweep_y} の左辺
$[\frac{1}{\Delta\tau}\mathbf{I} - L_{L}]$ を $L_L\,\delta p = d$ と比較すると：
\begin{equation}
  \delta p = \Delta\tau\,(\mathbf{I} - \Delta\tau\,L_L)^{-1}\,d
  \label{eq:dc_omega_dtau}
\end{equation}
小 $\Delta\tau$ の極限で $(\mathbf{I} - \Delta\tau L_L)^{-1} \approx \mathbf{I} + \Delta\tau L_L + \cdots$ であるから
$\delta p \approx \Delta\tau\,d + \Ord{\Delta\tau^2}$ となり，
式~\eqref{eq:dc_three_step} の Step~3 で $\omega = \Delta\tau$ の対応が確認できる．
$\Delta\tau$ が大きい場合は $(\mathbf{I} - \Delta\tau L_L)^{-1}$ の非線形効果が加わるが，
Thomas 法の対角優位性（付録~\ref{app:sweep_thomas_coeff}）により無条件安定が保証される．

\subsubsection*{変密度場への拡張}

密度比 $\rho_l/\rho_g \gg 1$ の気液二相流では，
$L_H$ と $L_L$ の固有値比 $\alpha_j = \lambda_j/\mu_j$ が空間的に変動する．
特に：
\begin{itemize}
  \item 低密度相（気体）：$1/\rho$ が大きく $L_H$ の有効固有値が増大 → $\alpha_j$ が拡大
  \item 高密度相（液体）：$1/\rho$ が小さく有効固有値が低下 → $\alpha_j$ が縮小
\end{itemize}
この空間変動は $[\alpha_{\min}, \alpha_{\max}]$ の幅を広げ，
式~\eqref{eq:dc_rho_opt} の $\rho(M^*)$ を増大させる（収束が遅くなる）．

この剛性問題に対する解決策が\textbf{密度適応型局所時間刻み（LTS）}
（§\ref{sec:lts_dtau}，式~\eqref{eq:dtau_lts}）であり，
$\Delta\tau_{i,j} \propto \rho_{i,j}$ とすることで $\Delta\tau_{i,j} \cdot a_{i,j} = \mathrm{const}$
（$a_{i,j} = 1/\rho_{i,j}$）を実現し，
全セルの有効緩和が均等化される．

\begin{tcolorbox}[resultbox, title={欠陥補正法の収束特性まとめ}]
\begin{center}
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lcc}
\toprule

\midrule

等密度・等方格子（$\omega = 0.1$，非最適） & $\approx 0.76$ & $\approx 0.44$ \\
変密度（$\rho_l/\rho_g = 10$，LTS 適用） & $\approx 0.3$--$0.4$ & $\approx 0.03$--$0.06$ \\

\bottomrule

\end{center}
収束には $\omega \in (0,\, 2/\alpha_{\max}) = (0,\, 0.5)$ が必要であり，$\omega \ge 0.5$ では発散する．
高密度比では LTS を適用してもスペクトル半径が $0.5$--$0.6$ に増大するため，
$k = 10$--$20$ 回の反復が必要となる（それでもKrylov法より反復単価が低い）．
\end{tcolorbox}
