\subsection{PPE 解法の精度整理：本質問題と採用スタック}
\label{sec:pressure_accuracy_summary}
% ═══════════════════════════════════════════════════════════════════════════════

本章の結論は，複数の圧力解法を状況に応じて同列に併用することではない．
二相 pressure step の本質問題を
\[
  \text{密度ジャンプ}
  \;+\;
  \text{圧力ジャンプ}
  \;+\;
  \text{面演算子整合}
  \;+\;
  \text{高次片側データ}
  \;+\;
  \text{相別ゲージ}
\]
に分解し，各問題に必要な手段だけを残すことである．

\begin{tcolorbox}[mybox, title={閉包条件：§9 が production と呼ぶ pressure stack}]
\label{box:ppe_design_rationale}
§9 の production pressure stack は次の同時成立を要求する：
\[
  \boxed{
  \texttt{phase\_separated}
  +\texttt{affine\_jump}
  +\mathrm{HFE}
  +\mathrm{DC}\ k=3
  +\text{global gauge}
  }.
\]
ここで \texttt{affine\_jump} は毛管波専用の修正ではない．
液滴，気泡，毛管波，平坦 Rayleigh--Taylor 界面のいずれでも，
同じ $j_{gl}=p_g-p_l=-\sigma\kappa_{lg}$ と
同じ $G_\Gamma(p;j)$ を使うことが採用条件である．

この閉包が保証するのは，PPE と corrector が同じ面 flux 契約を共有し，
圧力ジャンプが regular 圧力自由度として消えないことである．
一方で，曲率 cap，HFE の実効精度，毛管 CFL，壁面影響，長時間の幾何エネルギー安定性は
§~\ref{sec:val_capillary} 以降の物理 benchmark に残る．
\end{tcolorbox}

\begin{tcolorbox}[warnbox, title={§9 で legacy として残すもの}]
\begin{itemize}[noitemsep]
  \item \textbf{smoothed-Heaviside 一括 PPE}：
        低・中密度比の比較経路として残すが，高密度比 pressure-jump production の解決策とはしない．
  \item \textbf{CSF body force}：
        Balanced--Force 原理を説明する基準経路として残すが，最終的な毛管閉包とはしない．
  \item \textbf{\texttt{jump\_decomposition}}：
        既存比較と退行検出のため残すが，新規 pressure-jump 設定の既定にはしない．
  \item \textbf{FVM/FD base solve}：
        DC の base operator や比較基準として使うが，CCD/FCCD 研究閉包の代替主張にはしない．
\end{itemize}
\end{tcolorbox}

したがって，9 章を読んだ後に残る物語は単純である：
圧力ステップの本質は「高次 Poisson solve」ではなく，
不連続な界面応力を同じ面演算子上で保存したまま投影することである．
そのために，分相化で密度ジャンプを外し，
向き付き affine jump で圧力ジャンプを PPE に入れ，
HFE で高次ステンシルの滑らかさを守り，
DC とゲージで実用的に解く．
