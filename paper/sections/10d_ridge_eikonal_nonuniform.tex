\subsection{Ridge--Eikonal の非一様格子定式化：D1--D4 と壁閉包}
\label{sec:ridge_eikonal_nu_grid}
% ═══════════════════════════════════════════════════════════════════════════════

\S\ref{sec:ridge_eikonal} で構築した一様格子上の Ridge--Eikonal 再初期化を
界面適合非一様格子（密度因子 $\alpha>1$）へ拡張すると，
(i) ガウス型半幅 $\sigma$ の物理セル分解能が不均一化，
(ii) $\xiidx$ 空間でヘッセ行列を評価すると狭帯域で連鎖律誤差が支配的になり，
(iii) FMM~\cite{Sethian1996} の二次式更新が単位格子幅を仮定して $|\bnabla_x\phi|=1$ を破る，
(iv) 壁面接触界面ではガウス型支持帯と FMM 種点集合が
閉領域 $\overline{\Omega}$ 上で閉じていない，
の 4 か所で一様格子不変量が破れる．これらを 4 補正
D1（$\sigma_\text{eff}$ の空間倍率）・D2（物理空間ヘッセ行列）・
D3（非一様 FMM）・D4（Ridge--Eikonal 再構成幅 $\varepsilon_\text{local}$）と
壁閉包補正 W で解消し，
\S\ref{sec:grid_density}（格子密度関数）・
\S\ref{sec:grid_coord}（メトリクス係数）・
\S\ref{sec:ccd_metric}（非一様 CCD）の幾何量へ数学的に接続する．
本節は「各補正がどの格子量に依存し，
どの離散演算子と整合するか」を明示することを目的とする．

\begin{remark}[本節の記号規約]
\label{rem:ridge_nu_jx_convention}
\S\ref{sec:grid_coord} は主本文メトリクス係数を
$J_x = \mathrm{d}\xi/\mathrm{d}x$（ヤコビアン）として定義したが，
本節ではセル幅計算および FMM 閉形式解との整合のため，
\textbf{逆数の幾何量}
$\mathsf{X}_\xi(\xi) \equiv \mathrm{d}x/\mathrm{d}\xi = 1/J_x^{\mathrm{main}}$ を用いる．
以降\textbf{本節では一貫して $\mathsf{X}_\xi$ 記号}を使用し，
$J_x$ 記号は登場しない．主本文の $J_x$ への
変換は $J_x^{\mathrm{main}} = 1/\mathsf{X}_\xi$ で読み替える．
\end{remark}

% -------------------------------------------------------------------------------
\subsubsection{D1--D4 と壁閉包の基本条件}
\label{sec:ridge_nu_d1d4_math}
% -------------------------------------------------------------------------------

上で列挙した一様格子不変量の破れを，
以下の D1--D4 と壁閉包 W で，依存する格子量ごとに閉じる．
本段落では各補正の基本条件を定め，後続節で同じ条件を CCD/FMM/FCCD の離散幾何へ接続する．

\paragraph{D1 — $\sigma_\text{eff}(\bm{x})$ の空間倍率}
基準格子幅を格子点数 $N_a$ に対して
$h_\text{ref}=(\prod_{a} L_a/(N_a-1))^{1/d}$，
局所幾何平均を $h(\bm{x})=\sqrt{h_x(i)\,h_y(j)}$ とし，
\begin{equation}
  \sigma_\text{eff}(\bm{x}) = \sigma_0 \cdot \frac{h(\bm{x})}{h_\text{ref}},
  \qquad \sigma_0 \in \mathbb{R}_{>0}\text{（}h_\text{ref}\text{ 単位）}
  \label{eq:sigma_eff}
\end{equation}
と修正する．
物理セル単位では $\sigma_\text{eff}(\bm{x})/h(\bm{x}) = \sigma_0/h_\text{ref}$ が定数となり
一様格子での基準値をそのまま $\sigma_0$ に引き継げる．

\paragraph{D2 — 物理空間ヘッセ行列（$\xi$ 空間ヘッセ行列を避ける理由）}
$\xiidx$ 空間でヘッセ行列を組み立て
連鎖律 $\bnabla_x^2 = J^{\!\top}\bnabla_\xi^2 J
+ (\bnabla_\xi J^{\!\top})\bnabla_\xi$
で物理空間に変換する経路は採用しない．

\begin{remark}[$\xi$ 空間ヘッセ行列の数値的不整合]\label{rem:xi_hessian_forbidden}
界面遷移幅が $\le 4$ セルのとき，
$\xi$ 空間ヘッセ行列を経由する評価は $\mathcal{O}(10^{-1})$ の
下限誤差を持つ（根拠は §~\ref{sec:xi_sdf_reinit}）．
リッジ定義 \eqref{eq:ridge_set_def} は
$\sigma_\text{eff}\sim 3h$ の範囲で評価されるため，
この条件は一般に違反される．
\end{remark}

許容される評価経路：
\begin{enumerate}
  \item\textbf{評価式 A（採用する高次評価）}——物理空間の非一様 CCD
        でヘッセ行列 $\mathbf{H}_{\xiridge}=\bnabla_x^2\xiridge$ を直接評価する．
  \item\textbf{評価式 B（低次の確認評価）}——
物理空間二階差分で $H_{xx},H_{xy},H_{yy}$ を近似し，
        符号判定 $\lambda_{\min}(H)<0$ または
        $\hat{\bm{n}}^{\!\top}H\hat{\bm{n}}<0$ を
        $H^{(2)}$ で行う．これは 2 次精度の確認評価に限り，
        Morse 非退化点でも固有値の符号余裕が小さい場合は評価式 A を用いる．
\end{enumerate}

\paragraph{D3 — 非一様 FMM}
標準 FMM の二次更新
$d_{\mathrm{cand}}=\tfrac{1}{2}(a_x+a_y+\sqrt{2-(a_x-a_y)^2})$
は $h_x=h_y=1$ を前提とする．
ここで $a_x, a_y$ は各方向で確定集合に属する近傍点（風上側）の $d$ 最小値である
（標準 FMM の風上選択規則；詳細は~\cite{Sethian1996}）．
物理空間 Eikonal
\begin{equation}
  \frac{(d-a_x)^2}{h_x^2}+\frac{(d-a_y)^2}{h_y^2}=1
  \label{eq:fmm_nonuniform}
\end{equation}
の閉形式解（大きい方の根）は
\begin{equation}
  d = \frac{a_x/h_x^2 + a_y/h_y^2 + \sqrt{D}}
           {1/h_x^2 + 1/h_y^2},\qquad
  D = \Bigl(\tfrac{1}{h_x^2}+\tfrac{1}{h_y^2}\Bigr)
      - \frac{(a_x-a_y)^2}{h_x^2 h_y^2}.
  \label{eq:fmm_closed_form}
\end{equation}
$D<0$ のときは一次元単調更新 $d=\min(a_x+h_x,\,a_y+h_y)$ を採用する．
粘性一意性は \eqref{eq:fmm_nonuniform} が物理空間で書かれている限り
§~\ref{sec:ridge_fmm_reconstruction} の粘性解一意性証明がそのまま成立する．

\paragraph{D4 — Ridge--Eikonal 再構成幅 $\varepsilon_\text{local}$}
再構成幅を
\begin{equation}
  \varepsilon_\text{local}(\bm{x})
  = \varepsilon_\text{scale}\cdot \varepsilon_{\mathrm{RE},\xi} \cdot h(\bm{x}),
  \qquad \varepsilon_{\mathrm{RE},\xi} = \varepsilon_{\mathrm{RE}}/h_\text{min}
  \label{eq:eps_local}
\end{equation}
と空間依存にする
（$h_\text{min} \equiv \min_{\bm{x}} h(\bm{x})$ は計算領域の最小格子幅；
非一様格子の定義は §~\ref{sec:grid}）．

\noindent
$\varepsilon_\text{scale}$ は「再構成された符号付き距離幅を
局所格子幅とどの程度整合させるか」を決める無次元因子であり，
一様格子の既定値を非一様格子へ移植する際にも
式~\eqref{eq:eps_local} の空間因子により自然に吸収される．
本稿では以降，代表値として $\varepsilon_\text{scale}=1.4$ を用いるが，
感度評価と補正条件は本節以降で扱う．

\paragraph{W — 壁閉包補正}
壁面に接する界面は領域内部の多様体ではなく，閉領域上の
\begin{equation}
  \overline{\Gamma}(t)=\Gamma(t)\cup C(t),\qquad
  C(t)=\overline{\Gamma}(t)\cap\partial\Omega
  \label{eq:wall_closed_interface}
\end{equation}
として扱う．
中立濡れ・Neumann 型 CLS 壁条件では，壁法線方向の幾何拡張は偶関数鏡像であり，
Ridge 補助場は
\begin{equation}
  \xiridge^{W}(\bm{x})
  =
  \sum_{\bm{c}_k\in S_\Gamma}
  \sum_{R\in\mathcal{R}_W}
  \exp\!\left(-\frac{|\bm{x}-R\bm{c}_k|^2}{\sigma_\text{eff}(\bm{x})^2}\right)
  \label{eq:ridge_wall_mirror}
\end{equation}
で評価する．
ここで $\mathcal{R}_W$ は恒等写像と，ガウス型支持帯内で関与する
壁鏡像（角では合成鏡像も含む）である．
これにより壁面で切断された半ガウス型関数によるリッジの内側バイアスを除く．

FMM の Dirichlet 種点集合も内部交差だけでは不十分である．
ただし種点は Eikonal 問題の境界条件そのものなので，
\textbf{実際の零集合または物理的に指定された接触線に限る}．
壁面トレース上の符号変化・厳密零点を含めて
\begin{equation}
  S_h
  =
  \mathcal{Z}_h(\phi_0;\Omega^\circ)
  \cup
  \mathcal{Z}_h(\phi_0;\partial\Omega_W)
  \cup
  S_{\mathrm{contact}}^{\mathrm{prescribed}},
  \qquad
  |\bnabla_x\phi|=1,\quad \phi|_{S_h}=0
  \label{eq:wall_contact_zero_set}
\end{equation}
を解く．
ここで $\partial\Omega_W$ は壁軸の物理境界であり，
周期軸の像平面は $\partial\Omega_W$ に含めない．
ガウス型リッジ候補関数は幾何候補の検出・壁鏡像補正には使えるが，
$S_h$ に人工的な零種点を追加する根拠にはならない．
人工種点は Eikonal の境界集合を変更し，
固定トポロジー再初期化で保存すべき零点集合を動かすためである．
さらに質量補正は接触線種点を動かしてはならないため，
\begin{equation}
  \phi \leftarrow \phi + \lambda\,\chi_{\mathrm{free}}(\bm{x}),
  \qquad
  \chi_{\mathrm{free}}=0\ \text{接触線帯上}
  \label{eq:contact_pinned_mass_correction}
\end{equation}
とし，$\lambda$ は通常の体積制約から決める．
この補正は Navier--Stokes の壁面速度・圧力条件を変更するものではなく，
Ridge--Eikonal と格子モニタが $\overline{\Omega}$ 上の界面閉包を
保存するための幾何契約である．

\subsubsection{\texorpdfstring{D1 の格子場：$\sigma_\text{eff}(\bm{x})$}{D1 の格子場：σ\_eff(x)}}
\label{sec:ridge_nu_sigma_field}
% -------------------------------------------------------------------------------

式~\eqref{eq:sigma_eff} のガウス型半幅
$\sigma_\text{eff}(\bm{x}) = \sigma_0 \cdot h(\bm{x})/h_\text{ref}$ は
格子幾何のみに依存するため，
\S\ref{sec:grid_gen} の格子点座標生成直後に
$\{\sigma_\text{eff}(\bm{x}_{i,j})\}_{i,j}$ として一度だけ定義する．
時間固定格子ではこの空間場は不変であり，時間発展中に再定義しない．

\noindent\textbf{幾何定義}：
\begin{itemize}
  \item 局所幾何平均 $h(\bm{x}) = \sqrt{h_x(i)\,h_y(j)}$ は
        本節ローカルな逆数メトリクス
        $\mathsf{X}_\xi = \mathrm{d}x/\mathrm{d}\xi = 1/J_x^{\mathrm{main}}$ から
        $h_x(i) = \mathsf{X}_\xi(\xi_i)\Delta\xi$ として得られる．
  \item 基準幅 $h_\text{ref} = (L_x L_y / (N_x N_y))^{1/2}$ は
        スカラ定数として格子生成時に決定．
  \item ALE（\S\ref{sec:grid_ale}）時間変化格子では
        $\sigma_\text{eff}$ を格子更新ごとに再計算する．
        壁面接触界面では式~\eqref{eq:ridge_wall_mirror} の
        鏡像交差も格子更新後に再生成する．
\end{itemize}

一様極限（$h(\bm{x}) \equiv h_\text{ref}$）で
$\sigma_\text{eff} \to \sigma_0$ となり一様格子版の既定と一致する．

\subsubsection{D2 の離散ヘッセ行列：非一様 CCD で直接評価}
\label{sec:ridge_nu_hessian_field}
% -------------------------------------------------------------------------------

注記~\ref{rem:xi_hessian_forbidden}（$\xi$ 空間ヘッセ行列の数値的不整合）により，
リッジ検出 \eqref{eq:ridge_set_def} で必要な
ヘッセ行列 $\mathbf{H}_{\xiridge}=\bnabla_x^2 \xiridge$ は
物理空間で直接評価しなければならない．
\S\ref{sec:ccd_metric} の非一様 CCD（座標変換法）は
ノード値から直接物理空間の第一・第二導関数を返す演算子
$\mathbf{D}_1^\text{CCD,nu}, \mathbf{D}_2^\text{CCD,nu}$ を提供するため，
評価式 A は以下の完全なヘッセ行列で定義する：
\begin{equation}
  \mathbf{H}_{\xiridge,i,j}^{\text{nu}}
  =
  \begin{pmatrix}
    D_{xx}^{\text{CCD,nu}}\xiridge &
    D_{xy}^{\text{CCD,nu}}\xiridge \\
    D_{xy}^{\text{CCD,nu}}\xiridge &
    D_{yy}^{\text{CCD,nu}}\xiridge
  \end{pmatrix}_{i,j},
  \qquad
  D_{xy}^{\text{CCD,nu}}
  = D_y^{(1),\text{CCD,nu}}D_x^{(1),\text{CCD,nu}},
  \label{eq:ridge_nu_hessian}
\end{equation}
ここで軸ごとに \S\ref{sec:ccd_metric} 式~\eqref{eq:transform_2nd_correct} を
本節記号で書き直した
$\partial_x^2 = \mathsf{X}_\xi^{-2}\,\partial_\xi^2
  - (\mathsf{X}_{\xi\xi}/\mathsf{X}_\xi^3)\,\partial_\xi$
の修正済み行列を適用する
（$\mathsf{X}_{\xi\xi} = \mathrm{d}^2 x/\mathrm{d}\xi^2$；
主本文 $J_x^{\mathrm{main}} = 1/\mathsf{X}_\xi$ と $J_x^2 \partial_\xi^2 + J_x\partial_\xi J_x \partial_\xi$
の表現は数学的に等価）．
混合項 $D_{xy}^{\text{CCD,nu}}$ は §~\ref{sec:ccd_mixed} の逐次 1 次 CCD 適用で評価する．
リッジ判定は §~\ref{sec:ridge_eikonal} の定義と同じく
$\lambda_{\min}(\mathbf{H}_{\xiridge}^{\text{nu}})<0$ を用い，
法線曲率形式で確認する場合も
$\hat{\bm{n}}^{\!\top}H\hat{\bm{n}}
= n_x^2H_{xx}+2n_xn_yH_{xy}+n_y^2H_{yy}$ として
混合項を保持する．

\noindent\textbf{評価式 B（低次確認評価）}：
物理空間二階差分
\[
  [\partial_x^2\xiridge]_{i,j}
  = \frac{2}{h_x^+(i)+h_x^-(i)}
    \left[\frac{(\xiridge)_{i+1,j}-(\xiridge)_{i,j}}{h_x^+(i)}
          - \frac{(\xiridge)_{i,j}-(\xiridge)_{i-1,j}}{h_x^-(i)}\right]
\]
は 2 次精度に留まるが，Morse 非退化リッジ\textbf{かつ曲率半径 $R \gg h_\text{min}$}では
符号判定に影響しない（小曲率半径の薄いリッジでは $\Ord{h^2}$ 誤差が
$\hat{\bm{n}}^{\!\top}H\hat{\bm{n}}$ の符号を反転させうるため評価式 A を採用する）．
混合成分 $H_{xy}$ は §~\ref{sec:ccd_mixed} と同じ 2 段適用で評価し，
対角成分だけに落とさない．
したがって二階差分経路は確認計算に限る．

\subsubsection{D3 の Eikonal 幾何：非一様 FMM の格子幅条件}
\label{sec:ridge_nu_fmm_geometry}
% -------------------------------------------------------------------------------

式~\eqref{eq:fmm_closed_form} の閉形式解
$d = (a_x/h_x^2 + a_y/h_y^2 + \sqrt{D})/(1/h_x^2 + 1/h_y^2)$ を
格子層で組み立てるには，
各ノード $(i,j)$ に対する隣接セル幅 $(h_x(i), h_y(j))$ を
FMM の確定順序を定める各更新で参照する必要がある．
必要条件は以下：
\begin{itemize}
  \item \textbf{セル幅配列}：
        \S\ref{sec:grid_gen} の $\{x_i\}_i, \{y_j\}_j$ から
        $h_x[i] := x_i - x_{i-1}$，$h_y[j] := y_j - y_{j-1}$ を定義
        （\S\ref{sec:fccd_nu_geometry} の $H_i$ と同義；本節の $h_x[i] = H_i$ として参照）．
  \item \textbf{判別式負時の一次元単調更新}：
        $D<0$ のとき一次元単調更新
        $d = \min(a_x + h_x(i),\,a_y + h_y(j))$．
        コーナー近傍や急な密度勾配帯で発生し，
        粘性一意性は物理空間 Eikonal \eqref{eq:fmm_nonuniform} の
        弱解性質により保たれる．
  \item \textbf{サブセル種点}：
        実際の零集合 $S_h$ は一般にノード上に載らないため，
        符号反転セルの線形補間位置から最寄りノードへ
        物理空間距離で種点値を分配する．
        非一様格子では $(h_x(i),h_y(j))$ で正しく重み付ける必要があり，
        単位セル仮定では最大 $\alpha$ 倍の誤差を生む
        （$\alpha$ は格子密度比）．
        壁面接触界面では式~\eqref{eq:wall_contact_zero_set} の
        $\mathcal{Z}_h(\phi_0;\partial\Omega_W)$ も同じ Dirichlet 種点として扱い，
        厳密零点の壁ノードも種点に含める．
        一方，周期軸の端点は商空間同一性 $q_N=q_0$ で閉じる像であり，
        壁種点や接触線固定の対象ではない．
\end{itemize}

\noindent この結線により，一様版と非一様版は同じ Eikonal 境界値問題の
物理空間離散化として比較できる．

\noindent\textbf{離散解の等価性要件．}
非一様 FMM は候補集合から最小の未確定値を順に確定する
因果的 Eikonal 離散化であり~\cite{Sethian1996}，固定回数の掃引緩和や Jacobi 型緩和に置き換えると
式~\eqref{eq:fmm_nonuniform} の粘性解を同じ離散問題として近似したことにならない．
したがって同じ非一様 FMM と呼ぶには，距離場，確定・凍結集合，
候補列に相当する因果順序を保ち，同じ非一様風上二次更新
\eqref{eq:fmm_closed_form} によって隣接点を緩和する必要がある．
これは便宜的な置換可能性ではなく，曲率・圧力ジャンプ・BF 投影が共有する
符号付き距離幾何を定義する離散化契約である．

% -------------------------------------------------------------------------------
\subsubsection{\texorpdfstring{D4 の再構成幅場：$\varepsilon_\text{local}(\bm{x})$}{D4 の再構成幅場：ε\_local(x)}}
\label{sec:ridge_nu_eps_field}
% -------------------------------------------------------------------------------

式~\eqref{eq:eps_local} の再構成幅
$\varepsilon_\text{local}(\bm{x}) = \varepsilon_\text{scale}\cdot\varepsilon_{\mathrm{RE},\xi}\cdot h(\bm{x})$ を
D1 と同様に格子点上の空間場として定義する．
ここでの $\varepsilon_\text{local}$ は Ridge--Eikonal 再構成幅であり，
CLS/CSF 界面厚み $\varepsilon_\Gamma$ とは別の幾何量である．
したがって CSF 表面張力の $\delta_{\varepsilon_\Gamma}(\phi)$ に空間可変幅を使う許可ではない
（§~\ref{box:eps_xi_csf_warning}）．
$\xiridge$ から $\phi$ への再構成式
\begin{equation*}
  \phi(\bm{x}) = \operatorname{sgn}(\xiridge - \xi_\text{th})\cdot d_\mathcal{R}(\bm{x}),
  \qquad \xi_\text{th} = \xi_\text{ridge,min} - \varepsilon_\text{local}(\bm{x})
\end{equation*}
では閾値 $\xi_\text{th}$ がノードごとに変わるため，
ノードごとに $\varepsilon_\text{local}(\bm{x}_{i,j})$ を用いる．

\noindent\textbf{体積保存の制約}：
$\varepsilon_\text{scale}$ は一様格子での較正値を基準に置くが，
非一様格子で許容されるのは，Ridge--Eikonal 再構成後の体積誤差が
指定した保存許容範囲内に留まる場合に限る．
許容範囲を満たさない場合は，$\varepsilon_\text{scale}$ を空間倍率
$h_\text{ref}/h_\text{min}$ と組み合わせる別較正問題として扱い，
本章の固定格子標準閉包には含めない．
界面適合格子族（\S\ref{sec:grid_density}）では
$h(\bm{x}) \approx h_\text{ref}$ が界面近傍で成立するため，
適応スケーリングは密度勾配帯のみで発動する．

% -------------------------------------------------------------------------------
\subsubsection{界面適合格子族との統合}
\label{sec:ridge_nu_interface_fitted}
% -------------------------------------------------------------------------------

\S\ref{sec:grid_density} の格子密度関数
$\omega_\text{grid}(\xi)$ を界面近傍で集中化すると，
D1--D4 の効果が次のように簡素化される：
\begin{itemize}
  \item \textbf{D1 $\sigma_\text{eff}$}：界面帯では $h(\bm{x}) \approx h_\text{min}$ 一定．
        リッジ検出は等価密度帯内で評価されるため実効 $\sigma_\text{eff}$ は一様．
  \item \textbf{D2 ヘッセ行列}：界面法線 $\hat{\bm{n}}$ は密度勾配方向と
        ほぼ平行になる単純帯では対角寄与が支配的になりやすいが，
        離散化では $H_{xy}$ も保持して
        斜め界面・曲率帯での符号反転を避ける．
  \item \textbf{D3 FMM}：界面から距離 $\mathcal{O}(\varepsilon_\text{local})$ の
        狭帯域のみで定義する．$D<0$ の一次元単調更新は
        密度勾配帯が急すぎないことを確認する診断量となる．
  \item \textbf{D4 $\varepsilon_\text{local}$}：界面帯で一様値
        $\varepsilon_\text{local}\approx \varepsilon_\text{scale}\cdot\varepsilon_{\mathrm{RE}}$．
\end{itemize}
つまり界面適合格子では Ridge--Eikonal の非一様補正は
「界面帯での一様近似 + 密度勾配帯での正則化」として作用する．
ただし，これは D1--D4 の数式上の接続であり，
移動界面，分裂合体，分相 PPE，HFE，圧力ジャンプを同時に含む
長時間 NS 計算での標準閉包としての実証は別の検証課題として残る．
一方，壁面接触については式~\eqref{eq:wall_closed_interface}--\eqref{eq:contact_pinned_mass_correction}
により，幾何再構成が有限 gap を作らないことを離散契約として要求する．

% -------------------------------------------------------------------------------
\subsubsection{幾何量と FCCD との接続}
\label{sec:ridge_nu_cost}
% -------------------------------------------------------------------------------

\noindent\textbf{固定格子で定まる幾何量}：
$\sigma_\text{eff}[i,j]$，$h_x[i]$，$h_y[j]$，$\varepsilon_\text{local}[i,j]$ の
4 つは格子構築時に定まる空間場である．
追従格子を採用する場合は格子更新ごとに再定義する．

\noindent\textbf{同じ格子幾何から定まる演算子}：
D2 の非一様 CCD ヘッセ行列，D3 の非一様 Eikonal 更新，D4 の再構成幅は，
いずれも同じ $(h_x,h_y,\mathsf{X}_\xi,\mathsf{Y}_\eta)$ を参照する．
したがって再初期化後の符号付き距離場と FCCD 面幾何は，別々の格子尺度ではなく
同一の離散幾何から定義される．

\noindent\textbf{FCCD との接続}：
\S\ref{sec:ridge_fccd_handoff} で示した通り，
Ridge--Eikonal で再構成された $\phi$ は
\S\ref{sec:fccd_def}/\S\ref{sec:fccd_nonuniform} の FCCD 面中心追跡に
そのまま引き継がれる．
非一様 FCCD の係数 $(\mu_i, \lambda_i)$ と
本節の $\sigma_\text{eff}, \varepsilon_\text{local}$ は
\textbf{同じ格子幾何量} $(h_x, h_y, \mathsf{X}_\xi, \mathsf{Y}_\eta)$ から派生するため，
両者は同じ離散幾何に基づいて定義される
（主本文 $J_x^{\mathrm{main}}, J_y^{\mathrm{main}}$ は逆数 $1/\mathsf{X}_\xi, 1/\mathsf{Y}_\eta$ で得る）．
