\section{多相流ソルバの統合検証実験}
\label{sec:verification}

\paragraph{本章の位置付け．}
\autoref{sec:numerical_integrity}（§12）で個別コンポーネント
（CCD/DCCD/UCCD6/FCCD 演算子族 U1，PPE BC U2，非一様空間 U3，Godunov Eikonal U4，
Heaviside--$\delta$ U5，分相 PPE+DC+HFE U6，BF 静止液滴 U7，時間積分 U8，DCCD 圧力禁止 U9）が
\textbf{単体で}設計仕様（slope, 境界整合，演算子トレース性）を満たすことを確認した．
本章は，これらコンポーネントを\textbf{統合}した完全 Navier--Stokes / 二相流ソルバが
\textbf{物理整合性を保つ領域}と\textbf{設計収束次数が崩れる領域}を，10 個の独立した統合検証実験
V1--V10 で評価する
（古典 Level Set 系の検証文化と benchmark の代表的整理は~\cite{OsherFedkiw2003,Sethian1999,Hysing2009,AlandVoigt2019}）．
本章の V 系列は integration verification 専用であり，
前章の U 系列（単体 component verification）とは実験目的と判定基準を分ける．

\paragraph{判定の読み方．}
本章では，\checkmark を「設計通り（あるいは文献根拠で再定義した revised criterion）に対して合格した結果」，
$\triangle$ を「物理的・文献的説明はつくが検証目的に対して条件付きの結果」，
$\times$ を「本稿の保証範囲外または設計未達」とする．
V1-b/V2/V4 は §\ref{sec:error_budget} で当初の理想化基準
（AB2 単体 2 次，CCD 内点 6 次，厳密 Galilean ゼロ残差）から，
\textbf{各実装縮約実験の identity に対する文献根拠付き revised criterion} に判定基準を見直し，
それぞれ \emph{standard projection の $O(\Delta t)$ 律速}~\cite{Chorin1968,BrownCortezMinion2001,Guermond2006},
\emph{wall one-sided CCD 境界閉包による global compact pollution の $O(h^4)$
基準}~\cite{Lele1992,VisbalGaitonde2002},
\emph{固定 Eulerian + no-slip 壁での原理的 $O(\|\bm U\|)$
bound}~\cite{LeVeque1996,DoneaHuerta2003} に対する合格へ再分類した．
これは「合格基準を恣意的に緩めた」結果ではなく，
連続方程式レベルの設計次数と実装縮約実験の identity との差を
文献根拠で分離した結果である．
V1-a / V1-b / V2 / V3 / V4 / V5 / V6 / V8 は主張を支持する結果である．
V7（二相 IMEX-BDF2 coupled-stack slope $1.58$ — §14 stack 化により $\times$ 区分から $\triangle$ 区分へ移動；
TVD--RK3 (3 次) と IMEX-BDF2 (2 次) の Lie 演算子分離は 1 次 splitting 誤差を生む~\cite{Strang1968,HoldenKarlsenLieRisebro2010}
ため，coupled-stack 実効 slope の理論的下限は $1$ 程度であり，
観測値 $1.58$ はこの下限を実質上回る），
V10-a（$\alpha=1$ Zalesak — slot 解像度律速の sharp-corner CLS 移流~\cite{Zalesak1979,OlssonKreiss2005};
Type-B 修正（\S\ref{sec:zalesak_cls_advection} 内 mass correction 採用）後の再評価値で判定する），
V10-b（$T=8$ 単一渦 reversal — Rider--Kothe 1998 標準
$T=2$~\cite{LeVeque1996,RiderKothe1998} より重い stress-test 領域での
形状復元診断；filament 厚 $< h$ に到達する sub-grid 構造は固定一様格子では
原理的に追跡不能~\cite{EnrightFedkiwFerzigerMitchell2002}）は条件付き結果である．
V9（局所 $\varepsilon$）は §14 stack と同時に使う条件付き採用結果として扱う．
残る $\triangle$ は，格子不足だけでなく operator splitting，norm 選択，
参照解品質，reinitialization cadence，非可逆な形状復元を含む多軸仮説として扱い，
本稿では理論的下限を超えた合格化を試みない．
本章ではこれらを隠さず，アルゴリズムの適用範囲を定めるための検証結果として扱う．

\paragraph{V1--V10 の設計指針．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{Tier 構造} — V1--V2 は単相 NS（密度比 $1$）で空間/時間収束を確立し，
        V3--V7 で二相 NS（密度比 $\le 833$）の物理整合性と時間精度を，
        V8--V9 で非一様格子を，V10 で一様格子上の CLS 強変形移流を検証する．
  \item \textbf{設定忠実性} — 各実験は表に示す物理設定
        （$\sigma$, $\rho_l/\rho_g$, $\Re$, BC）で評価する．
        スキームは §14 の本計算条件（FCCD 界面輸送，Ridge--Eikonal,
        HFE/分相 PPE，UCCD6 対流，IMEX--BDF2）を基準とし，FD/単純 upwind/解析輸送は
        比較基線または条件付き縮約テストとして明示する場合に限る．
        V10-b の単一渦 reversal は，界面追跡なしの固定一様格子上で Ridge--Eikonal と質量修正を含む強変形テストとして扱う．
  \item \textbf{対応関係} — 各実験は「目的 → 設定 → 結果 → 判定」の順に記述する．
  \item \textbf{数値の扱い} — 表には再生成した数値のみを記載し，過去表の値は再利用しない．
\end{itemize}
