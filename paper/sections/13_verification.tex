\section{多相流ソルバの統合検証実験}
\label{sec:verification}

\paragraph{本章の位置付け．}
\autoref{sec:numerical_integrity}（§12）で個別コンポーネント
（CCD/DCCD/UCCD6/FCCD 演算子族 U1，PPE BC U2，非一様空間 U3，Godunov Eikonal U4，
Heaviside--$\delta$ U5，分相 PPE+DC+HFE U6，BF 静止液滴 U7，時間積分 U8，DCCD 圧力禁止 U9）が
\textbf{単体で}設計仕様（slope, 境界整合，演算子トレース性）を満たすことを確認した．
本章は，これらコンポーネントを\textbf{統合}した完全 Navier--Stokes / 二相流ソルバが
\textbf{物理整合性を保つ領域}と\textbf{設計収束次数が崩れる領域}を，10 個の独立した統合検証実験
V1--V10 で評価する
（古典 Level Set 系の検証文化と benchmark の代表的整理は~\cite{OsherFedkiw2003,Sethian1999,Hysing2009,AlandVoigt2019}）．
本章の V 系列は integration verification 専用であり，
前章の U 系列（単体 component verification）とは実験目的と判定基準を分ける．

\paragraph{判定区分と Type 分類．}
本章は判定マーク 3 種と Type 分類 3 種を組合せて各 V test を評価する．
表~\ref{tab:V_verdict_types} に凡例を示す．Type-A は「実装縮約実験の identity に対し
文献根拠で基準を再定義した結果の合格」，Type-B は「構造的アルゴリズム修正
（mass correction 等）による合格化」，Type-D は「連続方程式・離散構造に由来する
theoretical hard limit 上の near-optimal として条件付き合格」を意味する．
これらは「合格基準を恣意的に緩めた」結果ではなく，連続方程式レベルの設計次数と
実装縮約実験の identity との差を文献根拠で分離した結果である．
本章ではこれらを隠さず，アルゴリズムの適用範囲を定めるための検証結果として扱う．

\paragraph{演算子 stack の構成．}
本章は §14（本計算 reference path）と同じ operator stack の \emph{subset} を，
各 V test の検証目的に応じて部分的に使用する縮約 (reduced) 路線を取る．
§14 stack の正式構成は下記であり，各 V test では \paragraph{設定} 内で
含有 operator を明記する．
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{interface transport}: FCCD flux-form (高次保存形)
  \item \textbf{convection}: UCCD6（6 次 upwind compact）
  \item \textbf{curvature}: HFE（Heaviside-Flux-Embedded）
  \item \textbf{PPE}: pressure-jump phase-separated FCCD PPE + defect correction (DC)
  \item \textbf{reinitialization}: Ridge--Eikonal（典型 $10$ step ごと）
  \item \textbf{mass correction}: Olsson--Kreiss（典型 $10$ step ごと）
  \item \textbf{time integration}: TVD--RK3（界面）+ IMEX-BDF2（運動量）
\end{itemize}
本章での縮約路線は次の四系統に大別される：
(i) \textbf{単相縮約}（V1, V2）— CCD 微分 + standard projection PPE のみ；
(ii) \textbf{二相縮約 BF}（V3, V4, V5, V8）— CCD 微分 + Neumann FVM PPE
（参照点 pin gauge 固定）+ CSF $\kappa$（pressure-jump PPE は含まない）；
(iii) \textbf{§14 stack subset}（V6, V7, V9）— FCCD interface transport
+ UCCD6 momentum convection + HFE curvature + pressure-jump phase-separated
PPE + DC（reinitialization / mass correction の有無は test ごとに異なる）；
(iv) \textbf{CLS-only advection}（V10-a/b）— FCCD flux-form + TVD--RK3
（+ Ridge--Eikonal 再初期化 + Olsson--Kreiss mass correction; momentum/PPE
は含まない passive scalar 経路）．

\paragraph{圧力出力の取り扱い．}
PPE 経路により出力 $p$ の物理的意味が異なるため，本章は二種の圧力指標を併用する．
\textbf{Neumann PPE 経路}（V3/V5/V8）では出力 $p$ は physical pressure であり，
判定指標は Laplace 圧 $\sigma/r$ に対する相対誤差
$|\Delta p_{\mathrm{rel}}| \equiv |\Delta p_{\mathrm{meas}} - \sigma/r|/(\sigma/r)$ を用いる．
\textbf{pressure-jump PPE 経路}（V6/V9）では jump-decomposition により $\sigma/r$ が
界面結合に埋め込まれるため，出力 $p$ は projection-corrected pressure であり
physical Laplace 圧そのものではない．本章は
$|\Delta p_{\mathrm{corr}}|/(\sigma/r)$ を solver diagnostic として測る
（Laplace 絶対圧誤差ではない）．

\paragraph{寄生流れスケールの参照値．}
寄生流れ振幅の典型上界として capillary Reynolds 数
$\mathrm{Re}_c \equiv \rho\,\sigma\,h/\mu^2$ から導かれる
$\sigma/\mathrm{Re}_c \sim O(10^{-2})$ を本章共通の参照スケールとして用いる
（具体値は \autoref{sec:csf}, \cite{Popinet2009,DennerVanWachem2015}; V3/V5/V8 verdict
で参照する）．

\paragraph{各 V test の判定再分類．}
V1-a / V1-b / V2 / V3 / V4 / V5 / V6 / V8 は主張を支持する結果（\checkmark）である．
V7（二相 IMEX-BDF2 coupled-stack slope $1.58$）は §14 stack 化により $\times$ 区分から
$\triangle$ 区分へ移動した：TVD--RK3（3 次）と IMEX-BDF2（2 次）の Lie 演算子分離では
splitting 誤差が $O(\Delta t)$ で支配するため漸近 slope は $\to 1$ となる
~\cite{Strang1968,HoldenKarlsenLieRisebro2010}．
観測値 $1.58$ はこの Lie 1 次分割漸近域より高く，
完全な BDF2 単体（$2.00$）と Lie 漸近（$1$）の中間に位置する Type-D（near-optimal）として記録．
V10-a（$\alpha=1$ Zalesak; §\ref{sec:zalesak_cls_advection}）は Type-B 修正（mass correction）により
質量保存軸 \checkmark / 形状軸 $\triangle$（slot under-resolution の Type-D
~\cite{Zalesak1979,OlssonKreiss2005}）の二軸構造を取る．
V10-b（$T=8$ 単一渦 reversal; §\ref{sec:single_vortex_reversal}）も同 Type-B により
質量保存軸 \checkmark / 形状軸 $\triangle$（filament 厚 $<h$ Eulerian 固定格子の Type-D
~\cite{EnrightFedkiwFerzigerMitchell2002}; Rider--Kothe 1998 標準
$T=2$~\cite{LeVeque1996,RiderKothe1998} の 4 倍の積分時間範囲）の二軸．
V9（局所 $\varepsilon$）は §14 stack と同時に使う条件付き採用 ($\checkmark$)．

\paragraph{N 軸 vs.\ 時間軸 monotonicity の区別．}
本章では V3/V5/V8 で寄生流れ $u_\infty$ の挙動を測る際，
\textbf{時間軸 monotonicity}（200 ステップ trajectory 内での secular growth の有無）と
\textbf{N 軸 monotonicity}（解像度 sweep に対する単調収束）を区別して扱う．
時間軸での secular growth 不在は CSF $\kappa$ 推定の安定性を表し合格条件に含めるが，
N 軸での非単調挙動（$\Delta t = h/4$ 固定下で短波長 spurious mode が解像度ごとに
異なる累積振幅を示す）は CCD 高次微分の特性であり Type-D （収束 regime ではない）として
個別 verdict で記述する．

\paragraph{V1--V10 の設計指針．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{Tier 構造} — V1--V2 は単相 NS（密度比 $1$）で空間/時間収束を確立し，
        V3--V7 で二相 NS（密度比 $\le 833$）の物理整合性と時間精度を，
        V8--V9 で非一様格子を，V10 で一様格子上の CLS 強変形移流を検証する．
  \item \textbf{設定忠実性} — 各実験は表に示す物理設定
        （$\sigma$, $\rho_l/\rho_g$, $\Re$, BC）で評価する．
        スキームは §14 の本計算条件（FCCD 界面輸送，Ridge--Eikonal,
        HFE/分相 PPE，UCCD6 対流，IMEX--BDF2）を基準とし，FD/単純 upwind/解析輸送は
        比較基線または条件付き縮約テストとして明示する場合に限る．
        V10-b の単一渦 reversal は，界面追跡なしの固定一様格子上で Ridge--Eikonal と質量修正を含む強変形テストとして扱う．
  \item \textbf{対応関係} — 各実験は「目的 → 設定 → 結果 → 判定」の順に記述する．
  \item \textbf{数値の扱い} — 表には再生成した数値のみを記載し，過去表の値は再利用しない．
\end{itemize}
