\begin{abstract}
気液二相流数値シミュレーションにおける最大の数値的障害は，
静止液滴界面に生じる人工的渦流れ——\textbf{寄生流れ（spurious currents）}——である．
その根本原因は，圧力勾配項と表面張力項に異なる差分演算子を用いることによる離散化の不整合であり，
格子を細分化しても下限が消えない誤差体積力を生む．
本稿は，この問題を結合コンパクト差分法（CCD，内点 $\Ord{h^6}$），
Hermite 場延長法（HFE；\cite{Aslam2004} の Extension PDE に基づく），および Dissipative CCD 散逸機構の組み合わせにより，
寄生流れを CSF モデル誤差律速の水準まで根本的に抑制する設計を提案する．
その理論的基盤・実装詳細・検証手順を体系的に解説する．
界面追跡には Conservative Level Set（CLS）法と CCD スキームを採用して質量保存性と
数値拡散の抑制を同時に達成し，変密度圧力 Poisson 方程式は低密度比（$\rho_l/\rho_g \leq 5$）では
smoothed Heaviside 一括解法（FD 直接法 + CCD $\Ord{h^6}$ 勾配；必要に応じ欠陥補正法）で求解し，
高密度比では分相 PPE + HFE 解法（各相で定数密度の Poisson を独立化；必要に応じ欠陥補正法 DC）により求解する．
HFE が CCD ステンシルの界面 Gibbs 振動を除去し，
Balanced-Force 演算子整合と FCCD 面ジェット sub-system の協働により，
Rhie--Chow 補間に頼らずコロケート格子のチェッカーボード不安定を抑制する．
全手法は7ステップのタイムループとして統合される．
検証は三段構成で実施した：
数値要素の単体検証（CCD/FCCD/UCCD6，PPE 境界条件，非一様格子，HFE/分相 PPE 等），
NS パイプライン統合検証（静止液滴の Laplace 圧力誤差 $0.97\%$，
寄生流れは CCD が全表ケースで FD 以下，代表ケースで約 $2.5$--$5.9$ 倍抑制，密度比 $833$ までの分相 PPE + DC + HFE ロバスト性，
Zalesak / 単一渦 CLS 移流の 2D 可視化と体積・重心診断），
および物理ベンチマーク（毛細管波，気泡上昇，Rayleigh--Taylor 予備設定）．
一方，二相 IMEX--BDF2 の実効時間次数低下，local $\varepsilon$ の寄生流れ増加，
および $\alpha=2$ Zalesak の slot 重心追跡低次化は制約条件として明示した．
Hysing 級の terminal-velocity 比較や Prosperetti 級の dispersion 関係定量検証は今後の課題として位置づける．
\end{abstract}
