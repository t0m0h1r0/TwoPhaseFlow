\begin{abstract}
気液二相流数値シミュレーションにおける主要な数値的障害は，
静止すべき界面に生じる人工的渦流れ——\textbf{寄生流れ（spurious currents）}——である．
本稿は，その離散化起因成分を構造的に抑えるため，
結合コンパクト差分法（CCD，内点 $\Ord{h^6}$），FCCD 面ジェット，UCCD6，
Conservative Level Set（CLS）法，Hermite 場延長法（HFE；\cite{Aslam2004} の Extension PDE に基づく），
および Balanced-Force 七原則を一つの pressure-jump 型分相 PPE スタックへ統合する．
標準経路では，界面を FCCD 保存形 CLS と Ridge--Eikonal 再初期化で更新し，
表面張力は CSF 体積力ではなく $j_{gl}=p_g-p_l=-\sigma\kappa_{lg}$ の圧力ジャンプとして
分相 FCCD-PPE + HFE + 欠陥補正（DC $k=3$）へ組み込む．
この構成により，コロケート格子でも Rhie--Chow 補間に依存せず，
圧力勾配・表面張力・HFE 上流点・GFM ジャンプが同じ face footprint を共有する
Balanced-Force sub-system として閉じる．
時間発展は TVD-RK3（CLS），IMEX-BDF2/EXT2（運動量対流），
implicit-BDF2 + DC（粘性），IPC 圧力増分を組み合わせた 7 ステップ手順として整理する．

検証は，単体検証 U1--U9，統合検証 V1--V10，物理ベンチマーク第~\ref{sec:validation}章の
三層で構成した．単体検証では CCD $\Ord{h^{6.0}}$，FCCD $\Ord{h^{4.00}}$，
UCCD6 $\Ord{h^{7.00}}$，HFE 1D $\Ord{h^{5.91}}$，implicit-BDF2 $\Ord{\Delta t^{2.01}}$ などを確認し，
統合検証では静止液滴 Laplace 圧力誤差 $0.97\%$，
寄生流れの CCD/FD 比 $1/2.5$--$1/5.85$（代表ケース），
密度比 $\rho_l/\rho_g\le833$ での pressure-jump 分相 PPE + DC + HFE のロバスト性，
および Zalesak / 単一渦 CLS-only 強変形移流の質量・重心診断を示した．
一方，二相 coupled stack の時間精度は V7 で最終 slope $1.48$ に留まり，
capillary pressure-jump / projection の界面帯誤差に支配される Type-D 条件付き合格として扱う．
また，非一様 moving-interface CLS，長時間・完全連成 NS 移動界面，
Prosperetti 級の毛管波分散照合，Hysing 級の終端速度比較，
および Rayleigh--Taylor 線形成長率検証は今後の検証 gate として明示する．
\end{abstract}
