\subsubsection{V6：密度比 sweep ($\rho_r = 2 \to 833$)}
\label{sec:density_ratio_sweep}
\label{sec:interface_crossing}

\paragraph{検証対象．}
分相 PPE + DC + HFE 統合（\autoref{sec:split_ppe_motivation}〜\autoref{sec:U6_split_ppe_dc_hfe}）に対応する，
§14 と同じ FCCD interface transport / UCCD6 convection / HFE curvature
(``psi\_direct\_hfe'') / pressure-jump surface tension /
phase-separated FCCD PPE + defect correction + face-flux projection の構成が，
密度比 $\rho_r = \rho_l/\rho_g \in [2, 833]$（最大は水--空気の $\sim 833$；高密度比 LS の系統解析は~\cite{SussmanSmereka1997,Tryggvason2011}）で
\textbf{ブローアップしない}ことを確認する．
本検証は，一括 smoothed-Heaviside 解法で顕在化しうる密度ジャンプ越え問題に対する
分相・jump-decomposition 経路の応答評価でもある．
pressure-jump 形式では Laplace 跳び $\sigma/r$ は PPE の界面結合に埋め込まれるため，
出力される $p$ は絶対圧ではなく射影補正圧である．したがって本 V6 では
$u_\infty^{\mathrm{end}}$，CLS 体積 drift，および
$|\Delta p_{\mathrm{corr}}|/(\sigma/r)$ を solver diagnostic として測る．

\paragraph{設定．}
半径 $r=0.25$ の静止液滴，$\rho_l=\rho_r$, $\rho_g=1$, $\sigma=1$．
$\rho_r \in \{2,10,100,833\}$, $N \in \{24,32\}$．
各ケース $8$ step，CFL multiplier $0.50$．
スクリプト側で stack assertion を行い，通常 CCD/FVM-PPE 経路に落ちた場合は失敗させる．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 全 $8$ ケースで\textbf{ブローアップなし}（水--空気相当を含む）\checkmark．
  \item $|\Delta V_\psi|/V_{\psi,0} \le 2.4\times10^{-10}$ で CLS 体積は実質保存．
  \item $u_\infty^{\mathrm{end}}$ は $\rho_r$ 増加で抑制される．
        高密度比下では速度応答スケールが $\sigma/(\rho_l+\rho_g)$ に従い低下する．
  \item 本 V6 は stack assertion により通常 CCD/FVM-PPE ではなく
        FCCD/UCCD6/HFE/pressure-jump/phase-separated PPE/DC 経路を直接検証する．
\end{itemize}
検証対応：式~\eqref{eq:split_ppe}，\autoref{sec:split_ppe_motivation} の分相 PPE，
\autoref{sec:verify_split_ppe_dc}（DC），\autoref{sec:U6_split_ppe_dc_hfe}（HFE）に対し，
密度比 sweep の終端速度，CLS 体積 drift，補正圧スケールを測定する．

\subsubsection{V7：IMEX-BDF2 二相時間精度}
\label{sec:imex_bdf2_twophase_time}

\paragraph{検証対象．}
密度比 $\rho_r > 1$ の二相 NS において，IMEX-BDF2 時間離散~\cite{AscherRuuthSpiteri1997,HairerWanner1996}
（\autoref{ch:time_integration}）を含む §14 と同じ
FCCD interface transport + TVD--RK3，UCCD6 convection + IMEX-BDF2，
HFE curvature，ridge-eikonal reinitialization，pressure-jump phase-separated
FCCD PPE + defect correction を同時に使ったときの実効時間精度を，
参照解との差で評価する．これは BDF2 単体の次数検証ではなく，
界面輸送・再初期化・pressure-jump PPE を含む結合 stack 診断である．

\paragraph{設定．}
$\rho_l=10,\rho_g=1$ ($\rho_r=10$), 液滴半径 $r=0.25$, $\sigma=1$．
$N=24$, $T=0.02$．初期速度は液相内に
$u_0=0.02\cos(2\pi y)H_\epsilon$, $v_0=-0.02\cos(2\pi x)H_\epsilon$ を与える．
$n_{\mathrm{step}}\in\{8,16,32\}$ を粗解，$n_{\mathrm{ref}}=64$ を参照解とし，
ridge-eikonal reinitialization は全 step で有効化する．

\paragraph{結果．}

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 観測された最終局所 slope は $\mathbf{1.58}$．BDF2 単体の $2.00$ には届かないが，
        旧 reduced proxy の slope $0.56$ より §14 stack では改善する．
  \item 劣化要因の解釈：
    \begin{enumerate}\setlength{\itemsep}{0pt}
      \item \textbf{CLS 再初期化の頻度差}：参照解 ($n_{\mathrm{ref}}=64$) と粗解
            ($n_{\mathrm{step}}\in\{8,16,32\}$) は，各時間ステップで再初期化を実施するため
            \textbf{総再初期化回数が $n_{\mathrm{ref}}/n_{\mathrm{step}}$ 倍} 異なる．
            よって参照解差には BDF2 ではなく reinitialization cadence の差が混入する．
      \item \textbf{pressure-jump PPE の非線形結合}：
            curvature/HFE と jump-decomposition PPE は界面位置と $\epsilon$ に非線形依存するため，
            粗い時間刻みでは $u$ の差が単純な $\Delta t^2$ で減衰しない．
      \item \textbf{interface transport の TVD--RK3 と momentum IMEX-BDF2 の混成}：
            interface と momentum の時間離散次数が異なるため，結合問題としての観測次数は
            最小次数だけで決まらず，参照解差に operator-splitting 誤差が残る．
    \end{enumerate}
  \item \textbf{条件付き未達 ($\triangle$)} — §14 stack は安定に実行され，細かい側で
        $O(\Delta t^{1.58})$ まで近づくが，二相結合 stack 全体の 2 次収束主張には未達．
        本論文では $O(\Delta t^2)$ を momentum predictor 単体の設計次数，
        V7 を coupled-stack の実効次数診断として扱う．
\end{itemize}
検証対応：\autoref{ch:time_integration} の IMEX-BDF2 と
\autoref{sec:eikonal_reinit} の再初期化，\autoref{sec:split_ppe_motivation} の
pressure-jump PPE に対し，二相結合 stack の実効 slope を測定する．
