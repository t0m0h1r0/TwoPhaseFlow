\subsubsection{V6：密度比 sweep ($\rho_r = 2 \to 833$)}
\label{sec:density_ratio_sweep}
\label{sec:interface_crossing}

\paragraph{検証対象．}
分相 PPE + DC + HFE 統合（\autoref{sec:split_ppe_motivation}〜\autoref{sec:U6_split_ppe_dc_hfe}）が
密度比 $\rho_r = \rho_l/\rho_g \in [2, 833]$（最大は水--空気の $\sim 833$）の
全レンジで\textbf{ブローアップせず}，かつ Laplace 圧力差が $\rho_r$ にロバストに
収束することを確認する．本検証は，旧 §13g で提起された「Smoothed Heaviside 一括解法の
密度ジャンプ越え問題」に対する分相解法の応答評価でもある．

\paragraph{設定．}
V3 と同じ静的液滴設定．$\rho_l = \rho_r$, $\rho_g = 1$, $\sigma = 1$, $r = 0.25$
→ $\Delta p_{\mathrm{exact}} = 4.0$．
$\rho_r \in \{2, 10, 100, 833\}$, $N \in \{32, 64, 128\}$．
$\Delta t = h/2$ (CFL ターゲット 0.4)．

\paragraph{評価．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 全 $12$ ケース blew\_up = False で\textbf{ブローアップ無し}（水--空気を含む）\checkmark．
  \item $|\Delta p_{\mathrm{rel}}| \le 1.8\%$ で BF 平衡を保つ → 分相 PPE + DC + HFE 構成が

\subsubsection{V7：IMEX-BDF2 二相時間精度}
\label{sec:imex_bdf2_twophase_time}

\paragraph{検証対象．}
密度比 $\rho_r > 1$ の二相 NS において，IMEX-BDF2 時間離散
（\autoref{ch:time_integration}）が
設計次数 $O(\Delta t^2)$ を保つかを，参照解（$n_{\mathrm{step}}=640$）と粗解の差で評価．

\paragraph{設定．}
$\rho_l = 10, \rho_g = 1$ ($\rho_r = 10$), 静的液滴 $r = 0.25$, $\sigma = 1$．
$N = 64$．参照 $n_{\mathrm{ref}} = 640$．
$n_{\mathrm{step}} \in \{40, 80, 160, 320\}$ ($\Delta t \in \{1.25\!\times\!10^{-3}, 6.25\!\times\!10^{-4}, 3.13\!\times\!10^{-4}, 1.56\!\times\!10^{-4}\}$)．

\paragraph{評価．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 観測 slope $\mathbf{0.56}$ — V1 単相の $2.00$ から\textbf{大幅劣化}（4 段階差以上の未達）．
        $O(\Delta t^{0.56})$ は実質的に $O(\Delta t)$ より悪く，
        BDF2 設計次数 $O(\Delta t^2)$ の主張は\textbf{二相結合下では維持されない}．
  \item 劣化要因の量的論証：
    \begin{enumerate}\setlength{\itemsep}{0pt}
      \item \textbf{CLS reinit の頻度差}：参照解 ($n_{\mathrm{ref}}=640$) と粗解
            ($n_{\mathrm{step}}\in\{40,\dots,320\}$) は，各時間ステップで reinit を実施するため
            \textbf{総 reinit 回数が $n_{\mathrm{ref}}/n_{\mathrm{step}}$ 倍} 異なる．
            reinit 1 回の $\psi$ 補正は両解で同符号・同オーダーだが
            \textbf{時刻のずれ}が存在し，差分 $u_{\mathrm{ref}} - u_{\mathrm{coarse}}$ を取っても
            cancel せず $\Delta t$ に比例する切片誤差として残る．
            （reinit が両解で完全に同タイミングなら共通分は cancel するが，
             reinit は CLS 進展時刻に同期するため粗解と参照で必ず非同期．）
      \item \textbf{曲率の explicit lag}：本検証では CSF $\kappa$ を
            $\kappa^n$（前ステップ値）で評価しており $\kappa^{n+1/2}$ ではない．
            これは Crank--Nicolson 補正されていない explicit treatment であり，
            $\sigma\nabla\!\cdot(\kappa^n \nabla H)$ 項に系統的 $O(\Delta t)$ 誤差を残す．
            この誤差は両解で $\Delta t$ 比例で異なり，差分でも cancel しない．
      \item BDF2 自体の最大次数は $\Delta t^2$ だが，上記 2 つは
            「両解で頻度・lag が異なる」ため両解差で cancel せず律速する．
    \end{enumerate}
  \item \textbf{要改善 ($\times$ — 設計未達)} — slope $0.56$ は
        $|p_{\mathrm{measured}} - p_{\mathrm{design}}| = 1.44$ で
        合格基準 $\le 0.5$ から大幅外。改善には：
        \textit{(i)} CLS reinit を 2 ステップに 1 回に間引いて頻度差を縮める（短期）；
        \textit{(ii)} $\kappa^{n+1/2}$ Crank--Nicolson 化（中期）；
        \textit{(iii)} 界面厚 $\varepsilon \to 0$ 極限の sharp interface 化（長期）．
        本論文では，二相時間精度の設計次数は\textbf{$O(\Delta t^{0.56})$ が実効上限}
        として\autoref{sec:error_budget}に明示する．
\end{itemize}
検証対応：\autoref{ch:time_integration} の IMEX-BDF2 と
\autoref{sec:eikonal_reinit} の再初期化に対し，二相時間精度の実効 slope を測定する．
