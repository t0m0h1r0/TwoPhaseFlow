\subsubsection{\texorpdfstring{V6：密度比 sweep ($\rho_r = 2 \to 833$)}{V6：密度比 sweep (rho_r = 2 to 833)}}
\label{sec:density_ratio_sweep}
\label{sec:interface_crossing}

\paragraph{検証対象．}
分相 PPE + DC + HFE 統合（\autoref{sec:split_ppe_motivation}〜\autoref{sec:U6_split_ppe_dc_hfe}）に対応する，
range-projected pressure-jump stack の FCCD interface transport / UCCD6 convection /
Hermite Field Extension (HFE) による片側延長・曲率データ供給 / pressure-jump surface tension /
phase-separated FCCD PPE + defect correction + projection-native face closure /
capillary range projection / affine pressure-history faces の構成が，
密度比 $\rho_r = \rho_l/\rho_g \in [2, 833]$（最大は水--空気の $\sim 833$；高密度比 LS の系統解析は~\cite{SussmanSmereka1997,Tryggvason2011}）で
\textbf{ブローアップしない}ことを確認する．
本検証は，一括 smoothed-Heaviside 解法で顕在化しうる密度ジャンプ越え問題に対する
分相・jump-decomposition 経路の応答評価でもある．
pressure-jump 形式では Laplace 跳び $\sigma/r$ は PPE の界面結合に埋め込まれるため，
出力される $p$ は絶対圧ではなく射影補正圧である．したがって本 V6 では
$u_\infty^{\mathrm{end}}$，CLS 体積 drift，および
$|\Delta p_{\mathrm{corr}}|/(\sigma/r)$ を solver diagnostic として測る．

\paragraph{設定．}
半径 $r=0.25$ の静止液滴，$\rho_l=\rho_r$, $\rho_g=1$, $\sigma=1$．
$\rho_r \in \{2,10,100,833\}$, $N \in \{24,32\}$．
各ケース $8$ step，CFL multiplier $0.50$．
検証経路は上記 operator stack に固定し，通常 CCD/FVM-PPE 経路とは分離する．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item \textbf{主指標（合格条件）}：(a) 全 $8$ ケースで\textbf{ブローアップなし}
        （水--空気相当 $\rho_r=833$ を含む）\checkmark，
        (b) $|\Delta V_\psi|/V_{\psi,0} \le 1.3\times10^{-16}$ で CLS 体積は実質保存\checkmark，
        (c) $u_\infty^{\mathrm{end}}\le 5.8\times10^{-10}$ で capillary face balance が丸め誤差近傍に閉じる\checkmark．
  \item \textbf{補助指標（scale 確認）}：
        $|\Delta p_{\mathrm{corr}}|/(\sigma/r) \sim 0.74$--$1.01$ は
        pressure-jump PPE の補正圧 diagnostic（§\ref{sec:verification}「圧力出力の取り扱い」参照；
        Laplace 絶対圧誤差ではない）であり，合格基準には含めない．
  \item 本 V6 は通常 CCD/FVM-PPE ではなく
        FCCD, UCCD6, HFE, pressure-jump phase-separated PPE/DC,
        capillary range projection, projection-native face closure, affine pressure-history faces 経路を直接検証する．
\end{itemize}
検証対応：式~\eqref{eq:split_ppe}，\autoref{sec:split_ppe_motivation} の分相 PPE，
\autoref{sec:verify_split_ppe_dc}（DC），\autoref{sec:U6_split_ppe_dc_hfe}（HFE）に対し，
密度比 sweep の終端速度，CLS 体積 drift，補正圧スケールを測定する．

\subsubsection{V7：IMEX-BDF2 二相時間精度}
\label{sec:imex_bdf2_twophase_time}

\paragraph{検証対象．}
密度比 $\rho_r > 1$ の二相 NS において，IMEX-BDF2 時間離散~\cite{AscherRuuthSpiteri1997,HairerWanner1996}
（\autoref{ch:time_integration}）を含む range-projected pressure-jump stack の
FCCD interface transport + TVD-RK3，UCCD6 convection + IMEX-BDF2，
HFE curvature，ridge-eikonal reinitialization，pressure-jump phase-separated
FCCD PPE + defect correction，capillary range projection，projection-native face closure，affine pressure-history faces
を同時に使ったときの実効時間精度を，
参照解との差で評価する．これは BDF2 単体の次数検証ではなく，
界面輸送・再初期化・pressure-jump PPE を含む結合 stack 診断である．

\paragraph{設定．}
$\rho_l=10,\rho_g=1$ ($\rho_r=10$), 液滴半径 $r=0.25$, $\sigma=1$．
$N=24$, $T=0.02$．初期速度は液相内に
$u_0=0.02\cos(2\pi y)H_\varepsilon$, $v_0=-0.02\cos(2\pi x)H_\varepsilon$ を与える．
$n_{\mathrm{step}}\in\{8,16,32\}$ を粗解，$n_{\mathrm{ref}}=64$ を参照解とし，
ridge-eikonal reinitialization は全 step で有効化する．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 観測された最終局所 slope は $\mathbf{1.59}$．BDF2 単体の $2.00$ には届かないが，
        capillary coupled-stack の実効次数として $1.5$ 近傍に位置する．
  \item \textbf{分離診断（capillary pressure-jump/projection 界面帯律速）}：
        参照解品質，再初期化 cadence，界面輸送，表面張力，二相係数を分離した補助診断では，
        $n_{\mathrm{ref}}=128$ でも slope は $1.59$ 近傍に留まり，
        界面輸送を止めても同程度の実効次数が残る．一方，$\sigma=0$ では velocity error が
        数桁低下し，最大誤差は $\psi\simeq0.5$ の界面帯に局在する．
        したがって支配因子は BDF2 係数や reinitialization cadence ではなく，
        range-projected capillary pressure-jump / affine FCCD projection の界面帯誤差である
        （対照実験表は付録~\ref{app:v7_imex_bdf2_rca}）．
  \item \textbf{Type-D 合格 ($\triangle$, capillary coupled-stack-limited)} —
        本稿では V7 を「BDF2 単体の純粋次数」ではなく「pressure-jump capillary stack 下の coupled-stack
        実効次数」として測定する diagnostic とする．
        観測 slope $1.59$ は，pressure jump を含む界面帯低正則性を含む縮約下では
        \emph{capillary coupled-stack-limited} であり，range-projected capillary-jump/projection の
        時間離散を構造的に上げない限り $2.0$ への合格化は理論上望めない．
        これは Type-D（理論的 hard limit; §\ref{sec:verification} の
        Type 分類および表~\ref{tab:V_verdict_types} 参照）として記録する．
        BDF2 単体の設計次数 $2$ は U8（§12）で既に独立検証済み．
\end{itemize}
検証対応：\autoref{ch:time_integration} の IMEX-BDF2 と
\autoref{sec:eikonal_reinit} の再初期化，\autoref{sec:split_ppe_motivation} の
pressure-jump PPE と capillary range projection に対し，二相結合 stack の実効 slope を測定する．
