\subsubsection{V6：密度比 sweep ($\rho_r = 2 \to 833$)}
\label{sec:density_ratio_sweep}
\label{sec:interface_crossing}

\paragraph{検証対象．}
§14 と同じ FCCD interface transport / UCCD6 convection / HFE curvature
(``psi\_direct\_hfe'') / pressure-jump surface tension /
phase-separated FCCD PPE + defect correction + face-flux projection の構成が，
密度比 $\rho_r = \rho_l/\rho_g \in [2, 833]$（最大は水--空気の $\sim 833$）で
\textbf{ブローアップしない}ことを確認する．
pressure-jump 形式では Laplace 跳び $\sigma/r$ は PPE の界面結合に埋め込まれるため，
出力される $p$ は絶対圧ではなく射影補正圧である．したがって本 V6 では
$u_\infty^{\mathrm{end}}$，CLS 体積 drift，および
$|\Delta p_{\mathrm{corr}}|/(\sigma/r)$ を solver diagnostic として測る．

\paragraph{設定．}
半径 $r=0.25$ の静止液滴，$\rho_l=\rho_r$, $\rho_g=1$, $\sigma=1$．
$\rho_r \in \{2,10,100,833\}$, $N \in \{24,32\}$．
各ケース $8$ step，CFL multiplier $0.50$．
スクリプト側で stack assertion を行い，通常 CCD/FVM-PPE 経路に落ちた場合は失敗させる．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 全 $8$ ケースで\textbf{ブローアップなし}（水--空気相当を含む）\checkmark．
  \item $|\Delta V_\psi|/V_{\psi,0} \le 2.4\times10^{-10}$ で CLS 体積は実質保存．
  \item $u_\infty^{\mathrm{end}}$ は $\rho_r$ 増加で抑制される．
        高密度比下では速度応答スケールが $\sigma/(\rho_l+\rho_g)$ に従い低下する．
  \item 本 V6 は stack assertion により通常 CCD/FVM-PPE ではなく
        FCCD/UCCD6/HFE/pressure-jump/phase-separated PPE/DC 経路を直接検証する．
\end{itemize}
検証対応：式~\eqref{eq:split_ppe}，\autoref{sec:split_ppe_motivation} の分相 PPE，
\autoref{sec:verify_split_ppe_dc}（DC），\autoref{sec:U6_split_ppe_dc_hfe}（HFE）に対し，
密度比 sweep の終端速度，CLS 体積 drift，補正圧スケールを測定する．

\subsubsection{V7：IMEX-BDF2 二相時間精度}
\label{sec:imex_bdf2_twophase_time}

\paragraph{検証対象．}
密度比 $\rho_r > 1$ の二相 NS において，IMEX-BDF2 時間離散
（\autoref{ch:time_integration}）が
設計次数 $O(\Delta t^2)$ を保つかを，参照解（$n_{\mathrm{step}}=640$）と粗解の差で評価．

\paragraph{設定．}
$\rho_l = 10, \rho_g = 1$ ($\rho_r = 10$), 静止液滴 $r = 0.25$, $\sigma = 1$．
$N = 64$．参照 $n_{\mathrm{ref}} = 640$．
$n_{\mathrm{step}} \in \{40, 80, 160, 320\}$ ($\Delta t \in \{1.25\!\times\!10^{-3}, 6.25\!\times\!10^{-4}, 3.13\!\times\!10^{-4}, 1.56\!\times\!10^{-4}\}$)．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 観測 slope $\mathbf{0.56}$ — V1 単相の $2.00$ から\textbf{大幅劣化}（4 段階差以上の未達）．
        $O(\Delta t^{0.56})$ は実質的に $O(\Delta t)$ より悪く，
        BDF2 設計次数 $O(\Delta t^2)$ の主張は\textbf{二相結合下では維持されない}．
  \item 劣化要因の量的論証：
    \begin{enumerate}\setlength{\itemsep}{0pt}
      \item \textbf{CLS 再初期化の頻度差}：参照解 ($n_{\mathrm{ref}}=640$) と粗解
            ($n_{\mathrm{step}}\in\{40,\dots,320\}$) は，各時間ステップで再初期化を実施するため
            \textbf{総再初期化回数が $n_{\mathrm{ref}}/n_{\mathrm{step}}$ 倍} 異なる．
            再初期化 1 回の $\psi$ 補正は両解で同符号・同オーダーだが
            \textbf{時刻のずれ}が存在し，差分 $u_{\mathrm{ref}} - u_{\mathrm{coarse}}$ を取っても
            相殺されず $\Delta t$ に比例する切片誤差として残る．
            （再初期化が両解で完全に同タイミングなら共通分は相殺されるが，
             再初期化は CLS 進展時刻に同期するため粗解と参照で必ず非同期．）
      \item \textbf{曲率の陽的遅れ評価}：本検証では CSF $\kappa$ を
            $\kappa^n$（前ステップ値）で評価しており $\kappa^{n+1/2}$ ではない．
            これは Crank--Nicolson 補正されていない陽的評価であり，
            $\sigma\nabla\!\cdot(\kappa^n \nabla H)$ 項に系統的 $O(\Delta t)$ 誤差を残す．
            この誤差は両解で $\Delta t$ 比例で異なり，差分でも相殺されない．
      \item BDF2 自体の最大次数は $\Delta t^2$ だが，上記 2 つは
            「両解で頻度・遅れ評価が異なる」ため両解差で相殺されず律速する．
    \end{enumerate}
  \item \textbf{要改善 ($\times$ — 設計未達)} — slope $0.56$ は
        $|p_{\mathrm{measured}} - p_{\mathrm{design}}| = 1.44$ で
        合格基準 $\le 0.5$ から大幅外．改善には：
        \textit{(i)} CLS 再初期化を 2 ステップに 1 回に間引いて頻度差を縮める（短期）；
        \textit{(ii)} $\kappa^{n+1/2}$ Crank--Nicolson 化（中期）；
        \textit{(iii)} 界面厚 $\varepsilon \to 0$ 極限の sharp interface 化（長期）．
        本論文では，二相時間精度の設計次数は\textbf{$O(\Delta t^{0.56})$ が実効上限}
        として\autoref{sec:error_budget}に明示する．
\end{itemize}
検証対応：\autoref{ch:time_integration} の IMEX-BDF2 と
\autoref{sec:eikonal_reinit} の再初期化に対し，二相時間精度の実効 slope を測定する．
