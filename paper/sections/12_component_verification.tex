\paragraph{目的．}
本章は，Part 2 各章に分散して提示した
数値要素のうち，単体実験へ分離できるものを，
Part 2（§\ref{sec:CCD}--§\ref{sec:grid}）の
理論定義から検証可能な実験指標まで対応付け，
\textbf{各数値要素が単体で設計通りの数学的精度を有する}
ことを格子細分化 / MMS（製造解法~\cite{Roache2002,SalariKnupp2000}）/ 解析参照解との比較によって
系統的に確認する（Richardson 外挿に基づく観測収束次数の運用については~\cite{Roy2003}）．

本章は最終スキームを構成する各数値要素について，
式，離散化，観測指標，合格基準を一対一に対応させる精度評価である．
ここで検証されるのは，各式・離散化・実験指標が一貫した精度主張を支えること，
および単体実験で観測された限界を次章の統合実験へ持ち越すことである．第~\ref{sec:verification}章
（統合実験）はここで合格した数値要素のみを用いて，
組み立てられたアルゴリズム全体の物理的妥当性を検証する．
§\ref{sec:algorithm} の 7 ステップ更新順序，履歴圧力面加速度，
射影後の面速度，表面エネルギー毛管閉包，および
圧力ジャンプ結合系の時間発展は，単体テストではなく
第~\ref{sec:verification}章の V 系列で扱う．

\paragraph{本章の数式と表の読み方．}
U 系列の式と表は，最終ソルバの成功を一度に示すものではなく，
部品ごとの「契約」が個別に守られているかを確認する検査票である．
CCD の収束表は微分演算子の契約，
PPE の表は境界・ゲージ・係数の契約，
HFE や BF の表は界面を跨ぐデータ受け渡しの契約を読むために置く．
ここで重要なのは，数値が小さいことそのものよりも，
その数値がどの式のどの前提を検査しているかである．
したがって本章を読むときは，各 U-test を
「どの式を検査しているか」「何をまだ統合検証へ送っているか」
の二列で見ると，次章の V 系列との関係が明確になる．

\paragraph{構成（10 層 / 12 単体テスト U1--U12）．}
本章の U 系列は単体検証専用であり，
次章の V 系列（統合検証）とは実験目的と判定基準を分ける．
Part 2 の手法を依存関係の下流方向にトポロジカル整列し，
\textbf{基礎演算 → 特殊離散 → 結合系 → 否定検証 → 状態空間条件 → PhaseRegion 毛管経路の事前ゲート}の順に 10 層へ分類した：

\begin{description}\setlength{\itemsep}{0pt}
  \item[Tier I（基礎 CCD 族 + Poisson）]
        U1 = CCD/DCCD/UCCD6/FCCD 演算子族（§\ref{sec:CCD}），
        U2 = CCD--Poisson 楕円型ソルバ + PPE 境界条件（§\ref{sec:ppe_solve}）．
  \item[Tier II（非一様格子 + メトリック）]
        U3 = 界面適合非一様格子 + D1--D4 計量診断（§\ref{sec:grid}）．
  \item[Tier III（界面表現）]
        U4 = Godunov Eikonal 再初期化 + DGR バンド厚（Ridge--Eikonal 単発再構成経路とは別の数値要素；§\ref{sec:eikonal_reinit}），
        U5 = 平滑化 $H_\varepsilon$ / $\delta_\varepsilon$ モーメント精度（§\ref{sec:eps_spatial_variable}）．
  \item[Tier IV（圧力結合系）]
        U6 = 分相 PPE+HFE へ渡す DC/HFE 数値要素
        （§\ref{sec:split_ppe_framework}, §\ref{sec:hfe}）．
  \item[Tier V（界面--圧力結合の単発実験）]
        U7 = 縮約 BF 静止液滴 1-step + 面 $\mu$ 補間；FD2 不整合は合格経路ではなく低判別力を示す負対照（§\ref{sec:balanced_force}）．
  \item[Tier VI（時間積分）]
        U8 = TVD-RK3 / EXT2 / implicit-BDF2 3 種積分子（§\ref{sec:time_int}）．
  \item[Tier VII（否定検証）]
        U9 = DCCD on pressure 禁止規則の根拠（§\ref{sec:bf_p2_adjoint}）．
  \item[Tier VIII（保存形共通フラックス台帳）]
        U10 = CLS $\psi$ の FCCD 面フラックス台帳を密度と運動量へ同一写像として渡す保存条件
        （§\ref{sec:cls_advection}, §\ref{sec:algorithm}）．
  \item[Tier IX（制約付き面速度状態空間）]
        U11 = $F_w=\ker C_w$，$P_w$，$G_w=P_wG_A$，および周期壁商空間の階数条件を確認する小型代数検査
        （§\ref{sec:split_ppe_framework}, §\ref{sec:algorithm}）．
  \item[Tier X（PhaseRegion 毛管経路の事前ゲート）]
        U12 = $R_p$ を定義しない全圧力反力分解が毛管波の面上駆動を消すこと，
        成分体積 Hodge 診断が非静的波を検出すること，
        PhaseRegion graph chart の測度・HFE ジャンプ候補と支配法線方向の正則層ガードが
        圧力・速度連成前の前提になること，
        および $R_p$，Hodge 逆適用済み面補間，移動格子で保存する面量，圧力履歴座標を欠く近似分解を
        標準構成へ入れない境界を確認する
        （§\ref{sec:capillary_pressure_reaction_subspace}, §\ref{sec:ao_fast_state_space}）．
\end{description}

各 U-test は独立した単体実験として実施し，表と図には本章で採用する検証値のみを記載する．

\paragraph{Part II 主要内容の分担境界．}

複数の数値要素を同じ面空間・同じ時刻レベル・同じ界面幾何で接続して初めて意味を持つ主張は
第~\ref{sec:verification}章で統合検証する．

\paragraph{検証値の読み方．}
本章の U 系列は，性能値や実行環境の比較ではなく，
各数値要素が後続の統合検証へ渡せる数学的条件を満たすかを読む．
U12 も同じ位置付けであり，PhaseRegion 毛管経路を成功した物理検証として扱うのではなく，
全圧力像への過射影反例，非静的波の診断，PhaseRegion graph chart の測度・HFE 候補，
および追随格子の正則層ガードを通じて，
最終条件を満たさない近似分解を標準構成へ混入させないための境界を確認する．
