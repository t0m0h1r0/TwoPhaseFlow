\section{圧力 Poisson 方程式（PPE）の導出と離散化}
\label{sec:pressure}

\subsection{変密度 Projection 法の段階的導出}
\label{sec:projection_derivation}

第\ref{sec:collocate}章（§\ref{sec:helmholtz}）では，\textbf{等密度場（$\rho = \mathrm{const}$）を仮定した}
Helmholtz 分解を用いて Projection 法の幾何学的意味と「発散を取ると PPE が得られる」数学的論理を説明した．
本節ではこの定式化を気液二相流に不可欠な\textbf{変密度場}へ拡張し，
なぜ可変係数演算子 $\bnabla\cdot(1/\rho\,\bnabla p)$ が現れるかを段階的に導出する．

Projection 法（Chorin, 1968 \cite{Chorin1968}）の本質は，Navier-Stokes 方程式を
「\textbf{速度予測ステップ}」と「\textbf{圧力・速度補正ステップ}」の2段階に分解することにある．
本稿では van Kan（1986）の \textbf{IPC（Incremental Pressure Correction）法}を採用し，
予測ステップに前時刻圧力 $-\bnabla p^n$ を組み込む（§\ref{sec:time_int}，式~\eqref{eq:predictor_ab2_ipc}）．
これにより，標準 Chorin 法に固有の $\Ord{\Delta t}$ スプリッティング誤差が $\Ord{\Delta t^2}$ に低減される．
変密度場ではこの分解に特別な注意が必要であり，以下に丁寧に導出する．

\subsubsection{Step 1：予測速度（Predictor）}

時刻 $t^{n+1}$ の離散 NS 方程式（無次元形）は：
\begin{equation}
  \rho^{n+1}\frac{\bu^{n+1} - \bu^n}{\Delta t}
  = \underbrace{-\rho^{n+1}(\bu^n\cdot\bnabla)\bu^n
    + \frac{1}{Re}\bnabla\cdot[\mu^{n+1}(\bnabla\bu^{n+1}+\bnabla^T\bu^{n+1})]
    + \bm{F}_\text{ext}^{n+1}}_{\displaystyle \equiv \bm{R}^{n+1}}
  - \bnabla p^{n+1}
  \label{eq:NS_full}
\end{equation}

\noindent\textbf{時間離散化の注意：} 対流項は AB2（式~\eqref{eq:predictor_ab2_ipc}）で離散化し IPC 法に従い前時刻圧力 $-\bnabla p^n$ を陽的に加える．粘性項の表記 $\bnabla\cdot[\mu^{n+1}(\bnabla\bu^{n+1}+\cdots)]$ は概念的な後退 Euler 形式であるが，実装では Crank--Nicolson（$\mathcal{O}(\Delta t^2)$）を採用する（詳細は§\ref{sec:advection}，式~\eqref{eq:crank_nicolson}）．本節の導出では Backward Euler 形式で統一し Projection 法の圧力--速度分離構造を示す．

IPC 法では，前時刻圧力 $-\bnabla p^n$ を陽的に加えて\textbf{予測速度} $\bu^*$ を定義する：
\begin{equation}
  \rho^{n+1}\frac{\bu^* - \bu^n}{\Delta t} = \bm{R}^{n+1} - \bnabla p^n
  \quad\Longrightarrow\quad
  \bu^* = \bu^n + \frac{\Delta t}{\rho^{n+1}}\bigl(\bm{R}^{n+1} - \bnabla p^n\bigr)
  \label{eq:predictor}
\end{equation}
$\bu^*$ は一般に発散フリーではない（$\bnabla\cdot\bu^* \neq 0$）．

式~\eqref{eq:predictor} に現れる $\rho^{n+1}$ は，同一時間ステップ内で事前に完了している CLS 移流ステップ（§\ref{sec:algorithm}，Step 1）が更新した密度場 $\rho^{n+1}_{i,j} = \rho_l + (\rho_g - \rho_l)\,\psi^{n+1}_{i,j}$ から得る．全体アルゴリズムの実行順序（CLS 移流 → Predictor → PPE → Corrector）については §\ref{sec:algorithm} を参照のこと（CLS 再初期化・曲率評価を含む合計7ステップ構成）．

\subsubsection{Step 2：圧力補正（Corrector）}

式 \eqref{eq:NS_full} から式 \eqref{eq:predictor} を引くと，
圧力増分 $\delta p \equiv p^{n+1} - p^n$ と速度補正の関係が得られる：
\begin{equation}
  \rho^{n+1}\frac{\bu^{n+1} - \bu^*}{\Delta t} = -\bnabla p^{n+1} + \bnabla p^n = -\bnabla(\delta p)
  \quad\Longrightarrow\quad
  \bu^{n+1} = \bu^* - \frac{\Delta t}{\rho^{n+1}}\bnabla(\delta p)
  \label{eq:corrector}
\end{equation}

\subsubsection{Step 3：発散フリー条件から PPE を導く}

$\bu^{n+1}$ に非圧縮条件 $\bnabla\cdot\bu^{n+1}=0$ を適用する．
式 \eqref{eq:corrector} の両辺に $\bnabla\cdot(\cdot)$ を作用させると：
\begin{align}
  \bnabla\cdot\bu^{n+1}
  &= \bnabla\cdot\bu^* - \Delta t\,\bnabla\cdot\!\left(\frac{1}{\rho^{n+1}}\bnabla(\delta p)\right) = 0
  \notag \\
  &\Longrightarrow\quad
  \bnabla\cdot\!\left(\frac{1}{\rho^{n+1}}\bnabla(\delta p)\right)
  = \frac{1}{\Delta t}\,\bnabla\cdot\bu^*
  \label{eq:PPE_derivation}
\end{align}

以上の3ステップ（Predictor・PPE・Corrector）が変密度 IPC 射影法の骨格であり，
完全な7ステップ実装順序は§\ref{sec:algorithm}（特に図~\ref{fig:ns_solvers}）を参照のこと．

\noindent\textbf{PPE ソルバーの実装詳細（付録~\ref{sec:ppe_pseudotime} を参照）：}
得られた離散 PPE（式~\eqref{eq:PPE_varrho}）をどのように高速・安定に解くか――仮想時間スウィープ法，
欠陥補正，最適 $\Delta\tau$ の選択，局所時間刻み，ゲージ固定――の実装詳細は
付録~\ref{sec:ppe_pseudotime} にまとめる．
本章（§8）は数学的物理と離散化ロジックのみを扱う（Modification IV: 実装詳細は付録へ）．

\noindent\textbf{初期条件 $p^0$ の設定（$t=0$ での圧力場）：}
IPC 法の Predictor は前時刻圧力 $p^n$ を用いるため，$n=0$ における $p^0$ が必要となる．
推奨される初期化戦略は問題に応じて以下から選択する：
\begin{itemize}
  \item \textbf{静止液滴（検証問題）：} Young-Laplace 解 $p^0 = -\sigma\kappa_0 = \sigma/R$（液相内），$p^0 = 0$（気相）を解析的に与える（$\kappa_0 = -1/R < 0$；$[p] = p_{\text{gas}}-p_{\text{liq}} = \sigma\kappa_0 < 0$ より $p_{\text{liq}} = -\sigma\kappa_0 > 0$）．
  \item \textbf{一般問題（高密度比を含む）：} $n=0$ でのみ PPE（式~\eqref{eq:PPE_derivation} に $\delta p = p^{n+1}$ を代入）を $p^0 = 0$ 初期値で解き，得られた圧力場を $p^0$ として設定する（1回追加求解のコスト）．
  \item \textbf{簡易設定：} $p^0 = 0$ のまま発進する．IPC の O(Δt²) 精度は定常後に達成され，初期の数タイムステップ（$O(1)$ 回）では O(Δt) の過渡誤差が生じる可能性があるが，多くの実用問題では影響は無視できる．
\end{itemize}

一様密度（$\rho=\text{const}$）では $\bnabla^2 p = \rho\,\bnabla\cdot\bu^*/\Delta t$ になる．
変密度場では密度が空間的に変化するため\textbf{可変係数の拡散型演算子}が現れる．
本稿の実装ではこれを product-rule で展開した微分形式：
\begin{equation}
  \nabla\cdot\!\left(\frac{1}{\rho}\nabla p\right)
  = \frac{1}{\rho}\nabla^2 p - \frac{\nabla\rho}{\rho^2}\cdot\nabla p
  \label{eq:varrho_product_rule}
\end{equation}
として CCD 法（$\Ord{h^6}$）で求解する（§\ref{sec:ccd_poisson_matrix}，§\ref{sec:ccd_2d_full}）．
FVM 離散化との連続体レベルでの等価性と調和平均の根拠については §\ref{sec:fvm_face_coeff} 参照．

\noindent\textbf{展開式の2項の精度評価（界面近傍の注意）：}
式~\eqref{eq:varrho_product_rule} の2項をそれぞれ CCD で評価するとき，
平滑領域では両項とも $\Ord{h^6}$ 精度で評価される．
しかし界面近傍では $|\nabla\rho| \sim \rho_l/\varepsilon$（密度勾配が界面厚さ $\varepsilon \sim h$ の逆数で増大）となるため，
第2項 $-(\nabla\rho/\rho^2)\cdot\nabla p$ の係数 $|\nabla\rho/\rho^2|$ が $O(1/h)$ に増大する．
この増幅により，第2項への CCD の打ち切り誤差 $\Ord{h^6}\cdot O(1/h) = \Ord{h^5}$ が界面近傍で顕在化しうる．
ただし，Balanced-Force 条件（§\ref{sec:balanced_force}）のもとでは
PPE 左辺と DCCD 散逸機構の間で密度勾配項が整合的にキャンセルされるため，
実質的な寄生流れへの影響は GFM 界面処理誤差に吸収される
（詳細は §\ref{sec:balanced_force} 参照）．
実用上，$\varepsilon \sim h$ の設定では界面近傍での有効精度は $\Ord{h^4}$ 以上に保たれることが期待される（理論的には $\Ord{h^5}$；§\ref{sec:balanced_force} 参照）．

\subsection{Projection 法と PPE}
\label{sec:ppe}

\begin{equation}
  \bnabla\cdot\!\left( \frac{1}{\rho^{n+1}} \bnabla(\delta p) \right) = \frac{1}{\Delta t}\, \bnabla\cdot \bu^{\ast}
  \label{eq:PPE_varrho}
\end{equation}
