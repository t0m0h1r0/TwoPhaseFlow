\subsection{毛管波：PhaseRegion グラフチャートの正規経路}
\label{sec:val_capillary}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と理論基準}

毛管波ベンチマークは，平坦界面に小振幅の正弦波摂動を与え，
表面張力による復元振動を観察する最も単純な移動界面テストである．
本節では，第~\ref{sec:ao_fast_state_space}節で導いた
PhaseRegion 正規経路を標準の入口として用いる．
紙面に載せる可視化時刻と図は，後から都合よく選ぶのではなく，
同じ再現設定で固定された時刻列から読む．
すなわち，所有者は
\[
  \Omega_g(t)=\{y>\eta(x,t)\},
  \qquad
  \Gamma_h(t)=\partial\Omega_g(t)
\]
であり，セル測度は $q_g=Q_h(\Omega_g)$，
液相測度は $q_l=|C|-q_g$ として従属的に読む．
$\eta$ はグラフチャート，$\phi$ は必要なら作るゲージであり，
どちらも単独では物理状態の所有者ではない．

初期界面は
\begin{equation}
  y_\Gamma(x,0)=y_0 + A_0\cos(kx+\theta_0),
  \qquad
  k=\frac{2\pi m}{L_x}.
  \label{eq:capillary_wave_initial}
\end{equation}
初期条件は水--空気の実スケールとして
$y_0=0.01\,\mathrm{m}$，$A_0=2.0\times10^{-4}\,\mathrm{m}$，
$m=2$，$L_x=0.02\,\mathrm{m}$，$\theta_0=0$ とする．
上下壁の間に置かれた有限深さ二流体の小振幅理論では
\begin{equation}
  \omega^2
  = \frac{\sigma k^3}
         {\rho_l\coth(kh_l)+\rho_g\coth(kh_g)},
  \qquad
  h_l=h_g=0.01\,\mathrm{m},\quad g=0,
  \label{eq:flat_capillary_dispersion}
\end{equation}
を位相基準に置く~\cite{Prosperetti1981}．
本設定では
\[
  \omega_\sigma
  =134.419859151\,\mathrm{s^{-1}},\qquad
  T_\sigma=\frac{2\pi}{\omega_\sigma}=0.046742983863\,\mathrm{s}
\]
である．

離散経路は次である：
\[
  \eta
  \longrightarrow
  q_l=Q_h(\eta)
  \longrightarrow
  q_g=|C|-q_l
  \longrightarrow
  \Omega_g\text{ を所有する PhaseRegion}
  \longrightarrow
  E_h^{(2)}[\eta].
\]
$Q_h$ は列ごとの厳密 P1 グラフ列積分で評価し，
$E_h^{(2)}$ は同じグラフ周長の二次変分として
線形毛管波モードを進める．
時間発展は velocity-Verlet 法による縮約チャート時間発展であり，
圧力・速度場へ PhaseRegion の面力を渡す段階は未接続である．
したがって本節の合格条件は，厳密解との振幅・速度・エネルギー比較，
$Q_h$ 測度残差，体積ドリフト，GPU 経路，および
力連成を採用しないことの明示である．

\begin{itemize}[noitemsep,leftmargin=2em]
  \item グリッド：$[0,0.02]^2\,\mathrm{m}^2$，$32^2$，
        $x$ 周期，$y$ 壁面，非一様診断格子
  \item 相領域所有者：上側気相
  \item チャート：$x$ 周期グラフチャート
  \item 測度：列ごとの厳密 P1 グラフ列積分
  \item 力連成：圧力・速度場へは未接続
  \item 時間：$T=T_\sigma=0.046742983863\,\mathrm{s}$，
        CFL 係数 $1.0$ から毛管 CFL を評価し，
        $3103$ step，$\Delta t=1.506380401644\times10^{-5}\,\mathrm{s}$
  \item GPU：測度集約と所有者写像を GPU 配列上で実行
  \item 可視化時刻：$0,T/4,T/2,3T/4,T$
\end{itemize}

\subsubsection{結果と判定}

再現設定で固定した 1 周期計算は，GPU 経路で合格した．
結果は表~\ref{tab:ch14_phase_region_capillary_graph} に示す．
終端時刻は $T_\sigma$ と一致し，
終端振幅 $1.999999999999\times10^{-4}\,\mathrm{m}$ は
厳密基準 $2.0\times10^{-4}\,\mathrm{m}$ と
$1.65\times10^{-10}\,\mathrm{m}$ 以下の最大誤差で一致した．
測度残差は全時刻で 0，体積ドリフトは
$5.43\times10^{-20}$ 以下である．
最大 step 時間は $3.86\times10^{-2}\,\mathrm{s}$ であり，
$0.2\,\mathrm{s/step}$ の GPU 目標を満たした．

この結果の判定は，
「PhaseRegion が界面を所有し，$q$ は測度，
$\phi/\eta$ はチャートである」という第~\ref{sec:ao_fast_state_space}節の
状態所有者が，毛管波グラフチャートでは可視化つきで動作した，
である．
本節の正規経路は，$\Omega_g$ から $Q_h$ と $E_h$ を同時に読み，
測度残差を 0 として確認する縮約チャート問題である．

一方で，図~\ref{fig:ch14_capillary_yaml_velocity_pressure_snapshots} の
圧力・速度場は，固定時刻で保存した可視化診断であって，
PhaseRegion から得る面力を PPE や速度補正へ渡せたことを
主張する証拠ではない．
その契約は別ゲートであり，本節では \texttt{force\_admissible=0} を
「力連成未採用」として明示したまま合格とする．
