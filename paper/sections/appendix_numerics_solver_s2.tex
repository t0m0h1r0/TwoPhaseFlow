\subsection{Thomas 三重対角係数の導出}
\label{app:sweep_thomas_coeff}
% ═══════════════════════════════════════════════════════════════════════════════

本付録は §\ref{sec:defect_correction} で定義した欠陥補正法（式~\eqref{eq:defect_correction_linear}）において，
左辺の有限差分演算子 $[\frac{1}{\Delta\tau}\mathbf{I} - \mathcal{L}_{FD}]$ を
1方向 Thomas 法で求解するための三重対角係数を導出する．
スウィープ型前処理を採用する場合の係数表であり，現行標準の
\texttt{PPESolverDefectCorrection} では base solver の選択に応じて参照する．

\subsubsection*{変密度 FD 演算子の離散化}

PPE の可変係数 Laplacian を $x$ 方向（格子幅 $h$，節点数 $N+1$）に対し 2 次精度中心差分で離散化する：
\begin{equation}
  (\mathcal{L}_{FD,x}\,q)_i
  = \frac{1}{\rho_i}\,\frac{q_{i-1} - 2q_i + q_{i+1}}{h^2}
    - \frac{(\partial\rho/\partial x)_i}{\rho_i^2}\,\frac{q_{i+1} - q_{i-1}}{2h}
  \label{eq:lfd_x_disc}
\end{equation}
ここで $\rho_i$ は密度，$(\partial\rho/\partial x)_i$ は CCD で事前計算した密度勾配（反復中凍結）．

\subsubsection*{各方向の更新方程式}

欠陥補正の各反復（式~\eqref{eq:defect_correction_linear}）を
$x$・$y$ 方向の2段スウィープに因子分解する：

\textbf{第1スウィープ（$x$ 方向）：}
\begin{equation}
  \Bigl[\tfrac{1}{\Delta\tau_{i,j}}\,\mathbf{I} - \mathcal{L}_{FD,x}\Bigr]\,q_{i,j} = R_{i,j}
  \label{eq:sweep_x}
\end{equation}

\textbf{第2スウィープ（$y$ 方向）：}
\begin{equation}
  \Bigl[\tfrac{1}{\Delta\tau_{i,j}}\,\mathbf{I} - \mathcal{L}_{FD,y}\Bigr]\,\Delta_{i,j} = q_{i,j}
  \label{eq:sweep_y}
\end{equation}

各スウィープでは「解く軸」に沿った三重対角系を独立な「断面バッチ」として同時求解する（全断面ベクトル化 Thomas 法）．

\subsubsection*{三重対角係数（内部節点 \texorpdfstring{$1 \le i \le N-1$}{1 ≤ i ≤ N-1}）}

式~\eqref{eq:lfd_x_disc} を $-\mathcal{L}_{FD,x}$ として式~\eqref{eq:sweep_x} の係数を整理する：
\begin{align}
  a_i &= -\frac{1}{\rho_i\,h^2} + \frac{(\partial\rho/\partial x)_i}{2\rho_i^2\,h}
          &&\text{（下対角：$q_{i-1}$ の係数）}
  \label{eq:coeff_a}\\
  b_i &= \frac{1}{\Delta\tau_{i,j}} + \frac{2}{\rho_i\,h^2}
          &&\text{（主対角：$q_i$ の係数）}
  \label{eq:coeff_b}\\
  c_i &= -\frac{1}{\rho_i\,h^2} - \frac{(\partial\rho/\partial x)_i}{2\rho_i^2\,h}
          &&\text{（上対角：$q_{i+1}$ の係数）}
  \label{eq:coeff_c}
\end{align}

\textbf{主対角優位の確認：}
$\rho_i > 0$，$h > 0$，$\Delta\tau_{i,j} > 0$ より $b_i = 1/\Delta\tau_{i,j} + 2/(\rho_i h^2) > 0$．
密度場が十分に滑らかで $|(\partial\rho/\partial x)_i|\,h/(2\rho_i) < 1$ が成立するとき
$|a_i|+|c_i| = 2/(\rho_i h^2)$ となり，$|b_i| - |a_i| - |c_i| = 1/\Delta\tau_{i,j} > 0$ であるため，
行列は\textbf{厳密対角優位}（strict diagonal dominance）を満たし，
ピボッティングなしの Thomas 法は数値的に安定である（$b_i - a_i c_{i-1}/b_{i-1} > 0$）．

\subsubsection*{境界節点（Neumann BC：ゴーストセル反射）}

壁面節点（$i=0$ および $i=N$）では\textbf{ゴーストセル反射}により Neumann 条件
$\partial p/\partial n|_\text{wall} = 0$ を厳密に離散化する．
左壁（$i=0$）のゴーストセル則 $q_{-1} = q_1$ を式~\eqref{eq:lfd_x_disc} に代入すると
$\nabla\rho \cdot \nabla p$ 交差項（反対称差分 $q_1 - q_{-1}$）が消滅し，
境界スタンシルは：
\begin{equation}
  (\mathcal{L}_{FD,x}\,q)_0 = \frac{2(q_1 - q_0)}{\rho_0\,h^2}
  \label{eq:lfd_bc_ghost}
\end{equation}
$(1/\Delta\tau - \mathcal{L}_{FD,x})$ を整理した三重対角係数は：
\begin{align}
  &\text{左壁 }(i=0):\quad
    a_0 = 0,\quad
    b_0 = \tfrac{1}{\Delta\tau_0} + \tfrac{2}{\rho_0 h^2},\quad
    c_0 = -\tfrac{2}{\rho_0 h^2},\quad
    R_0 \text{ そのまま}
  \label{eq:sweep_bc_left}\\
  &\text{右壁 }(i=N):\quad
    a_N = -\tfrac{2}{\rho_N h^2},\quad
    b_N = \tfrac{1}{\Delta\tau_N} + \tfrac{2}{\rho_N h^2},\quad
    c_N = 0,\quad
    R_N \text{ そのまま}
  \label{eq:sweep_bc_right}
\end{align}
境界での右辺 $R_{0/N}$ はゼロに落とさず残差をそのまま与えるため，
補正量 $\delta p$ も壁面 Neumann 条件（$\partial\,\delta p/\partial n = 0$）を自然に満足する．

\noindent\textbf{旧実装（$b_0=1$，$c_0=0$，$R_0=0$）との比較：}
旧実装は壁面増分を強制ゼロとする弱形式であり，境界の CCD Laplacian 評価
（CCD は $\partial p/\partial n=0$ を片側コンパクトスタンシルで実装）との整合が取れない．
現行の Neumann ゴーストセル BC は $L_H$（CCD）と $L_L$（FD）が壁面での
$\partial p/\partial n = 0$ を\textbf{同一の物理的条件}として共有するため，
DC 反復が収束した後に解 $p^*$ が自然な壁面 Neumann 条件を満足する
（\texttt{FDPPEMatrix}，\texttt{thomas\_sweep\_1d} 参照）．

\subsubsection*{ゲージ固定}

PPE は Neumann BC のみでは定数不定（零空間）であるため，
中央節点 $(N/2, N/2)$ を圧力ゲージピンとして $(\delta p)|_\text{pin} = 0$ に固定する
（各スウィープ後に強制適用；§\ref{sec:pseudo_convergence} 参照）．
正方形領域の 4 つの対称操作（$x$・$y$ 反転，対角置換）に対して中心節点は不変であり，
ゲージ固定がシミュレーションの物理的対称性を破らない．

\subsubsection*{計算量}

格子点数を $N_x \times N_y$ とすると：
\begin{itemize}
  \item $x$ スウィープ：$N_y$ 本の $N_x$ 節点三重対角系を並列 Thomas 法で求解 — $\mathcal{O}(N_x N_y)$
  \item $y$ スウィープ：$N_x$ 本の $N_y$ 節点三重対角系を並列 Thomas 法で求解 — $\mathcal{O}(N_x N_y)$
  \item CCD 残差評価：$x$・$y$ 各方向 CCD 計算 — $\mathcal{O}(N_x N_y)$
\end{itemize}
1反復あたりの総計算量は $\mathcal{O}(N^2)$（$N = N_x = N_y$）であり，
Krylov 法（GMRES 等，前処理なし）の1ステップと同オーダーだが，
Thomas 法の定数係数が小さく実測上の反復コストが低い．
