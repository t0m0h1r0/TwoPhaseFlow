\subsubsection{U7：BF 静止液滴 1-step + face interp [Tier VII]}
\label{sec:U7_bf_static_droplet}

\paragraph{検証対象．}
第~\ref{sec:balanced_force}章 で導入した Balanced--Force（BF）結合~\cite{Francois2006}
（§\ref{sec:bf_p2_adjoint}：表面張力 $\sigma \kappa \hat{n}$ と圧力勾配の
離散レベル整合）の \textbf{Young--Laplace 圧力跳び再現精度}と，
CSF 表面張力モデルの基準実装~\cite{Brackbill1992}・寄生流れの抑制系統解析~\cite{Popinet2009,Renardy2002}を踏まえ，
セル界面での粘性係数補間（§\ref{sec:mu_face_interpolation}：算術 / 調和 / VoF 重み付け）の
\textbf{単体検証}を行う．

\paragraph{設定．}
\begin{description}\setlength{\itemsep}{0pt}
  \item[U7-a] BF 静止液滴 1-step（速度 0 初期化 + 1 ステップ pressure solve）．
        $R=0.25$ 円界面，$\sigma = 1$．Young--Laplace 解析値 $\Delta p^\star = \sigma/R = 4$．
        \textbf{match}（$\nabla p$ と $\nabla\psi$ がともに CCD）と
        \textbf{mismatch}（$\nabla p$ は CCD, $\nabla\psi$ は FD2）の 2 モード．
        密度比は両モードとも $\rho_l/\rho_g = 1000$．
        $N \in \{32,64,128\}$．
  \item[U7-b] face $\mu$ 補間：3 種（arith / harm / vof-weighted）．
        $N \in \{32,64,128,256,512\}$．$\mu$ ジャンプ $\mu_l/\mu_g = 100$，
        界面非整合グリッドで $L_\infty$ 誤差を測定．
\end{description}

\paragraph{face $\mu$ 補間の位置づけ．}
本 U7-b の 3 種補間は \textbf{実験スクリプト内 inline} で実装した
比較指標であり，現状 \texttt{src/twophase} には named API
（例：\texttt{face\_mu\_arithmetic}, \texttt{face\_mu\_harmonic},
\texttt{face\_mu\_vof}）として独立関数で公開してはいない．
production 経路では FCCD face value（§\ref{sec:mu_face_interpolation}）が
$\mu$ を含む全 face 量を統一的に扱うため，
本実験は『製造解法 + 高ジャンプ条件』下で 3 種補間の
収束次数（$\Ord{h^2}$）と相互優劣を 1 回限り定量化する
対比サンドボックスとして position する．
3 種いずれか単独を production 既定にするか，
ジャンプ大きさ依存で切替えるかは
§\ref{sec:mu_face_interpolation} の運用ガイドに従う．

\paragraph{判定と次章への接続．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item U7-a：$N \ge 64$ で BF 結合が Young--Laplace 跳び $\Delta p = \sigma/R$ を
        2\% 以下で再現．FD2 $\nabla\psi$ mismatch でも本 one-step 圧力跳び指標では match と同精度
        （差は $\le 4\times 10^{-3}$ で，$\Delta p$ 自体の絶対値 $\sim 4$ に対し $0.1\%$ オーダー）．
        \textbf{本 one-step 指標は match/mismatch の判別力が低い}：
        BF 結合の整合性破壊は時間積分による寄生流れ蓄積で初めて顕在化するため，
        単一 pressure solve では $\sigma\kappa\hat{n}$ と $\nabla p$ の差は曲率場
        $\kappa = \nabla\cdot(\nabla\psi/|\nabla\psi|)$ の評価誤差を介して間接的に
        現れるのみで，CCD vs FD2 の格子モード差を有意に分離しない．
        判別力の高い長時間版は次章 V (§\ref{sec:twophase_static_longterm}) の
        200--step 静止液滴で扱う（match/mismatch の分離は寄生流れ累積で 1 桁以上）．
        寄生流れ速度が $h$ 細分化で減衰（寄生流れ抑制設計通り）．
        §\ref{sec:bf_p2_adjoint} の理論と整合．\checkmark
  \item U7-b：3 種補間とも slope $\approx 1.9$（$\Ord{h^2}$）．
        高ジャンプ $\mu_l/\mu_g = 100$ でも安定．
        §\ref{sec:mu_face_interpolation} の選択基準（算術 = VoF が低分散，
        調和は係数のみ大きい）と一致．\checkmark
\end{itemize}
検証対応：
式~\eqref{eq:bf_discrete_balance}（BF 離散整合条件）に対し，
Young--Laplace 跳びと寄生流れを測定する．
face $\mu$ 補間については §\ref{sec:mu_face_interpolation} の 3 種平均の誤差を比較する．
