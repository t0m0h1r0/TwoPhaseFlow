\section{まとめと今後の課題}
\label{sec:conclusion}

\subsection{本稿の貢献のまとめ}

本稿では，気液二相流の高精度数値シミュレーションのための完全な定式化を示した．
採用した手法と，それが解決する具体的問題の対応を表\ref{tab:contributions}に整理する．

\toprule

特に以下の3点が技術的に重要な貢献である：

\begin{enumerate}
  \item \textbf{質量保存性：}
        CLS 法の導入により，標準 Level Set 法の根本的弱点（非保存形移流による質量損失）を克服した．
        $\psi(1-\psi)$ 型の圧縮項と拡散項の競合が界面厚さを安定に維持し，
        長時間計算でも $\int\psi\,dV \approx \text{const}$ が保たれる（第\ref{sec:verification}章）．

  \item \textbf{高精度空間微分：}
        CCD 法の係数を，テイラー展開による厳密な代数的手続きで導出した（第\ref{sec:CCD}章）．
        $O(h^6)$ 精度の曲率計算は，表面張力項の寄生流れを大幅に抑制する．

  \item \textbf{安定な圧力-速度連成：}
        Rhie--Chow 補正における調和平均密度の採用と，変密度 PPE の導出を整合的に示した（第\ref{sec:collocate}，\ref{sec:pressure}章）．
        特に密度比 $\rho_l/\rho_g = O(1000)$ の場合，算術平均と調和平均の差が250倍に達することを
        定量的に示し，調和平均の必要性を明確にした（第\ref{sec:collocate}章）．
\end{enumerate}

\subsection{今後の課題}

本稿で示した定式化は2次元の等温非圧縮流体を対象としている．
実問題への適用に向けた主要な拡張方向を以下に示す．

\begin{enumerate}
  \item \textbf{3次元実装と並列化：}
        3次元への拡張は概念的に直接的だが，CCD 法のブロック Thomas 法および
        Rhie--Chow 補正のフェイスフラックス計算を3方向に拡張する必要がある．
        大規模並列計算（MPI + OpenMP）での領域分割は，コンパクトスキームの
        非局所性（境界メッセージ通信）への対応が実装上の主要課題となる．

  \item \textbf{適応的格子細分化（AMR）との統合：}
        界面近傍のみを動的に高解像度化する AMR（Adaptive Mesh Refinement）と
        CLS 法を組み合わせることで，計算コストを大幅に削減できる．
        ただし，粗・密格子の境界での CCD スキームの整合性が技術的課題となる．

  \item \textbf{表面張力の陰的処理：}
        毛管波制約（$\Delta t_\sigma \propto \sqrt{\Delta x^3}$，第\ref{sec:time}章）を除去するため，
        表面張力項を陰的に扱う手法（Implicit Surface Tension）の導入により，
        高表面張力問題での計算効率を向上させる．
\end{enumerate}
\begin{enumerate}[resume]
  \item \textbf{相変化（蒸発・凝縮）の組み込み：}
        沸騰・蒸発問題では，界面でのエネルギー方程式と質量フラックスの結合が必要となる．
        CLS フレームワークは，界面での境界条件（Stefan 条件）の自然な実装を可能にする．

  \item \textbf{乱流モデルとの統合：}
        高 Reynolds 数流れでは，大渦シミュレーション（LES）または
        RANS モデルとの結合が必要である．
        変密度場での乱流モデリングは，特に気液界面近傍での渦粘性の扱いに課題がある．
\end{enumerate}

本稿の定式化が，気液二相流数値解析の研究・教育に貢献することを期待する．
