\section{まとめ}
\label{sec:conclusion}

本稿では，高密度比・高粘性比を持つ気液二相流の高精度シミュレーション手法について解説した．
Conservative Level Set 法による質量保存性の向上，CCD 法による高精度微分，
および Projection 法による堅牢な圧力速度結合を統合することで，
微細な界面構造を含む複雑な流動現象を正確に捉えることが可能となる．

今後の課題として，3次元問題への拡張，適応的格子細分化（AMR）の導入，および分散関係の定量的な検証が挙げられる．
