\label{sec:single_phase_ns}

\subsubsection{V1：Taylor--Green 渦エネルギー減衰}
\label{sec:energy_conservation}

\paragraph{検証対象．}
密度比 $1$ の Navier--Stokes 解 (Taylor--Green 渦~\cite{Brachet1983}; \autoref{sec:numerical_integrity} 式~\eqref{eq:CCD_I})
を CCD 空間離散 (\autoref{sec:ccd_def}) + AB2 predictor (\autoref{ch:time_integration}) +
PPE 圧力射影 (\autoref{sec:ppe_solve}) で解いた際，閉形 $E(t) = E_0 \exp(-2\nu k^2 t)$
に対する誤差と非圧縮拘束を確認する．
旧実装は FFT spectral projection であり CCD/PPE の検証ではなかったため，
現 V1 は project CCD 微分と FVM PPEBuilder 射影を直接用いる．
ただし V1-b の時間 sweep は production IPC/rotational projection ではなく，
非増分 standard projection~\cite{Chorin1968,BrownCortezMinion2001,Guermond2006} を測る
縮約実験であるため，判定基準は AB2 単体の 2 次ではなく
結合 stack の $O(\Delta t)$ trend とする．

\paragraph{設定．}
$L=2\pi$, $\nu = 0.01$（$\Re_k = 100$）, $T = 0.05$, 周期境界．
空間 sweep は $N\in\{32,48,64\}$, $\Delta t=10^{-3}$．
時間 sweep は $N=64$, $\Delta t\in\{3.846,2.000,1.000\}\times10^{-3}$．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 現 V1 は FFT spectral projection ではなく，CCD 微分 + PPEBuilder 射影を用いるため，
        紙面主張と実装 identity は一致した．
  \item 観測空間 slope は $\mathbf{2.00}$（$N=48\!\to\!64$ finest local；asymptotic
        $1.997$ on $N=32\!\to\!64$）．FVM PPE 射影が支配する結合 stack に対する
        revised criterion $O(h^2)$ trend に対して合格
        （V1-a：Type-A revised criterion；表~\ref{tab:V_verdict_types}）．
  \item 観測時間 slope は $\mathbf{1.00}$．本 script は old pressure を predictor に入れない
        standard projection であり，AB2+IPC 単体検証ではない．
        したがって revised criterion は $O(\Delta t)$ trend であり，V1-b は合格
        （Type-A revised criterion；表~\ref{tab:V_verdict_types}）とする．
  \item 非圧縮残差は $N=64$ で $6.4\times10^{-6}$．機械精度ではないが，
        FVM PPE 射影と CCD 勾配の不一致込みの residual として単調に低下する．
  \item TGV 渦度場 snapshot（\autoref{fig:V1_tgv_vorticity}）は，
        $t=0,25,50$ で 4 渦対 checkerboard 構造が粘性減衰係数 $e^{-2\nu t}$ に沿って
        振幅を失う様子を可視化し，理論減衰係数との一致 $\Ord{10^{-3}}$ を裏付ける．
\end{itemize}
検証対応：式~\eqref{eq:CCD_I}，\autoref{ch:time_integration} の時間離散，
\autoref{sec:ppe_solve} の PPE 射影に対し，TGV のエネルギー減衰と非圧縮残差を測定する．

\subsubsection{V2：Kovasznay CCD 空間残差}
\label{sec:kovasznay}

\paragraph{検証対象．}
Kovasznay 流~\cite{Kovasznay1948}（解析定常解）に対し CCD 演算子を反復適用したときの
空間残差 $R = \mathcal{L}_{\mathrm{CCD}}[u_{\mathrm{exact}}] - 0$ の格子依存性が
\textbf{真の 2D 定常 NS 解}上でどの実効次数を示すかを確認する．
純粋な CCD 微分演算子は U1 で 6 次を確認済みだが，本 V2 は非周期 outflow 方向を含む
Kovasznay 残差であり，非線形項・圧力勾配・wall one-sided CCD 境界閉包の結合を含む実験である．
境界閉包の二階微分は 4 次であるため，判定基準は CCD 内点 6 次ではなく
Kovasznay 統合残差の $O(h^4)$ trend とする．

\paragraph{設定．}
$\Re = 40$, 解析解 $u_{\mathrm{exact}}(x,y) = 1 - \mathrm{e}^{\lambda x}\cos(2\pi y)$,
$v_{\mathrm{exact}} = (\lambda/2\pi) \mathrm{e}^{\lambda x}\sin(2\pi y)$,
$\lambda = \Re/2 - \sqrt{\Re^2/4 + 4\pi^2}$．
$x\in[-0.5,1.0]$, $y\in[0,1]$．
$N \in \{32,64,128,256\}$, wall one-sided CCD boundary．
残差評価は境界 4 点を除いた interior $L_\infty$ とする．

\paragraph{判定と制約条件．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item 旧実装は manufactured periodic NS であり Kovasznay ではなかったが，
        現 V2 は解析 Kovasznay 解そのものを用いる．
  \item residual slope $\mathbf{3.95}$ → revised criterion $O(h^4)$ に対して合格
        （Type-A revised criterion；表~\ref{tab:V_verdict_types}）．
        これは CCD 単体の 6 次検証 (U1) ではなく，wall 境界閉包・圧力・非線形項を含む
        Kovasznay 統合 residual の observed order である．
  \item $\|\nabla\!\cdot u\|_\infty$ は $N=256$ で $1.7\times10^{-12}$ まで減少し，
        Kovasznay 解の非圧縮性は CCD 演算子上でも十分保たれる．
  \item Kovasznay 2D 場（\autoref{fig:V2_kovasznay_field}）の log-scale residual / divergence
        パネルは，wall 近傍 one-sided CCD stencil による境界層と bulk における
        $10^{-9}$--$10^{-13}$ までの sink 構造を直接示しており，
        wall 境界閉包・圧力・非線形項を含む統合 residual の収束性を裏付ける．
\end{itemize}
検証対応：式~\eqref{eq:CCD_I} に対し，Kovasznay 解の 2D 空間残差を測定する．
