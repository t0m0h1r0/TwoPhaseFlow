\subsection{Smoothed Heaviside 一括解法の適用限界}
\label{sec:interface_crossing}
% ═══════════════════════════════════════════════════════════════════════════════

§~\ref{sec:advection_pressure_coupling} では低密度比（$\rho_l/\rho_g \leq 5$）での
二相流パイプラインの整合性を確認したが，
§~\ref{sec:coupling_limitations} で 2 つの独立な破綻原因を特定した：
(1) IPC の $\bnabla p^n$ が界面ジャンプを横切る問題（HFE で回避可能），
(2) 変密度 PPE 自体が界面型密度場で精度劣化・発散する問題（分相 PPE でのみ解決可能）．
本節では原因 (2) を掘り下げ，密度比を系統的に増大させて
smoothed Heaviside 一括解法の\textbf{破綻閾値}を定量的に特定する．
これは「失敗を肯定的に記録する」実験であり，分相 PPE 解法の数値的根拠を提供する．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{界面型密度場での DC 収束特性}
\label{sec:varrho_dc_convergence}
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
界面型（smoothed Heaviside）密度場での変密度 PPE 欠陥補正が
密度比の増大に対してどのように劣化するかを定量的に記録する．

\noindent\textbf{§~\ref{sec:varrho_ppe_limitation} との関係：}
§~\ref{sec:varrho_ppe_limitation} では\textbf{滑らかな密度場}
（$\rho = 1 + A\sin\pi x\cos\pi y$）で $\rho_{\max}/\rho_{\min} \approx 999$ でも
$\Ord{h^{5.8}}$ を維持することを確認した．
本節はその補完として，界面型密度場での検証を実施する．

\noindent\textbf{手順とパラメタ：}
\begin{itemize}[noitemsep]
  \item 製造解 $p^* = \sin\pi x\sin\pi y$（ディリクレ BC，$p^* = 0$ on $\partial\Omega$）
  \item 密度場 $\rho = \rho_g + (\rho_l - \rho_g)\,H_\varepsilon(\phi)$，
        $\varepsilon = 1.5h$，円形界面 $R = 0.25$
  \item RHS $q = \bnabla\cdot\bigl(\rho^{-1}\bnabla p^*\bigr)$ を解析的に導出
  \item 密度比 $\rho_l/\rho_g = 2, 5, 10, 20, 50, 100$
  \item $N = 64$（固定），欠陥補正 $k = 3$
\end{itemize}

$\rho_l/\rho_g = 2, 5$ では DC は収束するが，

\textbf{3 桁以上劣化}している．
$\rho_l/\rho_g \geq 10$ では DC が発散し（$\|r_3\| > \|r_0\|$），
反復の増大や緩和係数では回復不能である．

\noindent
$\rho_l/\rho_g = 2$ では平均的な収束次数は $\Ord{h^{1}}$--$\Ord{h^{2}}$ 程度であり，
CCD の設計精度 $\Ord{h^6}$ からは\textbf{大幅に劣化}している．
$\rho_l/\rho_g = 5$ では収束次数が $\Ord{h^{0.1}}$--$\Ord{h^{0.3}}$ に留まり，
実用的な格子収束が得られない．

\noindent\textbf{劣化の原因：}
遷移層（幅 $\varepsilon = 1.5h$）内で $1/\rho$ が
$\rho_l/\rho_g$ 倍のジャンプを持つため，
界面を横切る CCD/FD ステンシルは $\Ord{1}$ の打ち切り誤差を生じる．
この「汚染」は遷移層近傍の格子点に局在するが，
PPE のグリーン関数の非局所性により解全体に伝播する．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{分相 PPE 解法の必要性}
\label{sec:split_ppe_motivation}
\label{sec:gfm_numerical_motivation}% backward-compat
% ─────────────────────────────────────────────────────────────────────────────

表~\ref{tab:varrho_breakdown} と表~\ref{tab:varrho_convergence} の結果を
統合すると，smoothed Heaviside 一括解法の適用範囲は
以下のように特定される：

\begin{tcolorbox}[resultbox, title={Smoothed Heaviside 一括解法の適用限界}]
\begin{enumerate}[noitemsep]
  \item $\rho_l/\rho_g \leq 5$：DC 収束するが，
        格子収束次数は $\Ord{h^{0.1}}$--$\Ord{h^{2}}$ に劣化
        （CCD の $\Ord{h^6}$ は界面近傍で回復不能）
  \item $\rho_l/\rho_g \geq 10$：DC 発散（反復行列のスペクトル半径 $> 1$）
  \item 水--空気系（$\rho_l/\rho_g \approx 830$）：使用不可
\end{enumerate}
\end{tcolorbox}

\noindent\textbf{なぜ破綻するか（理論的説明）：}
$1/\rho$ が遷移層幅 $\varepsilon = 1.5h$ で $\rho_l/\rho_g$ 倍のジャンプを持つとき，
界面を横切る FD/CCD ステンシルは $\Ord{1}$ の打ち切り誤差を生じる
（§~\ref{sec:ppe_condition_number}）．
欠陥補正の低次前処理 $L_L^{-1}$ も同じステンシルで構成されるため，
反復行列 $I - L_L^{-1}L_H$ のスペクトル半径が 1 を超えて発散する．
この限界は反復回数 $k$ の増大や緩和係数 $\omega < 1$ では回復不能であり，
smoothed Heaviside による一括解法の\textbf{本質的制約}である．

\noindent\textbf{結論：}
CCD の $\Ord{h^6}$ 精度を二相流で回復するには，
\textbf{すべての密度比}で分相 PPE 解法が必要である．
$\rho_l/\rho_g = 2$--$5$ でも界面近傍の精度は $\Ord{h^{1}}$--$\Ord{h^{2}}$ に劣化しており，
低密度比だから一括解法で十分ということはない．
分相 PPE 解法は各相で定数密度の PPE を独立に解き，
界面のジャンプ条件を HFE（§~\ref{sec:field_extension}）で課すことで，
不連続係数を横切るステンシルを完全に排除する．
次節で分相 PPE 解法による精度回復を実証する．

% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{分相 PPE 解法による精度回復の実証}
\label{sec:split_ppe_recovery}
\label{sec:gfm_ppe_recovery}% backward-compat
% ─────────────────────────────────────────────────────────────────────────────

\noindent\textbf{目的：}
§~\ref{sec:varrho_dc_convergence}--§~\ref{sec:varrho_grid_convergence} で観測された
密度比依存の精度劣化が，分相 PPE 解法により完全に解消されることを実証する．

\noindent\textbf{手法：}
分相 PPE 解法の核心は「各相で定数密度のラプラシアンを解く」ことにある．
Smoothed Heaviside 一括解法が $\bnabla\cdot(\rho^{-1}\bnabla p) = q$ を
変動係数 $\rho(\psi)$ のまま全領域で解くのに対し，
分相 PPE 解法は各相 $k$ で
\begin{equation}
  \nabla^2 p_k = \rho_k \, q \qquad \text{（$\rho_k$ は定数）}
\end{equation}
を独立に解き，界面のジャンプ条件は HFE で課す．
$\rho_k$ が定数であるため，離散化演算子は密度ジャンプの影響を受けない．

\noindent\textbf{手順とパラメタ：}
§~\ref{sec:varrho_dc_convergence} と同一の製造解
（$p^* = \sin\pi x\sin\pi y$，ディリクレ BC）を使用．
\begin{itemize}[noitemsep]
  \item Smoothed Heaviside：FD $\Ord{h^2}$ による変密度ラプラシアン
  \item 分相 PPE：FD $\Ord{h^2}$ による\textbf{定数密度}ラプラシアン（$\rho$ 非依存）
  \item 密度比 $\rho_l/\rho_g = 1, 2, 5, 10, 20, 50, 100, 1000$
  \item 格子収束テスト：$N = 16, 32, 64, 128$（$\rho_l/\rho_g = 10, 1000$）
\end{itemize}

Smoothed Heaviside は $\rho_l/\rho_g = 1000$ で格子精緻化により誤差が\textbf{増大}する
（収束次数が負値）．これは界面幅 $\varepsilon = 1.5h$ の縮小に伴い
$1/\rho$ ジャンプが急峻化し，ステンシル汚染が悪化するためである．
対して 分相 PPE 解法は\textbf{厳密に $\Ord{h^{2.0}}$} で収束し
（FD $\Ord{h^2}$ の設計精度通り），密度比への依存が完全にない．

\begin{tcolorbox}[resultbox, title={分相 PPE 解法の精度回復}]
\begin{enumerate}[noitemsep]
  \item 分相 PPE 解法は\textbf{すべての密度比}（$\rho_l/\rho_g = 2$--$1000$）で
        $\rho = 1$（単相）と同一の精度を達成する
  \item 格子収束次数は FD $\Ord{h^2}$（CCD に置換すれば $\Ord{h^6}$ が期待される）
  \item Smoothed Heaviside との改善比は $\rho_l/\rho_g = 100$ で $4 \times 10^5$ 倍に達する
\end{enumerate}
\end{tcolorbox}


% ─────────────────────────────────────────────────────────────────────────────
\subsubsection{PPE 離散化の選択}
% ─────────────────────────────────────────────────────────────────────────────

DC にはもう一つの\textbf{本質的制約}がある：
CCD 高次演算子 $L_H$ と低次前処理 $L_L$ の
\textbf{スペクトル不整合}により，定常反復が発散する（密度比非依存）．
7 種の代替法を系統的に検証した結果，
\textbf{FD 直接法 PPE + CCD 勾配}の併用が最適と結論された
（付録~\ref{sec:ppe_discretization_choice} にスペクトル解析と定量比較を収録）．

\begin{tcolorbox}[resultbox, title={PPE 離散化の選択（付録~\ref{sec:ppe_discretization_choice}）}]
\begin{enumerate}[noitemsep]
  \item CCD PPE の欠陥補正は $L_L^{-1} L_H$ の不定性により
        定常反復法では収束不能（密度比非依存の本質的制約）
  \item FD 直接法（\texttt{spsolve}）+ CCD 勾配の併用が最適：
        PPE 残差ゼロ＋$\Ord{h^6}$ Balanced-Force
  \item 投影不整合 $\Ord{h^2}$ は反復なし（1 回適用）のため有界であり，
        全体精度の律速因子（CSF 誤差 $\Ord{h^2}$）と同次
\end{enumerate}
\end{tcolorbox}

\begin{tcolorbox}[resultbox, title={PPE 離散化の選択（要約）}]
\begin{enumerate}[noitemsep]
  \item CCD PPE の欠陥補正は $L_L^{-1} L_H$ の不定性により
        定常反復法では収束不能（密度比非依存の本質的制約）
  \item FD 直接法（\texttt{spsolve}）+ CCD 勾配の併用が最適：
        PPE 残差ゼロ＋$\Ord{h^6}$ Balanced-Force
  \item 投影不整合 $\Ord{h^2}$ は反復なし（1 回適用）のため有界であり，
        全体精度の律速因子（CSF 誤差 $\Ord{h^2}$）と同次のため実用上問題なく，
        時間発展で蓄積しない
\end{enumerate}
\end{tcolorbox}

\noindent\textbf{本節の結論：}
smoothed Heaviside 一括解法の破綻閾値（$\rho_l/\rho_g \geq 10$）を定量化し，
分相 PPE 解法による精度回復（全密度比で $\Ord{h^2}$）を実証した．
PPE 離散化は FD 直接法 + CCD 勾配が最適である
（付録~\ref{sec:ppe_discretization_choice} にスペクトル解析を収録）．
HFE は smoothed Heaviside 圧力場には有害であり，IPC/分相 PPE 解法でのみ有効である
（付録~\ref{sec:hfe_smoothed_scope}）．
これらの知見を §~\ref{sec:error_budget} の精度バジェットに統合し，
本手法の能力と限界を総括する．
