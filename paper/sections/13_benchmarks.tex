\section{二相流ベンチマーク：設計と目標}
\label{sec:validation}
\label{sec:benchmarks}

第~\ref{sec:verification}章で確立したソルバの能力と限界を踏まえ，
本章では分相 PPE + HFE（§~\ref{sec:split_ppe_recovery}）による
高密度比対応を物理ベンチマークで検証するための設定と目標値を整理する．

§~\ref{sec:interface_crossing} において，smoothed Heaviside 一括解法の破綻閾値
（$\rho_l/\rho_g \geq 10$ で DC 発散）を定量化し，
分相 PPE 解法が全密度比（$\rho_l/\rho_g = 1$--$1000$）で
密度比非依存な $\Ord{h^2}$ 収束を達成することを実証した．
以下のベンチマークは，この分相 PPE + HFE の枠組みで実施する．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{静止液滴の寄生流れ格子収束}
\label{sec:val_static_drop}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Popinet (2009)，Francois~ら (2006)

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 静止液滴 $R = 0.25$，$\rho_l/\rho_g = 1000$，$\mathit{La} = \sigma \rho_l R / \mu^2$
  \item 格子 $N = 32, 64, 128, 256$，長時間安定性（$T = 10 / (c_\sigma / R)$）
  \item 分相 PPE 解法 + HFE IPC
\end{itemize}

\noindent\textbf{目標：}
$\|\bu_{\mathrm{para}}\|_\infty$ の格子収束（$\Ord{h^2}$ 以上），
$\Delta p$ 相対誤差 $< 0.1\%$（$N \geq 128$）．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{毛細管波の振動減衰}
\label{sec:val_capillary}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Prosperetti (1981) 解析解

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 円形液滴に $l=2$ モードの初期擾乱：$R(\theta) = R_0 (1 + \varepsilon \cos 2\theta)$，$\varepsilon = 0.05$
  \item $\rho_l/\rho_g = 1000$，$\mathit{Oh} = \mu / \sqrt{\rho_l \sigma R_0}$
  \item 分相 PPE 解法 + HFE
\end{itemize}

\noindent\textbf{目標：}
振動周波数と減衰率の Prosperetti 解析解との一致（相対誤差 $< 5\%$）．
移動界面＋表面張力の動的結合における CCD パイプラインの精度を検証．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{気泡上昇}
\label{sec:val_rising_bubble}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Hysing~ら (2009) ベンチマーク

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 2D 気泡上昇（Test case 1: $\rho_l/\rho_g = 1000$, $\mu_l/\mu_g = 100$, $\mathit{Re} = 35$, $\mathit{Eo} = 10$）
  \item $[0, 1] \times [0, 2]$，壁面 BC（上下），slip BC（左右）
  \item 分相 PPE 解法 + HFE + CLS 移流
\end{itemize}

\noindent\textbf{目標：}
終端上昇速度と気泡形状の Hysing~ら参照値との比較．
変密度＋移動界面＋表面張力の完全結合を検証．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{Rayleigh--Taylor 不安定性（$\sigma > 0$）}
\label{sec:val_rt_sigma}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Tryggvason (1988)，He~ら (1999)

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item §~\ref{sec:advection_pressure_coupling} の RT テスト（$\sigma = 0$，已完了）を拡張
  \item $\rho_l/\rho_g = 3$，$\sigma > 0$（Atwood 数 $= 0.5$）
  \item 分相 PPE 解法 + HFE + CSF/GFM
\end{itemize}

\noindent\textbf{目標：}
界面の非線形ロールアップ形状の先行研究との定性的一致．
$\sigma = 0$（§~\ref{sec:advection_pressure_coupling}）からの定量的差異を表面張力のみに帰属．
