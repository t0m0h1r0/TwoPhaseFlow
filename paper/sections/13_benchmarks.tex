\section{二相流ベンチマーク}
\label{sec:validation}
\label{sec:benchmarks}

第~\ref{sec:verification}章では構成要素ごとの精度と
NS 連成における物理的整合性を検証した．
本章では，公開ベンチマークとの定量比較により
ソルバの物理予測能力を評価する．

§~\ref{sec:verification}の結果から，balanced-force CSF + 一括 PPE パイプラインは
$\rho_l/\rho_g \leq 10$ で安定に動作する．
本章の 3 ベンチマークはこの範囲内で，
§~\ref{sec:verification}にない物理——
毛細管振動（§~\ref{sec:val_capillary}），
浮力駆動物体運動（§~\ref{sec:val_rising_bubble}），
せん断変形（§~\ref{sec:val_taylor_deformation}）——を
それぞれ解析解または公開 DNS データと比較する．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{毛細管波の振動減衰}
\label{sec:val_capillary}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Prosperetti (1981)

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 円形液滴に $l=2$ モードの初期擾乱：$r(\theta) = R_0(1 + \varepsilon\cos 2\theta)$，
        $R_0 = 0.25$，$\varepsilon = 0.05$
  \item $\rho_l = 10$，$\rho_g = 1$，$\mu = 0.05$（均一），$\sigma = 1.0$，重力なし
  \item ドメイン $[0,1]\times[0,1]$，壁面 BC，格子 $128\times128$，$T_\mathrm{final} = 5.0$
\end{itemize}

\noindent\textbf{解析解（Prosperetti 1981）：}
\begin{equation}
  \omega_0^2 = \frac{l(l-1)(l+2)\sigma}{(\rho_l+\rho_g) R_0^3},
  \qquad
  \beta = \frac{(2l^2+4l-1)\nu_l}{(2l+1) R_0^2}
  \label{eq:prosperetti}
\end{equation}
$l=2$ のとき $\omega_0 \approx 6.82$，$\beta \approx 0.048$，$T_0 \approx 0.92$．

\noindent\textbf{検証の意義：}
§~\ref{sec:advection_pressure_coupling}は毛細管 CFL の安定性限界（$\Delta t_\sigma \propto h^{3/2}$）
を検証するが，表面張力による振動周期や粘性減衰率の物理的精度は測定していない．
本ベンチマークは復元力としての表面張力と散逸の結合精度を検証する．

\noindent\textbf{目標：}
変形量 $D(t) = (L-B)/(L+B)$ の振動周期誤差 $< 10\%$，
減衰率誤差 $< 20\%$，体積保存誤差 $< 0.5\%$．

% ═══════════════════════════════════════════════════════════════════════════════
\subsection{気泡上昇}
\label{sec:val_rising_bubble}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Hysing~ら (2009) Test Case~1

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 気泡 $R = 0.25$，中心 $(0.5,\,0.5)$，ドメイン $[0,1]\times[0,2]$，壁面 BC
  \item $\mathit{Re} = 35$，$\mathit{Eo} = 10$，$\rho_l/\rho_g = 10$
        （Hysing 原典は 1000；$\rho_l/\rho_g = 1000$ には分相 PPE が必要）
  \item 導出：$\sigma = g\rho_l d^2 / \mathit{Eo} = 0.225$，
        $\mu = \rho_l\sqrt{gd}\,d / \mathit{Re} \approx 0.101$
  \item 格子 $64 \times 128$，$T_\mathrm{final} = 3.0$
\end{itemize}

\noindent\textbf{検証の意義：}
§~\ref{sec:verification}に浮力駆動物体運動のテストは存在しない．
RT 不安定性（§~\ref{sec:advection_pressure_coupling}）は界面全体の不安定性であり，
個体としての気泡の終端速度・形状変化とは異なる物理量を測定する．
本ベンチマークは浮力と表面張力の結合を動的に検証する．

\noindent\textbf{目標：}
気泡重心高さ $y_c(t)$，上昇速度 $v_c(t)$，真円度 $C(t)$ の時系列を計測し，
Hysing 参照値と定性的に比較する．
体積保存誤差 $< 0.5\%$．

\subsection{線形せん断流中の液滴変形（Taylor 変形）}
\label{sec:val_taylor_deformation}
% ═══════════════════════════════════════════════════════════════════════════════

\noindent\textbf{参照：}Taylor (1932, 1934)，Rallison (1984)

\noindent\textbf{設定：}
\begin{itemize}[noitemsep]
  \item 中性浮力液滴（$\rho_l = \rho_g = 1$），$R = 0.25$，ドメイン中央に配置
  \item Couette BC：$u(y=0) = -U$，$u(y=L_y) = +U$，せん断率 $\dot\gamma = 2U/L_y$
  \item ドメイン $[0,1]\times[0,1]$（$= [0,4R]\times[0,4R]$），重力なし
  \item $\mu_g = 0.1$，$\sigma = \mu_g \dot\gamma R / \mathit{Ca}$ より導出
  \item Capillary 数 $\mathit{Ca} \in \{0.1,\,0.2,\,0.3,\,0.4\}$，
        粘度比 $\lambda = \mu_l/\mu_g \in \{1,\,5\}$
  \item 格子 $128\times128$，$T_\mathrm{final} = 5\,\dot\gamma^{-1}$
\end{itemize}

\noindent\textbf{解析解：}
小変形（$\mathit{Ca} < 0.5$）における Taylor (1932) の公式
\begin{equation}
  D_\mathrm{theory} = \frac{19\lambda + 16}{16\lambda + 16}\,\mathit{Ca},
  \qquad D = \frac{L - B}{L + B},
  \label{eq:taylor_deformation}
\end{equation}
ただし $L$，$B$ はそれぞれ液滴の長軸・短軸．

\noindent\textbf{検証の意義：}
中性浮力により重力効果を完全に排除し，表面張力と粘性の結合精度を分離検証する．
粘度比 $\lambda = 1$ と $\lambda = 5$ の比較により，
界面を跨ぐ粘度不連続の離散化精度（variable viscosity $\mu(\psi)$）を評価する．

\noindent\textbf{目標：}
定常変形量 $D_\mathrm{ss}$ と解析値 $D_\mathrm{theory}$ の相対誤差
$|D_\mathrm{ss} - D_\mathrm{theory}|/D_\mathrm{theory} < 10\%$（$\mathit{Ca} \leq 0.3$）．

\subsection{実行結果}
\label{sec:val_results}
% ═══════════════════════════════════════════════════════════════════════════════

本章のベンチマークは §~\ref{sec:nonuniform_grid_ns}--§~\ref{sec:local_eps_validation}
で検証した非一様格子パイプラインで実行した：
界面適合格子（$\alpha = 2$），空間可変 $\varepsilon(\bx) = 1.5\,h_\mathrm{local}(\bx)$，
毎 $K$ ステップ格子リビルド．
リビルド頻度 $K$ は界面運動特性に応じて実験別に校正した（表~\ref{tab:ch13_rebuild_freq}）．

\begin{table}[htbp]
\centering
\caption{実験別のリビルド頻度校正}
\label{tab:ch13_rebuild_freq}
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{lll}
\toprule
実験 & 界面運動 & $K$ \\
\midrule
毛管波（§~\ref{sec:val_capillary}） & 定在振動（初期位置付近） & 20 \\
気泡上昇（§~\ref{sec:val_rising_bubble}） & 連続移動 $\Delta y \sim 0.1$/T=3 & 5 \\
Taylor 変形（§~\ref{sec:val_taylor_deformation}） & 非対称変形 & 1（範囲外） \\
\bottomrule
\end{tabular}
\end{table}

\noindent
毎ステップリビルド（$K = 1$）は短時間テストでは最良だが，
長時間（$>10^4$ ステップ）では線形補間の誤差が累積する．
校正された $K$ により補間呼出が $1/K$ に減少し，
誤差累積と数値拡散を抑制する．
一方，移動界面（気泡）では $K$ が大きすぎると
界面が格子の精細領域から外れる期間が長くなり物理が破綻するため，
界面運動量に応じた校正が必要である．

\paragraph{毛管波（§~\ref{sec:val_capillary}）．}
$K = 20$ で 24{,}350 ステップ（$T = 10.0$）を安定に計算した．
運動エネルギーは $l = 2$ モードに整合する振動パターンを示し（図~\ref{fig:ch13_capillary}），
体積保存誤差は $|\Delta V|/V_0 = 1.74 \times 10^{-2}$（一様格子 $1.89 \times 10^{-2}$ と同水準）．
非一様格子の質量補正が一様格子と同等の精度を維持できることを確認した．

\begin{figure}[htbp]
\centering
\includegraphics[width=0.75\linewidth]{figures/ch13_capillary_wave}
\caption{毛管波：変形量 $D(t)$ と Prosperetti 解析解の比較．
$N=64$，$T = 10.0$，$\alpha = 2$，$K = 20$．
体積保存誤差 $|\Delta V|/V_0 = 1.74\%$．}
\label{fig:ch13_capillary}
\end{figure}

\paragraph{気泡上昇（§~\ref{sec:val_rising_bubble}）．}
$K = 20$ では気泡の移動速度に対してリビルド頻度が低すぎ，
$y_c$ が初期 0.5 から 0.077 に急降下し物理が破綻した
（気泡の界面が精細領域から外れ，曲率計算が stale な格子上で劣化）．
$K = 5$ に下げたところ，単調上昇（$y_c : 0.5 \to 0.586$）が回復し
体積保存も良好（$|\Delta V|/V_0 = 7.5 \times 10^{-4}$）．
上昇速度は $v_\mathrm{rise} \approx 0.04$（一様格子ベースラインの $\sim 1/4$）と
補間誤差による数値拡散で減衰しているが，気泡形状・軌跡は定性的に正しい
（図~\ref{fig:ch13_bubble}）．

\begin{figure}[htbp]
\centering
\includegraphics[width=0.48\linewidth]{figures/ch13_rise_velocity}\hfill
\includegraphics[width=0.48\linewidth]{figures/ch13_bubble_snapshot}
\caption{気泡上昇：（左）上昇速度 $v_c(t)$，（右）$t = 3.0$ のスナップショット．
$N = 64 \times 128$，$\alpha = 2$，$K = 5$．
$y_c: 0.5 \to 0.586$，$|\Delta V|/V_0 = 7.5 \times 10^{-4}$．}

\paragraph{Taylor 変形（§~\ref{sec:val_taylor_deformation}）．}
8 ケース（$\mathrm{Ca} \in \{0.1,0.2,0.3,0.4\}$，$\lambda \in \{1,5\}$）全てが
$t < 0.05$ で blow-up した．
根本原因は明示粘性 CFL 制約で，Couette BC により初期から $u_\mathrm{max} = 1.0$
が強制されるため，$N = 128$ の細格子では $\Delta t_\nu \sim h^2/(4\nu)$ が
対流非線形性を収束させるには小さすぎる．
これは §~\ref{sec:verify_cn_viscous} で検証済みの Crank--Nicolson 陰解法が
\texttt{TwoPhaseNSSolver} に未統合であることに起因する既知の制限で，
本ベンチマークは\textbf{非一様格子の適用範囲外}とする
（粘性項の陰解法化は将来課題）．

\noindent\textbf{総括：}
3 ベンチマークのうち毛管波と気泡上昇は非一様格子パイプラインで
安定に実行でき，体積保存・定性的な物理（振動・上昇）を再現した．
Taylor 変形は明示粘性 CFL 制約により非一様格子でも依然 blow-up し，
陰解法化が必要である．
