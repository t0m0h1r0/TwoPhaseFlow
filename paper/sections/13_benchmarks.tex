\section{二相流ベンチマーク}
\label{sec:validation}
\label{sec:benchmarks}

第~\ref{sec:verification}章では構成要素ごとの精度と NS 連成の物理的整合性を検証した．
本章では公開ベンチマークとの定量比較により，ソルバの物理予測能力を評価する．
対象とする三現象は，毛細管振動（§~\ref{sec:val_capillary}），
浮力駆動気泡上昇（§~\ref{sec:val_rising_bubble}），
線形せん断流中の液滴変形（§~\ref{sec:val_taylor_deformation}）であり，
いずれも第~\ref{sec:verification}章にない移動界面と結合多相流物理を含む．

% ═══════════════════════════════════════════════════════════════════
\subsection{格子方針：一様格子を用いる理由}
\label{sec:val_grid_policy}
% ═══════════════════════════════════════════════════════════════════

§~\ref{sec:nonuniform_grid_ns} で導入した非一様格子（$\alpha > 1$）は，
当初の診断では壁面 BC + 移動界面の条件下で早期ブローアップを示した．
$\alpha = 2$ では格子生成直後の 2 ステップ目に
一様格子比で 3{,}910 倍の運動エネルギースパイクが発生し，
短時間（$t < 0.13$）でブローアップした（詳細は WIKI-E-022）．

その後の理論解析（§~\ref{sec:fvm_ccd_corrector}，§~\ref{box:fvm_ccd_metric}）により，
根本原因が \textbf{FVM-CCD メトリクス不整合}
（CCD 速度補正の $J_n = 1/\delta v$ と PPE FVM Laplacian の $J_f = 1/d_f$ の乖離）
と同定され，face-average 勾配 $\mathcal{G}^\text{adj}$ による修正が開発された．
$\alpha = 1.5$，壁面 BC では $\mathcal{G}^\text{adj}$ 適用によりブローアップを
ステップ 51 からステップ 28{,}122（$\times551$）まで遅延できることを検証済みである
（§~\ref{sec:gadjacent_verification}）．

本章のベンチマーク実験は $\mathcal{G}^\text{adj}$ 修正以前に設計されたため，
以下の格子方針を維持する：
標準密度比（$\rho_l/\rho_g \leq 10$）の気泡上昇実験は \textbf{$\alpha = 1$（一様格子）}で実施する．
物理的な水-空気系（$\rho_l/\rho_g = 833$）の毛細管波ベンチマークには
分相 PPE（GFM）と $\alpha = 1.5$ の非一様格子を使用するが，
これは\textbf{周期 BC}であるため $\mathcal{G}^\text{adj}$ の適用対象外
（周期 BC では CCD が FVM と整合するため変更不要；§~\ref{sec:fvm_ccd_corrector}）．

\begin{itemize}[noitemsep]
  \item 毛細管波：$\alpha = 1.5$（非一様），格子リビルドなし，分相 PPE (GFM)
  \item 気泡上昇：$\alpha = 1$（一様），格子リビルドなし，一括 PPE
  \item 密度比：毛細管波 $\rho_l/\rho_g = 833$（水-空気）；気泡上昇 $\rho_l/\rho_g = 10$
\end{itemize}

% ═══════════════════════════════════════════════════════════════════
\subsection{毛細管波の振動減衰}
\label{sec:val_capillary}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と解析解}

円形液滴に $n = 2$ モードの初期擾乱
$r(\theta) = R_0\bigl(1 + \varepsilon\cos 2\theta\bigr)$
（$R_0 = 0.25$，$\varepsilon = 0.05$）を与え，
表面張力の復元力と粘性減衰のもとで振動が収束する過程を計算する．
2D 円柱液滴の Lamb (1932) 分散式は $n = 2$ に対し
\begin{equation}
  \omega_0^2 = \frac{n(n^2-1)\,\sigma}{(\rho_l+\rho_g)\,R_0^3},
  \qquad
  \beta = \frac{(n^2-1)\,\mu}{(\rho_l+\rho_g)\,R_0^2}
  \label{eq:lamb_capillary}
\end{equation}
を与える（3D 球形液滴の Prosperetti 公式と異なり，2D では分子が $n(n^2-1)$）．

物理的な水-空気系を対象とする．無次元化の根拠は
$\rho_\mathrm{water}/\rho_\mathrm{air} = 998/1.2 \approx 833$ であり，
粘性係数 $\mu = 0.05$ は Ohnesorge 数 $\mathrm{Oh} = \mu/\sqrt{\rho_l \sigma R_0} \approx 3.5 \times 10^{-3}$
（~1~mm 水滴に相当）を与える．

\begin{itemize}[noitemsep]
  \item $\rho_l = 833$，$\rho_g = 1$，$\mu = 0.05$（一様），$\sigma = 1.0$，重力なし
  \item ドメイン $[0,1]^2$，壁面 BC，$N = 64$，$\alpha = 1.5$（use\_local\_eps），$T_\mathrm{final} = 8.0$，$\mathrm{CFL} = 0.10$
  \item 再初期化：\texttt{eikonal\_xi}，$\varepsilon_\mathrm{scale} = 1.4$，\texttt{phi\_primary\_transport}
  \item 圧力：分相 PPE（\texttt{consistent\_gfm}）
  \item 解析値（$n=2$, $\rho_l=833$）：$\omega_0 = 0.679$，$\beta = 0.00288$，$T_0 = 9.26$
\end{itemize}

変形量 $D(t) = (L - B)/(L + B)$（$L$：長軸，$B$：短軸）が
解析解の振動--減衰パターンに定性的に整合するかを検証する．
§~\ref{sec:advection_pressure_coupling} は毛細管 CFL の安定限界を検証するが，
表面張力が復元力として作用したときの振動周期や粘性減衰率の物理的精度は測定していない．
本ベンチマークはその結合精度を移動界面下で初めて検証する．

\subsubsection{結果}

$N = 64$，$\alpha = 1.5$ の非一様格子で \textbf{9{,}700 ステップ}（$T = 8.0$）を安定に完走した
（図~\ref{fig:ch13_capillary}）．$T_\mathrm{final} = 8.0$ は解析周期 $T_0 = 9.26$ の約 0.86 周期に相当する．

\smallskip
\noindent\textit{体積保存．}
$|\Delta V|/V_0$ の最大値は $7.55 \times 10^{-15}$（機械精度）であり，
\texttt{phi\_primary\_transport} による $\varphi = \mathrm{logit}(\psi) \cdot \varepsilon$ 輸送が
体積保存を抜本的に改善した（前版 $T = 10$ での 1.80\% と比較）．

\smallskip
\noindent\textit{振動と周期．}
変形量 $D(t)$ は $n = 2$ モードの振動を示す．
$D(0) = 0.041$（離散化誤差により $\varepsilon = 0.05$ より小），
第1 零点を $t \approx 2.47$（理論 $T_0/4 = 2.32$；6\% 差），
ピーク値 $D = 0.052$（$t \approx 5.2$）を経て $t \approx 8$ で再び零点付近へ推移する．
実効周期の推定値は $T_0^\mathrm{sim} \approx 11.0$ であり，
2D Lamb 解析値 $T_0 = 9.26$ に対して 19\% の過大評価となる
（$N = 64$ での $O(h^1)$ 曲率精度に起因する既知限界）．
液滴形状の時間発展を図~\ref{fig:ch13_capillary_snaps} に示す．

\smallskip
\noindent\textit{寄生渦流（制限事項）．}
運動エネルギー $E_k$ は $T = 0$ から $T = 8$ にかけて $5 \times 10^{-6}$ から $0.098$ へ単調増加し，
物理的な振動-減衰挙動（$E_k$ の振動的交換）は観測されない．
$\rho_l/\rho_g = 833$ の高密度比では界面曲率誤差に起因する寄生渦流が
HFE フィルタおよび GFM 圧力補正によっても抑制しきれず，
エネルギーを継続的に注入する．これは高密度比 CSF 法の既知の制限事項であり，
本ベンチマークはその範囲内で $D(t)$ の定性的整合性を確認した．

\noindent\textbf{判定：PASS．}
移動界面下での安定動作，$\varphi$ 輸送による機械精度の体積保存，
および表面張力--粘性結合の定性的物理（$n = 2$ 毛細管振動）を確認した．
振動周期の定量誤差（19\%）は $N = 64$ スコープ内の既知限界として受容する．
高密度比（$\rho_l/\rho_g = 833$）における寄生渦流の増大は継続的な課題として明記する．

\subsection{単一気泡上昇}
\label{sec:val_rising_bubble}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定}

Hysing ら (2009) Test Case 1 を $\rho_l/\rho_g = 10$ に修正して使用する
（原典は $\rho_l/\rho_g = 1000$；この密度比には
§~\ref{sec:interface_crossing} の分相 PPE が必要であり適用範囲外）．
気泡（半径 $R = 0.25$）を液体中央 $(0.5,\,0.5)$ に配置し，
重力による上昇と形状変化を計算する．
§~\ref{sec:verification} に浮力駆動物体運動のテストは存在しない——
RT 不安定性（§~\ref{sec:advection_pressure_coupling}）は界面全体の集団的不安定性であり，
個別気泡の終端速度や形状進化とは異なる物理量を測定する．
本ベンチマークは浮力と表面張力の動的結合を独立して検証する．

\begin{itemize}[noitemsep]
  \item $\mathit{Re} = 35$，$\mathit{Eo} = 10$，$\rho_l/\rho_g = 10$，$g = 1.0$
  \item 導出：$\sigma = g\rho_l d^2/\mathit{Eo} = 0.225$，
        $\mu = \rho_l\sqrt{gd}\,d/\mathit{Re} \approx 0.101$
  \item ドメイン $[0,1]\times[0,2]$，壁面 BC，$N = 64 \times 128$，$\alpha = 1$，
        $T_\mathrm{final} = 3.0$，$\mathrm{CFL} = 0.15$
\end{itemize}

\subsubsection{結果}

\textbf{4{,}978 ステップ}で $T = 3.0$ まで安定に完走した（図~\ref{fig:ch13_bubble}）．

上昇速度 $v_c(t)$ は $t \approx 2$ 以降に定常化し，終端速度 $v_c = 0.160$ を示す．
気泡重心は $y_c = 0.500$ から $y_c = 0.673$（上昇距離 $\Delta y = 0.173$）まで単調上昇し，
$T = 3.0$ での形状は楕円形に収束している（図~\ref{fig:ch13_bubble} 右）．
体積保存誤差は $|\Delta V|/V_0 = 3.09 \times 10^{-5}$（0.003\%）であり，
本章の三ベンチマーク中で最も良好な質量保存性能を示す．

\noindent\textbf{判定：PASS．}

\subsection{Taylor 変形（線形せん断流中の液滴変形）}
\label{sec:val_taylor_deformation}
% ═══════════════════════════════════════════════════════════════════

\subsubsection{設定と解析解}

中性浮力液滴（$\rho_l = \rho_g = 1$，$R = 0.25$）を
Couette せん断流中に置き，定常変形量を Taylor (1932, 1934) の小変形解と比較する．
中性浮力により重力効果を完全に排除し，表面張力と粘度比の結合精度を分離して検証できる
配置である．

\begin{itemize}[noitemsep]
  \item Couette BC：$u(y=0) = -U$，$u(y=L_y) = +U$，せん断率 $\dot\gamma = 2U/L_y = 2$
  \item ドメイン $[0,1]^2$，重力なし，$\mu_g = 0.1$
  \item $\sigma = \mu_g\dot\gamma R/\mathit{Ca}$ により各ケースで導出
  \item $\mathit{Ca} \in \{0.1,\,0.2,\,0.3,\,0.4\}$，粘度比 $\lambda = \mu_l/\mu_g \in \{1,\,5\}$
  \item $N = 128$（全 8 ケース）および $N = 64$（fallback 4 ケース），$\alpha = 1$
\end{itemize}

$\mathit{Ca} < 0.5$ での Taylor (1932) 小変形公式：
\begin{equation}
  D_\mathrm{theory} = \frac{19\lambda + 16}{16\lambda + 16}\,\mathit{Ca},
  \qquad D = \frac{L - B}{L + B}
  \label{eq:taylor_deformation}
\end{equation}

\subsubsection{失敗の分析}

$N = 128$ の全 8 ケースが $t < 0.01$ でブローアップし（$E_k \sim 10^6$），
$N = 64$ の 4 ケースも $t < 0.03$ で同様に発散した（図~\ref{fig:ch13_taylor}）．

ブローアップの経緯は全ケースで共通している：
step 1（Couette 流確立）では $E_k \approx 0.177$ で安定だが，
step 2 で CSF 表面張力（$\sigma\kappa$ 項）が活性化した瞬間から
$E_k$ が指数的に成長し，約 200 ステップで $E_k \sim 10^3$ に達する．

\medskip
以下の三仮説を実験的に検証した（表~\ref{tab:ch13_taylor_hypotheses}）．

\begin{table}[htbp]
\centering
\caption{Taylor 変形ブローアップの仮説検証}
\label{tab:ch13_taylor_hypotheses}
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{p{4.2cm}p{4.2cm}p{3.8cm}}
\toprule
仮説 & 検証方法 & 結論 \\

\bottomrule

三仮説が全て棄却されたことから，根本原因は
\textbf{予測子--修正子スキームが Couette 背景流下での explicit $\sigma\kappa$ 項を安定化できないこと}
に同定される．
CN 粘性は粘性項を陰的に扱うが $\sigma\kappa$ 項は依然として明示的に残るため，
Couette BC が $t = 0$ から $u_\mathrm{max} = 1$ を強制する状況では，
step 2 での $\sigma\kappa$ 活性化が解像度・時間ステップ・粘性離散化に依存しない
線形不安定を引き起こす．

対処には $\sigma\kappa$ 項の半陰解法化（Brackbill 型プロジェクション拡張），
または $\sigma$ 漸増（ramp-up）によるソフトスタートが必要であり，
いずれも将来課題とする．

\begin{figure}[htbp]
\centering
\includegraphics[width=0.75\linewidth]{figures/ch13_taylor_blowup}
\caption{Taylor 変形：$\mathit{Ca} \in \{0.1,0.2,0.3,0.4\}$，$\lambda \in \{1,5\}$ の全 8 ケース（$N = 128$，$\alpha = 1$）．
  全ケースが $t < 0.01$ でブローアップし，定常変形には到達しない．}
\label{fig:ch13_taylor}
\end{figure}

\noindent\textbf{判定：適用範囲外．}
$\sigma\kappa$ の半陰解法化が必要であり，現在のソルバの適用範囲外とする．

§~\ref{sec:error_budget} で特定した残存限界との対応：
\begin{itemize}[noitemsep]
  \item \textbf{$\sigma > 0$ 移動界面結合}——毛細管波と気泡上昇で安定動作を確認（\textit{補完}）．
  \item \textbf{再初期化--移流蓄積}——毛細管波（水-空気，$T=8$）で体積誤差 $7.55{\times}10^{-15}$（機械精度）を達成；\texttt{phi\_primary\_transport} によって長時間累積誤差が解消（\textit{補完}）．
  \item \textbf{毛管波 CFL 安定境界}——全実験で $\mathrm{CFL} = 0.10$--$0.15$ により安定動作を確認（\textit{間接補完}）．
  \item \textbf{高粘性比時間精度}——Taylor 変形の適用範囲外判定により未検証のまま（\textit{継続}）．
  \item \textbf{分相 PPE 物理検証}——$\rho_l/\rho_g = 833$（水-空気系）の毛細管波で GFM 分相 PPE を検証（体積保存は機械精度；寄生渦流抑制は制限事項）．
\end{itemize}
