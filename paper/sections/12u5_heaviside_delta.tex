\subsubsection{U5：$H_\eps$ / $\delta_\eps$ モーメント精度 [Tier III]}
\label{sec:U5_heaviside_delta}

\paragraph{検証対象．}
第~\ref{sec:eps_spatial_variable}章 で導入した平滑化 Heaviside $H_\eps(\phi)$ の導関数
$\delta_\eps(\phi) = H'_\eps(\phi)$ の \textbf{0 次モーメント精度}と，
スケーリング規則 $\eps = c h$（§\ref{sec:eps_spatial_variable}）の係数 $c$ 依存性を
1D 線形 $\phi(x) = x - 0.5$ 上で確認する．

\paragraph{設定．}
$N \in \{16,32,64,128,256\}$，$h = 1/N$．
\begin{description}\setlength{\itemsep}{0pt}
  \item[U5-a] スケール $\eps = 1.5 h$ の $\delta_\eps$ 0 次モーメント
        $M_0 = \sum_i \delta_\eps(\phi_i)\,h - 1$，
        1 次モーメント $M_1 = \sum_i x_i\,\delta_\eps(\phi_i)\,h - x_\Gamma$ の
        格子収束．
  \item[U5-b] $c \in \{0.5, 1, 1.5, 2\}$ で $N \in \{32, 64, 128\}$ の
        $\delta_\eps$ 0 次モーメント誤差 $\|M_0\|$ 比較（$c$-最適性検証）．
\end{description}

\paragraph{結果．}

\paragraph{判定と次章への接続．}
\begin{itemize}\setlength{\itemsep}{0pt}
  \item U5-a：0 次モーメント誤差は $N \le 64$ 域で $h^{11}$ 相当の超収束．
        $N \ge 128$ は機械精度床．§\ref{sec:eps_spatial_variable} の
        $C^\infty$ logistic 連続化の高次効果を確認．\checkmark
  \item U5-b：$c < 1$ では $\eps$ がカーネル幅未満となり stall．
        $c \in [1.5, 2.0]$ が実用最適．§\ref{sec:eps_spatial_variable} の
        スケール選択基準と整合．\checkmark
\end{itemize}
検証対応：
平滑化 Heaviside / $\delta_\eps$ 定義と §\ref{sec:eps_spatial_variable} に対し，
0/1 次モーメント誤差と $\eps=ch$ の感度を測定する．
