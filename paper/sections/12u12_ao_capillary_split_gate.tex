\subsubsection{U12：アクティブ幾何毛管分解の事前ゲート}
\label{sec:U12_ao_capillary_split_gate}

\paragraph{目的．}
U12 は，アクティブ幾何毛管分解を第~\ref{sec:verification}章以降の
標準物理経路へ混入してよいかを判定する事前ゲートである．
ここで検証するのは計算手順の性能ではなく，
式~\eqref{eq:capillary_pressure_reaction_projection} の
圧力反力部分空間 $R_p$ を定義しないまま毛管残差を読む論理が破綻すること，
式~\eqref{eq:ao_graph_hfe_jump}--\eqref{eq:ao_graph_hfe_face_law} の
$q$ 所有グラフ HFE ジャンプが現在の界面から構成されること，
および式~\eqref{eq:grid_regular_stratum_axis}--\eqref{eq:grid_regular_stratum_shift} の
正則層ガードが接線方向格子を壊さないことである．
したがって U12 は成功した物理ベンチではなく，標準経路を守るための
単体ゲートとして読む．

\paragraph{代数反例．}
固定アクティブ集合上で，節点幾何エネルギー勾配を $g$，
アクティブ制約ヤコビアンを $J_q$，SPD 計量を $W$ とする．
毛管 Riesz 代表と全セル圧力反力を同じ Schur 系
式~\eqref{eq:ao_full_pressure_cancellation} から作ると，
両者は同じ解を持つため，面上の 釣合い駆動 は厳密に $0$ になる．
このため，節点 Young--Laplace 残差 $e=g+J_q^T\pi$ が非零であることは，
面速度空間で実際に仕事をする毛管加速度の証明にならない．
アクティブ幾何毛管分解を物理経路として読むには，全圧力像ではなく
PPE 互換の $R_p(q_T)$ を先に定義し，
$r_\sigma-\Pi^{M_f}_{R_p}r_\sigma$ を直接評価する必要がある．

\paragraph{観測結果．}
表~\ref{tab:U12_ao_gate} に U12 のゲート結果を示す．
平坦界面では全経路がゼロ駆動でなければならない．
一方，毛管波では全圧力像への過射影が厳密分解で
駆動を消すこと，成分体積 Hodge 診断が非静的波を検出すること，
現在グラフから作る HFE ジャンプを圧力履歴として再利用しないこと，
および GPU 正則層ガードが支配法線方向だけを動かすことを確認する．

\paragraph{出口条件．}
U12 から第~\ref{sec:verification}章へ渡す結論は次の 3 点である．
第一に，全圧力像へ射影する古い分解と，
式~\eqref{eq:capillary_pressure_reaction_projection} で $R_p$ を定義する
アクティブ幾何毛管分解を同じ経路として扱わない．
第二に，アクティブ幾何毛管分解の精度は式~\eqref{eq:ao_fast_error_budget} の
共変量誤差と射影誤差で明示されなければならない．
第三に，$q$ 所有グラフ経路では式~\eqref{eq:ao_graph_hfe_jump}--\eqref{eq:ao_graph_hfe_face_law}
の現在グラフ HFE ジャンプを使い，前格子の圧力ジャンプ履歴を使わない．
第四に，追随格子では式~\eqref{eq:grid_regular_stratum_axis}--\eqref{eq:grid_regular_stratum_shift}
の正則層ガードを満たし，接線方向格子幅を界面退避の副作用で壊さない．
第五に，$R_p$ 未確定の近似分解，式~\eqref{eq:ao_projection_face_bridge}
の Hodge 逆適用済み面補間を満たさない分解，圧力履歴座標を持たない分解，
および式~\eqref{eq:ao_face_cochain_regrid} の移動格子面共鎖移送を欠く分解は
標準経路へ入れない．
このため，第~\ref{sec:benchmarks}章の毛管波結果は
U12 が棄却した全圧力像分解の成功証明ではない．
第~\ref{sec:ao_fast_state_space}節の最終契約を満たす経路だけを
アクティブ幾何毛管分解として読む．
