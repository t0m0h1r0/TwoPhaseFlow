\subsection{不連続界面通過時の整合性}
\label{sec:interface_crossing}
% ═══════════════════════════════════════════════════════════════════════════════

密度比 $\rho_l/\rho_g = 1000$ の二相場において，
界面を跨ぐ圧力勾配の滑らかさを検証する．
GFM 不連続面処理（§\ref{sec:gfm}）は界面に鋭い圧力跳びを許容しつつ，
速度補正項 $\bnabla p / \rho$ を界面の両側で正しく評価する．

§\ref{sec:verify_gfm} の静的テスト（表~\ref{tab:gfm_laplace}）では
GFM の Laplace 圧力跳び誤差が $N=128$ で $1.8\%$ に収束することを確認した．
§\ref{sec:verify_ppe} の変密度 Poisson テスト C-3（表~\ref{tab:ccd_test_c3}）では
密度比 $10$ での CCD-PPE 作用素の自己整合性が浮動小数点精度で成立することを確認した．

密度比 $10^3$ における界面通過時の速度連続性の定量的検証は，
GFM を統合した全二相 NS ソルバの完成後に実施する（§\ref{sec:future_work}）．
第~\ref{sec:validation}章の静止液滴ベンチマーク（§\ref{sec:val_static_drop}）が
この検証を包含する初の統合テストとなる．
