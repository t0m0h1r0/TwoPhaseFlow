\subsection{V11：アクティブ幾何毛管分解の統合前ゲート}
\label{sec:ao_fast_capillary_gate}

\paragraph{目的．}
V11 は U12 の代数診断を，第~\ref{sec:verification}章の統合検証文脈へ接続するゲートである．
標準の分相 PPE + HFE + pressure-jump + component-Hodge 経路は
V6/V7/V9 と第~\ref{sec:benchmarks}章で物理応答を評価する．
一方，アクティブ幾何毛管分解は，式~\eqref{eq:capillary_pressure_reaction_projection} の
$R_p$ が未確定のままでは標準物理経路に入れない．
V11 はこの境界を明示し，後続の毛管波結果を
アクティブ幾何毛管分解の成功証明として誤読しないための統合前検査である．

\paragraph{検査項目．}
V11 は以下を同時に要求する．
\begin{enumerate}[noitemsep]
  \item 平坦・静的ゲート：平坦界面および静的平衡では
        $a_{\sigma,\mathrm{bal}}$ が丸め誤差まで $0$ になる．
  \item 毛管波ゲート：毛管波では，全圧力像ではなく
        物理的に許容される $R_p$ を用いた
        $r_\sigma-\Pi^{M_f}_{R_p}r_\sigma$ が非零の復元向き仕事を持つ．
  \item 精度ゲート：近似分解を用いる場合は式~\eqref{eq:ao_fast_error_budget} の
        共変量誤差と射影誤差を満たす．
  \item 採用ゲート：未受理の近似分解を標準物理経路へ入れない．
\end{enumerate}

\paragraph{第14章への接続．}
V11 はアクティブ幾何毛管分解の未受理判定をもって標準毛管波の失敗とは読まない．
第~\ref{sec:val_capillary}節の毛管波は，検証済みの標準
FCCD/UCCD6/pressure-jump/component-Hodge 経路で走らせる．
逆に，その毛管波が完走しても，V11 未通過のアクティブ幾何毛管分解が
物理的に受理されたことにはならない．
この分離により，発展的な分解を保持しつつ，
未証明の圧力反力分解を本番ベンチへ混入させない．
