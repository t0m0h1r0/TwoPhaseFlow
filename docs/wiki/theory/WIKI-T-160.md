# Discrete Reinit-Aware Capillary Hodge Reference

## Curation Update

This card is retained as a discrete theory reference, not as an accepted
implementation route.  [[WIKI-T-161]] records the later fixed-stratum
trace-preserving conservative variational retraction candidate; that candidate
is now retired after ch14 N=32, T=10 long-time validation showed abnormal
droplet shape.  The surviving rule is narrower and more durable: capillary
work must use labelled transport endpoints, and reinitialization must expose
trace, surface-energy, volume, material, and topology defects rather than hide
them inside mass-only or profile-only repair.

## Claim

The capillary/reinit theory is a fully discrete contract, not a continuum
slogan and not a static/oscillating benchmark branch.

Use finite spaces:

```text
Q_h, Phi_h, U_h, P_hs
```

with production pairings:

```text
(x,y)_Q = x^T M_Q y
(u,v)_F = u^T M_F v
(p,r)_P = p^T M_P r.
```

The pairings are part of the theorem.  Replacing `M_F` by a bare Euclidean
norm changes the Hodge projection.

## State Split

The interface step must be labelled:

```text
q^n -- T_h(u_f) --> q_T
q_T -- Pi_Ridge --> q^{n+1}.
```

Only `q^n -> q_T` is generated by face velocity.  The reinitialization step is
a representation projection unless a separate metric/topology model is
declared.

## Capillary Cochain

Define a discrete surface-energy gradient by:

```text
(g_T,q_T-q^n)_Q = S_h(q_T)-S_h(q^n).
```

Then define the face cochain by transport virtual work:

```text
(c_sigma,w_f)_F = -sigma (g_T,L_T(q^theta)w_f)_Q
c_sigma = -sigma M_F^{-1} L_T(q^theta)^T M_Q g_T.
```

This is the production capillary force definition.  A curvature jump or scalar
pressure representative is valid only if it reproduces this identity in the
same weighted face space.

## Pressure-Hodge Rule

Use the full cochain in the pressure/corrector path:

```text
D_F B_F p = r_h + D_F c_sigma
a_f = B_F p - c_sigma
u_f^{n+1} = u_f^* - dt a_f.
```

The pressure-range representative is diagnostic:

```text
Pi_R c_sigma = argmin_{b in range(B_F)} ||b-c_sigma||_{M_F}
h_sigma = c_sigma - Pi_R c_sigma.
```

Static and dynamic behavior are separated by one weighted Hodge gate:

```text
static equilibrium        <=> h_sigma = 0
dynamic capillary release <=> h_sigma != 0.
```

Production replacement `c_sigma -> Pi_R c_sigma` is forbidden because it
deletes the dynamic Hodge component.

## Reinit KKT Contract

For fixed topology, Ridge-Eikonal/profile reinitialization is:

```text
minimize    1/2 ||q-q_T||_W^2 + 1/2 ||phi-phi_ref||_Z^2
subject to  q - H_eps(-phi) = 0
            C_E(phi) = 0
            V_h(q) = V_h(q^n)
            tau_h(q) = tau_h(q_T)
            S_h(q) = S_h(q_T).
```

The KKT multipliers are representation multipliers.  They are not pressure,
body force, or capillary force.

If the identities fail, record:

```text
Delta_tau_Pi = tau_h(q^{n+1}) - tau_h(q_T)
Delta_S_Pi   = S_h(q^{n+1}) - S_h(q_T)
Delta_V_Pi   = V_h(q^{n+1}) - V_h(q^n).
```

These are reinit defects or declared metric/topology events.

## Energy Ledger

The fixed-topology ledger is:

```text
K^{n+1} - K^n
+ sigma [S_h(q_T)-S_h(q^n)]
+ D_visc
+ D_metric
= R_num.
```

The stored surface change splits:

```text
S_h(q^{n+1}) - S_h(q^n) = Delta_S_T + Delta_S_Pi.
```

Only `Delta_S_T` is capillary work.  `Delta_S_Pi` cannot be used as evidence
of oscillation or damping.


## ch14 Reading

The failed oscillating-droplet result is not evidence that the deformed droplet
is a discrete static equilibrium.  It is evidence that the production path
deleted `h_sigma` and mixed transport with reinitialization.  The generic fix
is the labelled transport cochain plus explicit reinit identity/defect ledger.
