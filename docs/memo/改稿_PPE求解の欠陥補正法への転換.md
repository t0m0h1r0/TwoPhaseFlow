# 改稿案

## 1. 背景と目的

PPE（圧力ポアソン方程式）のデフォルトソルバを、従来の汎用反復法（LGMRES等）から「欠陥補正法（Defect Correction Method）」へ変更する。これに伴い、第8章の定式化、第10章の実験内容、および Appendix X の役割を抜本的に再定義する。

## 2. 構成の変更指針

### 第8章（定式化）への統合:
従来 Appendix X にあった「欠陥補正法」の説明を 8.1節（またはその直後の節）へ移動。6次精度CCD演算子 $L_H$ を直接解く困難さと、解きやすい低次演算子 $L_L$ を用いた「欠陥補正」による効率化のロジックを定式化の核心として記述する。

### 第10章（検証）の役割:
「アルゴリズムの説明」を排し、第8章で定義した手法の「性能証明」に特化させる。LGMRES等の既存手法に対する優位性と、反復回数 $k$ による精度の到達度をデータで示す。

### Appendix X の再定義:
数学的な「収束理論」および「安定性解析」の専門セクションとする。変密度場における固有値分布や、緩和係数 $\omega$ の理論的導出など、実装者ではなく理論家向けの深い内容に特化させる。

### 用語の統一:
「欠陥補正法 (Defect Correction Method)」を手法名とし、その実行過程を「方向分離スイープ (Directional Sweep)」と呼称して統一する。

## 3. 指示内容 (Tasks)

### Task A: 第8.1節付近の再構築（定式化の核心）

離散化されたPPE $L_H p = b$ に対し、直接反復が困難な理由を述べよ。欠陥補正のアルゴリズムを以下の形式で記述せよ。

$$\begin{aligned}
&\text{Defectの算出: } d^{(k)} = b - L_H p^{(k)} \\
&\text{補正ステップ: } L_L \delta p^{(k+1)} = d^{(k)} \\
&\text{解の更新: } p^{(k+1)} = p^{(k)} + \omega \delta p^{(k+1)}
\end{aligned}$$

$L_L$ を方向分離可能な三項差分等に設定することで、Appendix U (Thomas法) が直結し、計算コストが $O(N)$ になる利点を強調せよ。

### Task B: 第10章（数値検証）の改訂

実験「欠陥補正反復回数と空間精度の相関」の構成案を作成せよ。$k=1$ では2次精度に留まるが、$k=2, 3$ と増やすことで $O(h^6)$ へ漸近する様子をグラフ化するための論理を構築せよ。

### Task C: Appendix X の再構成

収束加速のための理論解析セクション。反復行列 $M = I - \omega L_L^{-1} L_H$ のスペクトル半径に関する数学的考察の骨子を作成せよ。

### Task D: 用語対照・定義表

「Defect」「Correction」「Sweep」「Virtual Time Step」の定義を整理し、特に「仮想時間刻み」が欠陥補正における「緩和係数」と数学的に等価である点に触れよ。
