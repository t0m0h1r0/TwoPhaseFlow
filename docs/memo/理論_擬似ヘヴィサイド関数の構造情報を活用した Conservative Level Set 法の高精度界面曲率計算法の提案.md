# 擬似ヘヴィサイド関数の構造情報を活用した Conservative Level Set 法の高精度界面曲率計算法の提案概要

Conservative Level Set (CLS) 法において、体積分率関数（擬似ヘヴィサイド関数）から符号付距離関数への逆変換時に生じる界面誤差と、それに起因する曲率計算の精度低下を解決する新しい手法を提案する。本手法は、Combined Compact Difference (CCD) 法等で得られる高精度な空間微分値と、擬似ヘヴィサイド関数の理論的な関数構造を直接結びつけることで、界面位置の特定および曲率の計算を代数的に行うものである。これにより、従来のロジット変換に起因する数値的不安定性を回避し、表面張力モデルにおける寄生流れ（Spurious Currents）の抜本的な抑制が期待される。

## 1. 緒言

多相流解析における Conservative Level Set (CLS) 法は、質量保存性に優れる一方で、界面の曲率 $\kappa$ を計算する際に、滑らかな体積分率関数 $\psi$ から符号付距離関数 $\phi$ への逆変換を必要とする。第10章における解析結果から、この逆変換プロセスにおいて界面誤差が顕著に増大し、結果として曲率の計算精度が著しく損なわれることが確認された。本稿では、$\psi$ の関数構造と、セル内で計算済みの勾配情報を最大限に活用することで、逆変換の悪影響を排除した高精度な曲率計算法を提案する。

## 2. 従来手法の課題

従来のCLS法では、界面プロファイルとして以下の擬似ヘヴィサイド関数 $H_\epsilon(\phi)$ を想定する。
$$\psi(x) = H_\epsilon(\phi) = \frac{1}{1 + \exp(-\phi/\epsilon)}$$
ここで、$\epsilon$ は界面厚みを制御するパラメータである。距離関数 $\phi$ を得るために、通常は以下のロジット変換（逆変換）が各格子点で行われる。
$$\phi = \epsilon \ln\left(\frac{\psi}{1-\psi}\right)$$

しかし、この変換は $\psi \to 0$ または $\psi \to 1$ の領域において勾配が特異的に増大するため、微小な数値振動が $\phi$ の巨大な誤差に変換される。さらに、点ごとに独立して逆変換を行った後で差分演算を用いて $\phi$ の高階微分（曲率）を計算するため、元々の $\psi$ が持っていた滑らかな関数構造が破壊されてしまうという構造的欠陥を抱えている。

## 3. 提案手法：核心的理論結果

### 3.1 レベルセット曲率の単調変換不変性（定理）

**メモ初版で提案したチェインルール経由の代数構成よりも、遥かに強い結果が成立する。**

**定理（レベルセット曲率の単調変換不変性）：**
$\psi = g(\phi)$ とし、$g$ を $C^2$ 級の狭義単調増加関数（$g' > 0$）とする。このとき、
$$-\nabla \cdot \frac{\nabla \psi}{|\nabla \psi|} = -\nabla \cdot \frac{\nabla \phi}{|\nabla \phi|} = \kappa$$

**証明：**
連鎖律より $\nabla \psi = g'(\phi)\,\nabla \phi$ である。$g' > 0$ より $|g'(\phi)| = g'(\phi)$ であるから、
$$|\nabla \psi| = g'(\phi)\,|\nabla \phi|$$
したがって、単位法線ベクトルは
$$\hat{\bm{n}}_\psi \equiv \frac{\nabla \psi}{|\nabla \psi|} = \frac{g'(\phi)\,\nabla \phi}{g'(\phi)\,|\nabla \phi|} = \frac{\nabla \phi}{|\nabla \phi|} \equiv \hat{\bm{n}}_\phi$$
$\hat{\bm{n}}_\psi = \hat{\bm{n}}_\phi$ が成立するから、$\kappa = -\nabla \cdot \hat{\bm{n}}$ は $\psi$ から計算しても $\phi$ から計算しても同一。□

**帰結：** CLS 法の擬似ヘヴィサイド関数 $H_\varepsilon(\phi) = 1/(1+e^{-\phi/\varepsilon})$ は狭義単調増加であるから、上の定理が直接適用できる。**ロジット逆変換は曲率計算において完全に不要である。**

### 3.2 2次元曲率公式（ψ 直接形式）

定理により、従来の $\phi$ ベースの曲率公式
$$\kappa = -\frac{\phi_y^2\,\phi_{xx} - 2\phi_x\phi_y\,\phi_{xy} + \phi_x^2\,\phi_{yy}}{(\phi_x^2 + \phi_y^2)^{3/2}}$$
において $\phi$ を $\psi$ に置き換えるだけでよい：
$$\kappa = -\frac{\psi_y^2\,\psi_{xx} - 2\psi_x\psi_y\,\psi_{xy} + \psi_x^2\,\psi_{yy}}{(\psi_x^2 + \psi_y^2)^{3/2}}$$
**この2つの式は数学的に厳密に同一の値を与える。**

### 3.3 チェインルール展開による検証（初版 §3.2 の完成）

初版で提案したチェインルールアプローチも同一の結果に到達することを検証する。

$H' = \frac{1}{\epsilon}\psi(1-\psi)$, $H'' = \frac{1}{\epsilon^2}\psi(1-\psi)(1-2\psi)$ とおくと：

**1階微分：** $\phi_x = \psi_x / H'$, $\phi_y = \psi_y / H'$

**2階微分：** $\phi_{xx} = (\psi_{xx} - H''(\phi_x)^2) / H' = \psi_{xx}/H' - (H''/H'^3)\psi_x^2$

曲率の分子を展開すると：
$$\text{分子} = \frac{1}{H'^3}\left[\psi_y^2 \psi_{xx} - 2\psi_x \psi_y \psi_{xy} + \psi_x^2 \psi_{yy}\right] - \frac{H''}{H'^5}\left[\psi_y^2 \psi_x^2 - 2\psi_x^2 \psi_y^2 + \psi_x^2 \psi_y^2\right]$$

**$H''$ 項の係数は $(\psi_x^2\psi_y^2 - 2\psi_x^2\psi_y^2 + \psi_x^2\psi_y^2) = 0$ となり完全に消滅する。**

分母は $(|\nabla\psi|/H')^3 = |\nabla\psi|^3/H'^3$ であるから、$H'^3$ が約分されて最終的に $\psi$ のみの公式が得られる。これは §3.1 の不変性定理の代数的確認にほかならない。

### 3.4 CCD 法との親和性

CCD 法は $\psi$ の解法過程で $\psi$, $\psi'$, $\psi''$ を同時に $O(h^6)$ 精度で算出する。したがって、式 (§3.2) の全ての入力が追加計算なしに利用可能である。

**従来法（$\psi \to \phi \to \kappa$）に対する利点：**
1. ロジット逆変換の**完全な省略**：飽和領域での対数関数の特異性による誤差増幅が原理的に排除
2. 逆変換後の $\phi$ に対する差分演算が不要 → $\psi$ の滑らかな関数構造がそのまま保存
3. 再初期化による Eikonal 条件 $|\nabla\phi| \approx 1$ の維持が曲率精度の前提条件ではなくなる

## 4. 実装戦略

### 4.1 ハイブリッド適用戦略

分母 $(\psi_x^2+\psi_y^2)^{3/2}$ は $\psi$ のプロファイル形状に依存する：
- 界面近傍（$\psi \approx 0.5$）：シグモイド勾配最大 $|\nabla\psi|_{\max} = 1/(4\varepsilon)$ → 安定
- 界面遠方（$\psi \to 0$ or $1$）：$|\nabla\psi| \to 0$ → ゼロ除算

実装上のハイブリッド戦略：
- **界面近傍**（$\psi_{\min} < \psi < 1-\psi_{\min}$, 推奨 $\psi_{\min}=0.01$）：ψ 直接法
- **界面遠方**：$\kappa = 0$（表面張力寄与なし）

ロバスト性：表面張力項 $\kappa\,\nabla\psi/We$ において、$|\nabla\psi| \to 0$ の領域では $\kappa$ が不定となっても積 $\kappa\,|\nabla\psi|$ は有界に留まるため、閾値選択に対する感度は低い。

### 4.2 実効界面厚み $\epsilon_{eff}$ による診断

$\psi$ による直接曲率計算では $\varepsilon$ は不要であるが、プロファイル品質の診断指標として：
$$\varepsilon_{\text{eff}, i} = \frac{\psi_i (1 - \psi_i)}{|\nabla \psi|_i}$$
- $\varepsilon_{\text{eff}} \approx \varepsilon$（設計値）：正常
- $\varepsilon_{\text{eff}} \gg \varepsilon$：数値拡散による界面幅増大

### 4.3 エルミート情報に基づく界面位置特定

サブグリッドスケールの正確な界面位置 $x_\Gamma$ （$\phi=0$ となる点）の特定において、単なる線形補間ではなく、隣接する格子点 $i, i+1$ の $\psi$ および $\psi_x$ を用いたエルミート的非線形フィッティングを行う。
$$\tanh \left( \frac{x - x_\Gamma}{2\epsilon_{\text{eff}}} \right) = 2\psi(x) - 1$$

この関係式と勾配情報を用いて関数形を再構築することで、ゴースト流体法（Ghost Fluid Method）等の境界条件処理に必要な界面位置を極めて高い精度で特定できる。

## 5. 結言

本稿では、CLS法における曲率計算の根本的改善手法を提案した。核心的結果は**レベルセット曲率の単調変換不変性**（定理 §3.1）であり、これにより：

1. $\psi \to \phi$ のロジット逆変換が**原理的に不要**であることが証明された
2. 曲率公式で $\phi$ を $\psi$ に置換するだけで数学的に厳密に同一の曲率が得られる
3. CCD 法の同時微分算出特性との組み合わせにより、追加計算なしで $O(h^6)$ 精度の曲率が得られる

初版で提案したチェインルール経由の代数構成は、不変性定理の代数的検証（$H''$ 項の消滅）として位置づけられる。今後は、多次元における混合微分の評価手法の確立と、実際の多相流ソルバーへの実装を通じた定量的な精度検証が課題である。
