## 1. 問題提起：現行 §11 の欠陥

### 1.1 二相流の時間積分テストが皆無

現行 §11 は以下の 6 節で構成されている。

| 節 | 内容 | 問題点 |
|---|---|---|
| §11.1 | 力学的バランス（静水圧・Laplace 圧単ステップ） | 単ステップのみ、動的検証なし |
| §11.2 | 保存則（TGV） | 単相のみ |
| §11.3 | NS 総合精度（Kovasznay/TGV） | 単相のみ |
| §11.4 | 二重剪断層 DCCD 安定性 | N=64 で CCD/DCCD 差が 0.01% 以下、結論が弱い |
| §11.5 | 界面通過（GFM 依存・延期宣言） | テストが実施されていない |
| §11.6 | 精度バジェット | §11.5 の欠落で根拠が弱い |

**最大の欠落**: 実装済みコンポーネント（CLS + 変密度 PPE + Hermite 延長 + Corrector）が **連携した二相流タイムステップループ**を通じて正しく機能するか否かを検証するテストが存在しない。

### 1.2 §11.5 の「延期宣言」は不十分

現行 §11.5 は「密度比 1000 の界面通過テストは GFM 未実装のため実施できない」と宣言するだけである。これは査読者に対して説得力を欠く。§10.3.1 が「界面型密度場での PPE 検証は今後の課題」と述べた空白を、**定量的な破綻データ**で埋める必要がある。

### 1.3 §11.4（二重剪断層 DCCD）の価値が低い

N=64 では DCCD と CCD の差が 0.01%、N=128 でも 0.09% にとどまる。§10.1.2 の DCCD 1D 検証（TV: 10.83 → 3.15）がより直接的な証拠であり、§11 での 2D NS 拡張は付加的な知見を与えない。

### §11.4 詳細：二相流時間積分テスト（NEW）

**目的**: 完全な二相流タイムステップループの物理的整合性確認
- CLS 界面移流 → 物性値更新 → 変密度 PPE → Hermite 延長 → Corrector
- §10.2.2 の Hermite 延長テスト（静的・1ステップ）を動的・多ステップに拡張
- Balanced-Force 条件が時間積分を通じて維持されるかの確認

**セットアップ**

| パラメータ | 値 |
|---|---|
| 計算領域 | [0,1]²、壁面 BC |
| 初期条件 | 円形液滴 R=0.25、中心 (0.5,0.5)、初速 (1,0) |
| ρ_l/ρ_g | 2（PASS期待）、5（PASS期待）、10（WARN期待） |
| We, Re | We=10、Re=100 |
| N | 32, 64, 128 |
| 積分時間 | T=0.5 |
| 時間刻み | CFL=0.5 基準 |

**合否基準**

| ρ_l/ρ_g | 判定 | 基準 |
|---|---|---|
| ≤ 5 | PASS | ‖div(u)‖∞ < 10⁻⁸、質量誤差 < 10⁻⁴ |
| = 10 | WARN | 収束劣化が観測されるが発散しない |
| ≥ 20 | 次節 §11.5 で検証 | — |

**§10 コンポーネントとの接続**

| §10 の検証 | 本テストでの役割 |
|---|---|
| §10.2.1 CLS 質量保存 O(10⁻⁵) | 動的液滴の界面追跡精度の保証 |
| §10.2.2 Hermite 延長（静的1ステップ） | 多ステップ動的問題での機能確認（「no-op」問題の解消） |
| §10.3.1 変密度 PPE（滑らか密度場） | 界面型密度場での動作の連続的検証 |
| §10.3.3 AB2 時間積分 O(Δt²) | ループ全体での時間精度の保証 |

**§11.1 との違い**: §11.1(b) は単ステップ投影（Hermite 延長が no-op）。本テストは多ステップ積分で非零圧力場での Hermite 延長が実際に呼ばれる初めてのケース。

### §11.5 詳細：変密度 PPE 破綻限界の定量化（NEW）

**目的**: smoothed Heaviside 一括解法の破綻閾値を特定し、GFM 導入の数値的根拠とする

> この実験は「失敗」を肯定的に記録する実験である。§10.3.1 が
> 「界面型密度場での検証は今後の課題」と述べた空白を埋める。

**セットアップ**: §11.4 と同一ドメイン（静止液滴、N=64、変密度 PPE）

**パラメトリックスイープ**

| ρ_l/ρ_g | 期待結果 | 判定 |
|---|---|---|
| 2 | DC 収束、Δp 誤差 < 5% | PASS |
| 5 | DC 収束、Δp 誤差 < 10% | PASS |
| 10 | 残差収束が遅化 O(10⁻³ 程度) | WARN / 境界 |
| 20 | DC 発散（‖r_k‖ 増大） | EXPECTED FAILURE |
| 50 | DC 発散 | EXPECTED FAILURE |
| 100 | DC 発散 | EXPECTED FAILURE |

**合否基準（「失敗」が合格）**: 破綻閾値が ρ_l/ρ_g = 10 前後に明確に特定できること

**§10.3.1 との関係**:
- §10.3.1 では **滑らかな密度場**（正弦波プロファイル）での O(h^5.8) を確認
- 本テストは **界面型（smoothed Heaviside）密度場**での補完
- §10.3.1 の表「varrho_interface_divergence」の N 固定・密度比変化版

**GFM への接続**: 破綻閾値の実測データは §11.6 の精度バジェットと §13（将来課題）の GFM 必要性の議論に直結する。
