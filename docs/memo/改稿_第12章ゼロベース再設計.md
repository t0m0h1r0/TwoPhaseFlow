## 1. 背景と動機

旧 §12 は「GFM 統合ソルバ」が動作する前提で設計されていた（§12.2 GFM 静止液滴、§12.3 毛細管波、§12.4 気泡上昇）。しかし以下の事実により、この前提は成立しない：

| 機能 | 状態 | 根拠 |
|------|------|------|
| GFM 統合 NS ソルバ（ρ ≥ 10） | **破綻** | exp12_3: 全9条件で2–3ステップで発散 |
| 移動界面 + σ > 0 | **破綻** | §11.4b テストB: IPC ∇p^n が界面ジャンプを横切り2ステップで発散 |
| CSF + HFE（ρ ≤ 5, 静止界面） | **動作** | exp12_2, §11.4a: 200ステップ安定 |
| RT 不安定（σ = 0） | **動作** | exp12_1: ω誤差 2.8%, 質量誤差 3e-4 |
| HFE 単体精度 | **動作** | §10.2: O(h⁷) 収束確認 |
| 分相 PPE 単体（製造解） | **動作** | §11.5: O(h²) ρ非依存 |

**結論**: §12 は現行ソルバで実際に動作する実験のみで構成し直す必要がある。

## 2. 再設計方針

1. **嘘を書かない** — 動かない実験の「予測結果」「理論的予測」ボックスは全廃
2. **実行可能な実験のみ** — CSF + HFE パイプラインで動くもの
3. **HFE の貢献を定量的に示す** — ablation study（除去実験）で効果を直接測定
4. **限界を誠実に記録** — §13 将来課題（分相 PPE 解法）への明確な接続

## 3. 実験計画

### EXP-12A: Rayleigh-Taylor 不安定性（σ = 0）

**目的**: 変密度 NS ソルバの大変形界面追跡能力を、線形理論の成長率と比較して検証する。

**設定**:
- 領域: [0, 1] × [0, 4], 壁面 BC（全辺）
- 密度: ρ_l = 3（上層, y > 2）, ρ_g = 1（下層）, At = 0.5
- 粘性: μ = 0.01（均一）, σ = 0（表面張力なし）
- 初期界面: y₀(x) = 2 + 0.05·sin(2πx)
- 格子: 64 × 256（h = 1/64）
- 時間積分: 非増分投影 + Forward Euler
- 浮力: f_buoy = −(ρ − ρ_ref)/ρ · g（静水圧分離）

**検証指標**:
- 線形成長率: ω_RT = √(At·g·k) = √π ≈ 1.7725 との相対誤差（許容 < 5%）
- 質量保存誤差: |Δm/m₀|
- 非線形段階のバブル-スパイク構造の定性的確認

**位置づけ**: σ = 0 のため HFE の圧力ジャンプ処理は不要。CLS 界面追跡 + 変密度 PPE の基盤能力を検証する。§11.4b テスト A（Galilean 不変性, σ = 0）の物理的帰結。

**実装**: experiment/ch12/exp12_1_rayleigh_taylor.py（既存、変更なし）
**状態**: **実行済み** ✓

### EXP-12B: 静止液滴 — Laplace 圧均衡と寄生流れの格子収束

**目的**: CSF + CCD 高次曲率による Laplace 圧均衡の精度と、寄生流れの格子依存性を定量化する。

**設定**:
- 領域: [0, 1]², 壁面 BC
- 液滴: R = 0.25, 中心 (0.5, 0.5)
- ρ_l/ρ_g = 2, We = 10, 重力なし
- 格子: N = 32, 48, 64, 96, 128
- 200 ステップ, 非増分投影
- HFE ON (extension_method = "hermite")

**検証指標**:
- Laplace 圧 Δp = p_in − p_out の理論値 σ/R = 0.1 との相対誤差
- 寄生流れ ||u_para||_∞ のピーク値と格子収束次数
- CSF 律速 O(h²) の確認

**位置づけ**: 表面張力が活性な最初の動的テスト。CCD O(h⁶) 曲率により寄生流れの**係数**が低減されることを確認する（収束**次数**は CSF 律速で O(h²)）。

**実装**: experiment/ch12/exp12_2_static_droplet_convergence.py（既存、変更なし）
**状態**: **実行済み** ✓

### EXP-12C: HFE 効果の定量的実証（ablation study）

**目的**: HFE の ON/OFF/ベースラインの3条件比較により、HFE が寄生流れ・圧力精度に与える効果を直接測定する。**本稿最大の新規実験。**

**設定**:
- EXP-12B と同一の静止液滴（R = 0.25, ρ_l/ρ_g = 2, We = 10）
- N = 64, 128 の2格子
- 200 ステップ, 非増分投影
- **3条件**:
  1. **HFE** — extension_method = "hermite"（本稿の提案手法, O(h⁶)）
  2. **Upwind** — extension_method = "upwind"（Aslam 2004 ベースライン, O(h)）
  3. **None** — extension_method = "none"（延長なし, Gibbs 振動あり）

**検証指標**:
- ||u_para||_∞ の時間発展（3条件を同一図に重ね描き）
- Δp 相対誤差の時間発展
- 界面近傍の CCD ∇p プロファイル（x = 0.5 断面, Gibbs 振動の可視化）
- 200 ステップ後の各指標の定量比較表

**期待される結果**:
- None: Gibbs 振動により寄生流れが増大（ステップ2以降、p^n ≠ 0 のため延長の有無が効く）
- Upwind: O(h) 精度の延長で部分的に改善
- HFE: O(h⁶) 精度の延長で最大の改善

**位置づけ**: §8.4 で定式化した HFE の理論的主張を、動的 NS パイプライン内で実証する ablation study。「HFE を外すと何が起きるか」を示すことで、手法の必要性を直接的に証明する。

**実装**: experiment/ch12/exp12_3_hfe_ablation.py（**新規作成**）
**状態**: **未実行**

### EXP-12D: 密度比スイープ — CSF 一括解法の動的限界

**目的**: NS パイプライン全体（時間積分を含む）での CSF + smoothed Heaviside 一括解法の破綻閾値を特定し、分相 PPE 解法の必要性を動的文脈で確認する。

**設定**:
- 静止液滴、EXP-12B と同一（R = 0.25, We = 10）
- N = 64（固定）, 200 ステップ（または発散まで）
- HFE ON
- **密度比スイープ**: ρ_l/ρ_g = 2, 3, 5, 10

**検証指標**:
- 安定/発散の2値判定（200ステップ完走 = 安定）
- 安定ケース: ||u_para||_∞, Δp 相対誤差
- 発散ケース: 破綻ステップ数, 最大 ||u||_∞

**位置づけ**: §11.5（PPE 製造解での破綻閾値 ρ ≈ 10）を NS パイプライン全体で追認する。§11.5 は PPE 単体の製造解テストだったが、本テストは Predictor → PPE → Corrector の完全パイプラインでの限界を記録する。結果は §13 将来課題（分相 PPE 解法）への直接的な根拠となる。

**実装**: experiment/ch12/exp12_4_density_ratio_sweep.py（**新規作成**）
**状態**: **未実行**

## 7. リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| EXP-12C で HFE ON/OFF の差が小さい | 新規実験の意義が薄れる | ρ = 2 は密度ジャンプが小さいため差が出にくい可能性あり。ρ = 5 でも追加実行し、差が最大になる条件を探索 |
| EXP-12D で ρ = 5 が発散する | §11.4a と矛盾 | §11.4a は N ≤ 128 で確認済み。N = 64 で再現しない場合は N = 128 に拡張 |
| 非増分投影の O(Δt) 分裂誤差が結果に影響 | 精度評価が不正確になる | 200ステップ安定→定常状態に近く、分裂誤差の影響は軽微。Δt は CFL 制約下で十分小さい |
