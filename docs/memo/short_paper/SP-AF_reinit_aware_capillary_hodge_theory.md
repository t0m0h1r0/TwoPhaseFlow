# SP-AF: Reinit-Aware Capillary Hodge Theory

**Status**: ACTIVE
**Date**: 2026-05-06
**Scope**: ch14 oscillating/static droplet theory, Ridge-Eikonal coupling, pressure-Hodge capillarity
**Companion papers**: SP-AD, SP-AE, SP-AC, SP-AA

## Abstract

The ch14 oscillating-droplet failure exposed two opposite hazards in the
pressure-jump route.  A static droplet must not receive a spurious
divergence-free capillary face force, but a deformed droplet released from rest
must not have its physical divergence-free capillary force deleted.  At the
same time, Ridge-Eikonal reinitialization can change deformation diagnostics
even when velocity is zero, so a theory that ignores reinitialization cannot
close the physical interpretation.

This memo completes the capillary theory as a split-map variational contract:

```text
q^n -- T_h(u_f) --> q_T
q_T -- Pi_h    --> q^{n+1}
```

`T_h` is the projection-native physical interface transport driven by the
canonical face velocity.  `Pi_h` is the Ridge-Eikonal/profile representation
projection.  Capillary work is paired with the velocity-generated transport
increment `q^n -> q_T`; the projection increment `q_T -> q^{n+1}` is either
trace/energy preserving or is charged to an explicit metric/topology defect
budget.

Under this split, the same weighted face-space Hodge gate decides both cases:

```text
static equilibrium        <=> P_h c_sigma = 0
dynamic capillary release <=> P_h c_sigma != 0
```

The result is not a benchmark-specific rule and not a switch between static and
oscillating droplets.  It is the discrete variational statement that pressure
may absorb only pressure-range capillary work, while Ridge-Eikonal may repair
only representation unless explicitly modeled as a metric dissipation.

## 1. Problem Statement

The old range-projected capillary closure solved one real problem: on a static
droplet, a raw capillary face cochain can have tiny divergence but nonzero
solenoidal/Hodge content, and that content injects kinetic energy.  But the
same closure also deletes the physical Hodge component of a deformed droplet,
making the N32 ch14 oscillating droplet stay at machine-zero velocity.

The reinitialization schedule adds a second ambiguity.  In the current
oscillating-droplet probe, the signed deformation changes even when the
computed velocity is zero.  Such a change cannot be capillary motion because
no face velocity performed mechanical work.  It is a representation-projection
effect unless it is explicitly promoted to a modeled metric/topology process.

The theory must therefore satisfy all of the following at once:

1. a Young-Laplace equilibrium remains static;
2. a non-equilibrium deformation releases capillary acceleration;
3. Ridge-Eikonal reinitialization can be used without creating fictitious
   capillary work;
4. no benchmark-name branch is introduced.

## 2. State Split

Use three interface layers:

```text
a       trace/interface degrees of freedom defining Gamma
q       conservative diffuse carrier, such as psi
phi     metric/profile/gauge field, such as signed-distance-like phi
```

The physical surface energy is a trace functional:

```text
S_h = S_h(a) = |Gamma_h(a)|.
```

The carrier and metric fields are allowed to support transport, curvature
estimation, and conditioning, but their off-interface gauge values are not
surface-energy variables.  This distinction is what allows reinitialization to
be included without turning it into a physical capillary force.

## 3. Split-Map Step

### 3.1 Physical transport

The interface transport step is:

```text
q_T - q^n + dt D_f((P_f q^theta) u_f) = 0.
```

For a virtual face velocity `w_f`,

```text
delta q_T[w_f] = -dt D_f((P_f q^theta) w_f).
```

This is the only fixed-topology displacement generated by momentum.  Only this
increment can be paired with reversible capillary work.

### 3.2 Ridge-Eikonal/profile projection

After transport:

```text
(q^{n+1}, phi^{n+1}) = Pi_h(q_T).
```

In fixed-topology validation, `Pi_h` is an admissibility projection:

```text
q = H_eps(-phi),
E_h(phi) = 0 in the interface band,
V_h(q) = V_h(q^n),
Gamma(q) = Gamma(q_T),
S_h(Gamma(q)) = S_h(Gamma(q_T)).
```

The projection multipliers stay inside the representation problem.  They do
not become unlabelled forces in the momentum equation.

If the projection changes `Gamma` or `S_h` during fixed-topology motion, the
step is projection-defective:

```text
Delta S_Pi = S_h(Gamma(q^{n+1})) - S_h(Gamma(q_T)).
```

That quantity is not capillary work.  It must be rejected or charged to a
named metric/topology budget.

## 4. Capillary Cochain

Construct a discrete gradient `g_T` on the physical transport endpoint:

```text
<g_T, q_T - q^n> = S_h(Gamma(q_T)) - S_h(Gamma(q^n)).
```

The capillary force covector is defined by virtual work:

```text
<F_sigma, w_f>_f = -sigma <g_T, delta q_T[w_f]>.
```

Using the transport map,

```text
c_sigma,f = -sigma (P_f q^theta) ((-D_f)^T g_T)
```

up to the exact face/cell pairings and sign convention.  The important
contract is:

```text
c_sigma is built from T_h^* dS_h,
not from an unlabelled transport-plus-reinitialization endpoint.
```

If `Pi_h` preserves the trace,

```text
S_h(Gamma(q^{n+1})) - S_h(Gamma(q^n))
= S_h(Gamma(q_T)) - S_h(Gamma(q^n)).
```

Thus the reinitialized state can be stored as the next state without changing
the capillary work identity.

## 5. Pressure-Hodge Gate

The pressure projection and corrector use:

```text
D_f A_f G_f p = r_h + D_f c_sigma,
a_f = A_f G_f p - c_sigma,
u_f^{n+1} = u_f^* - dt a_f.
```

The range/Hodge decomposition is:

```text
Pi_R c_sigma = A_f G_f p_sigma,
D_f A_f G_f p_sigma = D_f c_sigma,
h_sigma = c_sigma - Pi_R c_sigma.
```

With the physical face mass/metric pairing `M_f`,

```text
static equilibrium        <=> ||h_sigma||_M <= tol_static ||c_sigma||_M
dynamic capillary release <=> ||h_sigma||_M >= tol_release ||c_sigma||_M.
```

Replacing `c_sigma` by `Pi_R c_sigma` in production is invalid.  It gives
zero acceleration for every zero-predictor release and therefore freezes the
deformed droplet as well as the static one.

## 6. Discrete Energy Ledger

The completed fixed-topology energy target is:

```text
K^{n+1} - K^n
+ sigma [S_h(Gamma(q_T)) - S_h(Gamma(q^n))]
+ D_visc
+ D_metric
= R_num.
```

For identity-preserving reinit:

```text
D_metric = 0,
S_h(Gamma(q^{n+1})) = S_h(Gamma(q_T)).
```

For a declared metric/topology event:

```text
D_metric accounts for Delta S_Pi and the declared regularization.
```

The oscillation claim must be based on `Delta S_T` and kinetic response, not on
`Delta S_Pi`.

## 7. Theorems

### T1. Gauge-invariant reinit

If `S_h=S_h(Gamma(q))` and `Gamma(Pi_h(q_T))=Gamma(q_T)`, then:

```text
dS_h[Pi_h T_h(u)]/du = dS_h[T_h(u)]/du.
```

Ridge-Eikonal can be included in the algorithm without entering reversible
capillary force.

### T2. Projection defect is not capillary force

If `Pi_h` changes `Gamma` at zero face velocity, then no reversible fluid work
pairing explains the change:

```text
delta q_T[w_f=0] = 0,
Delta S_Pi may be nonzero.
```

The change is a projection defect or declared metric dissipation, not a
surface-tension acceleration.

### T3. Static/dynamic separation survives reinit

With identity-preserving reinit and a transport-adjoint cochain:

```text
P_h c_sigma = 0
```

if and only if the constrained surface-energy first variation vanishes on the
admissible incompressible face velocity space.  Static droplets and
oscillating droplets are therefore separated by one variational condition.

### T4. Range replacement remains invalid

Even with reinit accounting:

```text
c_sigma -> Pi_R c_sigma
```

is a force deletion.  It cannot be the production law for dynamic capillary
release.

## 8. Verification Gates

### G0. State split

Diagnostic runs must expose:

```text

psi_after_transport_before_reinit,
psi_after_reinit.
```

### G1. Projection identity

For fixed-topology runs:

```text
||Gamma(q^{n+1}) - Gamma(q_T)|| <= tol_Gamma,
|S_h(Gamma(q^{n+1})) - S_h(Gamma(q_T))| <= tol_S.
```

### G2. Virtual work

For sampled admissible face velocities:

```text
<F_sigma, w_f>_f + sigma dS_h[T_h(w_f)] = 0.
```

### G3. Static Hodge

On constrained static equilibria:

```text
||h_sigma||_M / max(||c_sigma||_M, eps) <= tol_static.
```

### G4. Dynamic release

On non-equilibrium deformations released from rest:

```text
||h_sigma||_M / max(||c_sigma||_M, eps) >= tol_release,
K^1 = 1/2 dt^2 ||h_sigma||_M^2 + O(dt^3) + viscous terms.
```

### G5. Reinit defect

At zero velocity in fixed topology:

```text
Delta_Gamma_Pi = 0,
Delta_S_Pi = 0.
```

### G6. Energy ledger

Report:

```text
Delta K, Delta S_T, Delta S_Pi, D_visc, D_metric, R_num.
```

## 9. Implementation Consequence

The current code already has useful pieces: transport-adjoint capillary
cochains, P2 trace surface-energy gradients, ALE/discrete-gradient endpoint
logic, and reinit projection diagnostics.  The missing contract is endpoint
labelling.

The next implementation must build capillary work from:

```text
psi_before_transport -> psi_after_transport_before_reinit
```

and separately audit:

```text
psi_after_transport_before_reinit -> psi_after_reinit.
```

Using `psi_previous -> psi` after reinit as if it were physical transport is
the exact ambiguity this theory removes.
