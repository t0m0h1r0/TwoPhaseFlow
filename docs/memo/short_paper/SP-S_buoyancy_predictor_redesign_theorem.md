# SP-S — Buoyancy-Aware Predictor Redesign Theorem and Algorithm Spec

Date: 2026-04-25  
Status: ACTIVE  
Author: ResearchArchitect

## 1. Abstract

This note turns the current ch13 diagnosis into a theorem-level redesign
statement.

The main result is:

> if the dominant instability is generated by a buoyancy-driven mismatch in the
> raw intermediate predictor state `u_pred`, supported on the full two-axis
> dilated interface band `I_1`, then the minimal consistent redesign is to
> assemble a **buoyancy-carrying predictor substate** first, repair it on `I_1`,
> and only then compose the final `u_pred` used in `V(u_pred)`.

The note formalises:

1. the discrete object that must be controlled,
2. why late witness/corrector patches are insufficient,
3. what algorithm class is mathematically admissible,
4. how that redesign should be traced back into the code.

## 2. Setting

We start from the one-fluid variable-density incompressible system

\[
\rho(\psi)\left(\partial_t \mathbf{u} + \nabla\cdot(\mathbf{u}\otimes\mathbf{u})\right)
= -\nabla p + \nabla\cdot\bigl(2\mu(\psi)\mathbf{D}(\mathbf{u})\bigr)
+ \rho(\psi)\mathbf{g} + \mathbf{f}_\sigma,
\qquad
\nabla\cdot\mathbf{u}=0,
\]

coupled to the conservative level-set transport

\[
\partial_t \psi + \nabla\cdot(\psi \mathbf{u}) = 0.
\]

The current CFD step uses an explicit convection/body-force branch,
a CN-type viscous predictor, and a variable-density pressure projection.

Let

- `C(u^n)` be the explicit convection branch,
- `B(psi^n)` be the explicit buoyancy branch,
- `V(u)` be the viscous predictor operator,
- `P_\beta` be the variable-density projection with `\beta = 1/\rho`.

The raw intermediate predictor state is

\[
\mathbf{u}_{\mathrm{pred}}^{\mathrm{raw}}
= \mathbf{u}^n + \Delta t\left(C(\mathbf{u}^n) + B(\psi^n) + \mathbf{V}(\mathbf{u}^n) + \cdots\right).
\]

This is the state on which the CN closure later evaluates `V(u_pred)`.

## 3. Interface-Band Assumptions

Define the strict interface set

\[
I_0 = \{x : 0 < \psi(x) < 1\},
\]

and the full two-axis one-cell dilation

\[
I_1 = I_0 \cup \mathcal{N}_x(I_0) \cup \mathcal{N}_y(I_0).
\]

The probe ladder established the following empirical hypotheses as working
assumptions:

### A1. Support assumption

The dominant predictor defect is supported mainly on `I_1`.

### A2. Buoyancy dominance

The leading unstable sub-assembly is carried by the buoyancy-containing part of
the explicit branch.

### A3. Pre-closure amplification

The instability becomes dynamically important before projection, through the
evaluation of `V(u_pred^{raw})`.

### A4. Geometry requirement

Neither `I_0` alone, nor x-only dilation, nor y-only dilation reproduces the
useful signal; the relevant neighbourhood is the full two-axis band `I_1`.

## 4. Defect Model

Write

\[
\mathbf{u}_{\mathrm{pred}}^{\mathrm{raw}}
= \mathbf{u}_{\mathrm{pred}}^\star + \delta \mathbf{u}_{I,B},
\qquad
\mathrm{supp}(\delta \mathbf{u}_{I,B}) \subset I_1,
\]

where `u_pred^*` is the ideal closure-compatible intermediate state and
`delta u_{I,B}` is the buoyancy-driven interface-band mismatch.

Then

\[
\mathbf{V}(\mathbf{u}_{\mathrm{pred}}^{\mathrm{raw}})
= \mathbf{V}(\mathbf{u}_{\mathrm{pred}}^\star)
+ \mathcal{L}_V[\delta \mathbf{u}_{I,B}]
+ \mathcal{N}_V(\delta \mathbf{u}_{I,B}),
\]

where `\mathcal{L}_V` is the Fréchet linearisation of the viscous response and
`\mathcal{N}_V` collects higher-order terms.

The observed instability diagnostics imply that the dangerous component of
`\mathcal{L}_V[\delta u_{I,B}]` couples strongly into the closure-sensitive
subspace seen by the projection stage.

## 5. Theorem 1 — Late Witness Insufficiency

**Statement.**  
Assume A1–A3. Let `\mathcal{W}` be any late witness correction that acts only
after `u_pred^{raw}` has already been assembled, e.g.

\[
\mathbf{u}_{\mathrm{pred}}^{\mathrm{late}}
= \mathbf{u}_{\mathrm{pred}}^{\mathrm{raw}} + \mathcal{W}\bigl(\mathbf{u}_{\mathrm{pred}}^{\mathrm{raw}}\bigr).
\]

Then `\mathcal{W}` cannot be expected to remove the dominant instability unless
it reconstructs the same interface-band state family as `u_pred^\star` itself.

**Reason.**  
The dominant defect is carried by the state entering `V(u_pred)`, not merely by
one derived witness such as `\nabla\cdot u_pred` or a scalar pressure-like
compensator. A late witness term does not change the assembly path that created
`delta u_{I,B}`.

**Meaning for this repo.**  
This theorem explains why reduced-pressure proxies, hydrostatic corrector
patches, trace-like witness additions, and scalar-weighted diagonal corrections
underperform the best predictor-side branch.

## 6. Theorem 2 — Necessity of Full-Band Repair

**Statement.**  
Assume A1 and A4. Let `\mathcal{T}_{I}` be a repair operator localised to a set
`I`. If

\[
I \in \{I_0,\ \mathcal{N}_x(I_0),\ \mathcal{N}_y(I_0)\},
\]

then the repaired predictor branch cannot reproduce the useful signal of the
successful branch using `I_1`.

**Reason.**  
The probes show that strict-interface, x-only, and y-only localisation all
collapse back to the same baseline trajectory. Therefore the useful signal
depends on the full two-axis narrow-band geometry, not on a one-direction or
zero-neighbour truncation.

**Meaning for this repo.**  
Any redesign that keeps only a strict band or a one-axis band is mathematically
mis-specified relative to the observed defect support.

## 7. Theorem 3 — Minimal Admissible Redesign

**Statement.**  
Under A1–A4, the minimal admissible redesign class is

\[
\widetilde{\mathbf{u}}_B
= \mathcal{T}_{I_1}\!\left(\mathbf{u}^n + \Delta t\,B(\psi^n)\right),
\]

followed by composition

\[
\mathbf{u}_{\mathrm{pred}}^{\mathrm{BA}}
= \widetilde{\mathbf{u}}_B
+ \Delta t\left(C(\mathbf{u}^n) + \mathbf{V}(\mathbf{u}^n) + \cdots\right),
\]

or by its buoyancy–viscous coupled variant

\[
\widetilde{\mathbf{u}}_{BV}
= \mathcal{T}_{I_1}\!\left(\mathbf{u}^n + \Delta t(B(\psi^n)+\mathbf{V}(\mathbf{u}^n))\right),
\]

\[
\mathbf{u}_{\mathrm{pred}}^{\mathrm{BV}}
= \widetilde{\mathbf{u}}_{BV} + \Delta t\,C(\mathbf{u}^n) + \cdots.
\]

Any admissible redesign must act on a **buoyancy-carrying substate before**
`V(u_pred)` is re-evaluated.

**Reason.**  
That is the earliest point at which the harmful state mismatch can be removed
without changing unrelated operators or relying on late compensators.

## 8. Algorithm Spec

The preferred algorithm is:

1. Update `psi` and material fields `rho(psi), mu(psi)`.
2. Build the buoyancy-carrying substate

   \[
   \mathbf{u}_B = \mathbf{u}^n + \Delta t\,B(\psi^n).
   \]

3. Apply a repair operator on the full two-axis band

   \[
   \widetilde{\mathbf{u}}_B = \mathcal{T}_{I_1}(\mathbf{u}_B).
   \]

4. Compose the final predictor state

   \[
   \mathbf{u}_{\mathrm{pred}}
   = \widetilde{\mathbf{u}}_B + \Delta t\left(C(\mathbf{u}^n)+\mathbf{V}(\mathbf{u}^n)+\cdots\right).
   \]

5. Evaluate `V(u_pred)`.
6. Form the CN/trapezoidal state `u_*`.
7. Apply the variable-density projection `P_\beta`.

## 9. A3 Traceability to Code

Equation-to-code chain:

- **Equation**: `u_pred = u^n + Δt(E^n + V(u^n))`
- **Discretisation**: explicit buoyancy branch + explicit assembly repair on `I_1`
- **Code focus**:

## 10. Non-Admissible Shortcuts

The following are excluded by the theorem chain above:

- late witness additions without state-family reconstruction,
- strict-interface-only repair,
- x-only or y-only band repair,
- corrector-side reduced-pressure substitution as the primary cure,
- scalar hydrostatic/pressure weights that do not rebuild the substate.

## 11. Acceptance Criterion

A redesign qualifies as a meaningful solver-side fix only if it:

1. improves upon the current best branch (`buoyancy_local`) on the short debug
   benchmark, or
2. extends the stable horizon without moving to a harsher failure mode.

Otherwise it remains a diagnostic probe, not a production candidate.
